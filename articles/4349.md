**国内要闻**

阿里 CEO 吴泳铭新年家书：今年几场大仗打得很漂亮

昨日，阿里巴巴给全体员工寄出了 2026 新年家书，里面除了家书、春联、红包、日历、淘公仔外，还有一封阿里首席执行官吴泳铭的书信。信中吴泳铭称，“今年几场‘大仗’我们都打得漂亮 —— 淘宝闪购破局逆袭，‘吃住行游购’大消费服务体系加速成型；高德‘扫街榜’让商家专心经营、消费者放心消费，促进线下服务业更健康可持续发展；千问 App 公测一周下载量破千万，阿里 AI 正在从千行百业走向人人可用…… 这些成果，不只是单一业务的突破，更是阿里生态协同释放出的化学反应。它们让我们真切看到，AI 带来的不是替代，而是升级和更多可能性。”

华为云空间“全新临时备份”功能下线，云端数据可转存至会员云存储空间

华为宣布华为云空间中的“全新临时备份”今天正式下线，后续将无法继续新增临时备份，用户此前使用临时备份功能备份的数据（未超过 60 天）仍将保存在云端，用户可查看恢复临时备份记录或转存至会员空间，若未转存将在存储最多 60 天后被删除。

“临时空间”指华为云空间为临时备份提供的免费空间，不占用已购买或已领取的云存储空间。在换机、重大系统升级等场景，华为用户可以使用临时备份将设备数据备份到云空间免费提供的临时空间中，备份数据的保存有时效限制，临时备份没有大小限制。

消息称飞书将推出首款硬件，与安克协作打造“AI 录音豆”

有消息称，飞书将与安克创新联合发布一款名为“AI 录音豆”的智能录音硬件，这是飞书旗下首款硬件产品。该产品重量仅 10 克，外观呈豆状，机身搭载双 MEMS 麦克风阵列，整机配合充电舱重量约 48 克。产品支持蓝牙与 Wi-Fi 两种传输模式，主打无感佩戴和随时录音。

定位方面，该产品在飞书内部属于探索性项目，飞书产品团队负责软件部分研发工作，产品本身强调“无感佩戴”和“全天候随身”，用户可将其夹在衣领、口袋边缘，或通过磁吸配件固定在手机背面，让用户在日常会议、客户拜访、甚至通勤路上都能随时开启录音，而不必专门掏出手机打开 App。（智能涌现）

字节跳动发布新一代折叠模型 SeedFold，性能超越谷歌 AlphaFold 3

字节跳动 Seed 团队推出新一代折叠模型 SeedFold，为生物分子结构预测领域的基础模型缩放提供新思路。该模型在多个蛋白质相关任务性能上超越 AlphaFold 3，揭示了三个关键「缩放配方」：模型缩放主张加宽而非加深，实验表明加宽主干比加深主干或结构模块效果更好。架构创新引入线性三角注意力机制，将计算复杂度从立方级降至平方级。数据扩展通过大规模知识蒸馏，将训练数据集扩充至 2650 万个样本。在 FoldBench 基准测试集上，SeedFold 及其线性版本在多个关键任务上取得当前最佳性能。最后，研究人员指出未来可探索专家混合网络和训练后缩放两个方向，SeedFold 为构建下一代生物分子基础模型奠定了基础。

机器人将再度登上春晚舞台引热议，网友：绝不是小心翼翼扭秧歌

据央视新闻报道，日前，中央广播电视总台《2026年春节联欢晚会》完成首次彩排。据介绍，继蛇年春晚扭秧歌的智能机器人跻身“顶流”后，机器人将再度登上春晚舞台。有网友表示：“可以期待，今年春晚舞台的人形机器人表演足以惊艳全世界，绝不是小心翼翼地扭秧歌。”博主“小特叔叔”也表示，去年春晚机器人还只会扭秧歌，下台都要人扶着，但是今年再度登台，他们能干什么了不敢想。

**国际要闻**

马斯克向 OpenAI、微软索赔最高 1340 亿美元

马斯克已正式起诉 OpenAI 及其合作伙伴微软，索赔 790 亿至 1340 亿美元，指控其背弃非营利创始使命、转向营利模式并与微软深度结盟的行为构成欺诈。索赔额基于 OpenAI 目前约 5000 亿美元的估值及马斯克 2015 年提供的 3800 万美元种子资金。

本案将于 4 月下旬在加州奥克兰开庭，此前联邦法官已驳回 OpenAI 与微软避免陪审团审理的动议，OpenAI 称诉讼「毫无根据」。马斯克与 OpenAI 的矛盾始于 2024 年，源于他对公司转为营利性结构不满，他于 2018 年退出董事会，2023 年创立自己的 AI 公司。OpenAI 转型引发争议，马斯克质疑其合法性，指控其背离「以造福人类为先」的使命，还计划进一步主张惩罚性赔偿。

OpenAI 将在美测试 ChatGPT 广告，8 美元 Go 订阅同步上线

OpenAI 宣布将在 ChatGPT 的免费版和入门订阅层级「ChatGPT Go」中引入广告功能。这项测试将在未来几周内率先面向美国的成年用户展开。好消息是广告并不会强行打断你的对话，它们只会在系统判断存在相关赞助产品时，安静地出现在回答内容的底部，并配有清晰的标注。如果你是 Plus、Pro 或企业版的付费用户，那么恭喜你，你的界面依然会保持纯净；但如果你打算尝试刚刚推出的月费仅 8 美元的 ChatGPT Go 订阅服务，那么你依然属于被广告覆盖的目标人群。

Anthropic 将 Claude Cowork AI 智能体扩展至 20 美元 Pro 订阅用户

Anthropic 现已将 Claude Cowork AI 助手下放给所有 20 美元（现汇率约合 139.8 元人民币）Pro 订阅用户，让他们的电脑能够更简单地处理日常事务。

此前 Anthropic 只向 Max 订阅用户提供该功能，这一档为的订阅费用高达每月 100 美元，不过可以获得更多高成本推理模型使用额度、抢先体验实验性功能等。Anthropic 表示，Pro 用户使用 Claude Cowork 时，可能会比 Max 用户更早达到使用限额。

Claude Cowork 是一种 AI 智能体，能够操纵用户的电脑并自主完成任务，例如可以根据用户保存的文件创建文档、整理文件夹、处理基础系统操作等，同时还可以使用“连接器”（connectors）和 Chrome 插件，让智能体与其他 App、网页协同工作。不过需要注意的是，这款智能体目前只支持苹果 macOS 系统。

马斯克：特斯拉将重启 Dojo 3 项目

今天早间，马斯克在 X 平台发文宣布，随着 AI5 芯片设计完成，特斯拉将重启超级计算机项目 Dojo 3 的开发工作。同时，马斯克在这则帖子中发出了“招聘启事”，要求“有兴趣参与研发世界上产量最高芯片”的应征者在邮件中用 3 个要点介绍自己解决过的最棘手的技术难题。

2025 年 8 月曾有消息称，特斯拉已全面叫停 Dojo 超级计算机项目，其负责人也将离职。这一决定意味着特斯拉正式放弃开发自动驾驶所需自研芯片的计划。知情人士透露，马斯克亲自下令关闭该项目。当时，马斯克回复网友评论称，特斯拉分散资源并同时开发两种截然不同的 AI 芯片设计是没有意义的。特斯拉的 AI5、AI6 及后续芯片在推理方面将表现出色，至少在训练方面相当不错，所有努力都将集中在这些芯片上面。

Neuralink 首例受试者：无需手术升级脑机接口，类似特斯拉 OTA 更新

全球首例 Neuralink 侵入式脑机接口受试者诺兰・阿博透露，其植入的脑机芯片可通过 OTA 远程升级，类似特斯拉汽车。目前 Neuralink 脑机接口系统更新途径有三：通过「心灵感应」应用程序软件更新。植入体固件无线升级。再次手术植入更先进硬件。阿博曾遇电极线脱落问题，通过升级解决，他还表示未来愿接受手术升级芯片或植入双脑机系统。此外，脑机接口行业致力于降低设备植入与更新的「侵入性」，Neuralink 计划 2026 年启动大规模量产并推动手术全自动化。脑机接口正与多领域加速融合，有望推动商业化落地。

微软最新安全更新导致部分 Windows 11 PC 无法正常关机

根据微软官方公告，它的最新安全更新导致部分 Windows 11 23H2 PC 无法正常关机。受影响的 PC 无法进入关机或休眠状态，而是一直处于唤醒状态，顽固的抵抗关机指令。该 bug 与 Secure Launch 相关。Secure Launch 是基于虚拟化的保护功能，旨在确保启动过程中只加载受信任组件。在启用了 Secure Launch 系统上，安装了最新安全更新的 PC 无法执行关机、重启或休眠等操作。微软表示权宜之计是命令行下输入“shutdown /s /t 0”强制关机。目前没有修复方法。

腾讯向逾 30 个 GitHub 微信相关项目发出 DMCA 通知

腾讯的微信以体积随时间不断膨胀著称，这促使很多开发者对微信进行了分析，其中一部分人将他们的分析方法或相关清理工具发布在代码托管平台 GitHub 上。但本月初，腾讯法务向超过 30 个 GitHub 项目发出了 DMCA 通知，导致这些项目被迫下架。腾讯法务指控开发者们违反了 DMCA 绕过技术保护措施的条款、违反了微信禁止逆向工程条款、威胁用户隐私和安全，以及侵犯知识产权。

Linux 6.19-rc6 内核发布，修复苹果 M1/M2 系列 Mac 设备 USB-C 端口控制器问题

据 phoronix 报道，即将于今天晚间发布的 Linux 6.19-rc6 内核版本修复了苹果 M1 Pro / M1 Max / M1 Ultra 以及 M2 / M2 Pro / M2 Max / Ultra 机型 USB-C 端口控制器无法正确识别 USB 2.0 和 USB 3.x 设备的情况。

此前 USB-C 端口控制器会触发多余的中断，导致 USB 角色反复切换到当前已处于激活状态的角色，从而引发异常。为此，Linux 6.19-rc6 内核现在会在 dwc3-apple 已处于目标状态时忽略这些 USB 角色切换，确保 USB-C 端口控制器与 dwc3-apple 始终保持一致状态，其行为也与 core.c 中的 \_\_dwc3\_set\_mode () 保持一致。此外，Linux 6.19-rc6 还引入了“在 DWC3 初始化之前提前设置 USB2 PHY 的工作模式”特性，主要用于解决在开机前或启动过程中已插入 USB2 设备时偶发的识别问题。

轻量 JavaScript 框架 jQuery 推出 4.0 正式版，放弃支持微软 IE 10 浏览器

JQuery 是一款受到广泛赞誉的轻量 JavaScript 框架，目前官方推出了 JQuery 4.0 正式版本，号称是近 10 年来的首个重大版本更新（https://blog.jquery.com/2026/01/17/jquery-4-0-0/）。在兼容性方面，jQuery 4.0 正式放弃对 IE10 及更早版本的支持（官方同时预告 IE11 将在后续 jQuery 5.0 中逐步淘汰）。同时也停止支持 Edge Legacy、过旧的 iOS 与 Firefox 版本以及 Android Browser。如果用户仍需要使用这些老旧浏览器，应当继续使用 jQuery 3.x。
