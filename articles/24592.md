**国内要闻**

字节跳动向 Seed 员工开放「豆包股」认购权

据四位了解该激励计划的知情人士透露，字节跳动首次针对特定业务单元发行股权，为旗下 Seed 人工智能部门员工提供与该部门挂钩的低价股票期权「豆包股」，当前认购价为每股 13 美元。该「豆包长期激励计划」始于 2025 年四季度，初期估值参考业务投入成本，未来将结合豆包及大模型 toB 业务发展重新估值，以加强对大模型人才的长期吸引和激励。此外，字节此前已为 Seed 部门核心技术员工实施期权增发，按绩效与职级分档，员工每月可拿到约 9 万、11 万、13 万元人民币价值的字节期权，按月归属，首批连续发放 18 个月，最高可达百万元级别。（新浪财经）

滴滴出行 App 大规模故障，客服回应：目前已恢复正常

滴滴出行 App 出现大规模系统故障，涉及北京、深圳等地，用户集中反映问题多在行程管理和支付环节。多位用户表示无法开启行程、司机接单后无法结束订单，甚至出现无法取消订单的情况。针对此事，滴滴出行官微发布声明称，非常抱歉，因云厂商网络专线故障，造成今天 17 点左右滴滴 App 部分服务出现短暂故障，目前服务已全部恢复。（新浪科技）

卢伟冰：小米不会把内存涨价成本转嫁给消费者，今年七八月将发布新一代 OS

据小米集团总裁、集团合伙人卢伟冰在小米财报电话会上表示，尽管这个季度全行业承压，小米手机在关键指标上展现出了强韧性：全球市场份额连续 23 个季度稳居前三，手机均价及海外均价均创历史新高，手机毛利率超出市场预期。卢伟冰表示，一方面小米会坚定推进高端化和全球化战略，另一方面坚定用 AI 重塑手机业务，“今年超级小爱与 miclaw 合体，会是一个关键节点。”

面对存储涨价周期，卢伟冰表示：“不能简单地把内存成本的上涨转移给消费者，要去对我们的产品做重新定位”，将通过产品矩阵的升级和软件的优化在规模和利润中取得平衡。他透露，今年七八月将发布新一代 OS，带来不一样的交互体验，“整个产业在面临一场变革。”（IT 之家）

蚂蚁集团：Token 消耗将迎来指数级增长，预计 2030 年消耗量增长超 300 倍

5 月 26 日，支付宝举行 AI 支付生态大会。会上，蚂蚁集团研究院院长庄蹯表示：“Token、数据和工具等数字资源成为最重要的生产要素，Token、数据与工具是智能体必不可少的燃料、感官和四肢。”他预测：“Token 消耗将迎来指数级增长，2030 年预计 Token 消耗量将增长超 300 倍，拥有 22 亿的活跃智能体，年执行任务可达 400 万亿次”。（财联社）

中国人形机器人全球市场占比超八成

据央视财经，海关统计，今年一季度，我国机器人产业海外市场拓展步伐加快。各类单独列名的“机器人”合计出口额达 113.2 亿元，产品远销至全球 148 个国家和地区。目前中国人形机器人整机企业超 140 家，年出货量 1.44 万台，全球占比 84.7%，每 10 台人形机器人就有 8 台来自中国。数据显示，中国具身智能产业的市场规模预计在 2030 年达到 4000 亿元，并有望在 2035 年突破万亿元。（格隆汇）

美团推出「跑腿 Skill」，可对接各大 AI 助手实现「一句话点单」

美团推出「跑腿 Skill」，可对接各大 AI 助手，用户通过自然语言输入命令即可让 AI 助手调用美团跑腿完成下单，系统自动完成场景识别、地址匹配、价格预估与订单提交，将多步操作压缩为一步。该 Skill 采用零开发接入方式，安装后即可使用，兼容 OpenClaw 本地版、云部署版及各类第三方客户端，代码已在 GitHub 开源。用户用日常语言说清需求，AI 即可理解执行，不熟悉服务边界的用户会被引导可行方案，高频用户可口述完成从下单到支付全流程。使用时，Skill 接入美团跑腿地址簿，登录后自动拉取常用地址模糊匹配，物品信息由 AI 自动识别品类，下单前展示完整订单预览卡片，用户确认后提交，下单后可直接询问 AI 助手查询骑手实时配送进度。（IT 之家）

**国际要闻**

消息称埃隆 · 马斯克已与同事讨论过合并 Tesla 与 SpaceX 的可能

据美媒 CNBC 当地时间 26 日报道，消息人士表示埃隆 · 马斯克已与同事讨论过合并其掌控的两家科技产业巨头 Tesla（特斯拉）与 SpaceX（太空探索技术公司）的可能性。尽管 Tesla 和 SpaceX 的主营业务分别是电动汽车和航空航天，但由于汽车行业智能化加速和对 xAI 的合并，这两家企业都正逐步蜕变为“AI 驱动型企业”，双方在打造强大算力基础设施、构建高能效端侧算力系统上目标一致。

不仅如此，两家企业在资源、技术上存在长期共享，双方之间也经常进行交易：SpaceX 是 Tesla 的 Cybertruck 皮卡、Megapack 储能系统与太阳能设备的客户之一，Tesla 委托 SpaceX 开发合金材料。Tesla 与 SpaceX 合并的关键阻力之一来自财务方面：将哪家企业设为存续公司、如何为两家企业设定估值和换股计划、如何保护少数股东的权益，这些都是可能引发股东担忧的问题。

谷歌 CEO 解释了 AI 编程背后的原因：缺少 Cursor 这样的用户入口

谷歌 CEO 桑达尔·皮查伊在接受《纽约时报》专访时，首次坦承谷歌在前沿 AI 编程、工具调用与长程任务上目前落后于行业最前沿。他将落后原因归为：谷歌缺少像 Cursor 或 Claude Code 这样的用户入口，导致流失了最关键的动态工作流数据。皮查伊表示，谷歌在去年 11 月便已将首代 Antigravity IDE 推向公开市场，但因配额锁死 bug、服务器频频过载崩溃以及第三方开源 API 接入等运营事故，导致首代公测版未能建立稳定的开发者信任生态，中部早期用户流向了竞品。专访中，皮查伊展示了 Antigravity 在谷歌内部闭环运行时的测试数据，包括内部 token 总量每周翻倍，以及在极端性能测试中用 12 从小时零搭建出后台。上周发布的 Antigravity 2.0 开始全面公测，试图用技术窃取，让利动作赢回开发者的信任。

微软官方确认：Windows 11 定位 AI 原生操作系统！是战略资产

5 月 26 日，微软发布一份 14 页电子书，系统阐述 Windows 11 作为 AI 原生操作系统的定位。微软提出，企业 AI 投入的最大回报并非来自添加更多 AI 工具，而是要将 AI 直接嵌入工作流程。其引用相关数据指出，80% 的员工没有时间和精力完成日常工作，82% 的高管计划引入 AI Agent 提升生产力，但实际效果不佳，新增过多 AI 工具反而会额外占用员工的学习时间。微软提出将AI能力直接带入用户熟悉的原有体验，明确三大AI集成方向，重点介绍任务栏上的“Ask Copilot”功能，说明 AI 原生操作系统无需安装额外工具即可完成相关操作的核心特征。（快科技）

英伟达黄仁勋：AI 时代孩子学什么专业没那么重要，真正要紧的是会不会用 AI

据《商业内幕》报道，英伟达创始人兼 CEO 黄仁勋最近接受媒体采访时表示，随着 AI 不断发展，家长和孩子们不应该只寻找不会被 AI 取代的学科，而是应该学会利用 AI 帮助自己成长。黄仁勋认为，与其拼命寻找所谓“不会被 AI 取代的铁饭碗”，不妨关注如何借 AI 深化学习、提升技能水平，让自己变得更好。他认为，即使 AI 未来会变得越来越强大，新闻学、艺术学、设计等领域仍会具备价值。优秀的主持人之所以出色，是因为他们能随时投入现场、认真倾听。他对此表示：“未来，向受众讲述故事的能力会和今天一样重要”。

Anthropic 即将发布新一代模型 Mythos，或引发安全隐忧

Anthropic 公司疑似准备公开最新模型 Claude Mythos 预览版，该模型是公司迄今最大 AI 模型之一，开发可追溯至 4 月 7 日，官方定位为针对计算机安全任务的前沿模型，与现有旗舰模型 Opus 4. 相比，在代码理解和自主方面有显著提升。因该模型能自动具备专业水准的网络攻击手段，发布可能对数字基础设施带来严重风险，公司未急于全面开放。此前用户曾在 Claude、Claude Security 及 Code 的公开版本中短暂发现启用该模型的选项，但随后被撤回，目前 Anthropic 尚未正式宣布具体公开发布时间。同时，Anthropic 确认正在推进名为「Glass」的新项目，旨在与其他公司联合保护全球关键软件系统防止 AI 驱动的漏洞利用，该项目使用尚未发布的 Claude Mythos 预览版，并已与多家组织伙伴合作，还展示了用于汇总 Mythos 预览版发现的不同严重等级漏洞的源漏洞面板。（aibase）

国产第一：阿里 Qwen3.7-Max 模型 AI 编程能力超 Claude Opus 4.6

5 月 25 日，全球权威编程榜单 Code Arena 更新，阿里旗舰模型 Qwen3.7-Max 以 1541 分位列全球第四、大模型厂商第二，仅次于 Claude 系列，其编程能力超过 Claude-opus-4-6、glm-5.1、kimi-k2.6 等模型。该榜单通过用户随机盲测评估模型实际编程能力，防止刷榜。此外，Qwen3.7-Max 在 Design Arena 榜单中位列第十，Design Arena 是基于真实用户盲测的高含金量 AI 模型评测平台，被称为「AI 界的奥林匹克」。（IT 之家）

小米自研大模型 MiMo-V2.5 系列 API 永久降价：最高降 99%，不再区分上下文长度

5 月 27 日，小米宣布旗下 MiMo-V2.5 系列大模型 API 迎来永久降价，调价已于北京时间当日 0:00 正式生效，全球同步调整。本次调价不再区分上下文窗口长度，覆盖 MiMo-V2.5、MiMo-V2.5 Pro 两个版本，较原始 API 定价最高降幅可达 99%。其中两个版本输入缓存命中价格分别低至 0.02 元/百万 tokens、0.025 元/百万 tokens，输出价格分别降至 2 元/百万tokens、6 元/百万tokens。同步优化的 Token Plan 计费体系实现加量不加价，用量提升至原来的 5 至 8 倍，引入 Credits 概念，小米通过多项推理系统相关技术优化，在保障服务质量的前提下降低单位 token 服务成本。（快科技）
