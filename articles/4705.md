# 为什么 Claude Code 不用 RAG 向量检索代码

Boris（Claude Code 创始人）解释为什么 Claude Code 不用 RAG 向量检索代码：  
  
在开发 Claude Code 的早期版本时，我们曾尝试过 RAG 搭配本地向量数据库的方案。  
  
但很快我们就发现，Agent 使用关键字搜索在实际应用中的表现通常要出色得多。  
  
这种方案不仅实现起来更加简洁，而且还完美避开了 RAG 模式下那些令人头疼的“老毛病”：比如数据安全性、隐私泄露风险、信息滞后以及系统可靠性等问题。
