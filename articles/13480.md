# 曝贝索斯融资100亿美元，押注物理AI

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqXmCQpfA2JLNR9iaW7JCh9cOvGbw6UfRRpQtic5u2icvQvIZaLbQC3PibzuCAVnEgNPbjYufpWpIzBsTFXknBZfu7V9zJBJftTeNk/640?from=appmsg)

**贝索斯的AI创企估值将达到380亿美元。**

**编译 |**  杨京丽  

**编辑 |**  李水青  

智东西4月21日消息，据彭博社援引英国《金融时报》报道，**亚马逊创始人杰夫·贝索斯（Jeff Bezos）的AI创业公司Project Prometheus即将敲定一笔100亿美元（约合人民币682亿元）的新融资**，估值将达到380亿美元（约合人民币2591.6亿元）**。**

知情人士称，本轮投资方包括摩根大通（JPMorgan）和贝莱德（BlackRock）等，融资尚未最终敲定，预计很快完成。

这家公司由贝索斯与科学家维克·巴贾杰（Vik Bajaj）共同创办，**主攻能够“理解物理世界”的AI模型，目标落地场景是航空航天、汽车等领域的工程制造**。据《纽约时报》此前报道，Project Prometheus去年**成立之初就拿到62亿美元（约合人民币422.84亿元）融资**，贝索斯本人也是出资方之一。

  

**_01_****_._**

**100亿美元融资即将落地**

**摩根大通、贝莱德入场**

据英国《金融时报》援引的知情人士消息，本轮融资规模约100亿美元（约合人民币682亿元），估值预计将达到380亿美元（约合人民币2591.6亿元），摩根大通与贝莱德均出现在投资人名单中。叠加此前62亿美元（约合人民币422.84亿元）的首轮融资，Project Prometheus已成为**目前全球资金储备最充裕的早期AI创业公司之一**。

Project Prometheus成立于2025年，据《纽约时报》报道，这是贝索斯自2021年7月卸任亚马逊CEO以来，**首次在一家公司担任正式的经营岗位**。贝索斯此前曾创办的航天公司蓝色起源（Blue Origin），其官方头衔也仅是“创始人”。

**另一位联合CEO是物理学家和化学家巴贾杰**，他曾在Google X与谷歌联合创始人谢尔盖·布林（Sergey Brin）共事，参与过后来独立为Waymo的自动驾驶项目。他还联合创办了Alphabet旗下生命科学实验室Verily，以及AI创业孵化器Foresite Labs并出任CEO，去年已从Foresite Labs离任，全力投入Project Prometheus。

**_02_****_._**

**让AI从物理世界学习**

**团队核心成员多来自头部AI公司**

据《纽约时报》2025年11月援引知情人士披露，彼时Project Prometheus已招募近百名员工，其中**多位核心研究人员来自OpenAI、DeepMind和Meta等头部AI公司。**通过高薪集结一支既懂前沿模型、又懂工程制造的复合团队，是其62亿美元首轮融资最直接的用途之一。

技术路线上，Project Prometheus与当前主流的大语言模型路线区别较大。《纽约时报》指出，以ChatGPT为代表的大语言模型通过分析海量语料，学习模仿人类组织语言的方式，进而写代码、解数学题；但Project Prometheus选择的是另一条路径，**让AI能够从物理世界的试错中学习，服务于计算机硬件、航空航天、汽车等领域的工程制造**，这条路径也与贝索斯长期押注的太空业务直接呼应。

Project Prometheus并非物理AI潮中的孤例。2025年，一批从Meta、OpenAI、谷歌DeepMind等大厂离职的研究人员创办了Periodic Labs，已获3亿美元（约合人民币20.46亿元）融资，计划在北加州自建实验室、让机器人大规模运行科学实验，使AI系统通过物理试错自主积累经验。

另外，李飞飞的World Labs和图灵奖得主杨立昆主导的AMI Labs都在**押注“世界模型”路线，****用视频训练AI理解真实物理世界**。据知情人士称，Project Prometheus的系统与世界模型有所不同，更偏向工程制造侧的仿真理解。就贝索斯个人而言，2024年他投资了将AI应用于机器人的创业公司Physical Intelligence，此次创办Project Prometheus，是其在物理AI方向上的系统化加码。

  

**_03_****_._**

**结语：资本市场押注物理AI**

Project Prometheus本轮融资后估值将达到380亿美元，迅速实现显著估值跃升，一方面有摩根大通、贝莱德这类主流金融机构的下注，同时也有贝索斯本人的效应，可以看出资本市场对于物理AI抱有较高期待。

与此同时，Project Prometheus、World Labs、AMI Labs以及Physical Intelligence等路径各异的玩家接连入场，物理AI、世界模型赛道的竞争进一步升温，围绕顶尖人才与算力的争夺，接下来会更加激烈。
