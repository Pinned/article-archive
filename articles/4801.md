# 我入选了 Claude for Open Source

`Claude for Open Source` 是 Anthropic 给活跃开源开发者的一个福利项目，刚查邮件的时候，发现自己进了

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdjbQzWW2E73h1OewmAZTD3HuCmPS1xMdPnQWichic8HpydjibkVXzRIPB2qDJajTuP9vrmsrTKjoOyJRKXAic63SrrIqeswmeXjPt0/640?wx_fmt=png&from=appmsg)

录取邮件

**You're in! Welcome to the Claude for Open Source Program**

然后我的一些项目，也可能会被 Anthropic 用作联合营销

在这里，也把这个项目分享给大家

* * *

### **项目给什么**

6 个月的 Claude Max 20x，免费，正常每月 200 美元

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdhL1I9bpIcGkjaI4oFnrQP18NGZRSutcOdfe8qUl6W5yBGd0rSH9oPgMYpsEiakhiaib0dWGUjUBTib2YXo6aS1mvpg4DG5w6nWm28/640?wx_fmt=png&from=appmsg)

价值 1200 美金

Max 20x 在 Pro 基础上额外提供 5x 或 20x 的用量选择、所有任务的更高输出上限、高峰期优先访问，以及新功能的 Early Access

条款里还有一条：Anthropic 有权公开你的名字、GitHub 用户名和关联项目，标注你是项目成员。也可能会被邀请参与联合营销活动，但属于自愿

被 Anthropic 官方公开点名，也算一种背书了

* * *

### **怎么加入**

2 月 27 日，Lydia Hallie 正式宣布了这个计划，给热门项目背后的开发者，赠送 6 个月的 Claude Max 20x 权益

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdhGKMDDBYh5nZndsjZTH8Ir4ZwbMjpELBibkRdVdjsd796SKuLzT3NfNJdzAverm386Ar5kvSFLFlSwdsZsgcosl20JxkGMxUQ0/640?wx_fmt=png&from=appmsg)

Lydia Hallie 宣布推文截图

Claude Code 负责人 Boris Cherny 随后转发，说「So much of what makes Claude Code great came from feedback from OSS developers. Excited we can give back a little.」  

大意是，Claude Code 能做到今天这样，很大程度上来自开源开发者的反馈，很高兴能回馈一点

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdiaSVzgiaCBdgReEwYKOfxcCMmPnSicL6Aib5sEk94guXaDnwGGv2U3RKKiceqXNgPzUAxL1lASoIRndp5GdAy5yJJt4qcbWmuG4k40/640?wx_fmt=png&from=appmsg)

Boris Cherny 是 Claude Code 的创建者兼负责人

> **申请条件**  
>   
> 
> 你需要是某个知名项目的主要维护者或核心团队成员，该项目在 Github 上有 5000+ Star  或每月 NPM 下载量超过 100 万 次，且过去 3 个月内有实质贡献

如果项目规模没到这个量级，但维护的是生态里被广泛依赖的底层包，也可以提交说明单独申请

然后，这个项目，并没有提没有国籍或地区限制，所以...可以直接申请

申请入口：`claude.com/contact-sales/claude-for-oss`

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdjbYDZxHFYaJGHdZTgG8eaiaTOrRHfatLhx8KFGqVLdlpZLmA13E2hFdaeKR9z0fObBkNXuqrnmzHBBrXWkUHQZXE38ajL1R8SY/640?wx_fmt=png&from=appmsg)

最多 10,000 个名额，滚动审核

顺带一提，OpenAI 那边也有一些面向开发者和各领域专家的社区，给到些福利，大家可以留意一下
