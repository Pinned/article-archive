#### ******要事解读①******  **未来走向何方？Agent 创企 2025 生存现状一览**

**引言：**2025 年，Agent 成为了人工智能领域的核心议题。然而，在热度之下，市场关注度与实际商业化进展之间的落差给部分 Agent 创企带来了挑战。各类创企因此也在通过不同的路径探索 Agent 时代的生存法则，一如 Manus 的高价收购，Cursor 的持续融资，以及 Robin AI 的破产清算等。

  

#### **2025 Agent 热度如何表现？**

1、2025 年 12 月 30 日，Meta 宣布以逾 20 亿美元收购 Manus。这场交易在业内引发了广泛关注，被称为是 Agent 模式的里程碑事件，并引起了业内的多元解读。

① GPU 算力平台 Hyperbolic Labs 的联合创始人兼 CTO Yuchen Jin 指出，无论未来是 OpenAI、Anthropic 还是谷歌开发出了更强大的模型，Manus 都可以轻松替换该模型，同时保留底层的任务编排能力、工具链集成、用户交互界面和分发渠道。\[1-1\]

② Resemble AI 的开发者关系负责人 Dev Shah 则认为 Meta 并不是买了一家「模型公司」，而是买了一家「环境公司」，且「智能无法脱离环境而独立存在」。他将这个概念描述为 Situated Agency，即 Agent 的行动能力源于模型与工具、记忆及执行环境的深度耦合。\[1-1\]

2、除却年底的 Manus 收购案引发的讨论，Agent 在 2025 年度几乎已成为 AI 应用进展的代表，且话题持续火热。围绕 Agent 话题爆火的原因，有观点认为，Agent 的热度主要来自于 tool-use 能力的突破。\[1-2\]

① 《知识增强大模型》《知识图谱》作者、前达观数据副总裁，王文广表示，从编程到 browser-use，再到 computer-use，随着 MCP 通用接口的普及，Agent 能够更高效地调度外部系统并获取信息。

② 平安保险技术平台组负责人张森森，随着协议的成熟，市场竞争的焦点已经发生转移。过去比拼的是模型参数，现在则更看重「模型+生态工具」的组合能力。

3、在技术突破之上，Agent 的市场接受度在 2025 年急剧上升，谷歌云、BCG、Salesforce 等机构的研究数据均从各自的维度反应了当前市场对 Agent 的关注和积极态度。\[1-1\]

① 根据谷歌云发布的《2025 年人工智能投资回报率报告》，目前已有 52%使用生成式人工智能的企业在生产环境中部署了 Agent。

② BCG 的研究指出，应用高效的 Agent 可以让业务流程提速 30% - 50%。

③ Salesforce 的研究表明，预计到 2026 年，80% 的企业级应用将嵌入 Agent 能力。

  

#### 2025 年 Agent 玩家众多，哪些赛道被资本青睐？

1、伴随着市场信心而来的，是资本对 Agent 赛道的层层加码。过去一年，融资过亿美元的美国 Agent 创企已超过 20 家，涵盖了编程、B2B 客户服务、医疗健康、法律、科学发现和搜索等垂类领域。\[1-3\]

2、2025 年融资过亿美元的美国 Agent 创企中，有 3 家的业务涉及法律业务。

① Harvey 于 12 月完成 1.6 亿美元 F 轮融资，估值达 80 亿美元。投资方包括 Andreessen Horowitz、WndrCo、Sequoia 和 Kleiner Perkins 等现。其平台可帮助律师自动完成合同审阅、尽调等任务，目前已有 7.4 万律师用户。\[1-3\]

② EvenUp 于 10 月完成 1.5 亿美元 E 轮融资，投资方包括 General Catalyst 、Floodgate 和 Sierra Ventures 等。该公司过一系列专用 Agent 和知识库帮助企业法律团队自动化处理例行事务，如合同起草和案件跟踪等。\[1-7\]\[1-8\]

3、在搜索赛道，Parallel 和 You.com 2 家专注搜索业务的 Agent 创企均在 2025 年获得了超过 1 亿美元以上的融资。此外，AI 搜索独角兽 Perplexity 被曝于 9 月完成 2 亿美元融资，但官方尚未确认。

① Parallel 于 11 月完成了 1 亿美元 A 轮融资，估值 7.4 亿美元，投资方包括 Kleiner Perkins 和 Index Ventures 等。该公司由前 Twitter CEO Parag Agrawal 创办，是一个面向 Agent 的实时网络搜索引擎。\[1-9\]

② You.com 于 9 月完成 1 亿美元 C 轮融资，估值 15 亿美元，投资方包括 Cox Enterprises、Georgian 和 Norwest 等。该公司原先定位为大众搜索引擎，后转型为面向企业和 Agent 的搜索基础设施。\[1-10\]

4、除了主流的 Agent 垂类赛道外，音频明星创企 ElevenLabs 于 12 月融资 1 亿美元，并明确了向「对话式 Agent 平台」转型的意向。

① ElevenLabs 于 12 月通过由红杉资本和 ICONIQ 领投的要约收购筹集了 1 亿美元，估值突破 66 亿美金。该公司 CEO 指出，纯语音模型很快将成为「商品化」的基础能力，因此 ElevenLabs 正向「对话式 Agent 平台」转型，拓展业务至 Agent 驱动的音乐生成、深度伪造检测等新领域。\[1-11\]\[1-12\]
