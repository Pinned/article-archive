然后有了算力之后，Claude 从今天开始作出以下改变

*   **Claude Code 的五小时速率限额翻倍**，Pro、Max、Team、企业版都翻。之前被卡得最难受的 Claude Code 重度用户，额度直接乘以二
    
*   **Pro 和 Max 用户的高峰时段限额削减取消**，之前高峰期用 Claude Code 会被额外降速，现在不分高峰低谷，额度全天一致
    
*   **Claude Opus 的 API 速率限额大幅提升**，Tier 1 的每分钟输入 token 从 3 万涨到 50 万（16 倍），输出从 8 千涨到 8 万（10 倍）。Tier 4 的输入从 200 万涨到 1000 万，输出从 40 万涨到 80 万

Colossus 超酸中心
-------------

Colossus 1 是 xAI 在田纳西州孟菲斯建的超算中心，去年上线，是全球最大的单体 AI 算力集群之一。装了超过 22 万张 NVIDIA GPU，覆盖 **H100、H200 和 GB200** 三代加速器

Anthropic 签的是全量租约，整座数据中心的算力全拿走。300 兆瓦的新增容量，**本月内就能用上**。Anthropic 产品负责人 Ami Vora 在开发者大会上确认了这个消息

这笔交易的时间窗口很巧。Anthropic 过去一个月反复因为算力不够被用户投诉，Claude Code 和 API 的限额问题已经影响到了付费用户的日常使用。上个月 Anthropic 自己公开承认过，需求激增给基础设施带来了「inevitable strain」，影响了可靠性和性能

而 SpaceX 这边，xAI 的训练任务已经迁移到了更新的 Colossus 2。据 The Information 报道，**Colossus 1 的模型利用率只有大约 11%**，远低于行业平均 40% 左右的水平。空着也是空着，租出去是纯收入

SpaceX 计划六月 IPO。在招股书里多一个 Anthropic 这样量级的算力客户，对估值叙事有直接帮助

--------

不得不说，这个世界上有两大奇人：懂王和马斯克...你永远想不到他俩将要干啥

回想一下，今年年初 SpaceX 和 xAI 合并，合并后的估值 1.25 万亿美元。马斯克在 X 上骂 Anthropic 的频率比骂 OpenAI 还勤快，说 Anthropic「hates Western civilization」，说 Dario 是在「projecting」

三月份五角大楼把 Anthropic 列为供应链风险，马斯克的 xAI 拿到了军方合同。从外面看，这就是两个阵营的对立格局....然后今天，交易官宣

Anthropic 联合创始人 Tom Brown 在 X 上写了一条帖子

We're going to need to move a lot of atoms in order to keep up with AI demand, and there's nobody better at quickly moving atoms (on or off planet Earth).

我们需要搬运大量原子来跟上 AI 的需求，没有谁比 SpaceX 更擅长快速搬运原子（不管是在地球上还是在太空中）

马斯克在这条帖子下面回复，说自己上周花了很多时间跟 Anthropic 高层会面

Everyone I met was highly competent and cared a great deal about doing the right thing. No one set off my evil detector.

我遇到的每个人都非常有能力，都很在意做正确的事。没有人触发了我的 evil detector

从「misanthropic」到「no evil detector」，中间只隔了**一周的高层会面和一笔数据中心租约**

不过马斯克也加了一句：SpaceX 保留在 Anthropic 的 AI「做出伤害人类的行为」时收回算力的权利

  

算力拼图
----

客观来说，这笔交易是 Anthropic 目前**唯一能立刻到位的大规模算力**

Anthropic 签了一系列总量惊人的算力协议，但大部分产能要到 2026 年底或 2027 年才能上线

**亚马逊**高达 5GW 的协议，其中近 1GW 的新增容量计划 2026 年底前到位

**Google + Broadcom**5GW 协议，2027 年开始陆续上线

**微软 + NVIDIA**300 亿美元的 Azure 容量

**Fluidstack**500 亿美元的美国 AI 基础设施投资

四大 deal 的总容量超过 10GW，但远水解不了近渴。SpaceX 的 Colossus 1 本月就能用，直接缓解了眼前最急的算力缺口

Anthropic 在三种芯片上同时跑 Claude：**AWS Trainium、Google TPU 和 NVIDIA GPU**。多线押注的策略在 2026 年的算力争夺战里提供了更多腾挪空间

  

海外和太空
-----

Anthropic 在博客里还透露了两个信息

第一，**国际扩展**。金融、医疗、政府这些受监管行业的企业客户，越来越需要数据留在本国的基础设施。Anthropic 跟亚马逊的合作已经包含了亚洲和欧洲的推理节点。Anthropic 强调只会在「民主国家」部署，要求合作方的法律框架和供应链都安全可控

第二，**太空数据中心**。Anthropic 在协议中正式表达了跟 SpaceX 合作开发「多吉瓦级轨道 AI 算力」的兴趣。今年一月 SpaceX 向 FCC 提交了部署百万颗卫星轨道 AI 数据中心的申请。Anthropic 成了第一个公开表态要参与的大模型公司

另外一个细节：Anthropic 之前宣布会承担因其美国数据中心导致的居民电费上涨，现在表示在国际扩展中也会探索类似的承诺

  

同一周的密度
------

SpaceX 这笔交易只是 Anthropic 本周密集发布中的一环

**周一**Anthropic 宣布跟 Blackstone、Hellman & Friedman 和高盛成立 15 亿美元的企业 AI 服务合资公司

**周二**在纽约金融服务专场，推出 Claude Opus 4.7、10 个面向金融行业的 Agent 模板、Microsoft 365 全套集成、Moody's 深度接入。摩根大通 CEO Jamie Dimon 和 Dario Amodei 同台

**周三**Code with Claude 旧金山站开幕，SpaceX 算力交易官宣，Claude 限额当天翻倍

Anthropic 目前的融资估值谈判已经到了 **9000 亿美元**。从算力布局到产品线扩展到行业渗透，节奏非常密集
