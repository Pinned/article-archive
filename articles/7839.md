# DeepSeek网页版大升级！随后宕机11小时崩上热搜，新模型真的来了

##### 梦晨 发自 凹非寺  
量子位 | 公众号 QbitAI

DeepSeek崩上热搜！

宕机持续超过8小时，写论文的、角色扮演的和心情不好找AI吐槽的人也都崩溃了。

But！这不是一次普通的服务中断，反而被解读为模型升级的前兆。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGGrpz5wEibNA1x1cMVuI6qRCKLqiaNf23L3PCXt1wJBACh6Tiatg4b9UyfTvuzv4fcSV5goP9hCMkWbWTP9x67vddNNsxCibPQibeos/640?wx_fmt=png&from=appmsg)

就在宕机前，很多用户报告DeepSeek网页版它变了，模型能力大幅提升。

比如经典的用SVG画鹈鹕骑自行车任务，3月29日的DeepSeek，构图和色彩都明显比一周前强。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHFRK8IJ3YLwIwIP658r9iahFBOSbtg4H2DvM5ibIzoBnGs0fGMPAiaA3KlnJHLcp95wVzENc9q2XNucbeLTocLe2NpEJKoPFxWoI/640?wx_fmt=png&from=appmsg)

而且DeepSeek这家公司，不预告、不公告，瞧瞧静默升级模型，已经不是第一次了。

DeepSeek真的变了
------------

模型到底有没有真的升级，答案就藏在对话记录里。

首先是身份标识更新，3月29日的版本会稳定的介绍自己为DeepSeek-V3模型。

一周前，同样的问题，它的自我介绍还是模糊的“我是DeepSeek，一款纯文字AI助手……“，对自己的版本号只字不提。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGnulPdJjylLOB31ofPd6niatNRxM7H8JU6CDT5aD7n9vBic3YaYSR5SgtVkjVJJFHbmnBEG9mukaK9oBGjvaDPXDyyNV64bibf84/640?wx_fmt=png&from=appmsg)

如果说这种变化也可能是偶然，那么知识截止日期的变化更有说服力。

在不开启联网搜搜的情况下，现在它知道25年的美国选举结果。但不知道26年2月的大事，有人据此推测新版本的知识截止日期可能是26年1月。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGUOHwsib4Zboe2Loe1wfR87yeKsOibx4TR4QMWuyQkd2A3wEicmCmsSicEicKI65YPicCz7tfkrgpfWhiastCzvpSXVDqMZdyngdfy9Q/640?wx_fmt=png&from=appmsg)

最后，3月29日DeepSeek一次性写代码生成前端页面的效果也有大幅提高。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHHRseA7Klt6icumhHNE4MBbVHs7DKPm4GKyVxaYV3RVE9zJj6v4UuYz4XtWAVFbFYfKG1Q7Pa9gLopoFOsddtVUgGlKVTmf7Ac/640?wx_fmt=png&from=appmsg)

所以到底是微调后的V3，还是直接升级了V4？

DeepSeek官方一个字都没说。

One More Thing
--------------

目前DeepSeek网页版已恢复服务。

不过还是有点小问题，开启深度思考模式的话，模型在思考结束后就会停止输出，并不会把答案输出在正文里。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHS4XcuJxkpclMwFaXXxaWVhXpC5MXUWQhhCcoZSGCMrEN0IMia1vgK3O6YJQbUj3IHkJwhK6S77hme9R5n3EOer20gic0llvwOM/640?wx_fmt=png&from=appmsg)

不开深度思考模式的话，从模型自我介绍看似乎已恢复成老版本。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHzf63pyscaMpLqRuGOPVwFGB6mxj9WQianZAuYR0qJdQQO7AtDGBxjtH7vU8e8ZkAwKXEjqMc7f7CwMjJvYibx0ZKj4IGyfqeKc/640?wx_fmt=png&from=appmsg)
