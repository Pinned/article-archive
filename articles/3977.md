# Nano Banana 2，泄露！

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acuozy5wJiauTWStQDX3pygTjqvEPW0GjCMgJAWFPwN198fygNe1XOJT39D3JsYZIyP7ichWsmic8zCSWmc83nQLy8rnibVkLvBHLyp4/640?from=appmsg)

**Nano Banana 2泄露证据被扒：4K生图，速度离谱。**

**作者 |**  李水青  

**编辑 |**  云鹏  

智东西2月25日报道，过去48小时，**Nano Banana 2**成为AI开发者圈的热议话题。在海外社交平台X上，关于谷歌这款最新图片生成模型（又名**Gemini 3.1 Flash Image预览版**）将发布的帖子层出不穷，**4K图片**四处流传，各种猜测也甚嚣尘上。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurccMicJV8ib2gozUjiax1K8Z5M8iaBZp1UL1OpFe8r0Cap3MWefEQPR0dg49zCM3dVDZ0TVia1CV0ZomlvuJKoia05zfFD4zX8DNFbY/640?from=appmsg)

▲社交平台X上关于Nano Banana 2的猜测截取

众所周知，基于Gemini 3 Pro的Nano Banana Pro在AI图片生成领域接近“封神”的存在，而Nano Banana（Gemini 2.5 Flash Image的昵称）去年发布时也引起业界沸腾。**如果Nano Banana 2真的发布，其相对于Nano Banana Pro在性能和价格上会有多大程度的升级？引起产业高度关注。**

根据X平台多方自媒体互证，Nano Banana 2为Gemini 3.1 Flash Image预览版的昵称，具备**4K图像生成能力、更快速度、价格比Nano Banana Pro更低**等特征。此前2月20日，谷歌发布其新一代旗舰模型Gemini 3.1 Pro，通常情况下，Gemini 3.1 Flash及图像功能有望在近期内发布。

TestingCatalog News是AI领域关注者较多的自媒体，其在今日的推文中称，谷歌正在为即将发布的Gemini 3.1 Flash镜像预览版做准备。基于Gemini 3 Flash的**Nano Banana 2（Flash）模型已于去年12月进行了测试**，但随后推迟发布了，或许很快将发布。他还随之发布了一张疑似Nano Banana 2生成的图片。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuoaYLatIvJtptCe31P5lKyK1vic0m2nBqp8wSwrpjOs4q5mIK4Cc1E24qIA3GtvPtjc0I6BqjFUwKHABp0iaCDn19p1vkibYNI78A/640?from=appmsg)

▲社交平台X上的自媒体发文

他引用了另一位X平台网友MarsEverythingTech在2月24日发布的推文，推文中Nano Banana 2（Gemini 3.1 Flash Image预览版）的早期测试4K生成图随之流出。如下面图片所示，四张图片在细节生成和文字渲染等方面看起来表现力不错。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqqDQtpFyNBnm8Bu7SpG0p1ZTTjd5CLBgoeLvfl42o7iaIta6UwAicmXccNT6M1OLpialHB469KzX97OuAkcdxW3bNWianJ6DZuExs/640?from=appmsg)

▲社交平台X上的网友发文

随后，一位名为Legit的关注度颇高的开发者发布推文称，一个新的匿名模型**anon-bob-2**已经上线，它很可能是竞技场Arena.ai（原LMArena）上的新款Nano Banana Flash模型，由Gemini 3.1 Flash Image驱动。不过智东西登陆Arena.ai后未发现这一模型，可能是测试版已下线。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuoBn1ZlBhwWcQx5BfGDSwvbh3kydhicQ5V77nmmEgiaUNZFF3asyA1TwfbdDjaGyIw0gCr0TWrPBFyU1jYlcpZOR8vOvAp2tasQ8/640?from=appmsg)

▲社交平台X上的网友发文

他还提供了在谷歌云的企业级AI平台Vertex AI网站上，发现Gemini 3.1 Flash Image镜像的证据。如下图所示，其输入Gemini-3后下拉页面出现了Gemini 3.1 Flash Image模型的名字。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acur40NAHoggUbaoz6icnthibwpptrnkhk9v9q6HMAFoumj0BPWhsedjdVE8WCum1YkOG6WcQvVDxC6CLV0ODQ8cKp2tOUcxYdwSl8/640?from=appmsg)

▲社交平台X上的网友发文

另一位X平台用户称：“Nano Banana 2（Gemini 3.1 Flash Image预览版）已以匿名用户anon-bob-2的身份活跃于竞技场Arena.ai的图像对战模式，社区目前取得的成果之一：它很棒！”

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuqRpn32u7oIm5uoVDVL10e7vpqvc8dnn5ycty6pG1tibD437Pq2hgG80s0ohJic8abaaKbH6kicmLMNpo7ibPXPz2bnxDJs5hInf54/640?from=appmsg)

▲社交平台X上的网友发文

如果泄露信息属实，Nano Banana 2或许会结合前两代产品的优势： **Flash系列的速度和价格优势，以及接近或优于 Nano Banana Pro的视觉质量。**

比如，其生成图像或许会具备**原生4K分辨率**、更出色的**多角色场景处理能力**，以及在人体结构、反射、光照等方面**更一致的细节表现**。据测试人员称，所有这些都实现了**闪电般的生成速度**，远超以速度慢著称的Pro版。

据Stable Diffusion博客推测，原定于2025年12月进行的（Nano Banana 2）内部测试因质量校准问题而被推迟。现在模型已准备就绪，图像功能**可能就在未来几天或几周内发布**。

截至发稿，谷歌尚未就此发布任何官方公告。

  

**结语：头部玩家密集发新**

**AI图像生成竞赛或升级**

如果泄露的信息属实，Nano Banana 2或许会是一款超高速、支持4K图像生成、价格比Nano Banana Pro版更低，画质却与之不相上下的模型，有望引起产业的新热潮。

但Nano Banana 2未必能建立起绝对的优势。近一段时间，字节Seedream 5.0、阿里Qwen-Image-2.0、智谱GLM-Image等国产模型相继发布，在指令遵循、图像质量、文字渲染等发面发力，或许都能够与谷歌新图像模型掰腕子，AI图像生成竞赛将再度升级。
