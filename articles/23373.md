# 最快本周五！OpenAI紧随SpaceX递招股书，Anthropic提前两年盈利

![](https://mmbiz.qpic.cn/sz_mmbiz_png/lYDricib2fsm7F2JnxaykTrWFVps7ZErgUzKp8uzKS35O7uaazyKQicrQU8AcyNibNpK3xkF8I7OxNfdcRCdfsGkGyHnKs0SLDFdkCt0lWwB6WM/640?wx_fmt=png&from=appmsg)

作者 | 四月

本周，OpenAI、Anthropic、SpaceX（xAI） 三枪齐发，华尔街可谓是烈火烹油了。

最新消息，OpenAI 最快将于本周五（5 月 22 日）秘交 IPO 招股书草案。按照目前一级市场 8520 亿美元的估值推算，上市后市值大概率破万亿，**有望成为史上最大规模的上市交易。**

如果流程顺利，OpenAI 今年 Q4 即可挂牌。这个节奏，可比年初爆出的计划快了将近两个季度。奥特曼之所以快马加鞭，还是马斯克和达里奥两位“老冤家”已经逼到了家门口。

同一天，马斯克的 SpaceX （xAI）公开 S-1 招股书，启动路演准备。其目标估值 1.75 万亿美元、计划募资 750 亿美元。

这个体量预计将打破另一项纪录——**史上最大募资规模**。2019 年，沙特阿美一次性募资 294 亿美元。在这之后，该纪录已沉寂七年了。

同样在今天，华尔街日报还披露了另一组数据。正在进行新一轮融资的 Anthropic，向投资人亮出了底牌：公司 Q2 营收预计达 109 亿美元，环比暴增 130%，**创下成立以来的首个盈利季度。**

按彭博社此前报道，Anthropic 最早将于今年 10 月上市。加之近期 Anthropic 开放员工股权兑现，多数人选择持股而非套现，可以解读为给上市前景投下的信任票。

Anthropic 迎头反超

OpenAI 上市提速，最直接的压迫感来自 Anthropic。这一点，从无论两家公司被动，亦或是主动，选择在同一天释放消息，角力的意味已经很浓了。

Vibe Coding 的浪潮，可以说是直接带飞 Anthropic。今年二季度，Anthropic 预计将实现 130% 的增速，不仅超过了谷歌和 Facebook 当年 IPO 前的历史峰值，更是突破了 Zoom 疫情期间最高增速。

**更让奥特曼刺眼的，还有 Anthropic 的盈收能力。** 披露数据显示，其 Q2 运营利润将达 5.59 亿美元。

要知道，去年夏天达里奥给投资人的画饼还是“2028 年盈利”，**如今硬生生提前了整整两年。**Anthropic 与 OpenAI 较劲的底气，可以说是全写在财务报表里。

Claude Code 积攒的产品口碑，让 Anthropic 在 B 端市场形成虹吸效应，并逐步向 C 端渗透；加上抗打的 Sonnet 和 Opus 模型稳住基本盘，顺利让 Anthropic 在这一局完成了对 OpenAI 的反超。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/lYDricib2fsm5FYibjNLL81XNgemUUXzx8HZTIj8SIkF6Ymm2neuiaD6RoFMSgZk5icib56Ruica20ib6EMBW1JgviawH08tygq2u6TXXW70hNrIkXVc/640?wx_fmt=png&from=appmsg)

一级市场的估值曲线最能说明问题：OpenAI 当前估值 8520 亿美元，而 Anthropic 已飙至 9300 亿。两家公司的走势在本月交叉后，开始分道扬镳。**而随着本轮融资谈完，两家公司的差距可能进一步拉开。**

比钱更敏锐的信号，是人才的流向。

昨天，Andrej Karpathy 宣布加入 Anthropic 的 pre-training 团队。作为 OpenAI 联合创始人、特斯拉自动驾驶的技术底座，网友评价他“是可以去任何公司、拿到任何条件的人”。Karpathy 作为全球开发者精神领袖，**他的职业选择本身就是一种无声的表态。**

过去半年，Instagram、Box、Workday 等大厂 CTO 纷纷放弃高管头衔，转投 Anthropic 甘当普通研究员。真正懂行的人在用脚投票，且方向出奇一致。

这种人才磁场的转向，对两家公司未来三五年模型能力的影响，可能比今天的财务数字要更锋利。

于是，上市成了奥特曼的必选项，在窗口期彻底关闭前，把定价权从私募市场夺回，让二级市场来接力讲这个故事。

SpaceX 递表

但它早已不是一家火箭公司

如果说 OpenAI 和 Anthropic 的竞争，还能被理解为两家头部独立模型公司在厮杀份额，那 SpaceX 今天的招股书则说明，这场战争的边界远比想象中宏大。

SpaceX 今年 2 月全股票收购了 xAI。翻开 S-1，"近期进展"部分有这样一条记录：Anthropic 与 SpaceX/xAI 签署协议，每月向旗下 Colossus 和 Colossus II 超算中心支付 12.5 亿美元算力费用，合同期到 2029 年。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/lYDricib2fsm4NApFd2GYZQiaIXWIAuOIT0SoKxOkU8Hjb3VYB9ib9Lwfeb6cnr8ic5buGOzlDtj7ERDunJMZMfGcib80dyPIYkeibXZrUiawoSclaU/640?wx_fmt=png&from=appmsg)

这意味着，Claude 的日常运转和迭代，有一部分算力是向 Grok 的东家租的。大模型赛场的刺刀见红，和算力生意的甲方乙方，在马斯克和达里奥之间奇妙同框。

这个细节也折射出两个信号：一是算力已经紧缺到了连竞争对手的数据中心都要抢的程度（友商的友商就是盟友）；二是 SpaceX 的 AI 叙事不是附属品，而是 S-1 核心故事的一部分。

Colossus 和 Colossus II 两座超算中心算力加起来接近 1GW。但硬币的另一面是，xAI 去年营收 32 亿、亏损 64 亿，Q1 亏损继续扩大。这个窟窿由谁来填？

答案是 Starlink。SpaceX 的卫星互联网去年创造了约 80 亿美元运营利润，是整个集团真正的现金奶牛。所以，这场 IPO 的真实叙事是：**“用卫星互联网的稳定利润，给 AI 军备竞赛输血”。**

往后真正能拉动股价的，不会是下一次发射，而是 Grok 什么时候不再亏钱。

结    语

三巨头同赴二级市场，底层逻辑如出一辙：AI 基础设施的军备竞赛，已经突破了一级市场的承接极限，到了不得不让公众二级市场来接盘的时刻。

在 Hacker News 和知乎的评论区里，中美网友罕见达成共识：这是 AI 泡沫破裂前的强烈信号。
