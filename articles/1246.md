# 刚刚，微信支持连接小龙虾🦞了

终于，你的微信也能连接到小龙虾了！  
  
刚刚，微信团队重磅官宣，正式推出ClawBot插件，支持接入OpenClaw。  
  
接入后，你只要通过微信聊天的方式，就能快速调用自己的“龙虾”高效互动。  
  
目前，微信ClawBot插件正在逐步放量中，并且需要将微信更新至最新版本。  
  
无论你用的是云服务还是本地部署，都能通过微信扫码完成绑定。  
  
首先，你需要进入微信“我的”界面，然后点击“设置”，在“功能”栏目中找到“插件”。  
  
如果能看到P1这样的界面，就说明你已经被灰度到了。  
  
然后，使用下面的命令，让你的小龙虾帮你生成绑定二维码。  
  
npx -y @tencent-weixin/openclaw-weixin-cli@latest install  
  
如果你用的是腾讯云LightHouse、WorkBuddy、QClaw这些鹅厂自家的版本，还能直接通过桌面端UI界面直接获得二维码。  
  
快点开你的微信设置，到“插件”菜单里看看你有没有被灰度到吧~
