# 他给谷歌做出了最火CLI工具，结果被开了

##### 闻乐 发自 凹非寺  
量子位 | 公众号 QbitAI

谷歌老兵干了七年，结果因为写了个2.8万星的CLI工具，人被开了。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGG3QibTiaqib0UYExGLibdM3BHD1ZulbiaZtVNWUNibib81RV6qCjpvIRZuRatm2lPRnsmGG6hSXbvOUuvTQyAS4zQYKAIcR471c4CW8w/640?wx_fmt=png&from=appmsg)

然后…然后法务也来了——

“为什么Google的logo为什么出现在了这项目的GitHub仓库上？”

对，你没听错。一个在谷歌官方GitHub组织下、由谷歌DevRel团队成员开发的项目，被质疑“为什么用了谷歌的品牌”。

更离谱的是，**把人开除后，谷歌反手发布了一个官方版同款工具GWS**……

龙虾之父看不下去跑出来力挺：还好谷歌开除不了我。

网友们更直接，这个谷歌怎么总是功亏一篑功亏一篑功亏一篑……这家公司不是没有好东西，是好东西出来一个，自己干掉一个。

打压开发者的风波还没平息，谷歌又接连爆出人才出走连环噩耗。

两天走了两个诺奖级人物，连带两名Gemini核心开发者也纷纷跳槽。

这波**《置身谷内》**，不是只有我在期待吧（doge）。

2.8万星CLI工具的离谱结局
---------------

咱先说说这个让谷歌法务连夜出动的工具到底做了什么。

一直以来谷歌Workspace各个服务的API都是割裂分开的，调用邮箱、云盘、日历、表格都要各自对接接口、单独配置授权。

尤其在2026年，AI Agent满地跑，Agent想操作Gmail、Calendar、Drive这些服务却没有顺手工具。

Poehnelt做的这个Google Workspace CLI，核心设计就是运行时动态读取Google的Discovery Service，自动生成配套指令。

它把Gmail、Drive、Calendar、Sheets、Chat、Admin全线产品的API封装，只需一条命令就能完成各类操作，所有返回数据统一输出标准 JSON 格式，还内置四十多个适配 AI 智能体的功能模块。

连谷歌元老**Addy Osmani**都跑来站台叫好。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFdwqMPhhibPe00ooO8rXiak8N8KyQVpnXkDxXYWJTvhYRLdsnKb79icH2enOJ7p8ULI749j2l19JeYJEhxgst4QaB3Ltqv8ZlNyo/640?wx_fmt=png&from=appmsg)

这位Google Cloud AI总监，在谷歌干了14年，是Chrome DevTools、Lighthouse、Core Web Vitals背后的核心推手。

3月5日他亲自发帖推广，这条帖子后来被Poehnelt称为“那条害我被开除的推文”。

谁也没想到，工具爆火引来高层争相取经的同时，法务部门也很快找上门，拿仓库擅自使用谷歌Logo、品牌视觉为由追责施压。

更讽刺的是时间线，就在Poehnelt收到解雇通知的前两天，谷歌刚在Cloud Next大会正式官宣，要推出官方版Workspace CLI。

这款官方工具瞄准的需求、主打的AI Agent适配、统一命令行调用的核心思路，和Poehnelt业余做出来的工具高度重合。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGF3NfOwJN0xyTSSKPKlibpgcMj933O24gyfFicNTJs0uPhauRHdqA4RaZcVxTm9LX6jdJd1K4oYO1YWXBsuFsTOmMDwr9EwLGTQs/640?wx_fmt=png&from=appmsg)

而发推宣传CLI的Osmani，也已经确定离开了谷歌。

把别人做的东西官方化，再把人开了。这操作让很多路人都看不下去，直接喊话Sundar Pichai——

应该开除的是那些促成Poehnelt被开除的人，这才是谷歌真正该消灭的文化。

两天跑两个诺奖级
--------

一边是干活的被开除，另一边，牛人自己走了。

6月18日，Transformer共同发明人**Noam Shazeer**宣布离开谷歌，加入OpenAI。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEIfiaYtgurSjpGad0D382qlal1HpxAgG66ichrR2jTOsezMC7lyaQlCbVfRGaeZg57JLlF74cNMSo48vt8gtkQFgvs1YBTMWjNg/640?wx_fmt=png&from=appmsg)

而且这已经是他第二次离开谷歌了。上次他因不满谷歌商业化太保守而出走，创办了Character.AI。

后来谷歌花27亿美元把他弄回来，安排他联合领导Gemini。结果不到两年，人又走了，直奔最大竞争对手。

两天之后，又有另一颗重磅炸弹：**John Jumper**也走了，宣布加入Anthropic。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGG2X1ULia8tmsdnmMicsbWJyvhCKknxwF0KYGVTibnba6RTTJgRSR4GFPtrSYLel8NGfpeh50ibl4obHjfiaFkMV8PP0uxSrLXChic2Q/640?wx_fmt=png&from=appmsg)

这位是AlphaFold项目的领导者，和DeepMind CEO Hassabis一起拿了2024年诺贝尔化学奖，在谷歌待了近九年。

还没完，谷歌风波一波接一波。

据彭博社消息，这两天Gemini两位核心开发者Jonas Adler和Alexander Pritzel也双双跳到了Anthropic。

接连流失顶尖AI人才的消息重创市场信心，谷歌母公司Alphabet当日盘中一度大跌近7%，收盘跌超5%，创下近一年以来最差单日行情。

谷歌，真有点说法。
