Agent OS 是什么
------------

Agent OS 的架构可以拆成三层：`ONE`、`Real`、`悟空`

**钉钉ONE：桌面**

这个东西，定位是人和 Agent 的交互入口

打开钉钉，不再是消息列表，而是一个 AI 编排过的工作信息流——今天要处理什么、哪些消息重要、日程怎么安排，AI 帮你排好优先级

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUh7VrKU1YfiaS2cw6zAw4OwWv8vnJDgqtolFs2AamnxgJzexzfCbOaiaw/640?wx_fmt=png&from=appmsg)

底部是一排可滑动的 Agent 应用。消息 Agent 帮你过滤噪音、日程 Agent 帮你订会议室、差旅 Agent 帮你比价订酒店

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUts5GKfZicXNWnSk4P8TzM3UNqaJrPgaXibVDzOjOxQx5aa3jY4nxgWJA/640?wx_fmt=png&from=appmsg)

  

这里有两套操作模式：GUI 和 LUI 并存  
每个功能都有 A/B 两面，传统界面和 Agent 一键切换

高频确定性动作用按钮，需要推理的事交给 Agent

有个细节：ONE 团队在 1.0 阶段做过一次激进尝试，把消息列表完全干掉，改成「划短视频」式的卡片流，结果用户反馈分化，部分用户不太适应

最终，团队决定：多种交互会长期并存

**DingTalk Real：给 AI 一台专属电脑，物理硬件**

这是 1.1 最硬核的发布：  
**给 AI 用的专属硬件。与豆包 AI 手机助手类似，不过它同时可以运行电脑环境和手机环境**

![](https://mmbiz.qpic.cn/mmbiz_jpg/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUJiaHkPT0Sf870SNBEfDLMYIoNicOaalgumWMUG96agMxTjdF7YCtd2ibg/640?wx_fmt=jpeg&from=appmsg)

  

有了 Real，AI Agent 就有了自己的运行环境  
能用浏览器、能打开软件、能访问公司内网、能查知识库和业务系统

三个能力：

*   • `Real Entity`：AI 有主机，能用浏览器和软件干活，全程安全隔离
    
*   • `Real Data`：能访问公司的差旅、人事、知识库等内部数据，以及互联网数据
    
*   • `Real Time`：永不下线，并且一直可访问实时数据、实时执行
    

7×24 小时在线，人下班了 AI 还在跑

具体能干嘛？出差前说一句「帮我订明天去上海的行程」，Real 会规划路线、全网比价酒店、提交审批。你睡觉它在跑，醒来任务完成了

**悟空：Agent 总管**

这是个通用 Agent  
能理解钉钉上的聊天、日程、文档、业务数据，能调用各种 Agent 和工具，能自己拆解任务、执行、反思

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUks5bXHz8nBu69Gn4keaYenLEOWPo2Eu8qhDOeFgz0z3lA3gia9kgeZg/640?wx_fmt=png&from=appmsg)

  

比如「帮我写一份 Q4 销售分析报告」，悟空会：`找数据` → `分析趋势` → `生成报告` → `发给你确认`  
中间调用了 AI 表格、AI 搜问、文档 Agent

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUks5bXHz8nBu69Gn4keaYenLEOWPo2Eu8qhDOeFgz0z3lA3gia9kgeZg/640?wx_fmt=png&from=appmsg)

全称 Dingtalk Enterprise AI Platform，允许开发者帮助企业构建自己的 AI 生产力

**AI 表格**

两个新功能：

*   • `随口说`：语音录入，支持东北话、四川话等方言，说话就能填表
    
*   • `万能贴`：复制/截图/拍照任意内容，Ctrl+V 粘进去自动整理成结构化数据

**AI 搜问**

三个能力：  
`能搜`：打通企业知识库、聊天、工作等数据，实现企业知识全域汇聚  
`能问`：接入全球主流大模型，根据任务自动匹配最合适的模型  
`能干活`：识别任务意图时唤起悟空执行

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrKwLz7rGoHibO6EHC2O24PUZVBzISfDXr7SdRiaXFzZDJNjib22DbBrkn3Hh4Z5yhQjicFvNB5z5oSGQ/640?wx_fmt=png&from=appmsg)

  

有个 `安全模式`：  
脱敏模型加密敏感信息，调用外部模型后解密还原，企业可以安全使用全球任何模型

还有个「事实问答」能力。表格数据中潜藏着物理世界的业务逻辑，但字段是被压缩的。AI 搜问通过知识图谱引擎结合模型推理，还原物理世界的事实，对真实业务情况进行问答

AI 搜问 1.1 发布后，这些能力会沉淀为钉钉全局基础服务，覆盖 IM 聊天、AI 表格、AI 听记等场景。同步上线浏览器插件，网页阅读时划词即可调用 AI 问答
