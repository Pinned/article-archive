# 网易龙虾来了！生成式AI盛会最新嘉宾公布，腾讯混元领衔参与大模型记忆研讨会

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurclHZZJB67KWz7KEX3ptic1s1uY4hft7fqibGkhupFLibCrB6fndJMgeBoW89tRXVV7u0alau89nUiaRMGHJkSIEicxiaNo9C2rzE3U/640?from=appmsg)

**4月21-22日北京站将正式举行~**

4月21-22日，**GenAICon 2026 | 2026中国生成式AI大会（北京站）**将在北京富力万丽酒店正式举行。中国生成式AI大会已成功举办四届，现已成为国内人工智能领域最具影响力的产业峰会之一。

本次大会由智一科技旗下**智东西**联合**智猩猩**发起主办，为期两天，由**开幕式+专题论坛+研讨会+交流晚宴+展览区**组成，将以**“奔赴AGI 重塑未来”**为主题，邀请**70+位**重量级嘉宾与会带来致辞、报告、演讲和对话。

其中，**开幕式**将于4月21日上午在主会场+分会场一拉开帷幕；4月21日下午主会场将进行**AI算力基础设施专题论坛**，**大模型专题论坛、AI智能体专题论坛**则将于4月22日在主会场依次进行。

同时，大会首次开设了六场闭门技术研讨会，分别于分会场一和分会场二进行。

分会场一将于4月21日下午进行**大模型记忆技术研讨会**，4月22日则将依次进行**LLM强化学习技术研讨会、LLM推理系统技术研讨会**；分会场二将于4月21日下午进行**生成式世界模型技术研讨会**，4月22日则将依次进行**OpenClaw技术研讨会、视频生成技术研讨会**。其中，研讨会主要向持有大会通票和贵宾票的观众开放。

今年，在大会同期也设有**人工智能创新展览区**，以标展形式为主，将展示人工智能产业链优秀企业的创新技术、产品与方案，展商预计将覆盖**大模型、AI智能体、AIGC应用与AI Infra**等方向。

此前，我们已经公布了大会部分嘉宾。今天，将为大家继续揭晓开幕式和专题论坛嘉宾的最新进展以及大模型记忆技术研讨会的完整嘉宾阵容！更多确认与会分享和交流的嘉宾以及研讨会的议程，也将尽快公布，敬请期待。

此次大会也将组织六场技术研讨会，分别为大模型记忆技术研讨会、LLM强化学习技术研讨会、LLM推理系统技术研讨会、生成式世界模型技术研讨会、OpenClaw技术研讨会、视频生成技术研讨会。六场研讨会在分会场一、二进行，主要向持有大会通票和贵宾票的观众开放。

此前，我们已公布六场技术研讨会上将带来报告分享的部分嘉宾（[《人大赵鑫教授领衔、阿里“龙虾”技术碰撞！中国生成式AI大会要来了》](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652798652&idx=1&sn=bdd43dcd5dde5cc44d1a15c21b9808bc&scene=21#wechat_redirect)）。

**蚂蚁集团大模型算法专家邓勇，中国人民大学高瓴人工智能学院博士研究生、A²D一作陈志朋**将在LLM强化学习技术研讨会带来报告分享。**香港大学助理教授&博士生导师、Nanobot团队负责人黄超，HiClaw项目发起人、阿里云智能高级解决方案架构师付宇轩，中国科学技术大学副研究员程明月，Zilliz工程总监陈彪**等四位学者与专家将在OpenClaw技术研讨会上进行报告解读。**Sand.ai 创始成员、算法负责人李凌志，ModelTC开源社区负责人龚睿昊，快手可灵实习研究员、DreamID-Omni一作郭旭**则将参与视频生成技术研讨会并进行报告分享。

大模型记忆技术研讨会的嘉宾也已经全部敲定。**记忆张量联合创始人兼CTO、上海算法创新研究院大模型中心技术负责人、研究员李志宇，EverMind CEO兼盛大集团副总裁邓亚峰，阿里国际AIB团队算法专家、UMEM主要作者兰天，腾讯混元大模型团队Principal Researcher王锴，阿里云高级技术专家马腾，智悦云创及太忆TiMEM.ai创始人余宣庆**将出席并带来报告分享。

李志宇博士长期从事预训练和大模型应用方向的研发技术攻关，主要研究方向包括大模型记忆增强、高效评估与应用算法。近年来，李志宇博士先后和团队提出了首个记忆分层的创新架构大模型，以及业内首个大模型记忆操作系统（MemOS），MemOS开源7个月累计获得Star数超6100+，开发者数超11000+，为大模型的记忆增强落地提供了可行的探索路径。

由陈天桥和邓亚峰带队的EverMind此前发布过世界级长期记忆系统——EverMemOS。团队本周刚发表论文，提出了一种名为 MSA（Memory Sparse Attention，记忆稀疏注意力）的全新架构，就像是给大模型外挂了一个“无限容量”的记忆硬盘，成功将大模型的端到端长时记忆库扩展到了惊人的 1 亿Token 级别。EverMind CEO兼盛大集团副总裁邓亚峰将在本次研讨会带来深入分享。

兰天聚焦于Long-horizon Agent相关技术的前沿研究与业务落地，在人工智能与自然语言处理领域拥有扎实的学术积累与丰富的工业界实战经验，当前致力于推动大模型与复杂Agent技术在真实业务场景中的技术演进与创新应用。此次研讨会，兰天将就“UMEM：打破Agent“死记硬背”困局的可泛化记忆框架”这一主题展开介绍。

近期，腾讯混元发布了一篇名为《HY-WU (Part I): An Extensible Functional Neural Memory Framework and An Instantiation in Text-Guided Image Editing》的技术报告，提出了一种崭新的功能性记忆范式，在模型推理时实时生成个性化参数，由此对不同任务/分布/样本进行个性化适应。腾讯混元团队将HY-WU （混元无相）应用在各大开源图形编辑基模上，有效扩充了基模的功能性记忆，在图形编辑任务上普遍取得了更好的内容理解、指令遵循和生成质量。与此同时，该范式的数据效率、训练效率和推理速度也做到了真正的实用性与创新性并重。腾讯混元大模型团队Principal Researcher王锴博士将在此次研讨会进行针对性分享。

Engram 条件内存为大语言模型解耦了静态知识查找与动态计算，成为下一代 LLM 的潜在核心组件。由于 Engram 表现出稀疏的访问模式并支持预取功能，其庞大的嵌入表非常适合卸载到低层级存储器中。为此，北京大学联合阿里云等在“Pooling Engram Conditional Memory in LargeLanguage Models using CXL”这一论文中，首次提出使用 CXL（Compute Express Link） 内存池来存储 Engram。与 RDMA 相比，CXL 能够提供 Engram 所需的最小化且离散的检索模式要求的细粒度和低延迟访问。研究团队将基于 CXL 的 Engram 内存池集成到 SGLang 框架中，实现了接近本地 DRAM 的端到端性能。（[《Engram新范式！北大联合阿里云首次提出基于CXL的Engram内存池化架构，性能接近本地DRAM》](https://mp.weixin.qq.com/s?__biz=MjM5ODExNDA2MA==&mid=2450002799&idx=1&sn=1c5ef20d27a33aefd979cb0f2c862d9e&click_id=4&scene=21#wechat_redirect)）马腾博士是这篇论文的核心作者之一，也将参与大模型记忆技术研讨会并进行分享。马腾博士目前在阿里云主要从事大模型推理软件栈的开发和研究工作，并共同创建了大模型KVCache开源项目Mooncake（5K Star）。

余宣庆博士的主要研究方向为大语言模型、时态知识图谱推理与认知计算，本次研讨会上，他将以《持续观察、反思学习并预测未来的类脑认知引擎》为主题带来报告分享。
