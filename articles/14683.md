# 微软与OpenAI“分手”，独家协议正式走向终点！

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/S1iaf4GgGjEzsCDXwDzf1S1OqSNmPLjy9cHsRUh4B52Kmx7FBjGh16TfupyzhM253yuVZM3bo7kv4KfToxz9z9lPKvnS1qZiaz0FZq6pViaibMs/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)

整理 | 苏宓  

出品 | CSDN（ID：CSDNnews）

牵手 7 年，微软与 OpenAI 的关系，开始“松绑”了。

从双方发布的联合声明来看，微软放弃了对 OpenAI 模型及相关知识产权的独家使用许可。这意味着，OpenAI 不再只绑定 Azure，也可以与包括亚马逊在内的其他云服务商展开合作，为自身业务打开更多出口。

双方对此的解释也很直接：“快速的创新节奏要求我们持续演进双方的合作关系，以更好地服务客户并让两家公司共同受益。”

新协议中的关键变化

1\. 独家变非独家：OpenAI 不再“只此一家”

过去，微软拥有对 OpenAI 模型及产品知识产权的独家许可与访问权。现在，这一独家权被正式取消。

按照新协议，微软仍然拥有 OpenAI 模型及相关知识产权的使用许可，期限持续到 2032 年，但性质从“独家”变为“非独家”。

这也意味着，OpenAI 可以同时与其他云厂商合作，对外分发模型能力。

2\. 分成机制调整：微软拿得更久，但有上限

另外，收益分配方式也发生了变化。

微软将不再向 OpenAI 支付收入分成。

反过来，无论 OpenAI 在通用人工智能（AGI）上的进展如何，微软都可以持续获得分成，直到 2030 年。

对比此前的协议，一旦 OpenAI 实现 AGI，相关分成原本会终止。现在，这一“触发条件”被取消，改为固定周期，分成比例不变，不过设定了总额上限。

3\. 不再“绑死”，但仍是深度绑定

这次调整，并不意味着双方关系降级。

根据协议，微软依然是 OpenAI 的首选云合作伙伴。OpenAI 的产品仍会优先在 Azure 上发布，除非微软无法或选择不提供相关能力。

同时，微软仍是 OpenAI 的重要股东，继续参与其增长。此前在 OpenAI 向营利性公司转型的过程中，微软获得了约 27% 的股权，并在相关谈判中扮演了关键角色。

换句话说，关系还在，只是不再是“单一绑定”。

背后原因：双方各有现实压力

这次调整，并不难理解，本质上是双方各退一步。

一边是 OpenAI——模型规模越做越大，对算力的需求也迅速膨胀。仅依赖一家云厂商，已经难以支撑其扩张速度。

今年早些时候，亚马逊云业务就表示，将与 OpenAI 合作开发并托管相关产品。甚至有报道提到，微软一度考虑采取法律行动，认为这可能触及其原有的独家权利。

另一边是微软——这几年明显在推进“多模型策略”，不再把所有筹码押在单一模型供应商上，而是同时引入多个模型来源，降低依赖风险。

也正如声明中写道：“修订后的协议提升了可预期性，增强了我们在大规模构建与运营 AI 平台方面的协同能力，同时也为双方探索新的机会提供了更大的灵活空间。”

总结

这次调整，可以明显看出，微软从“唯一渠道商”，变成“优先合作伙伴 ”，而 OpenAI 从“绑定 Azure”，走向“多云分发”。

整体来看，这不是“分手”，而是把一段高度绑定的关系，改成了更灵活、更符合现实的合作方式。

对于这一变化，也有网友认为最终受益者中，Google 也算是一个，其表示：“此次协议的修订，最大的赢家可能是 Google。几乎所有前沿 AI 实验室都在使用 TPU，唯一没有用的是 OpenAI——主要是因为此前与微软的独家合作协议。随着本月刚刚发布的第 8 代 TPU，OpenAI 很可能也会开始考虑采用 TPU。”

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEwp7b7rPadvhuaTcpjEaqwOgf9v6LjgQS5dBryRN5bJqlqvw1bBnF1bGy5MgLw6sfcJ7a2xEjGVDNtibKsJcHJyuC4SgswMbIMk/640?wx_fmt=png&from=appmsg)

以下是 OpenAI 发布的声明全文：

快速的创新节奏要求我们持续演进双方的合作关系，以更好地服务客户并让两家公司共同受益。今天，我们宣布对合作协议进行修订，旨在简化合作模式与协作方式，以灵活性、确定性以及更广泛释放 AI 价值为核心。修订后的协议提升了可预期性，增强了我们在大规模构建与运营 AI 平台方面的协同能力，同时也为双方探索新的机会提供了更大的灵活空间。协议主要包括：

*   微软仍是 OpenAI 的首选云合作伙伴，OpenAI 的产品将优先在 Azure 上发布，除非微软无法或选择不支持相关所需能力。与此同时，OpenAI 现在可以在任何云平台上向客户提供其全部产品。
    
*   微软将继续拥有 OpenAI 模型和产品相关知识产权的使用许可，期限至 2032 年，但该许可将由独家变为非独家。
    
*   微软将不再向 OpenAI 支付收入分成。
    
*   OpenAI 向微软支付的收入分成将持续至 2030 年，与 OpenAI 的技术进展无关，分成比例保持不变，但设有总额上限。
    
*   微软仍作为主要股东，继续直接参与 OpenAI 的增长。
    

尽管此次修订简化了合作关系，但双方的合作目标依然雄心勃勃。从扩展以吉瓦计的数据中心算力规模，到协同研发下一代芯片，再到将 AI 应用于提升网络安全等领域，我们期待继续携手推进 AI 在全球范围内的落地与规模化应用。
