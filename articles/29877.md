# 马斯克奇袭Claude！速度翻倍，价格腰斩，杀进AI编程牌桌

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcuqbA6ESrAAmYyYetOMKLkqAxa652ckG36bUqGhuajcaLrCHIoaa7pHticEcP1oRA7nnav264ibXR6lQyhDibCKGeJibPM0nDEly06M/640?from=appmsg)

**马斯克预告，下月还有大招。**

**编译 |**  程茜  

**编辑 |**  心缘  

智东西7月9日报道，今日，马斯克旗下SpaceXAI宣布推出该公司迄今最强大模型**Grok 4.5**，这也是其首款专为编程和智能体任务开发的模型，**与Cursor共同训练**。

Grok 4.5的运行速度为**80 TPS**，同等任务下，其token利用效率达到Claude Opus 4.8（max）的**2倍**。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acuoaricp0L66wZ76YRpwHChcS8s7IFRt3T6FBUWvOHuXc5zsiap2jAuDouRpraNxlRA0f6tVCxUpicQYxYs8hbY8EuhWhzvoJfiabmg/640?from=appmsg)

马斯克发文称，Grok 4.5可以**对标Claude Opus系列模型**，但运行速度更快、效率更高、成本更低。他透露，Grok 4.5的上下文窗口可能会在下周提升到**100万**。Grok 4.5目前还没有使用其自研的C/C++推理软件，理论上，其运行速度有望提升1倍甚至更多。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acupdia5ianCNhh7cUoVPDdGrgFRqVFL7PjWN2atEJJkYGibjZTBlh0liaqs8TVPZ1ibNC5dfkQ0ibHtJJ4nIhSWVqSFNHjDyVeicibFiaQ6I/640?from=appmsg)

▲马斯克推文截图

有网友对比了Grok 4.5、Claude Opus 4.8、Claude Sonnet 5和GLM-5.2的效果，其基于SpaceX的一次发射任务进行了测试，模型生成火箭从发射、级间分离、进入轨道到卫星部署的整个过程。

结果显示，Claude Opus 4.8的生成速度最快但成本最高。与Sonnet 5和GLM-5.2相比，Grok 4.5的动画效果呈现更准确、生动，Grok 4.5成本介于两者之间，GLM-5.2最终只花了0.13美元就完成了任务。

Grok 4.5的定价为每百万token输入价格为2美元，每百万token输出价格为6美元。其官方博客提到，因为Grok 4.5能在更少的步骤内完成任务，所以**成本更低**。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuoDNZlCDmQ2IIZImrrNrH072GWn20PmPWm9ttrsAdynd1sWmdtDvicNXI87kXJfGeamQR0HAtBuIu3GV3XXI6R4eJ1MlbDibqpI8/640?from=appmsg)

开发者可通过命令行AI编程工具Grok Build、面向企业的SpaceXAI控制台**调用Grok 4.5 API**；Grok 4.5已成为**Cursor订阅用户的默认底层模型**。

此外，**Grok 4.5已接入OpenClaw**，开发者绑定X Premium或SuperGrok订阅就能调用该模型。

马斯克透露，下月发布的新版本将迎来又一次跨越式性能跃升，他们将完成业务闭环，落地解决特斯拉、SpaceX、Neuralink与隧道挖掘公司The Boring Company旗下各类真实工程难题。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqC9aNR32RiaiaKjx7qHdp7UkzKFFTiaq2uib9tk8vcjbSFoIvQgDX2p1u76dfayLNpyyQmwPyb9YRBVmTyt0oAiaeAMV9IlaEFvo0c/640?from=appmsg)

这也是马斯克旗下的商业航天、通信与AI巨头SpaceX上市、收购美国明星AI编程独角兽Cursor后首次发布的新模型。

  

**_01_****_._**

**多位开发者实测后爱不释手**

**夸Grok 4.5生成又快又好**

知名AI游戏开发者Danny Limanseta用Grok 4.5做了一款卡牌游戏，并对这一模型的效果给出了极高的评价。

他提到，Grok 4.5的激发创意、提出建议能力与Claude Opus 4.8相当，但Claude Fable 5略胜一筹；Grok 4.5速度极快，堪称所有前沿模型最快的之一，包括Opus 4.8、GPT-5.5等，使用Grok 4.5开发是一种享受；能轻松创建带有粒子效果的爆炸动画等视觉效果，动画流畅度以及时间控制都更好；从规划到执行的全过程，都可以使用同一个模型，无需换成本更低模型。

![](https://mmbiz.qpic.cn/mmbiz_gif/8Eyv4o4Acuqr4bibiaBRQbfmd8OZjJ0Vxp9ypvtnoMeByLc9wwrcS6yaSnZsYsKxhcSTLIOvRAamhuhTeAT6zbD4VJYXLyOKxC5P4AsHuJh0U/640?from=appmsg)

氛围编程社区BridgeMind的创始人Matthew Miller，用Grok 4.5制作了一款恐怖游戏，并称其相比Grok 4.3有“巨大的改进”。

![](https://mmbiz.qpic.cn/mmbiz_gif/8Eyv4o4AcuqCq1L8nTJekQYhyVmdE1UKjCrXISicRvVDxyPQO90Z9K7alOfcGAHcOrFBEO5cov110kibS7pyctk3vsIJcbaU1ibkxc3DRdO81g/640?from=appmsg)

Grok 4.5还能生成一套宇宙与太阳系仿真程序，支持时间变速调节，具备真实天体运动、行星轨道与恒星效果，基于Three.js实现；配套了界面平视显示器（HUD）。

![](https://mmbiz.qpic.cn/mmbiz_gif/8Eyv4o4Acuq6DzqvM08FPlkZYpgBxLkjxlRibrTByg2ISmQ9ggBsicl9eia5Umaqwx3d8xTk79vS2awuavmK5xSu8kHbfrj2Ar9qNj0rkEKqOQ/640?from=appmsg)

在办公方面，Grok 4.5目前是AI编程智能体Grok Build的默认模型，可以用来构建复杂的Excel表格，包括联网调取调研数据、跨工作表编写联动公式，以及在表格中插入批注备忘，方便用户后续查阅。

在Word和PowerPoint中，该模型能够利用自带的图形元素设计幻灯片内容，同时在Word中写出条理清晰的文字。Grok 4.5仅需一句提示词，就能制作一份5页的季度业务回顾报告。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcupOoE6dYVscCL00xuIgPoIa9z2CViaoDgQgy7GTDVib8OxeR32cHia22B557lRlVL6bURObQh2lE7bNsAWxf2Roicg1L4kY5uG9NVc/640?from=appmsg)

有开发者对比了Claude Fable 5和Grok 4.5的生成效果，根据其放出的截图可以看出，任务为支持用户在后台目录面板批量创建多个3D模型素材，无需保持弹窗常驻；统计完成该功能所需耗时与token消耗量。

最终Fable 5花了10分钟、消耗9.3万个token，Grok 4.5仅花费56秒、处理5.8万个token就搞定了任务。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acup74UCINpC9hBwBxXN34E3UAFWZznRKXpA6fADRAq4GSjRALqTGpZBoB2NUNypwMoKC8WP6TLmR7fEYHGJRf5WfxeLI2shQZrw/640?from=appmsg)

结合Grok 4.5和Grok Imagine，开发者打造了一个3D家居设计工具，将于明日上线。其推文提到，基于这一工具，不到1分钟就能设计出完整的2D和3D平面图。

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/8Eyv4o4AcupSZKASKbrG5UwkPKmibPKcUZsfp9VQH9WLHJxU2NJT5FWPp5KS9TsNW8e7YRtbKaygBibnasDSQP7YDhpfObpZj89yvF1ZCQ3rQ/640?from=appmsg)

**_02_****_._**

**基于数万块GB300训练**

**专为大规模工程任务打造**

Grok 4.5基于数万块英伟达GB300 GPU训练，并配套了专为大规模训练任务设计的训练与稳定优化技术。

在训练数据方面，研发团队在数据筛选与整理环节中，通过数据去重、质量评分、细分领域定向筛选等手段，打造了覆盖更广泛、有效信息密度更高的混合训练数据集。

其还在强化学习方面进行了优化。整套强化学习训练覆盖数十万类任务，围绕多步骤软件工程及其他技术类工作，配套自动化、模型驱动的评分机制。

整套技术栈为高异步训练架构打造，智能体轨迹推演可连续运行数小时，同时在数万块GPU上同步开展持续学习。

此外，Grok 4.5能够以更快的速度、更低的成本为用户提供精准的智能分析结果。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acuoh4y0T4LPwJ4QuTcWU0JkSNvEanr2UXtCXP2icBAlcsr1UGDxOsThURMGfkh4IhaKmnW4kS3tPHaqJNYZzLxoLzfnfhlWlsVT4/640?from=appmsg)

Cursor X帖子提到，相比其迄今发布的所有型号模型，Grok 4.5的表现都更出色，并已成为其团队中许多人的日常办公工具。并且，该模型仅是首批模型，还有更多内容即将推出。

在Cursor中，Grok 4.5与Composer分属两套完全不同的模型权重规格体系，其Composer 2.5版本会持续提供服务，后续Cursor将推出同规格量级的全新模型。

**_04_****_._**

**结语：马斯克放大招！**

**要用Grok拉通旗下实体企业闭环**

马斯克此番推出Grok 4.5，无论从模型性能还是调用成本来看，均对标OpenAI、Anthropic当前旗舰模型，并具有一定竞争优势。

与此同时，在收购编程独角兽Cursor后，SpaceXAI直接将Grok 4.5设为Cursor编辑器的默认底层模型，打通了模型训练、开发者落地实测、真实工程场景数据回流的闭环，有助于其迭代优化编程能力。

放眼长期布局，Grok系列模型还将持续接入特斯拉、SpaceX、Neuralink等马斯克旗下的实体企业，并获得一手工业数据，未来或能深度应用于高端制造、航天航空、脑机接口等场景。
