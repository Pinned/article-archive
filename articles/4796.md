# OpenAI 开发者计划，开放申请｜Codex for Open Source

前几天，[我入选了 Claude for Open Source](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247513921&idx=1&sn=ce2eb39484433f7a889f1c54a6b3afbb&scene=21#wechat_redirect)，里面提到一个事儿：类似的开发者补贴计划，OpenAI 也有（当时在等发布）

现在，OpenAI 正式宣布了 `Codex for Open Source` 计划，面向开源开发者提供支持

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdiay7uNbicoMCFwyW0q7kEqyW4wiaA4jXVRfBjTVALlGrlmwiaf5wDEVM2uBPXROVrc8QQD5ic0g0JYR3HfGnpWoHJE1g1tLeMssNvg/640?wx_fmt=png&from=appmsg)

https://x.com/OpenAIDevs/status/2029998191043911955

这是对标 Claude for Open Source 的，给的东西也差不多，都是 `6 个月顶配订阅`、`工具产品权限`、`额外 API 额度`

ChatGPT Pro 就是那个 200 美金每个月的套餐，6 个月下来价值 1200 美元

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdgLUngBliahHucCAppz7niaJ9WgYnTcXNlO2njbMcnPrtvBbVncJia5Vm50UwFZKWlSMjyvekadImiblsWT0icjDjHWXmMicKj84DOlo/640?wx_fmt=png&from=appmsg)

ChatGPT Pro

* * *

项目给什么
-----

选中的维护者可以拿到三样东西：

*   • 6 个月 ChatGPT Pro（含 Codex），用于日常编码、issue 分类、代码审查等维护工作
    
*   • Codex Security 的条件访问权限，针对需要更深安全覆盖的仓库，逐案审核
    
*   • API 额度，用于 PR review、维护自动化、发版流程等核心开源工作
    

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdgIS5pz5r1R21uySTAvyMsoLkp7X5t47de55LbS4MjjEChTXe2rUVhvfqnmG0kRU27f8EPoH2JwIWicUnMgsAAy3wYibgAxSutk0/640?wx_fmt=png&from=appmsg)

https://x.com/OpenAIDevs/status/2029998191043911955

Codex Security 这块值得单独说一句：  
**这部分访问权限会逐案审核**，不是申请就给

并且，这个权限只能用来扫描自己有权限维护的仓库，条款里明确禁止拿它探测别人的代码库

* * *

怎么申请
----

申请入口：  
`openai.com/form/codex-for-oss`

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdh5YibUD1pORfNJNGDTphX0kaIoPhf0p70gfbDyguORf855MwSTonINRntHPOtoib5Q5fdzPSGEowpYJkicaIicrJnFwpHdCR2VsPo/640?wx_fmt=png&from=appmsg)

openai.com/form/codex-for-oss

填的内容比较多，除了基础信息，还需要：

*   • GitHub 用户名和仓库链接（需设置为公开）
    
*   • 描述你在项目里的角色，主要维护者还是核心团队成员
    
*   • 说明项目为什么符合条件，Star 数、月下载量，或者在生态里的重要性（500 字以内）
    
*   • OpenAI Organization ID（在 platform.openai.com 的设置里找）
    
*   • 说明打算怎么用 API 额度（500 字以内）
    

官方没有给出像 Anthropic 那样明确的数字门槛，措辞是「有实质用量、被广泛采用，或对生态有明确重要性」。页面上还有一句话：「如果你的项目不完全符合标准，但在生态里扮演重要角色，还是可以申请，说明原因就行」

底层依赖、工具链里的关键包，也有机会

**滚动审核，没有截止日期**

* * *

两个都能申
-----

条款里都没有排他性要求，如果你同时维护着活跃的开源项目，两个都申请就行

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdj69oLqbdmWZoMwbhw9zRwLc0BvaomAv8q2w5pQC6pXGA3YunAicYHg0vJoCGhyXqrhAZib2k218tSQfr9Ub0X3pTjsPsbyo58o0/640?wx_fmt=png&from=appmsg)

对比

申请 Anthropic 的入口是 `claude.com/contact-sales/claude-for-oss`

OpenAI 的是 `openai.com/form/codex-for-oss`
