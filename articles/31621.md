# 视频解说 Skill：教你分清「柯基」与「吐司面包」

青年大学习

  

虽然大家都经历过标准的九年义务教育（应该）

但一些必要专业技能，大家可能都没有系统学习过，比如「如何吃饭」、「如何走神」，甚至是如何从「如何从百度搜索 Steam」这种高级技能，这些技能看似简单，实则一点也不容易

比如《如何吃饭》，这是一个相当复杂的任务，你需要了解餐具的持握方式、食物入口的时机，以及咀嚼与吞咽的先后顺序，稍有不慎，就可能导致任务失败，饿了肚子

  

  

万物说明书
-----

除了吃饭外，还有好多东西是需要学习的，我也做了一些，供大家学习

**如何呼吸**说明吸气与呼气的交替规律，帮助你在无人提醒的情况下持续维持这一行为

  

**如何在群里发「收到」**讲解发送的时机，以及这两个字之外是否需要附加其他内容

  

**如何假装在听**点头的频率、目光的落点，以及在适当位置发出简短回应的技巧

  

当然，还有更高级的技能，比如《如何用百度搜索》。你得先打开浏览器、找到输入框，这些都是比较难的操作，甚至后面你还得把问题原样贴进去...天呢，这一步应该能拦住很多的群友，真不是一般的困难

比如如果想知道「正版 Steam 的下载地址」，就可以通过百度搜索来获取这些有用信息，在完成复杂的输入之后，结果页上就会显示许多有用的信息

  

以后，如果群里再有人问「xxx 在哪下」的问题，就可以把这个视频发给他，赠人玫瑰、手有余香，日行一善总是好的  

除此之外，这一个 skill 还可以被看作科研利器。就比如说，我们可以去做一个科研视频，去聊一聊“柯基”和“吐司面包”的真实区别。因为这两个东西其实是极其高度相似的，很多人可能分不清。那我们不妨去尝试用一些更直观的方法，让大家能够分得清柯基和吐司面包

通过 LibTV 一键制作
-------------

这些视频全部出自 LibTV Agent，liblib 家的视频 Agent，**今天正式上线，大概有三：**  

一、Agent 交付成片

和各种大家熟悉 Agent 一样，LibTV Agent 有一个输入区，可以把需求、参考文件，产品图、视频啥的都丢给他，然后它自己就会把把剧本、分镜、角色、配音、配乐、剪辑整条流程跑完，给你的是一条能直接发的视频。
