# DeepSeek你怎么连HR都缺啊？！

##### 克雷西 发自 凹非寺  
量子位 | 公众号 QbitAI

不是，DeepSeek，我知道你缺人，但没想到你怎么连HR都缺啊？

这不刚好是毕业季嘛，DeepSeek官方放出了全新一轮的招聘公告。

其中除了算法数据运维这些技术型岗位，像HR法务行政等职能部门也都在招人。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGyxHtGibuegAOcnaxzMeXic7aRuaSCHcmGnwSeKSfnicsZKVefN7lfiaHr57F2fUa8pp8ic0j3sdrY3QDyh0hy5cpYicicJglqtfNbVA/640?wx_fmt=png&from=appmsg)

其中HR团队还特别加了一条横线，“地位”和AGI核心业务管培生一样，你品，你细品。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEDYqPZoSob6RD9fd65qeAEicVMWeRftSib3ynhnRDxOlRyibRcZugyibmrOAC5uRTe73NC2W7s6JXJia57rSyFlnnmiaktGUhb4cDOU/640?wx_fmt=png&from=appmsg)

那么，这次DeepSeek具体都要招聘什么样的人才呢？

技术&职能，30+岗位同时招人
---------------

DeepSeek官方的招聘页面，一共挂出了30+岗位，覆盖7个职能方向，横跨杭州、北京和内蒙古乌兰察布三地。

大概是这么个格局，全栈开发和算法方向10个、AI核心系统研发6个、模型数据策略5个、深度学习研究员4个，再加上产品、职能、管培各若干。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFaFjFX4htxoTDibmorwp6KJSz0e7N56CuYj4Qnq26KyHpfzdMvovuy8pgtMgguF09a1bE0IUnkGoAiaA38j5RJQpUzAsMsR07Sc/640?wx_fmt=png&from=appmsg)

技术线这块招的最多，这点也符合人们对DeepSeek的一贯印象。

特别是最新押注的Agent Harness方向，正在在疯狂加人，研究员、工程师、产品经理三个方向全部开放，实习全职都收。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGGzKFYEViaiabafs7FrQeIotpJuWYQDjQicGk3ic767UtEY7WvDGjdbXKNFk2VBwIFsTINubKanTlLHSibcPoTAbhGleUFJYYv07qh8/640?wx_fmt=png&from=appmsg)

搜索方向则是在招算法和架构两个子方向，DeepSeek给搜索写的团队使命是「AGI感知世界的原生感官」，要做的「让每次提问都被真正理解」。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGFya6UeibYVgqAOTibBNdhuRTlO1ju4BkRYqjFBRPic7QxcNzR92GdRicqWsyIEovyNCxict4XYuLyYw4cOBfOR3xZ7giaE43ETGbaD8/640?wx_fmt=png&from=appmsg)

Infra这块也招了不少。

大模型训练和推理框架工程师负责训练和推理的底层框架，高性能算子和通信编译器工程师负责榨干GPU的每一分算力，超算集群研发工程师负责让几千张卡协同工作不出问题。

AI平台运维管着整个算力平台的日常稳定，算力集群性能与可靠性工程师专门盯着集群跑得快不快、会不会出故障。

这几个方向拼在一起，基本把从模型训练到线上推理的整条链路都覆盖了。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHOCiaQ24wGrHeXiaklHOXjNS8m1y7rCibVlwHMicRjsphJNmSydTuibh5VpW2dNrqZyia80qhvicRnylWkESFvWSHLN2VpSDlgvzS0r4/640?wx_fmt=png&from=appmsg)

数据线也在招。

Agent要真正落地到具体场景，光有模型不够，得有人负责把各个领域的数据喂进去，代码、小语种、医学法律这些垂类有专人负责，办公、生活、搜索这些通用场景也都有对应的岗位。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFqJ9UftCE3FRKMdRnDR3cuHDkwTfVPytN8EenLVA3TJlniaa5OSxlbVZiaN1XAbYM64AEibhqdCaqjlse8KXEaEiayKfjYJBgg2Tk/640?wx_fmt=png&from=appmsg)

深度学习研究员方向招了四个，预训练、后训练、多模态理解，以及一个叫Frontier的方向，DeepSeek对这个方向的解释是「持续学习/自进化/新范式」，光看这个方向名字就知道是在摸还没人走过的地方。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGMetia6ibcaHppybuxXLyjfhoS395rpXFxN0M79cYa3eJia3ptm8ibb6W6rAVtE4MTbpCyFRSibKLrdrZDEymzv1nGhf0BOmnJq5Zs/640?wx_fmt=png&from=appmsg)

职能线方面，HR、法务、行政、财务、采购全上了，而且每个都招得很细。

HR分招聘和综合服务两个方向，招聘那边强调的是识别潜力而非简历，综合服务那边要求用AI工具把入职、考勤、薪酬这些流程自动化掉。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEVzdA95lTvjPpngceWibk4XEG28gpFHZuPq0Gw9VRicTxtDWMzuUoKqXsIGOF5YNG2FNDaAGMxncGia60nkicpQbqBeician0y2EvX8/640?wx_fmt=png&from=appmsg)

法务要求对AI技术本身有好奇心，不只是懂合规；财务连出纳都列了一整套职责说明；采购团队里有一个方向要求长期驻点乌兰察布机房，负责设备出入库和资产盘点。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEgs2rfjwgVyiaHHbFicynSiambhyDHOddLEmIFWACgrLMFze8UpFdcHcjDYMl7nPFMAUvmurXqhUqBq1gQl8QHqf6p5n6zul4sEk/640?wx_fmt=png&from=appmsg)

除了这些“常规”岗位，AGI核心业务管培生这个岗位值得单说。

这个岗位要求来自顶尖院校、应届或工作三年以内，进来之后根据个人特点分配到核心业务方向深耕，成长路径是「未来的业务骨干或团队负责人」。

这个定位放在DeepSeek这里，意味着管培生进去之后不再轮岗打杂，直接上核心业务。

用DeepSeek自己的还说，这就是他们的用人原则，也就是让新人直接承担最核心、最重要的任务。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFT9yBQjR2m4szHz7acM3DsEnboicCqpia1YxoIGyhAHK9RCrTEWibGlUDNpIGib4XH2W3zsQ5icIfiayUWwGU1w4oTQOVqRMZm5FgvQ/640?wx_fmt=png&from=appmsg)

DeepSeek，要把整条链路都攥在手里
--------------------

从这次招聘的结构能看出来，DeepSeek现在要做的事已经远远不只是出模型了。

算力基建和Agent产品等多线同时在走，职能部门也全线跟上，是一家公司在认真扩张的样子。

算力这条线上的动作一直没停。

DeepSeek在乌兰察布有自己的数据中心，这次招聘里从数据中心交付经理、运维工程师到IDC设计规划，再到采购团队的GPU硬件采购、IDC采购、数据中心资产管理，把选址规划、建设交付、日常运营这条链上的岗位都覆盖了。

一个以低成本算法著称的公司，在物理基建上下这么大力气，是要把算力的命脉握在自己手里。

DeepSeek选这里建自己的机房，并不是随机的决定。

乌兰察布是国家东数西算工程八大枢纽节点之一，年均气温低，散热成本天然就低，新建数据中心PUE能控在1.2以下，清洁能源装机占比超过65%，华为云、阿里云早就在这里落了大体量的数据中心。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHodjCRIujpk7CDz1QBHz6ibyrAJkrOnFrqWGxjtYzl0oj4tzSHU4MVRwXMl8oMGKTiab7TTBkjvhjfibZccMQJicg7kFSfXV3YaQI/640?wx_fmt=png&from=appmsg)

Agent那条线，Harness部门负责人崔添翼从5月开始在X上挂招聘，连招一个多月还没招够，最近又细化出了研究员岗位。

DeepSeek给出的公式是「Model + Harness = Agent」，模型之外的所有工作，上下文管理、长期记忆、Subagent与Multi-Agent协作、自进化Agent，全都属于Harness的范畴。

研究员岗位就是专门负责在这些方向上推进前沿研究，定义DeepSeek对Harness的理解，工程师负责把研究结果落成产品，产品经理负责连接研究员、工程师和用户。

三个方向缺一不可，崔添翼现在三个都在招。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGG4BxVXNWzfhRA58QZyiaBNpeoUyOD79iaCr5ErsVWttxgZoRVX2mOl1VzdnqtDc2ORLPcEvSaW4H8WicDK31IeiaKqJWAf5t8007A/640?wx_fmt=png&from=appmsg)

两条线同时走，加上职能部门这次全线跟上，DeepSeek要撑起的摊子越来越大。

现在的DeepSeek，研究员还在招，机房要自己建，Agent要自己做，数据要自己布，HR法务财务采购一起跟上。

这不只是在扩张，更是在把整条链路都攥在手里。

想要有更强的模型，就需要有自己的硬底气。
