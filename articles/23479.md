# 融资700亿元的DeepSeek，API价格直降3/4

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜泽南、杨文

  

「2.5 折优惠结束了，我们宣布将定价调整为原价的 1/4。」

  

本周五晚，DeepSeek 发布了 API 降价通知，让大家脑子转了半天。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFXHfF4CvQPffLAorm9IBpiaGhCDBO6RicU9QS27qvW7k3o3HUTsPafbCAHLIPqYcXv13ZYpqHhtGibQJbHKFcCiabyBYUsF31uBS4/640?wx_fmt=png&from=appmsg)

  

这次官方直接宣布把原价砍掉 3/4 变成永久定价，限时优惠变成了常态。转换后的实际费率在国际市场上极具杀伤力，价格比 GPT-5.5、Claude Opus 4.7 价格的数字要小，单位还从美元变成人民币。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEY2fCybbyLTafUd42UAM0WJdvY1ULteP3jt8l8H1l5unta50QEicUVMzK8G5BUJ1EAyPW0FSkGawNaGCXu5L113lf3JrRkNXHo/640?wx_fmt=png&from=appmsg)

  

在模型性能保持全球顶尖的同时，DeepSeek V4 Pro 通过独创的注意力架构、并行策略和显存量化带来了极低的推理成本，价格拉到了国际同级别模型的不到 1/10，这对于需要高频调用 API 写代码的全球开发者和企业来说，几乎成了不用做选择的选择。

  

最近大家写代码不是都说烧不起 token 了？那正好。

  

从国内到国外，大家一片欢呼。

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFEtqhXhrn0w6sRRdDnaumNjpMq6SWPkTzzCO1ybyaicPavib6QickKSowvGaNAjXr0m4VL3TFTQYicVdia9j2hgNRqgIHyCbZFP9FY/640?wx_fmt=png&from=appmsg)

  

以至于有的网友都开始替 DeepSeek 操心了：低价 AI 好是好，但这种低价策略能维持下去吗？

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHwJeajVRxyTsic5cNhNJRWCSa6XT7iaL9HPSz9ic2xfyT56wyh1f870Ca6850I9DJFibAic5SBDNhamPYYeP0EIRLRGLrxgE5ShHN8/640?wx_fmt=png&from=appmsg)

  

知名博主 Yuchen Jin 发帖猜测，DeepSeek 此次降价背后可能有两种逻辑：要么是 DeepSeek 在推理优化上取得了重大突破，要么是华为芯片成本比 Nvidia 低很多。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHVgMkwK0mT6QWWIVCqhahXUAFrgF28nUR8ZxRudQ3DXB8ZwHHazmoVQrpXTUh2wAEVwgotRD20F8lXXdbvNnaibrvDoLVUnibPs/640?wx_fmt=png&from=appmsg)

  

而就在前一天，DeepSeek 还传出了百亿美元融资即将收官的消息。

  

钱的问题，短期内大概不用担心。

  

DeepSeek 百亿美元融资临近收官

  

5 月 22 日，据彭博社报道，DeepSeek 正在推进一轮高达 700 亿元人民币（约合 100 亿美元）的融资，创始人梁文锋在与投资者的洽谈中明确表态：公司将把突破性 AI 研究置于短期商业化之前，并以实现 AGI 为最终目标。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFJibTYdquxZwwWy66SAmzh9RicRCvj7zO2WGvcic8O3ljiae7yc9rWZPKJXkic8E1eLQibbHVMeAJWjemBYYAo37wgN4ef7FDlmA8GA/640?wx_fmt=png&from=appmsg)

  

彭博社援引知情人士称，梁文锋在至少一场投资者会议上承诺，将持续开发开源 AI 模型，同时全力追求 AGI 这一宏大愿景。梁文锋态度明确，技术边界的拓展才是第一要务，而非急于变现。

  

DeepSeek 融资传闻经历了一个持续升级的过程。早在 4 月中旬，路透社援引 The Information 报道称，DeepSeek 正与投资者洽谈，计划至少融资 3 亿美元，估值约 100 亿美元，这是 DeepSeek 首次接受外部资本。进入 5 月，融资规模的传言进一步扩大，多家媒体报道指向 73 亿美元（约合人民币 500 亿元）量级，估值高达约 515 亿美元。

  

而此次彭博最新披露的口径，DeepSeek 融资已跃升至 100 亿美元（约合人民币 700 亿元），有望将 DeepSeek 的投前估值定格在约 450 亿美元，若最终落定，将创下中国科技初创企业首轮融资的历史纪录。

  

在投资方阵容上，国家人工智能产业投资基金参与其中，据悉拟出资约 100 亿元。与此同时，腾讯、IDG 资本、Monolith 资本均接近确认参与。据 The Information 报道，两名知情人士透露，中国电池巨头宁德时代正计划投资 DeepSeek，京东、网易也在谈判投资 DeepSeek，但各方投资额度及最终名单仍有变数。梁文锋本人据此前报道也可能个人注资约 200 亿元。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHhc3PIia5Tic28iclMVHwgXqyibZDAAj2yQsjLw4Bvab5tiaSIibDTZPtRnO98WA8p9da7fSsNMJuwlfs0Cc2xku6TtU9YwO2IDOH4Y/640?wx_fmt=png&from=appmsg)

  

组建 Harness 团队，产品化加速

  

DeepSeek 本周开始密集招募 Harness（工程外壳 / 智能体底座）团队，内部直接对标 Anthropic 旗下火爆的桌面端编程工具 Claude Code。

  

新增的职位包括 Harness 产品经理与研发工程师，新成员将全程参与「DeepSeek 桌面端 Agent 产品」研发全过程，这意味着 DeepSeek Code，甚至更多 Agent 产品就要来了。

  

在官方的招聘信息中，他们给出了一个非常有浪漫色彩的公式：Model+Harness = Agent。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqG3ibJmDicDjTeBCtLA7Yh9D11CYNYGqialicbBUiaDdxMzuRKZsa7BiadNLDZkkjwfSwhFzliaZRicSYpib9nke2LoU5hNDotdrGEEzZMU/640?wx_fmt=png&from=appmsg)

  

所谓的 Harness，就是给人工智能的大脑（Model）装上眼睛、手脚再进行规范 —— 规制它的输入、在本地环境中运行并验证代码、出错时自动纠错并收敛。

  

DeepSeek 的新动作或许说明其技术落地将会更加偏向产品化。他们正试图将 V4 系列强大的原生代码能力转化为体验丝滑、能够真正改变开发者日常工作流的桌面端 Agent 产品。

  

一手拿着降价 API 全球无差别吸粉，一手拿到融资钻研前沿方向，同时还要占领「AI 程序员」这个杀手级场景。

  

对于 DeepSeek 甚至中国 AI 产业来说，这个星期已经来到了拐点。
