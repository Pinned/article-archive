# 燧原科技科创板正式过会！国产GPU四小龙即将会师

##### 鹭羽 发自 凹非寺  
量子位 | 公众号 QbitAI

终于！国产GPU四小龙即将全员登陆——

**燧原科技**科创板首发过会，正式向IPO发起最后冲刺。

从今年1月22日IPO获受理起，燧原科技用时6个月，有望成为A股第三家实现上市的国产AI芯片厂商。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEzulw7DpPibTIenk4rfT5uEJm2xkfGPoC9Nc8lI1s7GEaBn3ncg4W6yzeUxDNKicfOElsNkcSzY84uibP0BOicd4sCtqaLl2OYHFk/640?wx_fmt=png&from=appmsg)

本次IPO，燧原科技拟募资**60亿**，计划用于支持后续产品研发相关。

其中根据招股书显示，腾讯重仓押注，从Pre-A到E轮连续6次投资，累计持股20.26%，是燧原第一大股东。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGE7671N6OrLxE2Q0PMfcITxXSDXwM3d2AoSbfSla0e06InLUicdDaBZsOPJbLJvibl3HXAeO5wmWjSH22OxicDibic2vt7LzmSR8Xog/640?wx_fmt=png&from=appmsg)

目前国产GPU四小龙已相继完成IPO进程，摩尔线程、沐曦股份均选择上市科创板，壁仞科技登陆港交所。

燧原冲刺IPO
-------

燧原科技此次计划筹集的60亿资金，预计将流入三个方向：

*   **15.03亿元：第五代AI芯片系列产品的研发及产业化项目；**
*   **11.96亿元：第六代AI芯片系列产品的研发及产业化项目；**
*   **33.00亿元：先进人工智能软硬件协同创新项目。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEaylEot3W2DaYKY1loFrE8LyZwZCI4vicoEoMEs355bAa1kcNkdlTCdiaCicDv4nb0ECo9ks8ibI7wBK2I3kLyYtKDvgHHmuzVNcg/640?wx_fmt=png&from=appmsg)

其中第五代云端AI芯片将作为第四代的升级推出，不仅支持FP4数据计算精度，还将全面覆盖文本、音频、图像及视频的生成功能，并实现跨模态生成能力。

第四代产品**L600**也是燧原首款训推一体加速模组，该芯片已完成流片回片，但暂未进入大规模量产阶段，预计今年下半年实现批量交付。

为满足市场对高性能低功耗算力解决方案的需求，第六代AI芯片也将进一步强化卡间互联能力与多芯粒异构集成技术。

另外，燧原将从超万卡集群核心技术研发、人工智能软件栈研发、核心IP国产化研发以及基于国产工艺的云端AI芯片研发四个方向，共同推动软硬件协同创新。

营收方面，公司合计业务收入从2023年的3.0亿元增长至2025年的9.9亿元，复合增长率达到**81.32%**。

AI加速卡及模组、智算系统及集群是公司的主要收入来源，前者主要面向头部互联网客户销售，随着越来越多AI业务场景适用，2025年度占比更是超85%。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGE6Ic4cEibHrbeib8l2KwXFIFJ7jbDLce96v2Yl4A47GsribHZtUU5zbzk9OW3UyLaOl2Uc0FlLJNApxIsYlQxr4bQY9ibyicylzjfM/640?wx_fmt=png&from=appmsg)

值得注意的是，**腾讯**不仅是燧原科技的最大股东，亦是最大客户。

2025年，对腾讯的销售额占燧原总收入的**83.79%**，远高于前一年的37.77%，其中AI加速卡及模组一项更是飙升至74.90%。

这些芯片落地于腾讯互联网及云平台，为其对话机器人、推荐系统、生成式AI等业务场景提供核心算力支撑。

同时燧原预计，鉴于腾讯仍是国内AI算力的主要需求方，未来一段时期内公司对腾讯销售占比较高的情况仍将持续。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFOIicB3fdTJcsSs54Sj9YL8QvkJ8cia3Arc0eyRiaia0XgcTJtr9hiaoR3Na8HXZWBjxbSr4JicrSOY8raGl02dWFGLG9jKxobL6sjQ/640?wx_fmt=png&from=appmsg)

由于云端AI芯片研发投入较大且客户验证周期长，截止到上会，燧原目前仍然尚未盈利。

但随着营收和毛利的高速增长，公司亏损已逐年收窄，下降至2025年的11.64亿元，并根据供应链交付进度等因素综合预估，**2026年或可实现基本盈亏平衡**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEmD9nbXzvh1q786lFL5mLDBlxeC8z0AWGRoJeAwsX3zb9WkakFc9kK2bNIcEShKlRIibeFbqqdLtiaAEgzR7qFZuj5jMPVjkTo0/640?wx_fmt=png&from=appmsg)

不过燧原科技也在招股书中明确表示，公司创始人目前合计持股**28.1357%**，低于安全线30%，与腾讯持股_（20.26%）_差距较小，一旦IPO上市，实际可支配的公司表决权比例将被进一步稀释，可能会对公司的控制权稳定造成不利影响。

事实上，燧原科技早在2024年8月就首次提交了IPO辅导备案，也是国产GPU四小龙中最早的一家，彼时辅导机构为中金公司，估值160亿元。

随后在2025年11月1日再次提交上市辅导备案，保荐机构更换为中信证券。

直到今年1月1日，公司终于完成上市辅导，并在1月22日，正式受理科创板IPO申请，再到如今**6月15日**成功上会。

作为四小龙中仅剩待上市标的，燧原科技正在成为多方关注的焦点。

关于燧原科技
------

燧原科技的两位创始人，赵立东和张亚林均出身**AMD**。

其中，**赵立东**在2007年到2014年期间先后担任AMD计算事业部高级总监、产品工程部高级总监，并牵头成立AMD中国研发中心。

他也是著名的清华EE85级的一员，在这一批毕业生中，走出了中国半导体产业的半壁江山，包括但不限于韦尔股份创始人虞仁荣、紫光集团董事长赵伟国、兆易创新创始人之一舒清明、卓胜微电子联合创始人冯晨晖、格科微电子创始人赵立新等。

**张亚林**2008年入职AMD，历任资深芯片经理、AMD中国研发中心技术总监，曾牵头全球团队定制XBOX-ONE系列主机芯片并主导开发全球目前最大的融合芯片APU。

随后他们在2018年3月共同成立了燧原科技。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGHf9hWOP0tYB91KfpNgatgaxAFfhYrHQoFUfo1vmznWT6ibRCiaD0aJd2GpFd2G9LqebtjxWrsBluxHuKw7f0jrMIYlKmUQueHO8/640?wx_fmt=jpeg)

公司成立8年来，已自研迭代了四代架构5款云端AI芯片，覆盖AI芯片_（燧思系列）_、AI加速卡及模组_（云端系列）_、智算系统及集群和AI计算及编程软件平台一整个全产品体系。

**底层硬件方面**，公司基于自主指令集，对标英伟达的Tensor Core加速计算单元和NVLink卡间互联技术，自研有GCU-CARE加速计算单元和GCU–LARE片间高速互连技术。

该架构不仅编程灵活性强，而且深度支持AI大模型高并行度加速计算，直面大模型训练与推理场景。

**软件层面**，燧原科技走了一条完全独立于英伟达CUDA生态的路。CUDA是目前全球应用范围最广的GPU通用计算开发平台，围绕着CUDA建立有深刻的护城河资产。

选择CUDA其实是一件相当讨巧的事情，但燧原科技决心从零出发，自研了包括驱动程序、编译语言与编译器、算子库、工具链的全栈AI计算及编程软件平台“驭算TopsRider”。

接着在软硬基础上，燧原科技搭建起**智算系统及集群**，已实现千卡、万卡智算中心项目营收，并已经开始联合客户研发超节点方案，合力打造万卡高速互联集群。

截至目前，燧原已参与58项AI芯片与智算系统的关键国家及行业标准的制定，并斩获三百多项发明专利。

国产GPU四小龙
--------

回到**国产GPU四小龙**身上，燧原科技加上摩尔线程、沐曦股份、壁仞科技这四家几乎是同期创业，也是国内AI算力浪潮下的第一批专注云端通用AI加速卡的公司。

核心团队均来自英伟达、AMD等国际芯片巨头，并先后登陆A股、港股资本市场。

目前沐曦股份还在积极推进H股发行上市，打造A+H双平台布局。壁仞科技也在同步开展A股IPO辅导，已更新至第六期，摩尔线程暂无新动作。

四小龙之间定位也并不相同，各有侧重。

**摩尔线程**专注于全功能通用GPU，自研MUSA架构，同时具备图形渲染和AI计算能力，对标英伟达的GeForce与数据中心双线布局。也是当中唯一覆盖BC双端的公司，既有消费级桌面显卡，也兼顾企业级AI智算。

**壁仞科技**则聚焦在高端云端大算力GPGPU，自研壁立仞架构，主打单卡极致峰值算力，对标A100/H100高端训练芯片，支持千亿参数大模型、异构混合训练。

**沐曦股份**直接对标AMD，自研XCORE通用GPU架构，以AI训推和通用计算为核心，兼顾图形渲染，重安全可控、国产供应链闭环。

**燧原科技**专为AI训练与推理优化，单算子能效、推理成本优势突出，核心聚焦AI推理市场。

除此之外，百度昆仑芯、阿里平头哥等大厂系国产芯片也在陆续加速资本化布局。昆仑芯已同步完成港股保密递表与科创板IPO辅导备案，双线推进两地上市，平头哥目前正在开展内部股权架构重组筹备工作，为后续IPO铺路。

显然，国产GPU IPO热潮已如火如荼，过去行业都依赖融资，现在集体上市后，GPU将进入真正的大规模研发竞争。

目的也很明确：**推进算力国产化。**
