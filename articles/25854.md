# 陆奇投了个「外卖骑手」：导师师从图灵奖得主

##### 林方舟 发自 凹非寺  
量子位 | 公众号 QbitAI

「十年留美，导师师承图灵奖得主，回国后，我却**黄袍加身**，成为一名全职外卖骑手。」

什么？！一位「外卖小哥」竟然登上了奇绩2026春季路演日的舞台？哥们这是参加变形计来了？要知道，台下坐着的可是1700多名投资人。

原来，这位小哥跑外卖，是为了亲身体验配送的痛点。他和团队自研AI眼镜等数采硬件，靠众包骑手网络，采集真实数据，开发具身智能的「导航大脑」。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGHaIficump2eibPb2iaiaHKvEqAKJzLdiaib2eMPuarejqs3g4QNeBkclngn6hhgu9Pqia0pgq59AhRBMegRrgeOTzBN0UxuRVvfVKnhg/640?wx_fmt=jpeg&from=appmsg)

在6月7日路演日的现场，还有许许多多类似的奇思妙想。

有人想和AI同居，有人想让机器人长出「电子肌肉」，还有人专门训练AI寻找下一个独角兽_（投资人：坏了，冲着我来的）_。

这些在奇绩创坛路演的项目，都处于创业的最早期阶段。这也意味着，它们瞄准的方向，就是当下AI创业热点的风向标。

搬好小板凳，我们总结了当下AI创业最值得关注的几个趋势。

Agent：解决真问题
-----------

「自从上一届路演日以来，我们经历了一场结构性变化。」奇绩创坛CEO陆奇开场点题。

具有里程碑意义的拐点是：2025年11月，Claude Opus 4.5和Claude Code的结合，使AI具备了能动性。

**过去，只有人能操作工具；而如今，工具本身即具备了「智能」**。

Agent到底有多火？这届路演日56个项目里，有39个都在做Agent。

Agent，正在深入各个行业的毛细血管中，掀起生产力革命。

有三家公司，不约而同地关注到芯片生产。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGGHnPibncbib05gaGt8sDUauvN3qo1OzZ3ou3j60mYicliaicibkXjmdVqAMxQibCVMaZcq4HzFHG3cqrXguyv7X4I6guAITnFUSS3uek/640?wx_fmt=jpeg&from=appmsg)

*   **心片科技**，让Chip Agent全自动完成芯片设计
    

*   **芯辰启源**，为模拟射频芯片设计公司提供 AI 模型、Agent 和下一代设计平台
    

*   **枢宇科技**，做模拟芯片的 Agentic 设计引擎
    

枢宇科技的CEO汪嘉成说，模拟芯片工程师，仍在花费70%的时间「手搓」调参数。

「缺乏数据集、没有标准答案、不能有误差」，是挡在自动化流程前面的「三座大山」。

「我们将打通光电子芯片从设计到量产应用全链路，带领中国芯片迈入光电子时代！」

当演讲者在台上喊出这句话时，我看到了年轻人的光荣与梦想。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGGQWz8l8G5GCiaWOOdzUqxWGl4mYhMSTEb7BCFjYwiarNE2qiak5tbCiaEfAkXMIq101735h2utJuKb5XFiaX6tTX34bTT6x6wB6KrQ/640?wx_fmt=jpeg&from=appmsg)

潇湘智控，将Agent引入了有色冶金这样的传统行业。它从铝电解切入，正在打造有色冶金行业首个物理智能底座。

有色冶金听起来有点「老登」，但CEO计昊天说，2025年全球有色冶金年产值1.2万亿美元，万亿级市场正在撬动千亿级的需求。

不久前，霍尔木兹海峡的油轮，牵动着全球资本市场的关注。

一家名为「天穹智算」的地理空间 Agent 公司，将卫星遥感、公开数据、AIS 航运数据、天气数据，转化为金融市场可用的参考信号，辅助交易员决策。

「AI领域的下一个独角兽，藏在论文里，GitHub仓库里，或者一条社交媒体的帖子里。等它出现在微信群里，已经晚了。」NextBanker的CEO王新宇说。

他们利用AI，帮助投资人第一时间识别高潜力的水下项目，每周能提供超过1000个项目。

内测期间，有100多家投资机构自发入驻，留存率超过40%，累计挖掘了600多家潜在独角兽。

「天啊，我们不会要失业了吧。」坐在我身边的投资人感慨。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGExYh6DDsZzYtBGVMlXYU9GufQeoZKlibJgAS0exQicib3TN13J85hFxEWgQiaJibSYhpV9XM8akxiaYdHFkoJqb8REKSSgiaKd1FNpbw/640?wx_fmt=jpeg&from=appmsg)

除了提高生产力，Agent还在重构**生产关系**。

几天前，Anthropic呼吁全员停止AI研究，正是担心AI能力的飞速进化导致失控。

AI带来了巨大的安全风险。明悉科技提出，要做「中国的Palantir」，利用自研的Agent，提前拿到情报并发出预警，帮助企业治理水军、监测黑灰产、识别商业间谍等。

Singularity Engine（奇点引擎）CEO曹瑞鹏说，企业正在大量尝试 AI，但真正卡点不是模型能力，而是如何进入真实业务、稳定交付并形成可复用资产。

本届路演日，有10家公司都开展了FDE & AI 咨询业务。AI行业的竞争，正在从模型能力竞争，转向**真实场景中的落地能力竞争**。

数据：最高的壁垒，最多的机会
--------------

占比第二多的创业公司类型是具身智能，共19家。

其中，开展数据业务的公司成为主角。数据是具身智能当前最大的瓶颈，也蕴藏着最多的机会。

正如陆奇所言：「过去，生产对象是原材料、商品、信息和流程；今天，数据既是投入，也是产出，正在成为新的生产对象。」

2026年，许多机器人公司都提出要采集百万小时级别的数据。「人欲善其事，必先利其器」，有创业公司瞄准了最基础的数据采集设备。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGGEviae5YnkvtXzjQzAVTVyEFUeIazNEcqiasl7Y5PySicwSVnBGyKHIZzGsKRb01SDsly7M7m9nib3BbHOYO1UIoYF4RcyGX7OAhc/640?wx_fmt=jpeg&from=appmsg)

人类依靠六条肌肉稳定眼球，确保在运动中也能看清外界。但机器人不同，跑半马的机器人之所以会摔倒，原因之一是运动令摄像头模糊，导致视觉感知质量下降。

小鹰锐视通过「电子肌肉」来主动防抖，配合自研的深层次防抖算法，想给机器人做「最清晰的眼睛」。

创始人付际26岁就拿到北大博士学位，还是视觉中国签约摄影师——**果然，摄影佬最懂防抖的刚需（doge）**。

还有一款灵巧手，竟然打通了碳基和硅基的需求。

SoftSync 柔偕科技开发了全球首款千元级软体仿生灵巧手。单看外形，它与人手十分接近。它先被用于人体义肢，再通过高频使用，积累真实手部操作数据，未来迁移到机器人训练。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGExCF0ZQmlhQBWwr1ic2CZI6RU6NgF9hbxBenccUE5WUrpiaFotmu1h21K97U0YjyKDo84JIBhWOkWV0XfXLWibbN75YvNYUDEC2w/640?wx_fmt=jpeg&from=appmsg)

还有的公司创新了数据采集模式。

你用的手机地图，在室外导航准确，但在室内经常「抓瞎」。这是因为，高质量的室内数据稀缺，不仅获取成本高昂，也难以规模化。

开头「外卖小哥」创立的**肆爱科技**想到了一个办法：建立一个「外卖骑手众包网络」，在日常配送场景中，采集第一人称视角的室内多模态数据。

一个「数据飞轮」形成了——更多数据训练更强的导航算法，更强算法帮骑手提效，进而吸引更多骑手，采集更多数据。

还有公司针对的是数据处理与标注场景。

例如，树为智能与歌渡科技，分别做的是「物理因果数据引擎」和「精细操作的数据引擎」，前者帮助机器人学会理解世界，后者帮助机器人学会接触世界。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGE1w7J05B5Df9JPWxmLsNNAxvzCdvvl20qiaThSbpic4nw1FloicsLFxViclAuyKgrsaVF2nAwxUroBDTcuAUn2IqJXQR5xkJ33iarM/640?wx_fmt=jpeg&from=appmsg)

陪伴：抓住时代情绪
---------

在全场关于技术的讨论之外，这届路演日还有一条柔软的线索。

「别再跟纸片人尬聊了，来和AI一起同居。」LYNOOK灵诺科技的罗镜民，穿着印着二次元小姐姐的T恤就上台了。

这位初中就担任二次元社区站长的创始人说，目前市面上超过60%的AI陪伴产品，生命周期只有五天。

根源在于：这些产品没有和用户在一起的生活空间，没有社交关系网，没有共同经历。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGH3Clan3U0Iw2eH0wlVVFlFJrZJy74iaiaY6SU4SN7jmYurzQSMW3Ao6UBeFGHC1vOlLyNFCwQmdLia4lWBvbf7DOPEaB65ibLwOicQ/640?wx_fmt=jpeg&from=appmsg)

他们开发的桌面陪伴产品，可以让用户通过语音/ 触摸/ 日常仪式与AI角色互动。还支持多设备磁吸堆叠，角色之间可互为「邻居」。

罗镜民让观众想象一下这些场景：你下班回家，她站在房间中间，叉着腰会吐槽你总是把衣服丢在沙发上；马上要下雨了，她想和你一起在阳台聆听雨声……

罗镜民描述这些场景的时候，现场不少人都笑了。

但笑着笑着，我突然发现一个问题：

如果TA真的记得你今天加班、知道你什么时候心情不好、会提醒你带伞、还愿意听你吐槽老板……那TA能提供的情绪价值，已经比许多真人伴侣还多了。

按照这个趋势，和AI女友/男友谈恋爱，可能真的不远了。

除了陪伴，更多人可能正在向AI寻求心理抚慰。

「想通了」的CEO是一个思维认知博主，过去几年，她收到过几千个年轻人的真实困惑。

他们表面上是在询问学业、就业或亲密关系的困惑，但归根结底的痛点只有一个：当卡在人生最关键节点时，没有一个能真正理解TA、支持TA，并持续给予引导的过来人。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGG2TObgaLiacmPZarlIaw3AOKNgyzCFlvSUmVDmJL5ibYFBaaS5szOGibQOSsV2DiaQcB5v1thXKafNR047uh6bJkc4XDbq2Q1wEoM/640?wx_fmt=jpeg&from=appmsg)

「想通了」试图用AI帮人理清困扰。它不做居高临下的诊断，而是以「过来人」的平等身份，通过长期记忆和思维模式识别提供个性化支持。

能为用户提供情绪按摩的，还有具备深度共情能力的智能手环、「修补钢铁丛林中破碎的情感」的游戏等。

AI，正在为现代社会中的人，提供一种**新型情感基础设施**。

Researcher Founder：当研究员成为生产者
----------------------------

这届路演日共有56个创业项目，录取率只有1%，这些项目，是真正的「百里挑一」。

活动聚集了大量关注AI领域的投资人。共有近3500名投资人报名，1700多人来到了会场。

几名创业者告诉我，路演后，他们的微信收到了400-500个好友申请，其中大多都是投资人。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGEVpB7rDeFFKzVtxDCL4mc96H82DslR7kjl16YRwrMQs7IoN1WbszBJibqoMianlzNcFcAb2rnnoWCTrtqmic2icxThrYLBu94tc5o/640?wx_fmt=jpeg&from=appmsg)

我们观察到一个重要的人才趋势：**Researcher Founder（研究型创业者）**，正在成为AI创业的主力军。

本届路演日中，Researcher Founder的占比达45%，几乎占据半壁江山。

复旦大学与上海创智学院联培博士马炜杰，两年达成博士毕业，复旦特奖得主，以独立一作身份，发表了国人团队在三维显示领域的第一篇 Nature 正刊。他的公司灵泷视界，做的是物理智能时代的无感视觉交互接口。

UseIt AI的CEO郜迪飞，中科院计算所博士，NUS 博士后，Computer Use Agent 方向最早期开拓学者，发表顶会论文 40 余篇，多次获最佳论文奖与 AI 顶会竞赛冠军，学术项目 Github 7K+ Stars。公司做的是 Computer Use Agent 的垂类商业化。

类似的例子还有很多，有的师从两院院士、图灵奖得主等学术大牛，还有的清华硕士毕业后，拒绝了CMU Robotics Institute 全奖 PhD ，成为智元机器人第一个应届生……

研究和创新之间的距离，正在被快速压缩。

陆奇将这种现象概括为**「从-1到1」**——过去，-1到0更多发生在大学、科研机构；0到1更多发生在企业。但在AI时代，研究正在更直接地进入生产过程，前沿创业也更多从一个科学问题、研究方向开始。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGHTOIdQVaO0ibnMTMe9GNmkbD2XvTKkQMba5asKJWIajALQabl1x6KGQwNLxacMNkwFI5nJTn55v8rDJ65iaXOQtemhl37nqE6tg/640?wx_fmt=jpeg&from=appmsg)

今天，研究过程正在成为生产过程的一环。Researcher Founder，正在走到时代的前台。

回头看，月之暗面、智元机器人、百川智能……这些资本市场的宠儿，都是从奇绩路演日出发的。

下一个独角兽，下一位明星创始人，很可能就藏在这次路演的项目中间。
