**OpenAI 为何开打补贴大战**

  

OpenAI 帮助文档列过一组数据，Codex 用户平均每月消耗 100 到 200 美元算力。

  

每送出去一次重置，背后就是实打实的 GPU 时间。

  

6 月，编程 Agent 赛道集体改计费规则。

  

Anthropic 宣布 6 月 15 日起 Claude Code 的自动化调用单独计费、按 API 价格走；

  

GitHub Copilot 积分制账单 6 月 1 日已经生效。

  

竞争对手在收口子，Codex 反手放开额度。

  

Codex 客户端还上线了「从 Claude Code 迁移」的一键导入，连搬家路径都替你铺好了。

  

抢开发者，抢的不只是订阅费。

  

编程 Agent 是目前所有 AI 实验室离 RSI（Recursive Self-Improvement，递归自我改进，用 AI 加速 AI 研发）最近的入口。

  

奥特曼 2025 年初就公开把目标从 AGI 调整到了 ASI。

  

OpenAI 和 Anthropic 都确认过，自家工程师的大量代码已经交给自家智能体写。

  

谁手里攥着更多重度开发者，谁就同时拿到三样东西：订阅收入、海量真实编程任务的训练数据、把数据喂回下一代模型的迭代速度。

  

今天多拉一万个重度用户进来，可能就是明年模型比对手快一个身位的差距。
