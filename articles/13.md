# 11人创始团仅剩2人，马斯克“向曾被拒、未获面试机会的人才致歉”：xAI从一开始就建错了，现正在重建

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/S1iaf4GgGjExRCstE0oaRaBjicibibgE9Bn11QuUZK6QVmYWibCCEiajWhVGoYej9lhu4c0nnDtlw4kVxky4DNWczA2qc5OCekHqrAamwaKbTP540/640?wx_fmt=jpeg&from=appmsg)

整理 | 苏宓  

出品 | CSDN（ID：CSDNnews）

“xAI 在最初构建时并不成功，公司正在从头开始重建。”

不到三年的时间，埃隆·马斯克创办的人工智能公司 xAI 正经历成立以来最剧烈的震荡——核心创始成员陆续出走，原本 11 人的创始团队仅剩 2 人，紧急之下，部分来自特斯拉和 SpaceX 的高管被调入参与 xAI 的管理和审查。

对此，马斯克本人也在社交平台上如是说道：

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEwxrZZbO1G8U1kgbkevwa6pdG9KqhytfNviaKKJcuNj5kDLkjNPldDBEvm0mx6A2FThXgicKRnO5I9QMBDFu5dAg0LSBsH42BhMQ/640?wx_fmt=png&from=appmsg)

这一切，与 xAI 刚成立时的高调开局形成鲜明对比。

回看 2023 年 7 月，[马斯克宣布成立 xAI](https://mp.weixin.qq.com/s?__biz=MzkzMDY1NDgyOQ==&mid=2247783289&idx=1&sn=0e40a9b511e9ddd1ae95afc2b4987f1e&scene=21#wechat_redirect)，公开挑战 OpenAI，并从 DeepMind、微软等机构挖来 11 位顶尖 AI 研究人员坐镇。仅四个月后，该公司便发布了 Grok 的首个版本，并在随后的几年间不断扩大算力规模，甚至于去年一度投入约 20 万张 GPU 训练新模型，带来 Grok 3、4 版本。xAI 也一度被视为硅谷 AI 赛道最激进的新玩家之一。

从高调入局、迅速扩张，到如今重新梳理团队和技术路线，这家成立不久、曾被寄予厚望的 AI 公司，为何会在短短几年间便开启重建，也成为硅谷近期最受关注的话题之一。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)

溃败的代码战场，被甩在身后的 xAI

在 AI 编码工具赛道上，xAI 的挫败尤为明显，也让马斯克与团队之间的矛盾更加突出。

2025 年 8 月，xAI 推出号称“极速编码”、“一位多才多艺的程序员”的 Grok Code Fast 1，主打智能体编程。xAI 内部对其持有高度期待。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjExD3PiamIJtQTnNy9kKOibp6BnHZKrTIHQbFCSMRr2ficbYKUYAzQnv5Fdr7hWdQU8pIag06eVEWd3fLqvdWJsvB4eM020oQtLoW0/640?wx_fmt=png&from=appmsg)

然而，现实并不理想。不少用户在深度体验后，称这款产品更注重速度而非深度推理。可对于大多数程序员而言，编码的速度只是加分项，推理才是核心重点。

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEzc7wMmNNvicbicV6ZL40ELNVoomlJtsib2EvqamUP5RgkAice4iaTwYPSMMfVkBwsUTNw1u5S4O6icjkUWSyp32w2LBlXa4A0RUibrCw/640?wx_fmt=png&from=appmsg)

正因此，Grok Code Fast 1 的上线，在与 Anthropic 的 Claude Code、OpenAI 的 Codex 的竞争中逐渐败下阵来。

据知情人士透露，Grok Code 的失利，核心症结之一在训练模型的数据质量上。与 Anthropic 投入巨资打造的高质量代码训练库，以及 OpenAI 依托 GitHub 海量开源代码的优势不同，xAI 的编码模型训练数据不仅体量不足，还存在大量低质量、重复的内容，导致模型在深度推理、复杂代码编写上远不如竞品。

而这一问题，也直接引发了马斯克的不满。

据外媒 ArsTechnica 报道，2026 年 2 月的一次全员大会上，马斯克公开批评编码团队的工作进度，这也成为公司此番新一轮震荡的导火索。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

11 人创始团队仅剩 2 人

产品的失利，最终转化为团队的崩塌。

上周，负责 Grok 模型预训练的 Guodong Zhang 在社交平台 X 上确认，3 月 13 日是他在 xAI 的最后一天。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjExm73beCbMbaPL4MwXbJQq07QAVZnxfHXibWvfxxBicJuAFO4X7EtA8IPHd39EaDibyhO4wgnMzb4wKKLwqDh1agsFMyIusOreyU0/640?wx_fmt=png&from=appmsg)

而另一位联合创始人 Zihang Dai 也在同一周早些时候宣布离职，他此前曾公开承认 xAI 在代码领域处于落后位置。

这两起最新离职，让 xAI 的创始团队几近凋零。此前，Greg Yang、Tony Wu 和 Jimmy Ba 等核心创始人均已相继离开。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEwNFV78eHnZia2pfBS5o5nJiawp5Pv9M4qzL7VTO7QRVxqk5Or3VevLp0XAmV7UktmnGAoGJpTHEoUrQkAiaoJNZb76f9PchHyWH0/640?wx_fmt=png&from=appmsg)

至此，三年前马斯克集结的 11 位 AI 领域的顶尖科学家组成创始团队，如今仅剩 Manuel Kroiss 和 Ross Nordeen 两人留守，逾 80% 的创始团队成员离职。

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEyqwjqibC4LWEjELxxfc9gKsoPv2VPKyVQV2AU2wpUdN8hNibNoiafEDy0gIfKKLNqf7rzicbSTLKbW8GezAOibBsDEpBeiabImkkpSk/640?wx_fmt=png&from=appmsg)

这种创始人的逃离，并非偶然，而是 xAI 内部长期动荡的必然结果。

马斯克对 xAI 的要求，始终带着他标志性的“极致速度”：从产品研发到模型迭代，都被设定了严苛的时间节点，而这种高压，也让团队不堪重负。

据一位曾在特斯拉工作的网友 jarrettcoggin 表示，「我在特斯拉工作的经历完全印证了这一点。埃隆一旦提出要求，就意味着“放下手头的事情去完成它”，然后你还得在原定的时间表下完成之前正在做的任务，哪怕被新任务打断了。」

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEw4oAia1BdnDiaCeWtPhShW4NWClLwWh337lBaMmicxCJAuickkDFvjxibcLPXACJe3QdbuomX3zMx6TARkgtNKBDcNaNPhruRlUnaY/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

特斯拉、SpaceX 高管来“救火”

随着团队动荡，马斯克开始调动特斯拉和 SpaceX 高管，对 xAI 进行全面审计和重组。

据两名直接知情人士透露，SpaceX 和特斯拉的管理人员已被借调至 xAI，审查员工工作情况，并在认定部分人员工作不达标后将其解雇。

另外，马斯克也宣布对旗下团队进行重组，让特斯拉与 xAI 联手重启 Macrohard（巨硬）项目，初衷是打造能“复制整个软件公司”的数字智能体，对标的公司是微软。

不过，这一项目的进展也是一波三折。上个月，马斯克找来前 DeepMind 研究员 Toby Pohlen 负责该项目。但讽刺的是，Toby Pohlen 上任仅 16 天，便选择离职，这个被寄予厚望的项目，也一度陷入停滞。  

Macrohard 项目的困境反映了 xAI 与特斯拉在技术路线上的分歧。据悉，xAI 原本采用基于静态屏幕截图的分析方法开发智能体，而有消息称，马斯克则多次要求改用特斯拉 FSD 自动驾驶系统的实时视频流处理模式，让 AI 能处理连续信息流。这种强行切换导致研发团队措手不及，一个由 600 多名外包人员组成的屏幕记录数据标注项目被紧急叫停，至今仍未恢复。  

无奈之下，马斯克只能调来了特斯拉 AI 软件负责人 Ashok Elluswamy，让其重启 Macrohard 项目，并审查此前的工作。马斯克表示，特斯拉与 xAI 将合作开发的 Macrohard、又称“数字擎天柱”，融合汽车与机器人制造商的现实世界 AI 技术，以及 Grok 大语言模型能力。

不过，对于这些变动，员工抱怨称，公司持续的动荡严重打击士气，也让 xAI 难以发挥其应有的潜力。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q3SAo8dibFicWvibnB5u4gdHLgb2AbA6UJ8VImKnyW92hibqZefwpDbPYAQ/640?wx_fmt=png)

招揽更多人才，马斯克下场为以前招聘致歉

为了填补当前 xAI 的人才缺口，马斯克不得不放下身段，在社交平台上公开道歉：“过去几年，许多优秀的人才被 xAI 拒绝，甚至连面试的机会都没有，我为此道歉。”

他表示，自己正与 xAI 的人才负责人一起，重新梳理公司的面试记录，向那些被拒绝的优秀候选人伸出橄榄枝。

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjExMjpuRgnicEEyQBsj2bL7UhnZeslFgJsKsFaNV0LceS1oWdciajF8MXUdib30FHnsF3SFEFVK7P3OHfalfRkks6ib7IbRgo5fKMhg/640?wx_fmt=png&from=appmsg)

与此同时，外媒透露，xAI 的招聘人员也开始联系此前面试失败的候选人，开出更优厚的薪资条件，只求能快速填补岗位空缺。

而马斯克也展现出了自己的人才吸引力，上周，SpaceX 宣布从炙手可热的人工智能编程初创公司 Cursor 聘请了两名程序员：Andrew Milich 和Jason Ginsberg ，希望能助力 Grok Code 的优化。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEx0Ibr2Lh0sf9hKl5xMbdXCh4EP8n9Ry0qvLtmsdQjSW5u6o20oznqXSTc7hW4EGoskiaEtyTBVZWZv2qG3H5zgsibAtqLdxTG84/640?wx_fmt=png&from=appmsg)

野心难敌现实

整体而言，现下 xAI 的动荡，是硅谷 AI 赛道竞争的一个缩影。如今的 AI 行业，早已不是靠概念和野心就能站稳脚跟的时代，而是进入了“硬实力比拼”的阶段：高质量的训练数据、稳定的技术团队、可持续的商业化路径，缺一不可。

Anthropic 凭借其顶尖的编码能力，赢得了企业客户的认可；OpenAI 依靠前沿的技术，仍能获得持续的资本支持；而 xAI 却陷入了“重速度、轻质量”的困境。

对于 xAI 如今的现状，有网友评论道：

*   「我觉得 xAI 的问题在于，它实际上只能雇两类研究人员——一类是在理念上与埃隆·马斯克一致的人，另一类是纯粹被金钱驱动的人（不是评判，只是事实）。而前沿 AI 研究领域汇聚了大量顶尖人才，这些人往往对自己的研究有强烈的理念驱动，而这些理念通常与埃隆的想法完全相左。OpenAI 和 Anthropic 在理念定位上更能吸引目前的顶尖人才，我实在看不出 xAI 如何与之竞争。」
    
*   虽然我认为 Grok 是一款不错的模型（在我们的一些内部使用场景中，它表现最好，直到 Gemini 2.5-pro 推出），但我忍不住对团队的运作方式感到遗憾。xAI（以及推特）当初宣传得最响亮的是六小时工作制、在办公室过夜以及“持续交付”。然而两年过去了，他们似乎一点成果都没有。我敢肯定，谷歌的工程师们可能每周只工作四天、每天两小时，其中一半时间还花在食堂里，但他们早就把 xAI 抛在身后了。
