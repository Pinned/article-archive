# 50%程序员被裁？微软Xbox启动史上最大规模重组，开发《DOOM》的传奇团队遭重创

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

微软游戏业务正在经历一次剧烈调整。

近日，微软宣布启动 Xbox 历史上最大规模的一次组织重组，计划在 2027 财年裁减约 3200 名员工。随即，Xbox 旗下多个团队陆续传出裁员消息，Bethesda、King、Turn 10 等工作室均受到影响。

但相比某个游戏项目取消，更让开发者震动的是另一则消息：据外媒 Game Developer 报道，负责 id Software 自研游戏引擎 idTech 的开发团队，疑似也在此次裁员中遭到大幅削减。

 有消息人士透露，id Software 大约一半员工受到影响，其中包括负责引擎研发的核心程序员团队，甚至有传言称“大部分甚至全部 idTech 开发人员”已经离开。

不久后，这一消息迅速冲上 Hacker News 热榜，引发大量开发者讨论。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjExp9PxomWRd38wrrvn7FDg8rkatj8OpRq1WTafCcBOI1GZatAeDKpPVycxmljggG60CbE98BVH1jm6gyqbKZIPFiam50AIP1xnk/640?wx_fmt=png&from=appmsg)

对于游戏行业而言，idTech 从来不只是一个普通引擎。它承载了《DOOM》《Quake》等经典作品背后的技术积累，也见证了现代 FPS 游戏三十多年的发展。如果相关消息最终得到证实，那么这套诞生于上世纪 90 年代的传奇技术体系，或许真的正在走向终点。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)  

Xbox CEO：今天的 Xbox 业务“不健康”

此次裁员消息的源头，是 Xbox CEO Asha Sharma 在 X 上公开的一封内部邮件。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEyYgRQXD6b65jwicOicc7Onkkj3Gib0Agvm7oO2NLwp2UiaVK1viau8qWeQk53HzRKErxJzic0bc4v4V9O2O5eFu94tEr2FtjiaicwNOKk/640?wx_fmt=png&from=appmsg)

她在邮件中表示，微软将启动 Xbox 历史上规模最大的一次组织调整，并计划在 FY27（2027 财年）期间裁减约 3200 个岗位，其中第一阶段将取消约 1600 个职位。

同时，还有 4 家工作室将脱离 Xbox 体系，由新的管理团队接手。

Asha Sharma 坦言：“今天的 Xbox 业务并不健康。”她解释称，与其他游戏平台和发行商相比，Xbox 的利润率只有竞争对手的十分之一到三分之一。

在第九世代主机周期开始时，Xbox 面临更小的硬件装机量，却承担着更高的成本结构。过去几年，微软尝试通过 Game Pass、多平台发行以及扩大内容阵容推动增长，这些战略确实创造了一定价值，但增长速度并没有达到预期。

与此同时，Xbox 不断扩大团队规模、增加投入，并延长游戏开发周期，但核心业务仍然承压。

Asha Sharma 表示：“因此，我们必须重塑 Xbox。”

随后，微软旗下多个游戏部门陆续传出裁员消息，包括 Activision Blizzard、Bethesda Softworks、King、Mojang Studios 以及 Xbox Game Studios 等。

而其中影响最为严重的，却是没有出现在此次 Asha Sharma 内部邮件中的 id Software。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

idTech 团队疑似大规模裁撤

作为《DOOM》《Quake》等经典 FPS 游戏的创造者，id Software 于 2010 年被 ZeniMax Media 收购，随着微软在 2021 年完成对 ZeniMax 母公司的收购，正式成为 Xbox 第一方游戏阵容的一部分。  

最初据资深游戏开发者、《毁灭公爵》创作者 Scott Miller 爆料，“id Software 的大部分员工正在被裁员，其中包括绝大多数（甚至可能全部）的程序员。”

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEyVpqibjITCJZGuuskQhT3UKGicIMgAnj9NxMkqribhGsofh5d6RCp7y0uREiaMAJu8F2VFh7Q5D0lu5GQwzoEh1WejvNvLx45Ibq8/640?wx_fmt=png&from=appmsg)

随后，多名 id Software 员工更新 LinkedIn，证实自己受到影响。

其中，在 id Software 工作超过 20 年的资深开发者 Michael Maynard 发文感慨：

「今天对 id Software 来说，是意义重大的一天！我们打造了业内公认最优秀的第一人称动作游戏。《DOOM》《DOOM Eternal》《DOOM: The Dark Ages》，堪称现代第一人称动作游戏的经典三部曲...

我们打造了业内公认最优秀的第一人称游戏引擎技术，堪称任何平台上最顶尖的技术，并拥有业内最优秀的程序员。过去 14 年里，连续三款作品都证明了这一点，毋庸置疑。

然而如今，微软/Xbox 仍决定裁掉团队约一半成员，尽管每一位设计师、程序员、艺术家、音频专家、关卡设计师、特效师和技术设计师都付出了令人惊叹的努力，完成了出色的工作。

就这样让他们离开……对于一个多年来始终追求卓越的团队来说，这是多么大的浪费...

我也是今天被裁撤的一员（约占公司总人数的 50%）。我已经在这里工作了 20 多年。虽然我感到非常悲伤，但我从事电子游戏行业已经超过 40 年，因此这样的事情对我来说并非完全意外。

真正让我难过的是，作为 FPS 动作游戏的先驱和创新者，id Software 如今竟然沦落到又一次进行所谓的资产‘重组’。」

id Software 联创、《DOOM》原制作人发声：别忘了这些人的贡献

消息传出后，id Software 的联合创始人，也是 DOOM、Wolfenstein 3D 和Quake 的原设计师 John Romero 也公开发声。

他表示，自己为所有受到影响的 id Software 员工感到难过。

“我知道离开 id，而 id 仍然继续前行是什么感觉。”Romero 回忆称，离开一个承载了大量工作、友谊和人生经历的地方，是一种非常痛苦的体验。

他认为，过去这些年，id Software 的团队一直努力延续《DOOM》《Quake》和《Wolfenstein》等经典 IP，并且做得非常出色。

同时，他也呼吁关注游戏历史保存问题。

Romero 表示，id Software 的历史对于整个游戏行业都非常重要。他保存了从 Softdisk 时代到 1996 年的重要资料和资产，其中一些内容甚至可能已经不再存在于 id Software 内部。

“希望有人也正在保存这家公司正在延续的历史遗产——包括作品、代码、资源，以及创造这一切的人。”

为什么 idTech 的命运会引发整个行业关注？

从早期的《DOOM》《Quake》，到后来的 idTech 3、idTech 4、idTech 5、idTech 6 和 idTech 7，idTech 代表的不只是一个引擎，而是一整套游戏技术路线，而这套引擎体系推动了 PC 游戏图形技术的发展。

许多经典游戏技术都能追溯到 id Software。

例如，Valve 最初开发《Half-Life》时，就曾获得 Quake 引擎授权；随后演化出的 Source 引擎，又支撑了《Half-Life 2》《Portal》《Left 4 Dead》等作品。

与追求”什么游戏都能做”的 Unreal Engine 不同，idTech 更强调针对特定游戏进行极致优化。

因此，对于很多开发者来说，倘若  id Software 自研游戏引擎 idTech 的开发团队整个被裁，那么被裁掉的不只是一个团队，而是一套持续三十年的技术传统。

对此，Hacker News 上的大量讨论，也都集中在这个问题上：如果微软放弃维护 idTech，是否意味着更多第一方工作室会转向 Unreal Engine？

要知道此前，微软旗下 Halo Studios 已经宣布未来作品转向采用 Unreal Engine。不少开发者认为，微软可能正在推动内部技术栈统一。

从商业角度看，微软的选择并非没有理由。

毕竟，近年来 idTech 已经很少对外授权，主要服务于 id Software 和 Bethesda 自家游戏。相比之下，Epic 依靠 Unreal Engine 建立了庞大的生态系统，覆盖大量第三方开发者。

对于微软而言，长期维护一支规模不小的底层引擎团队，需要承担巨大成本，但能够产生的商业收益可能有限。

一名 Hacker News 网友表示：“idTech 是伟大的技术，但如今微软内部使用它的团队已经很少。如果维护成本远高于收益，公司做出这样的决定并不奇怪。”

裁员引发热议，有开发者呼吁：如果不用了，就把 idTech 开源

在讨论中，还有网友直接提议，微软如今已经很难通过 idTech 获得授权收入，与其让这套技术逐渐消失，不如开放源码，让更多开发者继续维护。

事实上，早期 idTech 系列就曾部分开源，因此许多现代游戏引擎都能追溯到它的技术血脉。

如果未来 idTech 退出商业舞台，开源或许是延续其影响力的一种方式。

不过，无论 idTech 最终会走向何方，这场裁员都已给这支曾打造业内顶尖第一人称引擎的团队留下了沉重的一笔。
