# 一行代码都不会！花270元、烧光1500次请求，他和5岁儿子一周做出游戏：现在作业直接“玩上瘾”了……

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

整理 | 郑丽媛  

出品 | CSDN（ID：CSDNnews）

如果你点进一个 GitHub 仓库，发现里面全是又长又乱的 HTML、JS 混杂在一起的文件，函数名像随手敲出来的单词拼盘——你可能会毫不犹豫地关掉页面。

但这个仓库的主人，一位 ID 为 KiddFlash42 的网友却一点也不觉得羞愧：因为正是这堆“乱得像一盘散沙”的代码，让他和 5 岁的儿子一起做出了一款真正能玩的、《我的世界》风格的教育游戏。

最重要的是，他们俩，一个是完全不懂开发的 28 岁父亲，一个是刚上幼儿园的小孩，本身一行代码都不会写。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Pn4Sm0RsAujSA4EiaC8Nb7Wktic1GlJJBjepvKtianhf04Hz6Az43vicx3xYRGNYqQoNOfdh2OK8hjSelGmum3pcibA/640?wx_fmt=jpeg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)

为了让孩子好好学习，他去问了 AI

这个故事的起点，并不是什么宏大的技术理想，而是一个普通家长每天都在头疼的问题：孩子不爱学习。

KiddFlash42 的儿子特别讨厌练习阅读、写字和基础数学，每天写个家庭作业就像要打一场硬仗。他便想到一个点子：要不干脆把学习内容做成游戏？可问题是，他连“游戏该怎么写”都不知道。

后来偶然听说 “AI 能写代码”后，他就在手机上直接搜了一句：“AI 能做电子游戏吗？”

结果他的 Google Pixel 已经被 Gemini 接管搜索功能，搜索框没给网页链接，而是直接生成了一套方案：基于 HTML 的小游戏结构、项目目录建议，以及第一批示例代码。

那一刻，KiddFlash42 突然意识到：自己也许不需要成为程序员，只要会“问问题”，就能把这个游戏做出来。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

5 岁孩子当“产品经理”，AI 当程序员

于是，这个游戏项目就这样启动了。

据 KiddFlash42 介绍，他和儿子的整个开发流程极具“后现代感”：

● 儿子负责想点子：怪物、技能、光环、关卡规则等

● 然后用手机语音转文字，直接对 AI 下指令

● AI 写代码、改逻辑、调玩法

● 而 KiddFlash42 只负责复制粘贴、保存文件、报错就丢回去问 AI

他们几乎没有“设计文档”，一切纯靠孩子的想象力驱动。比如，他儿子会要求：“我要一个技能卡，怪物身上要有光环，还要像《我的世界》一样！”——AI 真的能照着实现。

在最开始的一两天里，Gemini 表现得相当不错，甚至还会用特别温柔的语气夸孩子的想法，这让项目更像一场亲子游戏，而不是开发工作。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

1200 行代码把 Gemini“挤爆”，付费 39 美元换来“生产力暴涨”

可当代码文件膨胀到 1200 行左右时，Gemini 开始频繁报错，称整个文件是“巨石系统（Monolithic）”：结构混乱、模块不可维护，Bug 像雨后春笋一样冒出来，甚至还发生了几次直接崩溃，导致未保存的进度全部丢失。

KiddFlash42 再次向 AI 求助，得到的建议是：迁移到 GitHub Copilot。于是，他开通了一个月 GitHub Pro，价格是 39 美元（约合人民币 272 元）。

这个决定，成了整个项目的转折点——生产力瞬间暴涨。在 Copilot 的帮助下，他儿子几乎成了“产品经理”，亲自指挥设计整套游戏的逻辑，功能需求也极其具体：

● 要求所有元素都是《我的世界》风格

● 技能卡要有光环特效

● 怪物需要有特殊触发机制

● UI 要更炫酷

Copilot 不但能听懂这些需求，还能保持项目稳定构建，开发效率直接起飞。

不过，让 KiddFlash42 最头疼的，并不是写代码或逻辑，而是一些对程序员来说再普通不过的事情，比如：文件到底该怎么组织？GitHub 仓库怎么建、怎么提交？PowerShell 到底该在哪打开？

甚至有一次，他们想把“技能卡上的光环效果居中”，结果AI 连续烧掉了 10 次高级请求都没解决问题。后来还是KiddFlash42 自己瞎琢磨，向AI提出：“要不要先在后台合并这些元素，再整体渲染？”——这一句话，居然歪打正着解决了问题。

“看来有些事儿，AI 暂时还真办不到，不过也可能是我给的指令本身就有问题，谁知道呢。”

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q3SAo8dibFicWvibnB5u4gdHLgb2AbA6UJ8VImKnyW92hibqZefwpDbPYAQ/640?wx_fmt=png)

谁是最强模型？

据统计，KiddFlash42 他们在一周内用光了 1500 次高级请求，总账面消耗约 108 美元，但实际只支付了 39 美元的订阅费。

与此同时，他还对各 AI 模型的表现进行了排名：

（1）Claude Opus 4.5：绝对的 MVP

它消耗了 1316 条指令，占了总数的绝大部分。也是唯一一个能读懂他儿子那些天马行空的想法，并且在实现功能时不会搞崩整个项目的模型。

（2）Gemini：性格最友好的选手

和儿子聊天的时候，它的语气特别亲切。但胜在“情商”，输在“技术稳定性”——频繁崩溃、丢失进度，最终他们选择弃用。不过加分项是，KiddFlash42的手机自带了其语音交互功能，用起来很方便。

（3）GPT-5.1：不太懂孩子的“直男”

它不太擅长在对话里加一些童趣的互动，而且面对小孩写的那些稚嫩指令，经常会 “卡壳”，理解起来很费劲。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0jqAkXB2hEohZkOqYzXVsmJtnNRQncYh54ZQOpcC5ZGSicYFgtBZP4w/640?wx_fmt=png)

最终成果：“我得把他从学习游戏里拽走”

一周后，这款《我的世界》风格的教育游戏真正跑了起来：阅读、拼写、算数被包装成一张张技能卡和怪物对战机制。而现在，KiddFlash42 的日常已经完全反转——他不再逼孩子学习，而是要拉着他别再玩了，去吃饭、去睡觉。

在分享的最后，KiddFlash42 感慨，虽然他还是不懂编程，但他已不再是“旁观者”。
