# Fable 5要回来了？「怪人」Dario Amodei被撤下谈判桌

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜Panda

  

距离 [Claude Fable 5 被美国政府一纸出口管制指令强行下线](https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651038995&idx=1&sn=6f0b514691740f05ed4f02172bdcd2a5&scene=21#wechat_redirect)，已经过去了将近两周。

  

这两周里，行业里有一句流传颇广的玩笑话：现在用 AI 最高效的方式，就是什么都不做，坐在沙滩上等 Fable 回来。

  

但这两天，等待似乎有了一点实质性进展。

  

Claude Code 悄悄改了几个字符串

  

6 月 24 日，有人在 Claude Code v2.1.190 的二进制文件里发现了一些不寻常的字符串变更。新增的提示文案是「您本周的 Fable 5 使用额度已用完，继续使用 Fable 5 将消耗使用积分」），而原本的「需单独购买，不包含在当前套餐内」字样被悄悄移除。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGXVo7bQbSZE11oSwJ0bftrDj9dEL5EGBFtDEsmjJGeh2BZtq6TzMJYjCPBoETaOia2pPTby9s8q47CzY7Mfoia4oI1bJHGQGKPo/640?wx_fmt=png&from=appmsg)

  

这个发现似乎最早由 X 用户 @synthwavedd 公开，其推文浏览量已经突破百万，表明大家都非常关心这件事。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHwhA1ibePEyIibOaO3rrMP79kGMAia9DZRENrhPgq4HzJ0Zre8M1lAct7LXdseia9ejmgjmPfzT5otGz2d08ZaYDpkyWD8seuYyQo/640?wx_fmt=png&from=appmsg)

  

科技媒体 Decrypt 随后用 npm pack 下载了 Claude Code 的 macOS Apple Silicon 包，解压后对内置的 claude 可执行文件运行 strings -a 命令，确认了这些字符串确实存在于生产二进制文件中，却并不在 GitHub 公开的更新日志里。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHylpSNPARxkKVG3KsMognPPibgdLz2e6dLWekibIZCIX7Rbc5Q7XalOqVqQcEpG8A1NN3UWAh0G6VPmafFHxj1LJ7PYfaRmIib6I/640?wx_fmt=png&from=appmsg)

  

这类字符串变更本身说明不了太多，因为产品团队提前几个月写好功能文案是常规操作，代码变化不代表明天就会上线。但它至少传递了一个明确信号：Fable 5 不会像最初设想的那样，作为订阅计划之外的单独付费项目存在，而是会被重新整合进 Pro、Max、Team 等订阅套餐的常规额度内，按周重置。

  

这与 Fable 5 最初的商业化路径（两周免费体验后转为按使用积分单独计费）形成了明显反差。

  

更有意思的是，已经有用户注意到，Amazon Bedrock 的模型目录页面上，Fable 5 的条目仍然挂在那里，标注为「由 Anthropic 出售」。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEqqC3gVG5FWkjUasOkXzGclkiaXYDiaj4DMv52IWFylhFEODCKKAo5vjLSfkOaA6rEzD3MoJeomrxuaykwFZ14VdA0icyrVk2b3g/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFYgamZ19IicAtyqdibsCcQMKuBGncRjSniaXmEFm9aPT2G75hVYNbQ8zmkibJIdSsAiblXE5kjwia1DcpO856GYSSM6gaJ3jNnRhuSE/640?wx_fmt=png&from=appmsg)

  

结合 Claude Code 里的字符串变化，不少观察者认为，Fable 5 可能真要回归了。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGE9S1UrZdS1WgKzCCdw1oxLBLoGZ8461xosxMR7Ud4mXBj2Bpe2B0DpBwHqkyLU2zTpcxHnC6kslKzwybiaOc1Ixy7KSvIPvPw/640?wx_fmt=png&from=appmsg)

  

事实上，到了今天下午 3 点多时，已经有少量用户在 𝕏 上表示 Fable 5 已经可用了。但我们目前还没看到更可靠信源的确切消息。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFiagggTjh25674jhC82qRFjKfVZs29jNgwSDxoszy8lNxxkRDDRLjR5nUJIibmjQT9OeIYFt2EiayK87xZ6QXEV6ZH49fYlicaRvM/640?wx_fmt=png&from=appmsg)

  

「不要再让 Amodei 去谈了」

  

伴随 Fable 5 可能回归的消息，白宫那边却传来了一个颇有戏剧性的人员替换。

  

据《连线》报道，美国政府近来与 Anthropic 的沟通明显顺畅了不少，原因很简单：他们不必再跟 CEO Dario Amodei 打交道了。在所有关于 Fable 5 重新上线的高层会议中，Amodei 已经被他的联合创始人 Tom Brown 取代。

  

一位直接参与相关通话的人士说：「Tom Brown 不会像 Dario 那样表现得像个『怪人（weirdo）』，他真的能正常沟通。」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFCdRZpfV6MDIXR5dvG1qjyWftEMp6yeZ5ozMlR7c8ccfGhXvWreHicpEaaRvzR3UQmZSW3pbOfURWLf7ln7csAJTsrjzdUsVh0/640?wx_fmt=png&from=appmsg)

  

Brown 的正式职务是 Anthropic 联合创始人兼首席计算官（Chief Compute Officer），自 6 月 15 日起，他与公司公共政策负责人 Sarah Heck 一起常驻华盛顿，与商务部长卢特尼克、国家网络总监 Sean Cairncross 等官员展开工作组级别的谈判。

  

Anthropic 红队负责人 Logan Graham 与资深安全研究员 Nicholas Carlini 也亲赴华盛顿，带去了具体的技术团队，而不只是公关层面的姿态。

  

转折点似乎可以追溯到 Brown 接手谈判之后。6 月 19 日，特朗普在接受 Axios 采访时改口表示，他不再认为 Anthropic 或 Amodei 本人是国家安全威胁；而就在一周前，他的措辞还要严厉得多。

  

「现在不是了，但一周前可能算。」特朗普这样说。

  

结语

  

当然，目前所有公开信息都还停留在「信号层面」。

  

Claude Code 的字符串变化、Bedrock 目录中的保留条目、以及华盛顿谈判团队结构调整，这些信息彼此独立，但方向高度一致：Fable 5 并未真正消失，而是在重新定义其商业与政策位置。

  

如果把这些碎片拼起来，一个更现实的可能性是：Fable 5 不会「恢复原状」，而是以一种更深度嵌入订阅体系的方式重新上线，同时还在监管框架下调整访问与计费结构。
