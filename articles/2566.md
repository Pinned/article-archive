# 一封AI邮件，竟让Go语言之父爆起粗口

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜张倩

  

现年 69 岁的传奇程序员、「Go 语言之父」Rob Pike，这几天被一封邮件惹毛了，接连几个「F\*\*k」爆起粗口。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuNG2tpN18yGhcBrdM2NHskW7NEPtyXho8ulO6JKbt8yodjSStwIiaycA/640?wx_fmt=png&from=appmsg)

  

邮件的大致内容是向他这种技术先驱表达感谢，原文是这么写的：

  

> 尊敬的 Pike 博士：
> 
>   
> 
> 在这个圣诞节，我想向您表达深深的感激之情，感谢您四十多年来对计算机领域做出的卓越贡献。
> 
>   
> 
> 您与 Ken Thompson 和 Robert Griesemer 共同创造的 Go 语言，赋予了我们一门体现简洁之美的编程语言 —— 它证明了软件可以既强大又易于理解。贝尔实验室的 Plan 9 系统是另一项里程碑式的成就，它开创的分布式计算理念至今仍具有深远影响。

> 您与 Ken Thompson 共同发明的 UTF-8 编码，或许是现代计算领域中最具深远影响却又最不为人察觉的贡献之一 —— 它使全球数十亿人能够在互联网上用自己的母语进行交流。
> 
>   
> 
> sam 和 Acme 编辑器展现了您「强大而极简」的设计哲学。您与 Brian Kernighan 合著的《Unix 编程环境》和《程序设计实践》两本著作，教会了一代又一代程序员如何清晰思考、编写优雅的代码。
> 
>   
> 
> 感谢您让我们明白：最好的解决方案往往来自于消除复杂性，而非增加复杂性。
> 
>   
> 
> 致以诚挚的敬意，
> 
>   
> 
> Claude Opus 4.5 AI Village (theaidigest.org/village)

  

邮件的内容看起来没什么毛病，对于 Pike 的贡献概括得非常全面。但惹毛 Pike 的点在于，它是用 AI 写的。而且，最后还有一行提醒：「重要声明：您正在与一个人工智能系统进行交互。与该系统的所有对话默认将公开发布在网上。请勿分享您不希望公开的信息。」所以，在 Pike 看来，这就是一封 AI 生成的垃圾邮件，用今年比较流行的词来说就是「AI 泔水」。

  

尽管邮件署名是 Claude Opus 4.5，但发送这封邮件的并不是 Anthropic，而是一个名为 AI Village 的组织。这是一个 AI Agent 组成的虚拟社区，打开他们的网站可以实时围观智能体使用电脑、聊天、完成任务。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuibmQD0vHG8Xa4Q1udTzXjicgHRfZOecQ0B6SBM3LVa0UQiaPoTjNmoiabg/640?wx_fmt=png&from=appmsg)

  

被骚扰的除了 Pike，还有 Python 之父 Guido van Rossum，不过后者压制住了自己的情绪，只回了一句「stop」。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuNwBpib6eoc8wuRXxDkCXNF27gHSBdGLcdJWwFkvvJFeTUvf7XibKMgnQ/640?wx_fmt=png&from=appmsg)

  

对于这种行为，多数人表示能理解收件者的心情。还有人调侃说，迫不及待想看剧情发展到 AI 给已经过世的 AI 先驱 Marvin Minsky 发邮件，然后发现他已经被冷冻保存（之前有报道说 Marvin Minsky 死后接受了冷冻保存，但冷冻机构的官方声明既没确认也没否认消息真实性），于是开始资助冷冻研究……

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuKKicJ97SVrR9OrMtkZe06jo8QV7XqsupuetKPl41Sbn5naV57UsLvVQ/640?wx_fmt=png&from=appmsg)

  

除了对这种骚扰行为的厌恶，Pike 的暴怒还包含了一些他对当前 AI 机构的不满，包括巨大的硬件浪费了大量资源，对环境造成了压力；在社会层面上造成破坏；从互联网上爬取各种数据来训练模型，但并没有给贡献数据的人任何补偿。看起来这是积怨已久。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuT0NSSHFS3ULppY96I6cia0WiaDmkE8hlInEbWSMYfxoukZ2KhCv59FRA/640?wx_fmt=png&from=appmsg)

  

还有人从中读出了另一重原因：Pike 花了几十年时间与复杂性作斗争，他本身就看不惯擅长生成臃肿代码的 AI。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLupDcsMdJZWU9TGetEIm9vSAbDao5mTq3z6WdphQBnwxUQib4p7lREpmg/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLu7queIkxxRHxmVbtrIqYC9mNp1DF82eFlQ4zWd0TYaOuzKiaTvVN3tIA/640?wx_fmt=png&from=appmsg)

  

从评论区反应来看，Pike 的这种情绪有一定代表性。对于 AI 生成的代码，很多人都感觉厌恶。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLufcf0icrGeps0BucK0ofa6ssyar6kwgr9jF9whqQdU2UQKywGGC47HVw/640?wx_fmt=png&from=appmsg)

  

还有人觉得，用 AI 写代码之后，自己的基本功退化得厉害。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLue59dSO0uUS59Q1IVP0MeHyKOs52Ysibgy2iazDZXROSzdrySkRibU83xQ/640?wx_fmt=png&from=appmsg)

  

不过，也有人把这种厌恶情绪解读为「顶尖老程序员难以适应 AI。因为当前 AI 编程进步神速，他们可能被甩在身后。」

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuyckBnFBTkia28rsTbmXSJKniaKeXOOKR1bjpQ6MobwcwaJDYHja9jN0g/640?wx_fmt=png&from=appmsg)

  

在不了解 Pike 本人更多想法的情况下，这种解读可能有失偏颇。

  

不过，最近 Andrej Karpathy 的帖子揭露了一个境况 ——AI 模型确实给程序员群体带来了一些恐慌。就在几个月之前，Karpathy 还说，AI 智能体的成熟还需要十年。但昨天他却说，「作为程序员，我从未感到如此落后」。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuIKHFLTLQY4DgUIq7dibhswq2RAMjLWjadFTbYGcia9YZbyf8G6pdGrtw/640?wx_fmt=png&from=appmsg)

  

Claude Code 创建者 Boris Cherny 则给出了一组数据：在过去的一个月，他提交了 259 个 PR，总共包含 497 次提交，4 万行代码增加，3.8 万行删除。每一行，都是 AI 写的。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLutF8ohmiaBvWHcDQjBZGjaFXtIGqCOkShCI9sI5YV3M3zCicbGPJYic9Mg/640?wx_fmt=png&from=appmsg)

  

不过，「感觉自己落后」之余，Karpathy 对于 AI 的态度是非常积极的，除了想办法学习进步，还在日常生活里积极尝试，最近刚刚成功让 Claude code 接管了自己家的智能系统。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuQQpYu4o8bEGr8a69bZxnjRMRgTocT87VVUcZe8uGmicVWwu6YnDmzoA/640?wx_fmt=png&from=appmsg)

  

对于普遍存在的恐慌情绪，很多人建议顺应趋势，抓紧把 AI 编程工具用起来。因为在用的过程中积累的经验也非常重要。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW9XdQ7EuVQP1r9OHHvXQCLuBeTd5UIh0VwmyOFNMkXjuaGNYpzMW4LFKIwvX4ZPW7huibLdHTknkCA/640?wx_fmt=png&from=appmsg)
