# 曝贾扬清离开英伟达！公司去年48亿卖身

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqFXyu7rJaOFClUQL9UGvopgW1xyY6pGvWSYneL5xSZmgmLrnicnRs73GB5u2Fy9M5HcG5IZLhJ0cDTjZ6vnBAfZ42U63PX36dE/640?from=appmsg)

**离职原因可能是项目失利、开源项目夭折。**

**编译 |**  程茜  

**编辑 |**  云鹏  

智东西6月29日消息，今日，半导体研究机构SemiAnalysis爆料，AI大牛、阿里云前副总裁、LeptonAI创始人兼CEO**贾扬清已离开英伟达**。

SemiAnalysis猜测，贾扬清离开的原因可能是其联合打造的AI超级计算云服务**DGX Lepton失败了**，未达到英伟达创始人、CEO黄仁勋预期的成功。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuruXic2hjtJxezmOUt7PUDeib27iaP5mGWg86bnustttuWx1Wt2R6T64W43f1cWH1qV2n5MEjwUp2IQC2QxOyq4KcibBao5hMs9G2A/640?from=appmsg)

▲SemiAnalysis爆料帖子（图源：X）

去年3月，英伟达收购了Lepton AI，当时对外仅披露交易金额为数亿美元，SemiAnalysis最新爆料称，这笔收购的实际价格高达**7亿美元（约合人民币48亿元）**。

Lepton成立于2023年，是被称为“Caffe之父”的AI领域大牛贾扬清在离开阿里之后创办的，其主要业务是出租英伟达GPU服务器，开发软件帮助创企在云中构建和管理自己的应用

外媒报道称，英伟达最初计划在2026年前开源Lepton的核心软件，但目前其还未开源。外界猜测贾扬清离职的原因还有可能是黄仁勋最终改变主意，**否决了Lepton项目的开源计划**。

此外，在企业收购项目中，股权分期兑现是通行惯例。因此，贾扬清入职英伟达仅一年左右就离职，要么意味着他放弃了巨额股权收益，要么当初谈判敲定股权解锁方案时，允许他在一年内拿到全部股权。

去年3月，外媒The Information认为，英伟达收购Lepton的原因可能是，试图进军云和企业软件市场，与AWS和谷歌等主要云服务商竞争。当时AWS、谷歌云等作为英伟达的最大客户，正试图以低价开发和出租替代芯片来削弱英伟达。

  

**_01_****_._**

**被收购不到俩月**

**就整合推出云平台**

去年5月，英伟达就宣布基于Lepton推出AI平台**NVIDIA DGX Cloud Lepton**，将AI开发人员与“来自全球云提供商网络的数万个GPU”连接起来，英伟达称之为“一个具有计算市场的AI平台”。

英伟达官方博客提到，当企业业务需要扩容时，依托DGX Cloud Lepton，开发者可以在全球多家GPU云服务商网络中完成定制开发与一键部署。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurGmy3HP05l3p2XiadtG5nVbt2UmB54lPFTtUpiajqR8A5KaQ95C4GSTZ8reNQsz7ia8BmHw4GL439Z9icgTLmuOsFsvMbf3ccdNds/640?from=appmsg)

▲DGX Cloud Lepton架构图（图源：英伟达）

去年收购Lepton时，英伟达的云和软件业务处于起步阶段，Lepton的主要业务就是将由英伟达GPU提供支持的服务器直接出租给企业、提供软件帮助公司开发AI模型和应用程序、提供管理训练AI的图形处理单元集群。

英伟达去年曾告诉投资人，从长远来看，英伟达将从云服务和软件业务中获得1500亿美元的收入 ，比英伟达、AWS目前的年收入还要多。

去年年初，英伟达披露其软件、服务和支持收入的年化收入为15亿美元（折合人民币约109亿元），这意味着每月收入为1.25亿美元（折合人民币约9亿元），但该公司此前预计，这一数字能攀升至20亿美元（折合人民币约145亿元）。

过去几年，云服务厂商已经购买了英伟达大约一半的AI服务器芯片。但黄仁勋认为，随着时间的推移，他预计其他类型企业购买芯片的销售额将远大于云服务厂商的销售额。

因此，他建议汽车制造商等公司有朝一日可以直接购买AI芯片，而不是从云提供商那里租用。

  

**_02_****_._**

**主要角色是“GPU经销商”**

**被收购时员工仅20人**

Lepton的命名来自于一种基本的物质粒子轻子，主要业务是出租英伟达GPU服务器，该公司自己不管理数据中心或服务器，而是从云供应商处租用后再转租给其他客户。被收购时，其员工人数大概在20名左右。

Lepton被称为GPU经销商，其直接竞争对手是创企Together AI等，这些公司会开发软件以帮助其他创企在云中构建和管理自己的AI应用。

这家创企的联合创始人是贾扬清、白俊杰、李响。

贾扬清毕业于清华大学，而后取得加州大学伯克利分校计算机科学博士学位。在读博期间，贾扬清开发了深度学习框架Caffe（Convolutional Architecture for Fast Feature Embedding），成为全球最受欢迎的开源深度学习框架之一。凭借出色成果，贾扬清毕业后进入谷歌大脑实习，参与了TensorFlow框架的开发。

2019年到2023年，贾扬青曾在阿里云担任智能计算平台事业部总裁，主要负责大数据计算和人工智能平台。2023年3月，贾扬清宣布离开阿里云，随后创立Lepton。

Lepton的另两位创始人中，白俊杰曾是阿里云前AI平台总监、MetaAI研究院，李响是阿里云前工程总监，曾在Meta和Uber任职。

  

**_03_****_._**

**结语：英伟达频频出手加码云服务**

**大佬离职或意味布局生变？**

英伟达的芯片此前因价格昂贵使得很多企业难以用较低的成本开发AI应用。因此从去年以来，英伟达一直在收购小型AI和云初创公司以降低开发人员使用其芯片运行AI模型的成本。

英伟达不断加码云和软件领域的布局，收购包括美国AI编程自动化初创公司Solver、加拿大初创公司CentML等在内的一众创企。但此次贾扬清被曝离开英伟达，或意味着英伟达与一众被收购公司之间的整合进程出现变数。
