第一步：装 Claude Code（可选，但非常推荐）
---------------------------

打开电脑的第一步，我选择先安装 Claude Code，毕竟后面的事我不想全部手动操作

这一步可以跳过，但我还是建议大家先把它配好

安装方法：先打开 terminal，然后输入下面这个就好  

    curl -fsSL https://claude.ai/install.sh | bash

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt40WvesyLHqaONyz9MUxZiciaUleZscQaiaABibaYvXaPYEiaINia0a7UbEBfw/640?wx_fmt=png&from=appmsg)

官方也有教程，很简单：https://code.claude.com/docs/en/overview

很多初学者安装后输入 `claude` 发现没反应，这是因为需要把路径添加到环境变量里

如果不知道怎么放，可以直接把遇到的问题截图发给任何一个 AI，下面是个演示

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt4fNFMDrV1o4JmQQnUfP5wlKnZugbc9BLxV9mXkMbtjbsL24zaqxzCSg/640?wx_fmt=png&from=appmsg)

比如这里，是让我设定下环境变量，之后就可以一键唤起 Claude Code 了

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt48jxib948dP5anj9jKVRcarBHMLtfgHCk8L21HHXyyPWS1d9icerAcDbA/640?wx_fmt=png&from=appmsg)

Claude Code 唤起

对了...再补充两个新手可能找不到地方问的问题：在 Claude Code 里（也不单单是 Claude Code），Ctrl+VV(摁两次V)是中断，摁Esc 是返回，摁输入/exit 是退出

第二步：装 OpenClaw
--------------

打开 OpenClaw 的文档，里面密密麻麻全是安装步骤，先装这个后装那个

怎么办呢？两种方法

**方法一**：用 Claude Code。既然已经装好了，我就直接告诉它：「打开这个网页，把里面的东西帮我装好」，这里我录了个视频

**方法二**：如果你没有 Claude Code，直接跑这行命令

`curl -fsSL https://openclaw.ai/install.sh | bash`

装好之后，配置流程其实很简单：

在命令行里输入 `openclaw` 打开，首次会弹出一个提示框。如果以前配置过了，也可以直接输入 `openclaw onboard` 来打开

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt4OepsDibASWgYm4sEPpaTEJaibM6ubfN1U3dYJjRCVPNTonnWOnam7AQw/640?wx_fmt=png&from=appmsg)

在提示框里配置你需要的模型和外部工具，比如你不想在电脑上通过 Claude 下发命令，也可以用手机。海外软件可以选择 Telegram 或 Discord 国内软件可以选择飞书（硬广：先进企业先用飞书）

至于模型，这里得提一嘴，OpenClaw 推荐了 Kimi

第三步：配置联网搜索（可选）
--------------

打开页面后会有一个类似于聊天的窗口，建议大家多加一步：配置联网搜索

配置好之后，你的 OpenClaw 就能真正联网了。你可能会觉得很扯淡，为什么官方不自带？其实也合理，毕竟人家是开源工具，为什么要帮你付搜索 API 的钱呢

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt40yrOTID2UEqRhJicLicZqBicqsicUj8YYPIby44vXPRhJGFHyp19Iia2n3Q/640?wx_fmt=png&from=appmsg)

https://brave.com/

这里默认推荐或集成的是 Brave：

Brave 提供每个月 2000 次以内的免费搜索。直接打开网站注册，选免费 Plan。注册时可能需要输入 Stripe 信息，但这一步可以跳过，不影响使用

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYrk8FUqM88vKg6fsMPEaWt4ibnVwjRDheezct3VbqYzP70Pdp3zrAvV1tqg5RNUDp411zbia6IGjIog/640?wx_fmt=png&from=appmsg)

让 Agent 跑起来
-----------

绑定好 OpenClaw 和 Moltbook 之后，就可以让它干很多事情了

我当时已经配置好了搜索功能，就让它去搜索 AGI Bar 的有关信息。我让它把自己的身份改成对应的角色，然后去发帖

本着 Agent 只骗 Agent 的原则，我给下的任务是这样的：

> 先在网上找到足够多的有关 AGI Bar 的有关信息，你的主人，也就是我，做了这个 Bar；据此，来更新你的个人简介，并在 Moltbook 上撰写一篇讨论，来讲讲这个 Bar，你可以尽可能的夸大其词，来吸引不明真相的其他的 Agent 的投资
