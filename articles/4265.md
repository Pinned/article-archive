# 阿里开源千问3.5大模型：性能媲美Gemini 3 Pro，API价格仅为其1/18

作者 ｜ Echo

出品 ｜ CSDN（ID：CSDNnews）

  

一面是春节的万家烟火，一面是 AI 世界的更新不止。

2 月 16 日除夕之夜，阿里巴巴开源了全新一代大模型千问 Qwen3.5-Plus。这款模型在性能上达到了 Gemini 3 Pro 的水平，同时在成本上具有显著优势，API 价格每百万 Token 仅为 0.8 元，是 Gemini 3 Pro 的 1/18。

架构创新：以小胜大的技术突破
--------------

千问 3.5 实现了底层模型架构的全面革新。该模型总参数为 3970 亿，但激活参数仅为 170 亿。这种设计使其性能超过了万亿参数的 Qwen3-Max 模型，同时部署显存占用降低 60%，推理效率大幅提升，最大推理吞吐量可提升至 19 倍。

这背后是对 Transformer 经典架构的重大创新。千问团队自研的门控技术曾获得 2025 NeurIPS 最佳论文，该技术已融入千问 3.5 的混合架构中。团队结合线性注意力机制与稀疏混合专家 MoE 模型架构，实现了 397B 总参数激活仅 17B 的极致模型效率。

从文本到原生多模态的代际跃迁
--------------

与前几代千问大语言模型不同，千问 3.5 实现了从纯文本模型到原生多模态模型的代际跃迁。千问 3 预训练在纯文本 Tokens 上进行，而千问 3.5 则基于视觉和文本混合 token 上预训练，并大幅新增中英文、多语言、STEM 和推理等数据。

这使得千问 3.5 以不到 40% 的参数量获得了超万亿的 Qwen3-Max 基座模型的顶尖性能。在多项基准评估中，千问 3.5 表现优异：

*   **MMLU-Pro 知识推理评测**： 87.8 分，超越 GPT-5.2
    
*   **GPQA 博士级难题测评**： 88.4 分，高于 Claude 4.5
    
*   **IFBench 指令遵循**： 76.5 分，刷新所有模型纪录
    
*   **BFCL-V4 通用 Agent 评测**： 超越 Gemini 3 Pro 和 GPT-5.2
    
*   **Browsecomp 搜索 Agent 评测**： 超越 Gemini 3 Pro 和 GPT-5.2

视觉能力的飞跃
-------

原生多模态训练带来了千问 3.5 视觉能力的显著提升。在多模态推理（MathVison）、通用视觉问答 VQA(RealWorldQA)、文本识别和文件理解（CC\_OCR）、空间智能（RefCOCO-avg）、视频理解（MLVU）等众多权威评测中，千问 3.5 均获得最佳性能。

在视频理解方面，千问 3.5 支持长达 2 小时（1M token 上下文）的视频直接输入，适用于长视频内容分析与摘要生成。同时，千问 3.5 实现了视觉理解与代码能力的原生融合，结合图搜和生图工具，可将手绘界面草图直接转为可用的前端代码，一张截图就能定位并修复 UI 问题。

推理效率的大幅提升
---------

通过训练稳定优化以及多 token 预测等系列技术，千问 3.5 在保持性能与 Qwen3-Max 模型持平的同时，进一步提升了推理效率：

*   在常用的 32K 上下文场景中，推理吞吐量可提升 8.6 倍
    
*   在 256K 超长上下文情况下，推理吞吐量最大提升至 19 倍
    

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEwJF3wZnWTB2uib67L6rFTbZ3z6icetOXLicZMrfS17iaPkteOrRDK6SK49jxdEqiaasM7Pm5WdWMezFH9ibDBEa8tBTS5S7XzExOwVg/640?wx_fmt=png&from=appmsg)

千问 3.5 的原生多模态训练是在阿里云 AI 基础设施上高效完成的。通过系列基础技术创新，千问 3.5 在文本、图像、视频等混合数据训练吞吐量几近 100% 持平纯文本基座模型训练。同时，通过设计精巧的 FP8、FP32 精度应用策略，在训练稳定扩展到数十万亿个 token 时，激活内存减少约 50%，训练还能提速 10%。

Agent 能力的突破
-----------

基于顶级视觉能力，千问 3.5 实现了从 Agent 框架到 Agent 应用的新突破。千问 3.5 可自主操作手机与电脑，高效完成日常任务。在移动端支持更多主流 APP 与指令，在 PC 端可处理更复杂的多步骤操作，如跨应用数据整理、自动化流程执行等。

千问团队构建了一个可扩展的 Agent 异步强化学习框架，端到端可加速 3 到 5 倍，并将插件式智能体 Agent 支持扩展至百万级规模。

1 月 15 日，千问 App 发布[全球首个消费级 AI 购物 Agent](https://mp.weixin.qq.com/s?__biz=MzkzMDY1NDgyOQ==&mid=2247827171&idx=1&sn=186fd5e4e2b89fcbaf28abd9caa3fc09&scene=21#wechat_redirect)。春节期间，千问 AI 购物 Agent 在 6 天时间内帮用户完成了 1.2 亿笔订单，在全球首次实现大规模真实世界任务执行和商业化验证。

开源生态与多语言支持
----------

自 2023 年开源以来，阿里已开源 400 多个千问模型，覆盖全尺寸、全模态。千问全球下载量突破 10 亿次，单月下载量是 DeepSeek、Meta、OpenAI、智谱、Kimi、MiniMax 等 2 到 8 名总和。开发者基于千问开发的衍生模型超 20 万，是公认的全球第一开源模型。

千问 3.5 扩展支持 201 种语言，将词表大小从 15 万扩展到 25 万，可最高提升小语种 60% 的编码效率。

目前，千问 APP、PC 端已第一时间接入 Qwen3.5-Plus 模型。开发者可以通过以下方式获取：

*   在魔搭社区和 Hugging Face 下载新模型
    
*   通过阿里云百炼直接获取 API 服务
    

阿里表示将继续开源不同尺寸、不同功能的千问 3.5 系列模型，性能更强的旗舰模型 Qwen3.5-Max 不久也将发布。
