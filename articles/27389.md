# 告别抽卡焦虑，OiiOii2.0升级开启AI创作“自动挡”模式

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/4KW1YfFgWQliaFTsXf2QXNj2qQrHMhRuM62s91TiaCjIHJemNqNSJ9XcHLJ1FwyuKC2b7MzF5xHEE6WHp1um78iaF6ISViacv1g7uefiaylecR9k/640?wx_fmt=jpeg&from=appmsg)

2026 年，看到的一个显著信号是，AI视频行业正在迎来全面爆发。

视频制作行业的痛点，也悄然发生转移：

大模型的能力越来越强，但模型强，并不代表每个人都能做出可被消费的优质内容。

怎么更好的控制AI产出，让Agent工具找到了突破口。

前几天6月10日，专业视频创作Agent OiiOii 官宣升级 2.0 版本，上线了「**智能画布、拉片复刻、Skill 库」**三大核心新能力。

直击 AI 视频创作门槛高、学习成本高、抽卡率高等行业痛点。

新版本覆盖了动画、游戏宣发、短漫剧等核心场景，让AI创作的过程更简单，让每个人都能轻松享受AI创作乐趣。

从2025年底内测引爆10万人排队，到现在 2.0全面升级，OiiOii正逐渐把模型能力，真正兑现为普通人的创作自由。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/4KW1YfFgWQlCUicoOw2bVy8mxuNSSHKv8cCNYUm2O8P6gc2UtOQLYpnL2u5ZYE0JEkThaaPBwiaRkRVuy98RBW7RZDyGJyw4vjpbQUwG4bNBI/640?wx_fmt=png&from=appmsg)

官网：https://www.oiioii.tv/

**01**

**智能画布：**

**不用写复杂提示词，和AI聊聊天就能做视频**

在网上搜索“AI教程”时，大概率会出现大段不同类型的提示词模板。

一个镜头需要撰写包含场景、人物、镜头运动等多维度的详细描述，学AI的第一步得先学会写复杂提示词。

同时，市面上很多自由画布类产品，本质上是把用户直接丢给大模型，如果不会写提示词、不会找参考图，出来的东西根本不可控。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQlUehXmO2yibvOsgxG7ccRenFZ4b1LgyiaoJAbKZ1CTiabiaXwib8pQAhxsTBcofgDksI5LUCIpXyFceuNXhxAIj5g6YAzhTxkygu10/640?wx_fmt=png&from=appmsg)

而OiiOii智能画布的解法很直接：你只要对着画布说话。

“把这个角色改得可爱一点”，“分镜1的节奏太慢了”，“帮我换个风格试试”。

不用解释上下文，不用琢磨提示词，AI自己理解画布里已经有什么，然后精准地改。

![](https://mmbiz.qpic.cn/mmbiz_gif/4KW1YfFgWQnthgkvT0lmADckVn5Vc5IRIBaDZevThricDv8icSHc1DI7bwHY4Ad5WgibQmlnOlrNozyBg2u6yglia81Wp0zZAFeHNe9tlBib6tCQ/640?wx_fmt=gif&from=appmsg)

我试了一下，针对每一个素材，选中以后，都能调用Agent，随时说出修改想法，二次编辑更方便了。

做了一个我自己头像的动态视频海报，最终的效果也很符合期待。

背后的逻辑是，系统自动调出对应的7个Agent——角色总监、分镜总监、音乐总监等，让AI去写提示词、去控效果。

用户只说要什么，不用管怎么做到。

这是OiiOii一直在做的事，只不过2.0把它从固定工作流里解放了出来，放进了画布里，变成了可以随时介入、自由调用的能力。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQld3fTUV4z05ia0k0mG8U63vNG1tPrmS1vQicnoGtd1bqLoYQUdPxMiaAIYJUmjoiahXVB9oC8INKbtuYzS9h6VAxiajeqLuKX0qaj8/640?wx_fmt=png&from=appmsg)

还有一个让专业创作者真正买账的细节：多个任务可以同时跑。

一边改角色图，一边改场景图，一边生视频，不用一个等一个结束。

同时OiiOii还接入了满血版不排队的Seedance2.0，解决了传统AI视频生产过程中的“熬夜排队”困境。

**02**

**拉片复刻 + Skill 库：**

**告别低效抽卡，做爆款更简单**

如果说智能画布解决了“做得顺”的问题，那么拉片复刻和 Skill 库，解决的就是怎么让人“上手快、出片稳” 。

如果想学习怎么做爆款，可以调用OiiOii拉片复刻功能。

上传任何一段想学的视频，系统自动从音乐音效、剪辑节奏、分镜功能、镜头叙事等最多18个专业维度拆解视频，提炼创作逻辑和提示词。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQlsSE5wGicnDvqxUtWSZeOHdC8EIdQXUyTJoiavQ4K0ZG62hMnT941b1rUicic10obSvF6g4TGM1aMVrcjcLYq4k32Wn0wiaJ1FKiaRU/640?wx_fmt=png&from=appmsg)

我上传了一段复杂打斗的科幻动漫视频，直接一键点击拉片复刻即可。

![](https://mmbiz.qpic.cn/mmbiz_gif/4KW1YfFgWQnl3RgVzyDfJficFhnEdZXfkyCyG5MV8A8Eia8FHPnMZW5qZQbLUuW5iaKGG8mXjhGicH5xH9ZJOz0hcOIuuJIfCfTCD0VQKtb3Eqg/640?wx_fmt=gif&from=appmsg)

一个很酷炫的动漫视频，叙事要素、镜头语言、影像处理、声音效果……很快就拆解了出来，非常专业且清晰。

本质上，是把电影学院导演课的拉片训练，变成了一键可得的能力。

再继续上传想替换的主体（角色/道具等），就能一键复刻同款视频。  

可以替换的元素，非常细致，不止是角色、场景、道具，能想到的都可以复刻。  

只一键，用户就可以精细化地复刻定制自己的视频。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQmNRFN6iasNVGgNKoqTsHj6O5zrYAgrMcG60u4kugC5PNuyTPaS5mEAInmLL5hibuJib3R2OwlJ2rJXIzCmqPwSZ8QHtYWJB2zTCs/640?wx_fmt=png&from=appmsg)

如果想快速做出一条适配行业需求的视频素材，那可以调用OiiOii的Skill库。

Skill的概念火了小半年，但OiiOii这回是真的把Skill放进了具体的使用场景里，让Skill 变成了真正的生产力工具：

自媒体故事短片怎么做，游戏投放素材怎么批量出，电商展示视频怎么快速生成。

每个Skill背后是一套预设的工作流和风格模板，选好场景直接出片。

当AI创作不用再靠反复 “抽卡” 碰运气，而是依托成熟的创作逻辑与模板，才能更好实现高效出片、稳定产出。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQmrt8SKMiad9zhtEmkOpdTwyu3ic41icmeoIhQGFicsjjdHxafiar786IzsjqOiayCEmicntMwrrPPkdtDISmJmD2yEicJIPBkpg5X3v9g/640?wx_fmt=png&from=appmsg)

在这个过程里，尽量减少“手搓”，让Agent帮你做。
