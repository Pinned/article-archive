# 李飞飞World Labs双模齐发，能「造」超复杂大场景，一手实测

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜Youli、杨文

  

李飞飞的 World Labs 又更新模型了。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFjZRoIa3GeyvULXqGhgv2GpceLfP3TbZgfwY7zJESicFcsLNEUynOaHibWUVng3pwsdD4VG8F7ZRv7w5DDSsOrQ09EqkL1T1GhA/640?wx_fmt=png&from=appmsg)

  

这次一口气推出了两款模型，分别是 Marble 1.1 和 Marble 1.1-Plus。

  

其中，Marble 1.1 在照明效果、对比度和整体视觉一致性上有了显著提升，大幅减少了视觉瑕疵；Marble 1.1-Plus 则是专为大规模场景打造的全新模型，能够生成比以往更大、更复杂的世界环境。

  

  

李飞飞在社交媒体评论称，逐步改进 Marble，就生成更大的世界而言，这可不是夸张。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqESOZ9x565QZkclAGYa8xJicE07YfEU2OXia1nsG50rEALzAH8yabeFm1peMic6Kia9BZc7DgFzESKCKKZdgK4PZQEGqOialicwLjlics/640?wx_fmt=png&from=appmsg)

  

目前，两款模型已经开放体验，官方建议初次使用时优先选择 Marble 1.1，其生成质量更高、更稳定；需要创建大型复杂场景时，再切换到 Marble 1.1-Plus。开发者也可以通过 API 立即使用这两个模型。

  

体验链接：marble.worldlabs.ai

  

接下来，我们就奉上一手实测。

  

一手实测

  

网站首页以各类生成效果展示为主，按风格化、现实主义、幻想、科幻等 6 大类别分类展示。左边侧功能栏中，点击画笔图标即可进入创作页面。

  

系统提供两种输入模式。一是 2D 输入，只需添加图片、视频或者全景图就能生成沉浸式 3D 世界；一是 3D 输入，使用内置的 Chisel 工具，根据我们自定义的 3D 布局来生成世界。支持导入外部 3D 文件，也可从系统内置的几何体开始构建。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqE8tRYibibJq16Q8Zlgo4LFWeU9qkibjLsKiaYEKicreS6Gtf2xNfMsQUQChGvEY00xBIk1ACusxHVrxgkAC8au7ZgGl2kI07ico7X5Y/640?wx_fmt=png&from=appmsg)

  

我们先来试下 Marble 1.1。

  

上传图片，输入提示词：A whimsical fairytale scene featuring glowing pink mushroom houses with golden windows, surrounded by lush greenery and twinkling lights in an enchanted forest.（发着粉色光芒的蘑菇小屋，窗户闪烁着金色光芒，周围环绕着茂密的绿植和闪烁的灯光，坐落在一片魔法森林中。）

  

  

再比如 Marble 1.1 生成的另一个世界，在一间霓虹灯照亮的复古房间，俯瞰着日落时的富士山。

  

  

此外，模型还支持全景编辑功能。在已生成的世界中选择编辑区域，输入新提示词，即可精准修改内容。比如我们在上述蘑菇小屋场景中添加了一只正在啃胡萝卜的小兔子。

  

  

它的 Expand 功能，可以通过创建无缝扩展，将世界扩展到其原始边界之外。不过该功能目前需要付费使用。

  

我们又上传了一张《荒野大镖客》图片，测试下 Marble 1.1-Plus。模型成功生成了规模更大、细节更丰富的复杂环境。

  

  

模型一经发布，广大网友也坐不住了，纷纷开始上手体验，下面我们就来看一些有意思的案例。

  

网友 Hugues Bruyère 是一位创意工作者，用 Marble 1.1 对自己的工作室进行重构。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFLGM3Jbx5kTicBGC9PqzyeZ1rWpTCz2oX6iaY6y4QXraObUH5sjcjB45rfaq24aFInCe2r6lNaiaBktK0qdfjbAjHsU8Yoy1EIjg/640?wx_fmt=png&from=appmsg)

  

他表示，整个过程非常简单，首先通过 Portal Cam 对真实空间进行一次 360 度捕捉，随后将这张全景图与一段简单的文本提示一起输入 Marble1.1。模型以此为基础，生成一个「平行版本」的空间，更复杂的设备布局、更丰富的光照层次，甚至延展出一个完全不同的外部环境。

  

  

而从这个案例中也可以看出，Marble1.1 是对「同一空间」的多种可能状态进行建模与重写。

  

甚至，Hugues Bruyère 的创意还引来了官方，World Labs 直接评论：「世界传送门，这太酷了！」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFoJUMhguibB3vkqWyUoLII2icYu9y6coMxsNtltb8nNA3JayqG0hxBeXIy12D4z3cRibp1SBsr5390FEiaD4NoAyP3Sib4xk9icZicC0/640?wx_fmt=png&from=appmsg)

  

而另一个网友使用 Marble 1.1 Plus 生成了多个长满绿色苔藓的古堡，效果相当逼真。

  

  

最让他感到惊喜的是，「Marble 1.1 Plus 可以看清建筑内部拐角处的情况。」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEvicvGsBqSic9LFia7dEM0q24g5JeGbt9VAyKz8CDANZ16ngU3MR5dK3WojKUNafcuomq7NlV5wUqEeOCtvYcxKfZ1m6sRLuNfOI/640?wx_fmt=png&from=appmsg)

  

Marble 1.1 被定位为新的默认模型

  

其实，从去年 11 月正式发布生成式世界模型以来，World Labs 短短几个月内接连完成了多项关键进展，优化了导出功能、推出了 API、完成了十亿美元级别的融资等。

  

而在此次更新中，从实测和网友分享的案例中也可以看出来，Marble 1.1 和 Marble 1.1 Plus，分别从不同维度提升了平台生成 3D 世界的能力，这是在生成质量和生成规模上的能力补齐。

  

从官网来看，Marble 1.1 被定位为新的默认模型。官方说法是，「如果你刚开始使用 Marble，那我们推荐从 Marble 1.1 入门。」而之前的版本则被标记为 Marble 1.0。

  

目前在价格方面，Marble 1.1 的生成价格为 1500 credits，而为持续探索而保留的旧版模型 Marble 1.0 生成价格为 1500 credits。

  

也就是说，这一升级，在固定成本和使用流程前提下，可以实现更高质量的世界生成，对用户来说就是一次「无感升级」。

  

Marble 1.1 Plus 作为用于构建最大规模世界的最新、最先进模型，其生成价格为 1500 credits 基础费用外加 0–1500 credits 的「可变世界」生成费用。

  

这是 Marble 首次引入了可变成本机制，以更好适配不同规模的需求。

  

另外，还有一个 Marble 1.0 Draft，是当前速度最快的模型，适合快速探索想法与测试提示词，生成价格为 150 credits。
