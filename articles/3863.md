# 刚刚，吴泳铭放话：打造最强模型是阿里AI第一优先级任务

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcuqI8nh7aXrepicqutIpqs4NVwZsrAfRLFQlCd8qcMN0TBBukbZNF0hZtXy7ibUp31Jqs7KlcxIpMawzHNrajIr79dD6QBjYUouco/640?from=appmsg)

**阿里云收入破千亿，AI连续10季三位数增长。**

**作者 |**  王涵  

**编辑 |**  云鹏  

智东西3月19日报道，刚刚，阿里巴巴集团发布2026财年Q3（即2025年Q4）财报。财报显示，截至2025年12月31日止三个月，阿里总收入为**2848.43亿元**，同比增长**2%**；净利润为**156.31亿元**，同比下降**66%**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcupVm1hgVEyFnyf0PRiaibjyxv1YjUxTlyyXgzMTjNDc4hL4XjNhvMRvcrf6RtzxbpGCuuT5Ep2PfqcsMWxd0lbtuXGzNqgTjU2KE/640?from=appmsg)

其中，云智能集团收入为**432.84亿元**，总收入及不计阿里巴巴并表业务的收入同比增长分别加速至**36%及35%**，保持住了上一季度的超30%的涨幅趋势。财报电话会上透露，阿里云2026财年截至2月底的外部商业化收入正式突破**1000亿元**。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acuqb1Y9PpHTPUKWkyyOTLIOQYGCoXYB4O6VgAGRG6DeolplZL71u9J6IGntdpuiagv7cqn8GrobZ4COa3ibF0N9gBLm1OtrE2b7WI/640?from=appmsg)

阿里巴巴集团首席执行官吴泳铭称：“AI是我们的主要增长引擎之一。阿里云收入强劲增长36%，其中**AI相关产品收入连续第十个季度实现三位数增长**。”

其还披露了集团AI战略的商业目标，即未来五年，包含MaaS在内的云和AI商业化年收入突破**1000亿美元（约合人民币6873.05亿元）**。

截至3月19日财报发布前收盘时，阿里巴巴集团股价**下跌4.14%**，报**132港元/股（115.82元人民币/股）**，总市值**25211亿港元（约合22191亿元人民币）**。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcupPicNjZE1BfdpcDVsuQASLAKDpMZEh93hDPBjYcg59cVroPAIdf0eClkFRYFz323UUcdoRyH2VtjtaticKqpffN6vmTIicv34nb0/640?from=appmsg)

  

**_01_****_._**

**阿里云六年蝉联第一**

**平头哥自研GPU大规模量产**

阿里云提供面向AI场景的云计算产品与服务，包括高性能网络、分布式存储、云操作系统，以及AI模型训练和推理服务。

在市场份额上，IDC发布的《中国金融云市场（2025上半年）跟踪》报告显示，2019年上半年以来，阿里云**连续六年**位居中国金融云整体市场**首位**，在金融公共云基础设施市场的份额为**43%**。

在IDC在2025年11月发布的《中国混合云市场份额，2024》报告中，阿里云同样位列中国混合云PaaS及服务市场**第一**。

2025年9月云栖大会上，阿里云发布Alibaba Cloud Linux操作系统，该系统面向云端AI负载优化，可支持万亿参数模型训练，并提升训练与推理效率。截至2025年12月31日，阿里云在全球**29个**地域运营了**92个**可用区。

平头哥半导体是阿里巴巴旗下的芯片设计子公司，目前已实现**自研GPU大规模量产**，产品可支持模型训练、微调和推理等全流程AI任务，并兼容主流AI框架。

该芯片与通义千问大模型、云计算结合对外提供AI服务，业务规模持续扩大，对阿里云算力基础设施已经形成**支撑作用**。

应全球存储价格上涨的趋势，阿里巴巴云业务部门周三称，将上调平头哥AI计算芯片真武810E的价格，涨幅在**5%至34%**之间。据《华尔街日报》报道，阿里云同时将存储服务价格上调**30%**。

  

**_02_****_._**

**吴泳铭定调“模型第一优先级”**

**千问月活超3亿**

财报电话会上吴泳铭透露，打造最强的模型是阿里AI第一优先级的业务。

截至2026年1月21日，千问模型在Hugging Face上的累计下载量**突破10亿次**。2026年2月，Qwen3.5正式发布，该模型在推理、编程、智能体任务和多模态理解等方面表现优秀，同时实现了更高的推理效率。

凭借Qwen3.5基础模型，阿里C端旗舰AI应用——千问App成为阿里AI战略的一个重要里程碑：**将自研模型的能力转化为可规模化落地的真实世界应用。**

2026年1月15日，阿里宣布对千问App进行重大升级，实现与淘宝天猫、淘宝闪购、高德、飞猪、支付宝等阿里生态业务的深度打通。

自2月6日启动春节推广以来，千问App用户活跃度显著提升。截至2月底，**近1.4亿用户**通过千问App的智能体功能首次体验AI购物，涵盖餐饮、日用品、票务服务及旅行预订等多个领域。2月千问C端应用月活跃用户数**超过3亿**。

3月16日，阿里正式成立Alibaba Token Hub（ATH）事业群，由阿里巴巴CEO吴泳铭直接负责。

吴泳铭在电话会问答环节提到，如今是Agent驱动的时代，与早期AI时代的区别就是，现在模型与应用之间的配合程度需要更加紧密。这句话其实就是阿里正式成立ATH事业群的原因。

ATH事业部旗下包括通义实验室、MaaS业务线、千问事业部、悟空事业部及AI创新事业部等五大事业部，覆盖从基础模型研发、模型服务平台，到个人与企业端AI应用的完整布局。

3月17日，在AI钉钉2.0发布会上，悟空事业部的首个产品、全球首个AI原生企业级Agent平台“悟空”正好似链条箱。悟空事业部定位打造B端AI原生工作平台，将模型能力深度融入企业工作流。

“AI智能体进入规模化办事时代。”吴泳铭如是说。

  

**_03_****_._**

**结语：阿里跑通“大模型+云+芯片”飞轮**

回顾过往趋势，阿里云本季度36%的营收增速，维持住了上一季度的高速增长的趋势。

从财报来看，“通（千问模型）+云（阿里云）+哥（平头哥芯片）”的全栈组合已经给阿里带来了显著的正向效益，不仅抵消了传统电商业务调整带来的利润波动，更成为了集团新的核心增长极。

吴泳铭在财报会上强调：“未来，依托‘大模型+云+芯片’全栈AI能力，以及与阿里商业生态的全面融合，我们（阿里）将继续在AI to B和to C方向齐发力。”
