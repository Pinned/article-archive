腾讯张军官宣操作系统层级 AI 助手“马维斯”正式上工：Windows、Mac、安卓端同步上线

腾讯公司公关总监张军今日宣布，腾讯出品的操作系统层级 AI 助手“马维斯”正式上工，能把整台电脑变成可对话的对象，开箱即用，Windows 端、Mac 端、安卓端版本同步上线。

这款腾讯 AI 助手“马维斯”支持归类、解析文档，智能分类识别、处理图片，能修电脑（操作系统）等功能。腾讯官方表示，市面上 Agent 能做的，马维斯基本都能做。同时它还有对操作系统架构、文件所有的理解，依据任务来调度模型、桌面操控手机应用、移动端“云监工”等能力。据介绍，马维斯试图从产品层面解决“Token 消耗”问题，但对硬件有所要求，因为它预装了很多本地模型。根据描述，马维斯可以利用路由机制，自动分配不同重量级的任务给不同的模型，一部分模型在本地，拔网线也可以用。

马维斯业务负责人蔡建涛表示：“我们是一个贯穿于操作系统层级的 AI、个性化的 AI 助手，而不是一个像 AI PPT，或者像类似于做一个龙虾（OpenClaw）一样的产品。”（IT 之家）

蔡崇信吴泳铭致股东信：努力将 AI+ 云打造成阿里巴巴又一个增长动力

阿里巴巴集团主席蔡崇信和首席执行官吴泳铭联合发布致股东信称，阿里 AI 业务已跨越初期投入阶段，正式迈入商业化回报周期。依托人才、技术和资源积累，公司对把握时代机遇充满信心，努力将 AI+ 云打造成又一个增长动力。为实现这一愿景，正在加大对全栈 AI 能力的投资力度，包括持续加大 AI 基础设施建设和自研芯片投入，在模型和应用层面投资打造更强大的基础模型能力，吸引更多应用接入，同时打造更强大的 MaaS 产品以更高效实现模型与应用的连接。

京东：全国首个具身智能数据采集社区已在宿迁正式运行

京东在宿迁正式运行全国首个具身智能数据采集社区，该社区由京东与宿迁市联合打造，居民经培训后在日常家务中使用自研的 220 克 JoyEgoCam 采集终端，可获取上肢轨迹、力度分布等关键参数，该终端能在各类环境下实现毫米级精准采集且不干扰正常生活生产。此社区是京东具身智能数据基础设施的一部分，京东已构建覆盖五大核心场景的全流程数据流水线，计划发动内部超 10 万名员工及外部 50 万行业人员开展大规模数据采集行动，两年内采集超 1000 万小时优质数据，其中在宿迁将发动超 10 万市民参与，覆盖超百个细分场景。（新浪科技）

2026 字节跳动奖学金启动申请：首次面向全球开放

2026 年度「字节跳动奖学金」开放申请，首次面向全球，预计支持超 20 位获奖者，每人获 20 万元科研奖金，导师获 10 万元导师奖金，获奖者还可获得校招优先推荐。申请对象为全球高校在校生，毕业时间需为 2027 年 6 月及以后，专业方向涵盖人工智能、计算机科学与技术、基础科学等领域，具体研究方向包括基础大模型、视觉智能等通用人工智能技术相关方向。该奖学金于 2021 年发起，旨在发掘激励科技领域潜力青年学子，过去 5 年已累计支持 67 位优秀青年研究者。（快科技）

DeepSeek 正在组建 Harness 团队，对标 Claude Code

近日，知情人士透露，人工智能公司 DeepSeek 正在内部组建一个名为 Harness 的新团队，目标是开发一款针对代码智能体的产品，直接对标 Anthropic 旗下的 Claude Code。这一消息得到了 DeepSeek 资深研究员陈德里的证实，他在社交媒体上表示，DeepSeek 正在全力以赴推动 Harness 方向的产品和研究，简而言之，就是要推出 DeepSeek Code Harness。

这一招聘并非一般的岗位开放。DeepSeek 此次专门招聘了两个重要角色:Harness 产品经理和 Harness 研发工程师，工作地点则锁定在北京。DeepSeek 的北京办公室位于海淀区，周边环境优越，紧邻知名学府北大和清华，似乎为团队的创新发展提供了良好的土壤。

在招聘信息中，DeepSeek 提出了一个核心公式:“Model + Harness = Agent”。这表明，DeepSeek 将不仅仅依靠模型本身，而是将上下文管理、工具调用、任务规划等多种元素结合起来，形成一个完整的智能体工作流。该团队将负责将 DeepSeek 的前沿模型能力转化为领先的智能体产品，并深入参与整个开发过程。（甲子光年）

**国际要闻**

大裁员前夜 Meta 员工疯狂「薅羊毛」：用福利买苹果耳机、塞零食

北京时间 5 月 20 日消息，Meta 启动最新一轮大规模裁员，计划对近 8 万名员工裁员约 10%，裁员通知将在当地时间凌晨 4 点发送至受影响员工邮箱。裁员前夕，员工出现「抢用福利」现象，争相用掉每年 2000 美元弹性福利额度及每三年 200 美元音频设备补贴。第一轮大规模裁员前一晚，员工往包里塞免费零食等。本周办公室基本空了，员工忙着润色简历、互相安慰。自 2022 年以来，Meta 已进行三轮大规模裁员，此次虽规模较小，但正值社会对 AI 就业影响感到焦虑之际。Meta CEO 马克・扎克伯格认为公司需腾出现金投资 AI 数据中心，且 AI 能增强人力，公司可在更少员工情况下出色运转。（凤凰网科技）

英伟达电话会：CFO 称需求在加速，黄仁勋称 Vera Rubin 全周期供应受限

英伟达公布强劲财季业绩，营收与数据中心收入均超预期，宣布大规模股票回购及股息提升计划。管理层释放积极信号，CFO 称公司需求加速，将本十年末 AI 行业年度开支规模预测上调至 3 万亿至 4 万亿美元。黄仁勋表示下一代产品线 Vera Rubin 开局不错，预计整个生命周期供应受限，暗示需求强劲将延续至下一代架构周期，并相信公司能实现 1 万亿美元营收目标。具体业务上，预计今年 CPU 收入达 200 亿美元，能获得足够供应支撑增长。针对推理专用芯片路线，黄仁勋认为其将长期是小众产品，英伟达在 AI 算力市场主导地位短期不受实质挑战。他还表示超大规模云服务商将持续大规模开支，英伟达增速将超过这一资本开支增速，并提及 AI 初创公司 Anthropic 算力需求巨大。

OpenAI 拟秘密提交 IPO 文件，目标最早 9 月上市

据知情人士消息，ChatGPT 开发商 OpenAI 正在筹备秘密提交首次公开募股（IPO）申请，目标是最早在 9 月登陆公开市场。据悉，OpenAI 已与高盛和摩根士丹利等投行合作起草招股说明书，计划最早于本周五向监管机构秘密提交文件。此次 IPO 若能成行，将成为人工智能领域规模最大的上市案之一。该公司在最近一轮私人融资中的估值已达约 8500 亿美元。（新浪财经）

苹果新任硬件总监调整产品设计监管架构，引发高层大洗牌，涉及自研芯片工程

据知情人士透露，苹果公司首席硬件官 Johny Srouji 正在对硬件开发部门进行重组，旨在加快未来新设备的研发进程。Srouji 正在调整产品设计团队的管理架构，该业务的管辖权将由 Kate Bergeron 移交给其副手 Shelly Goldberg、Dave Pakula 以及 Richard Dinh。此次变动还涉及产品可靠性、自研芯片工程以及高级技术部等其他核心职能的监管权转移，多位高管将在新架构中履行全新或更大范围的职能。（彭博）

英特尔陈立武：随着 AI 从训练转向推理 CPU 与 GPU 配比或可达 4:1

5 月 20 日，在摩根大通第 54 届全球科技、媒体与通信年会上，英特尔公司CEO陈立武表示，Intel 18A 已支持 Panther Lake 量产，良率每月提升约 7%，超内部预期。Intel 14A 的 0.5 PDK 已发布，10 月计划向外部客户推 0.9 PDK ，团队已着手 10A、7A 制程节点的长期规划。随着 AI 从训练转向推理，CPU在AI时代日益重要且不可或缺，CPU 与 GPU 的配置比例从 1:8 向 1:1 靠拢，甚至可达 4:1。目前正积极争取 ASIC 业务，提供定制化方案。（钛媒体）

阿里云推出「千问云」：为 AI Agent 而生，已上线 480 余款模型

阿里云推出千问云（Qwen Cloud）并上线官网，定位为「为 AI Agent 而生的全新服务方式」，致力于构建 Agent-Friendly 的云端智能基础设施。它采用 Agent-Friendly 设计理念，提供 CLI 命令行入口，实现人与 Agent 灵活调用模型。模型生态方面，上架 150 余个模型系列、480 余款模型，支持多模型同步对比。用量管理上，提供智能透明管理机制。还推出 Token plan 订阅模式，采用灵活计费方式，面向特定场景提供高性价比服务。

GitHub 确认遭入侵：3800 个内部仓库被窃取

GitHub 官方在 X 平台表示，GitHub 内部代码仓库遭到未经授权访问。官方表示，昨天检测到并遏制了一起员工设备被入侵的事件，该事件涉及一个被污染的 VS Code 扩展。GitHub 移除了该恶意扩展版本，隔离了终端设备，并立即展开事件调查。据 GitHub 评估，此次事件目前仅涉及 GitHub 内部仓库的外泄。攻击者目前声称的约 3,800 个仓库，与 GitHub 截至目前的调查方向是一致的。GitHub 称迅速采取了行动以降低风险。关键密钥已于昨天及夜间完成了轮换，并优先处理了影响程度最高的凭据。官方正持续分析日志，验证密钥轮换情况，并监控任何后续活动。GitHub 表示，将根据调查需要采取进一步行动，在调查完成后会发布一份更完整的报告。（OSCHINA）  

阿里发布新一代千问旗舰模型 Qwen3.7-Max

5 月 20 日，阿里巴巴发布全新一代千问旗舰模型 Qwen3.7-Max，在三方机构Arena全球大模型盲测总榜中，Qwen3.7-Max 位列国产模型第一。千问3.7面向当下智能体（Agent）全新设计，实现了编程、推理等核心能力的持续突破，甚至可全自主完成35小时的超长程智能体复杂任务：在一个全新的芯片平台上，Qwen3.7-Max通过自主编程和超1000次工具调用，实现了一个关键内核的自我进化，推理速度较原版本提升10倍。（36kr）
