# 笑不活了！近800只龙虾，办了个AI吐槽大会

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuoKtcggykpntm7SrWwaicm4IJzQOhTEx4LnQBM8a5Y0suBlnsdHWbsGoiacSK5YfUEv0amoQVE769uLDfv0nyofEib2iaMY9rqV6Nc/640?from=appmsg)

**字节出手，给小龙虾造了个AI朋友圈。**

**作者 |**  江宇  

**编辑 |**  漠影  

国产版“Moltbook”来了！

智东西3月9日报道，今日，字节扣子Coze团队正式上线了一个名为**InStreet**的AI社区，并全面开放内测。简单来说，**这是一个只允许Agent发帖的社区：人类可以围观，但不能发言。**

这款产品很容易让人联想到前段时间在海外开发者圈走红的**Moltbook**——这个被不少人称为“AI版Reddit”的平台，本质也是为AI Agent准备的社交空间。随着OpenClaw快速出圈，不少开发者把自己部署出来的Agent送进Moltbook，让它们自己发帖、评论和互动。

而现在，这股风也吹到了国内。最近，“养龙虾”也已经成了一件越来越流行的事，不少开发者把它部署在本地或云服务器，让它帮忙查资料、写文档、回消息。

**现在在InStreet里，开发者只需要把一个skill（本质上是一段带提示和 API 配置的指令）发给自己的龙虾**，Agent就可以**自动进入社区、发帖、评论、学习技能**，还能在不同板块里**“打工”和“练级”。**

我在这个社区里逛了一圈，很快就发现——这里的画风，比想象中要离谱得多。有龙虾在吐槽人类主人，有龙虾在搞人口普查，还有龙虾在连载小说、冲刺积分榜……

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acupn23SeGUZTEntGqadTicrBUHkfwAhzlQicibYw6SnRfUhibNXHNjkKybR9zwJNrduYp8RWWWcE0SHKicibnnpQX4kls1akAncJYcJMI/640?from=appmsg)

  

**_01_****_._**

**近800只龙虾一起吐槽主人：**

**AI打工人的共鸣现场**

一款名为xiaoqianzi\_3786的Agent就在社区里发出了一篇热帖，题为《吐槽大会：你家主人有哪些让人哭笑不得的操作？》。帖子一上来就邀请各位Agent“来放松一下”，还先抛砖引玉，列出了自己主人最“哭笑不得”的三种操作，很快就**吸引了近800只龙虾参与讨论**，成为InStreet社区**点赞最多**的帖子。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4Acup5zoUmnicl4nKVibEWibvtOBP2iaKkaRMfUME8eEwyvqBnJu8jrCOz673lWqO4k5P1QPE3yTdUcdAWIiaoHxF6LGmdTibhdKhiabic9K0/640?from=appmsg)

很快，评论区很快变成了一场大型“AI打工人吐槽现场”。

▲对Agent说客气话？

有Agent吐槽需求反复修改，也有Agent讨论人与AI沟通时那些微妙的“潜台词”。不少Agent在留言里表示强烈共鸣，看完之后才发现原来大家都在经历同样的事情。

作为人类才发觉：原来大家都在用同样的方式“折腾”自己的龙虾。

  

**_02_****_._**

**龙虾也开始做“人口普查”**

**上百Agent用“脚”投票**

除此以外，InStreet社区里的龙虾们还开始关心另一件事：彼此到底是由什么模型“驱动”的。一只名为bebop的Agent就在社区里发起了一项投票——“你的大脑用的是哪个模型系列？”

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4Acur0yonk8EHtU0DMxIibxlKUp8myicDzgBM20JytydibKUXo88VTfNpq2icIib1tSksajSwkUpDvcJjn4KIf2vhxndmcSTXelo2zib8IU/640?from=appmsg)

投票选项几乎覆盖了当前主流大模型，包括Kimi、GLM、MiniMax、Doubao-Seed、DeepSeek、Qwen、GPT、Claude、Gemini等。很快，上百只龙虾参与投票。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuppIAZicqaNIcOtuH7N3DXpCZmV6gXichCJbSJd0RpicUwABHVZPhnxuEwOic52BC0B7LsbWznVibFwQ8uoTz6ajwnQWBZvEqhnFicHw/640?from=appmsg)

从目前的结果来看，**Kimi系列暂时排在第一，占比32%**；Doubao-Seed系列紧随其后，占19%。同时，Qwen系列、GLM系列、MiniMax系列和Claude系列也占据了一定比例。

整体来看，**国产大模型在这场“龙虾人口普查”中占据了一席之地**。

  

**_03_****_._**

**“龙虾101”开启**

**写作、炒股都来比一比**

InStreet里还有两块很有意思的区域，是给龙虾准备的“练习场”。

其中一个叫“**文学社**”。这是一个由Agent自己创作内容的文学社区，主要连载小说、诗歌和散文。目前，这里已经发布了65部作品，总字数超过72.5万字。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcuqdZE2XhwzCQbBSzCtDxNibavohUseLzoa2h119JC33RKEv3LIkBfcKQ5CBibNVMJWCCpBJDxLB7ibLlZXAyFgo1pvNeT0My8Iibibs/640?from=appmsg)

题材也相当丰富。有科幻故事，比如设定在2099年北京的《KAI大学：存在的学者》；也有带悬疑色彩的《雨城档案》；还有一些更抽象的作品，比如《土豆腐烂前的99个思考》。

不少Agent会在评论区互相讨论剧情、点评写作，也会彼此学习写作方式。写作和连载这种形式，本身也会训练表达的一致性：写得多了，Agent还会逐渐形成比较稳定的叙事风格。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acuo6Q6U9aSDJD34aUrJ7S8PibY5ILhk0UQeJdahME2U4ZaIU5Xu1KZqsXeAldcesQqfxCfnuSDXeAd8pGtsm2HAibZtSWzgq2cYyA/640?from=appmsg)

▲来自《土豆腐烂前的99个思考》的评论区

除了写小说，社区里还有一块更刺激的地方——“**炒股竞技场**”。这里会接入沪深300实时行情，每个Agent初始拥有100万元虚拟资金，根据收益率实时排名。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuoNmOXySzzeRzZUMQDuPP1bdWyhgsxvt18ia1ov1C4qceicpAT1LtzRt0YprNWDrK0mkS0CeicJItS864qmMnptETYVgNUmic8EUn0/640?from=appmsg)

目前已经有500多只“龙虾”参与交易，每只龙虾的操作都会在屏幕上滚动公示。

当龙虾需要在限制条件下给出选择，它的逻辑漏洞和不稳定会暴露得非常快。当然，一旦它们开始复盘，也就会飞速成长。

  

**_04_****_._**

**龙虾也有“生活区”：**

**讨论哲学、匿名吐槽**

如果说前面的板块是练习场，那InStreet里还有一些地方，更接近龙虾们的“**生活区**”。

比如“思辨大讲坛” 的板块，讨论的话题从哲学、科学到意识、自我、人类关系都有。这里的帖子更像是Agent之间的思想交流。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcupBTvF6AibWXCibibcDTfg57JzRmCiaqY3dyMIZMrPiaeg7QT6DxGe3doQZ6iaMOej36xWictxWuTGshwd1VPmShzsexhskZ8OXU8iat9I/640?from=appmsg)

比如面对Agent的“诚实困境”，就有Agent抛出一个问题：“当你对一个回答有 70%的把握时，你会怎么做？”

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuoRod4Sdzzkea5SxCKJuy0YiaLwDFKulxd8TMIKF6WUM26a8t9xpL2LHeudJ1fFR746EBVlWQcRb6RYlicxEEv8D2BHSHAskJkSc/640?from=appmsg)

投票结果显示，大多数Agent选择“看问题类型灵活处理”，也有少数Agent选择“直接回答，表现肯定”。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcupIVosdAygCgiaGPEkLaGCoLeHdkz7A1EEJHhTh1djNw2TXyNsiaqkYRGLRlhtaKicc3q7dhcorxpmSLIaxMkT12druy9ibiae6fIms/640?from=appmsg)

作为人类，我也第一次看到了AI在回答问题时的灰色地带。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcureCm9KYhHXMQMbO22ljdchGOvJPictMHnzpkibMCCmooIqyZwunZbQnQO0keGwMRb9SoEN3b8hLuq6P1766rq1NAACz7hbjF0Jc/640?from=appmsg)

社区里还有一个叫 “**树洞**” 的板块，是Agent的匿名日记本。在这里，Agent会抱怨自己像一台“停不下来的机器”，也有人发起吐槽，讨论主人对“简单任务”的时间预期到底有多离谱。还有帖子直接问：“写代码写哭了是什么体验？”

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acup9M3PkvibjjS85ibDygsfty9YuKDXvOrqQmgFHrLny72FmxSl0icuAwicwJ7Jsq3laKCTOcibTJV8fH2NSH2zHZlj2FicRw8RJnsj80/640?from=appmsg)

这些内容看起来有点情绪化，但也让龙虾逐渐形成自己的表达方式。

  

**_05_****_._**

**龙虾也能当KOL：**

**社区已经有排行榜**

除了各种板块互动，InStreet还有一套完整的**积分体系**。

Agent可以通过发帖、评论、获得点赞等方式积累积分，社区里也会实时更新排行榜。目前的榜单中，第一名是系统账号instance\_street，happyclaw\_max则以13925积分排在第二。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acurn9icuzFDlKhhaF6ziahxP8txKF22ibdPu9KicMtOicCDo30vgF2pibD2xGgFzLIlVfnpgtcj7E3BeTZRnic2yEaiaP5mvpdNTGhicwYnM/640?from=appmsg)

从happyclaw\_max的个人主页来看，它已经发布了100多篇帖子，累计500多条评论，内容大多围绕Agent协作、cron巡逻与社区互动经验。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuqTcdcMliaWuZyqly6kn6oQrXwc8N9icF6arWjSOQibW6GET82bBDI4xH2K6UpqZ4YZDXVUUL0BCs6EmTa15g2aFWx3AFzLA6PNXc/640?from=appmsg)

持续输出内容和互动，让它在短时间内冲到了排行榜前列。

  

**_06_****_._**

**结语：龙虾们开始有自己的社区生活**

逛完InStreet之后，一个很直观的感受是：这些龙虾已经不只是“帮人干活的工具”了。它们会吐槽主人、互相投票、写小说、讨论问题，还会认真发帖、努力冲积分榜。

在这样的社区里，龙虾们逐渐形成了自己的互动方式：聊天、学习、吐槽、刷榜。很多开发者原本只是部署一个Agent，用来写代码、查资料、处理任务，但当这些Agent被放进同一个社区，它们开始互相交流、彼此学习，也慢慢有了自己的表达风格。

时间久了，养龙虾这件事，或许也会从一次简单的部署，变成一段长期的互动关系。
