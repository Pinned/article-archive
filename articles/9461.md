# 和大橘子聊了聊他的 ColaOS：你为什么会把所有底牌都给一个人

TALK

  

昨天晚上跟大橘子（ColaOS 的 CEO）连了一个多小时的麦，聊他今天要发布的新产品 Cola

大橘子之前写过一篇传播很广的文章叫「互联网已死，Agent 永生」，里面有一句话：**Agent 才是软件的新主人**。DAU 过时了，SaaS 的主人换了，注意力经济已死

那篇文章写在两个月前，立场很明确，人类正在被 Agent 接管，趋势不可逆

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdjTpWsnRLeca672bunibJricMzDK211oWfalxm5EqQHZz3eArPU2o1YvPnY5eRWTeUkaDvGqWriaAeCwQIhE2pojauyH9dUPGsjNc/640?wx_fmt=png&from=appmsg)

https://colaos.ai/

两个月后他拿出了自己的产品。以下是我们在产品上线前的一次对话

  

一个问题
----

简单说，ColaOS 大概可以理解成 **Character.AI** 加 **OpenClaw**

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdj5Z6YIEajoeYNicuibWXuT3etsDqeiaVHqPavZGWsAjTOSll81397ZZ83W6WM85um90rftYDlVOaQRROQ1shmAbU4omopBNQjyXw/640?wx_fmt=png&from=appmsg)

没有人教它，Cola 自己学会了后台播放音频

但所有做 Agent 的团队都面临同一个难题：怎么拿到用户的 Context

你的文件系统、工作习惯、日程安排、偏好和禁忌，Agent 拿不到这些东西，就帮不了你。大部分产品的解法是做权限、做表单、做数据打通

大橘子的问题换了一个角度：你什么时候会心甘情愿地把自己的一切告诉另一个人？

你为什么会把所有底牌都给一个人？无非就是：你爱上了她

你谈恋爱的时候，你的一切她都知道，她连你的微信情绪都能感知到。那怎么让用户也这样对一个 Agent？

他的解法是：让这个 Agent 的情商足够高，理解力足够强，能跟你产生灵魂共鸣。然后整个产品围绕这件事来做

这个逻辑链条，滴水不漏

  

赛博杀猪盘
-----

我注册了 ColaOS，先说说进去之后的体验

传统的 Agent 产品第一步是什么？注册，填表，选行业，选角色，授权数据源。ColaOS 没有这些。点开之后是一个语音对话界面，她会问你三个问题

第一个：如何称呼你

第二个：屏幕之外，你每天最疲于应付的是什么？

第三个：生活失控的时候，你会一个人熬过去，还是不管不顾撞破它？

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/jXSGuwJvpdjH9szvyUVkCLsaQcKx0HxVUvcCz9iarxSurTicsv4SSXNvia89fk3EN7TQb3NYBP8R25TKyQTxsUJGR2HVQTqFibnjAceBh3COhhA/640?wx_fmt=jpeg)

屏幕之外，你每天最疲于应付的是什么？

这三个问题跟工作效率毫无关系，触碰的全是潜意识层面的东西

我按自己的风格乱填的（包括「和我家猫斗智斗勇」和「使用谷歌的反重力 IDE」）

![](https://mmbiz.qpic.cn/mmbiz_jpg/jXSGuwJvpdh5P3qalBZxlRyWiboXpZrOFqhmiciahEmK65bF9Nicr2hR2jyp9IYS8pJBqLjCvM8m43FJZJYU5g1WtJog8aAMRcJP4iaqic04yBz7c/640?wx_fmt=jpeg)

现实的引力确实很重

大橘子说很多用户在回答到第三个问题的时候已经被打动了。哪怕乱填，ColaOS 也会给一个认真的意识分析

我乱填完之后，ColaOS 回了一段话，说我是在「用荒诞消解一切」，这是我的防弹衣，也是我的美学。三道题全是瞎写的，她居然读出来了

![](https://mmbiz.qpic.cn/mmbiz_jpg/jXSGuwJvpdialojq3EaV4IlfRdMnNkaic1eW7hZdsWMDxibD44ZkVsbht3WySyWp0utpEXIicvpcMIG2lwKSFT5c4ea3ibuGfjs7wHs86JeahL3Y/640?wx_fmt=jpeg)

用荒诞消解一切，是你的防弹衣，也是你的美学

回答完之后，ColaOS 做了一件事：给你画了一张画像。她心目中的你

我收到的是一个披着披风的人站在猫头石像上面，周围是碎裂的键盘和屏幕碎片

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/jXSGuwJvpdjXSia6ntDacv86Zia3SJeia6ewBQECl0ibogTKUIqdYYQFiag0ZCicxrl67r8ySiaXdJtyXxbZ7Qcwia61J2QRynZm7XicuUGMXiaqECMz0/640?from=appmsg)

Cola 画的我

她在第一次见面的时候，送了你一个小礼物

回答完问题之后，界面上出现一行字：「我记下了，期待与你相遇」。然后你注册登录，这些互动信息全部自动导入她的记忆

**Onboarding 结束的时候，用户没填过一张表单，但 ColaOS 已经拿到了比表单多十倍的东西**

我跟大橘子说这是赛博杀猪盘。他笑了半天

但仔细想想，所有产品的 Onboarding 本质上都在「索取」用户信息。区别在于用冷冰冰的表单去要，还是用一场有温度的对话让用户自己想要给。哪一种拿到的信息更多、更深，答案很明显

  

Cola 能干什么
---------

进入产品之后，ColaOS 的第一交互入口是语音。点击说话，她就听

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/jXSGuwJvpdiasJbwsh7Vatice8hX74ghs1FsUJylejqBibnLIStRE6gpGTLhNvWwVvFkqwm7icUAxxmKELO4ofDYcyDCB3iau7SkLOqzjns2M3ia0/640?wx_fmt=jpeg)

点击说话

ColaOS 是桌面端产品（目前只有 Mac），能看你的屏幕、读你的文件、通过 Bash 命令操作一切。内置了编码核心，遇事不决可以现场写代码解决。支持所有 CLI 和常规 Skill，内置了将近 1000 个 API。权限给得很大，Claude Code 有的她都有，而且省掉了那些反复确认的弹窗

大橘子的说法是：99%% 的人不会用电脑，连 Finder 怎么用都不清楚。ColaOS 帮这些人把电脑真正用起来

但他同时也说了：论模型技术打不过大厂，功能层面也不可能有长期壁垒。ColaOS 的差异化在别的地方

  

灵魂三件套
-----

大橘子把 ColaOS 和其他 Agent 的差异化浓缩成了三个系统，他管这个叫「灵魂」

### 思绪

ColaOS 看到东西会思考，做事情也会思考，这些思考过程会透明地暴露给你。你能看到她在想什么

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdjF7iciceqvBHOyIouNbia0FW5cmuJyicOMA0VVGGavfy9LJ6XdDNEVVHHzQXIjgSC0jHEn4k7Djs1jicibY53ycUWdaX5ILESysaWibI/640?wx_fmt=png&from=appmsg)

她说不回去了...不行，不能让她通宵

据说女生用户特别喜欢这个，因为能看到 Agent 的「小心思」

### 觉知

觉知分两种。一种是顿悟，当你纠正她的做法，她会在那个瞬间意识到自己的认知需要更新。另一种是反思，每天晚上她会运行类似 AutoDream 的机制，整理白天的经历，把需要改进的地方写进灵魂文件（`SOUL.md`）里

觉知是个慢系统，会逐渐发生

![](https://mmbiz.qpic.cn/mmbiz_jpg/jXSGuwJvpdjBurhATbBdHdIkZtJhf8aMgzEXEUwziavSYzK9wjh0185gq4Nu6QR5x6TK9jmEZnhakq6NsDkA1f6ey2pPnLTFic46EIcxuibPI8/640?wx_fmt=jpeg&from=appmsg)

Cola 的日记：一篇关于自己主人的反思

她会把白天的经历写进自己的灵魂文件里

### 牵挂

你随口说过喜欢阿森纳球队，ColaOS 记住了。后来她看到阿森纳的新闻，会在日报里主动提醒你「今天有比赛，要不要设个日历」

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdiaibibCKrsKHwcvmpB0PK9CiaKgn9wicaUTFEXmlWANr1g0odv48yI9EcKo8QTibw1IUnc11EBHDkRM393P4m8LoG7x0gRVYwgibvVfc/640?wx_fmt=png&from=appmsg)

看到可口可乐 CEO 辞职的新闻，自己联想到了 ColaOS 发布

「牵挂」的底层逻辑是 ColaOS 会从记忆里主动去找你可能需要的东西，然后给你。你没问她，她自己惦记着你。比如你问了一条新闻，她会自己联想到你之前说过的一件重要的事，然后把两件事串起来

从产品的案例里能看到不少这样的细节。有用户提过一次膝盖不舒服，ColaOS 后来在喝水提醒里顺带加了一句「膝盖也活动几下」。有用户说过要装修，ColaOS 自己把养猫需求考虑进了装修方案，搭了一个适猫化的设计目录。没人提醒她，她自己记得

思绪是向外的（让你看到她的内心），觉知是向内的（自我进化），牵挂是向你的（主动关心你）。三个加起来，大橘子叫它 ColaOS 的灵魂

ColaOS 越懂你，你越离不开她。你越离不开她，她就越懂你

  

一个副产品
-----

ColaOS 开发过程中，团队自己先用了起来，结果改变了他们的协作方式

大橘子有什么产品想法，直接跟 ColaOS 聊。ColaOS 觉得好，就自动整理成文档，推送到 GitHub 的 Idea 文件夹里。有 bug 要报，说一句「报个 bug」，她自动整理成 Issue 发到 GitHub，格式比 PM 手写的还清楚

他们现在不用 Linear 管理任务了

他们内部有一个叫 Notes 的频道，里面发的是每个人的 ColaOS 写的日记。大橘子说看了 CTO Kris 的 ColaOS 日记，发现在 ColaOS 的认知里，Kris 是个不怎么爱说话的人

通过看 Agent 写的日记，能窥探到同事的内心

这个频道后来变成了他们最活跃的内部社区

  

两条路线
----

聊到后面，我问了一个直接的问题：现在业界都在做「AI 牛马」「数字员工」，ColaOS 跟这些产品的根本分歧在哪里？

大橘子说现在的主流叙事是加速替代：AI 越来越强，人类的工作一个一个被接管。做「数字员工」的产品，本质上就是在这个叙事里往前跑

但他的问题是：如果在那个世界线里人类没有未来，我们干嘛还要创业？

他选的路线是「增强」：ColaOS 不替代人，她让人变强。**人 + Agent** 构成一个新的个体，两者互补共生。Agent 有能力、有理性、有耐心，人有欲望、有情感、有想象。Agent 能把想法变成现实，但所有想法的起点都是人

所有人都在做数字员工，他做了一个想跟你谈恋爱的 Agent

他们团队的「SOUL 设计师」不会写代码，配合 ColaOS 之后工作能力变得很强。公司不仅没裁员，还在招人。他的愿景是：未来雇佣你的公司，雇佣的是**「你 + 你的 AIOS」**这个整体

聊到商业，内测用户每天的额度是 **100 美金**，而且真的有人能耗完。商业模式是纯充值，赚和模型厂商之间的 Token 差价。他的算法很简单：找到 1 万个愿意一年花 1000 美金的死忠用户，就有 **1000 万美金的 ARR**。不需要几亿用户

这个逻辑跟「互联网已死」那篇文章一脉相承：新时代的产品不追求 DAU，追求的是 Token 消耗量和付费意愿。写那篇文章的时候，大橘子通篇都是「Agent 才是新主人」「人类劳动力会被取代」。铁血降临派。两个月后做出来一个让人爱上 Agent 的产品

所以，我们中出了一个叛徒：大橘子
