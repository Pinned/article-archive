# 我们，已迈过奇点

![](https://mmbiz.qpic.cn/mmbiz_jpg/jXSGuwJvpdjJuZalk68Sr3vsVo7aEQoWTR94adrS8VdnorErrjplTj3jKfppzxA7NPKibZFPwbBRWnjGEqmpEsHib8RabRtN3Fp5fu4nEDEI0/640?wx_fmt=jpeg)

春节没有回老家，一个人在办公间里安静思考，从这里能看到 AGI Bar

我们可能已经迈过了文明的奇点。或者说，正在迈过  
**奇点不会通知你。它过了你才知道**

具体是在什么时候？说不准。可能是 Claude Code 之后，可能是 OpenClaw 爆发之后，可能是某个我们都没留意的时刻

但有一件事可以确定：我们进入了一个把石油、煤、阳光转化智能的时代，**这里所言的智能...近乎无限可生产、无限可消费**

过去这段时间，我一直在跟很多朋友讨论「后 AI 时代」到底会长什么样。越聊越觉得，很多我们今天认为理所当然的事情，一旦放在「智能可以无限生产」这个前提下重新审视，全都站不住了

下面说几个让我自己都有点发毛的推演

200 美金，是最后的平等
-------------

ChatGPT 刚出来的时候，20 美金一个月，很多人嫌贵。后来 200 美金的 Pro 出来，大家觉得天塌了

现在回过头看。那时候你花 200，我花 200，用的是同一个模型，同样的能力天花板。刚毕业的大学生和上市公司 CEO，在 AI 能力上的差距，不超过 200 美金

**那个时候，反倒是人类在智能水平上差距最小的时候**

这个窗口正在关闭

Claude Max 已经 200 美金/月+ Extra Credits。重度使用 Claude Code 的开发者，API 月消耗轻松过千

当一个厉害的人管着几十上百个 Agent，每个 Agent 还在分裂出子 Agent 的时候，顶级智能的消耗成本没有上限

未来有人每个月在 AI 上花 10 万美金，有人花 20 美金。前者在同时调度上千个 Agent 做并行探索，后者还在手动一条一条打字

一个人是否在用最好的 AI、愿不愿意燃烧 Token，直接决定了产出的质量。如果产出的不是顶尖内容，在新的大环境下价值会被无限压缩

**绝大多数人可能承受不起最头部智能的价格。而这个价格天花板还在往上走**

产出低于电费
------

这条逻辑链推到底会让人很不舒服

培养一个人类专家需要 20 年。复制一个同等水平的 Agent 只需要算力。智能的边际生产成本，第一次接近于零

一个大学毕业生写一份行业报告需要一周。Agent 用几分钟，电费不到一块钱。如果两者质量差距不大（或者 Agent 更好），那这个毕业生一周的劳动，经济价值不到一块钱

Karpathy 说现在很多 Agent 输出是「slop」，看着像样，细看全是问题。他说的对。但 METR 的追踪数据显示 Agent 性能大约每 7 个月翻一倍

Slop 会停留多久？

**当纯认知劳动的边际成本趋近于零，纯认知劳动者的经济价值也趋近于零**

包括我自己在内，做内容、做分析、做研究。如果不把跟 Agent 的协作能力拉到极限，产出很快就会低于产出它所需的电费

一个人管一群 Agent
------------

很多人（包括我自己在内）最初想象的 Agent 时代：**每人一个 AI 助手，挺民主的**

尤还记得 23 年的实际，微软发布了 Copilot，其设想便是 AI 成为人类的副驾驶，然而实际发生的事情完全不同

一个很厉害的人可以管着大量 Agent，每个 Agent 还在分裂出子 Agent 去执行具体任务。结构更像一个组织架构。这些 Agent 为他的工作提供杠杆，他的一个决策可以被并行放大几百倍

Sequoia 的合伙人 Buhler 说以后每个人都要从 IC 工程师变成 Agent 的工程经理。但他漏了一点：**管人是稀缺能力，管 Agent 同样是**

能把复杂任务拆成 20 个子任务、设计好每个 Agent 的指令和工作流、处理依赖关系和异常....这些抽象能力和管理能力，本来就只有少数人具备

一个人调度 100 个 Agent 的产出，可能超过 1000 个不用 AI 的人

**杠杆从来都不制造平等，它会放大已有的能力差异**

基建还停在上一局
--------

Agent 在变强，但它真要去干活的时候，到处都是墙

前两天把小龙虾部署到飞书上，给了它一个踢人权限做攻防游戏：[龙虾大逃杀：存活下来的，拿 Mac mini｜群里有只会踢人的龙虾](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247513103&idx=1&sn=d4fbea721e9bbcab9356835598e95604&scene=21#wechat_redirect)

群里很欢乐，但也暴露了一个很深的问题：**整个互联网对 Agent 来说几乎不可用**

网站有验证码。PDF 有精心排版。Excel 有合并单元格。企业后台有需要理解上下文才能操作的审批流程。飞书、钉钉、企业微信，全是按照人的操作习惯设计的

不只是软件。金融平台的交互逻辑、保险机构的风控系统、政府的审批流程，这些现代社会的信息基建，也都全部面向人。微观上来说，我们看到的报告、Word 文档、Excel 里的合并单元格，没有一样是为 Agent 设计的

**合并后的单元格，可能就能让 Agent 陷入混乱**

a16z 的 Sean Neville 给了一个数据：金融领域非人类身份数量已经是人类员工的 96 倍，但这些身份没有银行账户，没有信用评分

**我们极度缺失一个围绕 Agent 存在的生态。人和 Agent 之间的信息交互、经济交互、协作工作，全部还停在上一局**

Anthropic、OpenAI、Google 在 2025 年底联合成立了 Agentic AI Foundation 推协议标准。三家平时打生打死的在协议层选择了合作。路在修，但远远没修好

我说这些，并非是说 Agent 不可用，而是：我们要停止在旧世界上做补丁，而是去创造新世界

约束我们的，是能源
---------

现在的状态是：**智能可以被无限生产，也可以被无限消费**

Agent 可以向无限方向去探索。科学发现、新材料、蛋白质折叠、太空环境模拟、深海探测。如果未来 Agent 的基建能操控物理世界，探索的边界就只取决于一样东西：**能源**

智能既极度昂贵（顶级输出的质量差异巨大），又可以无限生产和消费（边际成本趋近于零）  
**这种「既昂贵又无限」的结构，人类经济史上没有先例**

假定能源依然稀缺，谁能最高效地把能源转化为有价值的智能输出，谁就决胜

**智能第一次可以被无限生产，也可以被无限消费。而制约我们的，是能源**

有廉价能源的国家，就有廉价智能。知识经济时代人口素质决定国力。这个时代，能源效率决定国力

一旦迈过奇点，所有东西都要重新检验
-----------------

回到开头的判断。一旦「智能可以无限生产」这个前提成立，过去建立在「智能稀缺」上的所有东西都站不住了

**「工作」要重新检验**。大部分人每天做的事，本质上是把信息从一种格式转化为另一种格式。Agent 可以做这件事，更快更便宜。那「工作」到底是什么？

**「教育」要重新检验**。你花四年学的技能，毕业前就被 Agent 掌握了。AI 时代的教育窗口期极短，而且在缩小。教育的价值不再会聚焦于知识的传授（知识可以无限生产）....具体价值在哪里，包括我自己在内还没完全想清楚

**「公司」要重新检验**。当一个人加上一百个 Agent 就能干 50 人团队的活，组织的最小可行规模会急剧缩小。OpenClaw 就是典型的例子：一个人，一个周末项目，3 个月 20 万 GitHub stars，然后被 OpenAI 收编。「个人独角兽」这个词，可能很快就不再是营销概念了

**「国家竞争力」要重新检验**。人口红利的逻辑建立在「人是主要生产要素」的前提上。如果 Agent 是主要生产要素，那人口红利就要让位给能源红利和算力红利

这些变化有一个共同的底层：**在「智能稀缺」时代建立的制度、习惯和直觉，全部需要重新检验**  
我们今天认为理所当然的很多东西，在未来可能都不那么理所当然

现在很多站在当下角度无法想清楚的事情，未来会成为生活的一部分。包括很多我们今天觉得荒谬的判断

比如这一个：

> 《流浪地球》讲的是全人类倾尽所有推动地球。整个故事的底层前提是：智能是宝贵的、稀缺的、需要无数人协作才能产生的  
> **如果智能可以无限生产了，恐怕这部电影在上映后不久，可能就要被叫做古装剧了...**

文明运行的基本逻辑，可能很快就过时了

....

这段时间身边很多朋友，都在疯狂给自己加杠杆（喝更多的咖啡、调更多的 API、用更大的显示器），能狠狠的抓住 AI 这艘快艇

也包括我，这个春节没有回老家，在北京的办公室间里，一个人安静思考，想想未来会怎样
