##### **【新智元导读】两周前还是大厂团队专属，两周后20刀的Pro用户就能原封不动地用上——Claude Code把「会话实时变网页」这项企业级能力，直接下放到个人用户。**

刚刚，Claude Code又放了个大招！

Anthropic 通过官方开发者账号官宣：Claude Code 的 Artifacts 功能，正式向 Pro 和 Max 用户开放了。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYVuMuzQoWFg4YjAarAk60uumSicTRiauMaOJqJvERPD8glDagnCqbdwkxRTOlz3qGEg7mvBO7TfpkPvibwpblaxN124aqf2eNkbBA/640?wx_fmt=png&from=appmsg)

此前，这个功能只有 Team 和企业版才能用——6 月 18 日首发时，Anthropic 明确写着「Available in beta on Team and Enterprise plans」。

也就是说，这是掏钱最多的大公司团队专属。

现在，一个每月订阅 Pro（$20/月）或 Max（$100~200/月）的用户，也能享受到一模一样的能力。

从「企业专属」到「个人可用」，只隔了两周。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**Artifacts 到底是什么？**

简单说，它能把你在终端里和 Claude Code 的对话过程，实时变成一个可交互、可分享的网页。

![](https://mmbiz.qpic.cn/mmbiz_gif/Rvq8Ow69CYUAVcCE6ibGkWTCNLyxMztSol7umZIfANjicUjZoiabn7c91OSxkJ72bpJZIxoL8oQ4EcbP1S5jf8fFtFuctHh5IyJeDWYLWhuEZU/640?wx_fmt=gif&from=appmsg)

过去，Claude Code 的输出就是终端里一行行冷冰冰的文字。

你干完活想给同事看看进度，只能截图，截一张、再截一张，凑成一条长长的截图链；或者起个localhost，让别人凑过来看你屏幕。

现在，这套麻烦事全没了。

你在会话里让 Claude 写代码，它会顺手把工作进展发布成一个托管在 claude.ai 上的网页。

这个页面完全自包含——CSS、JavaScript、图表全内联，不依赖任何外部资源。

最关键的是，它会随着 Claude 继续干活，自动实时刷新。

你把链接甩给团队，所有人打开的都是同一个页面。

Claude 每往前推进一步，页面就原地更新一次，别人不用刷新、不用你再喊一嗓子，最新进展自己就长出来了。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**最能打的场景：调试**

在 Anthropic 内部测试里，一个高频用法长这样：

一位工程师开会前，顺手让 Claude Code 启动了一次事故调查。Claude 一边分析日志，一边发布了一个 Artifact——里面有时间线、可疑提交、错误率图表。她把链接往群里一贴。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYWsAlCE4HSfhxq7cLib9W59PicLv6lrJzPPALZFnYWRBNxgSQPGw9QOfeh75ibEhXIQica3dJfUJI7TwqvoLJhu1wUUeZEHuZuMH8Y/640?wx_fmt=png&from=appmsg)

等会议真开始时，随着调查推进，Claude 已经悄悄重新发布了两次页面，把最新信息都纳进去了。

团队成员打开链接看到的，不是十分钟前的快照，而是此刻最新的调查进展——所有人看到的，是同一个视图、同一份上下文。

信息差，被抹平了。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**远不止写代码的人在用**

更妙的是，Artifacts 的用户远不止程序员。

官方按角色列了一长串使用场景，几乎覆盖了一家科技公司的每一个工种。

安全团队做完一轮代码审计，生成的页面里每个漏洞发现都直接链到代码的具体行号，点一下就跳过去，不用在终端和编辑器之间来回切。

工程经理到了周五，Claude 从已合并的 PR 里自动攒出一份按项目分好组的团队周报，连手动整理的功夫都省了。

类似的，法务能一键出依赖许可证审计页、架构师能从 import 关系生成服务拓扑图、SRE 的事故页面能随调查自动「生长」最后直接变复盘文档。

角色不同，逻辑一样：把原本散落在终端里的 AI 产出，变成团队里任何人都能打开就看懂的页面。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYWMN7YCzETNCLUz3Mf9UcByMb5kFgBwkicy8ubNPsOBxFToqIpibEzNCTxyv2lXV7s1d0dCjCI5k4XRtZIV3QrYAYRhp5URs5RXo/640?wx_fmt=png&from=appmsg)

它不是一个只给程序员用的玩具，而是把整个团队的协作流，都装进了一个会自动更新的页面里。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**个人开发者也够得着了**

而这一次，Anthropic 把这套能力的门槛，直接拉到了个人开发者也够得着的高度。

要知道，「边写边播」这种实时协作能力，过去一直是企业级用户的特权。

现在，一个每月 $20 的 Pro 用户，也能原封不动地用上。

这意味着你哪怕单打独斗，也能像一支企业团队那样，把工作进展「边写边播」给客户和合作方看——

不用部署、不用截图、不用喊人凑过来看屏幕，一条私有链接搞定一切。

你甚至可以在终端里敲代码的同时，让客户在浏览器里实时看到进度在一步步推进。

Claude Code 的输出，终于不再只是终端里那一行行冷冰冰的文字了。

截图链和 localhost，可以正式退休了。
