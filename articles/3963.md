# 刚刚，OpenAI狂揽7500亿元，英伟达、亚马逊、软银抢投

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acuo3KcJMXcjUtWdYnbpVicF7SjUKkC5VLJXwz5yo2Vnsw2bnz6V2AD5E4HWa9Kdz3WNGOS77Q1Um4qUmE0P6Ef8ROhwxLNibxyQMg/640?from=appmsg)

**OpenAI拿下全球AI领域迄今规模最大单笔融资。**

**编译 |**  程茜  

**编辑 |**  云鹏  

阿尔特曼赢麻了！

智东西2月28日消息，刚刚，OpenAI宣布拿到**1100亿美元（约合人民币7544亿元）**新融资，其中**亚马逊**投500亿美元、**英伟达**投300亿美元、**软银**投300亿美元。这也是全球AI领域迄今规模最大的单笔融资。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurFFIczULUia2VUuYSmrgNH395HKnibEq03dbqxicstiaK9xxsUJzNHf9uvqPqVEBGazLzxq9wZJDRcXL4IuF4GHnjgcDicNkRLJqQs/640?from=appmsg)

其中，**亚马逊向OpenAI投资500亿美元（约合人民币3429亿元）**，首批投资为150亿美元（约合人民币1029亿元），在满足特定条件后未来几个月再投350亿美元（约合人民币2400亿元）。2月26日，据The Information援引知情人士消息，后续亚马逊的350亿美元，或**取决于OpenAI是否达成通用人工智能里程碑或实现上市**。

OpenAI与亚马逊的合作还包括，OpenAI将通过AWS使用**2吉瓦的Trainium AI芯片**；AWS将作为OpenAI企业级平台**Frontier的独家第三方云提供商**。

英伟达投资300亿美元，OpenAI博客提到，他们正在扩大与英伟达的长期合作，包括使用**3吉瓦专属推理算力，以及在Vera Rubin系统上提供2吉瓦训练算力**。

其官方博客提到，本轮融资前，OpenAI的预估值达到**7300亿美元（约合人民币5万亿元）**，未披露融资后的新估值。

阿尔特曼在社交平台X上感谢了三个投资方，顺便还安抚了微软。

阿尔特曼称，OpenAI很高兴与亚马逊合作，将新一代产品推向市场，期待能够充分利用Trainium芯片；英伟达一直是OpenAI的最重要合作伙伴之一，其芯片是人工智能计算的基石，很高兴能在AWS上运行他们的系统，对英伟达新产品很期待；软银是优秀合作伙伴，很高兴其再次成为OpenAI重要投资方，期待展开更广泛合作。

他还提到**微软将仍与OpenAI保持合作**，OpenAI的无状态API将继续独家面向Azure提供，并且将与微软一同扩建更多算力资源。

**_01_****_._**

**亚马逊为OpenAI投钱、供应芯片**

**还成Frontier平台独家第三方云供应商**

OpenAI宣布与亚马逊建立为期多年的战略合作伙伴关系，亚马逊将向OpenAI投资500亿美元，首批投资为150亿美元，在满足特定条件后，未来几个月再投入350亿美元。

其达成的合作伙伴关系包括：

**AWS与OpenAI将共同创建由OpenAI模型驱动的有状态运行时环境（Stateful Runtime Environment）**，该环境可在亚马逊Bedrock平台上提供，供AWS客户在生产规模上构建生成式AI应用和Agent。有状态运行环境预计将在未来几个月内上线。

其博客提到，有状态开发者环境是前沿模型的下一代应用，使模型能够无缝访问计算、内存和身份等元素。有状态运行环境允许开发者保持上下文，记住之前的工作，跨软件工具和数据源协作，并访问计算。其被设计用于处理正在进行的项目和工作流程。

**AWS将成为企业级AI平台OpenAI Frontier的独家第三方云分发提供商**，该平台帮助组织构建、部署和管理AI Agent团队。

Frontier使企业能够构建、部署和管理跨越真实业务系统的AI Agent团队，拥有共享上下文、内置治理和企业级安全，而无需管理底层基础设施。

**OpenAI将通过AWS基础设施消耗2吉瓦的Trainium算力**，以支持对有状态运行环境、Frontier及其他高级工作负载的需求。

2025年11月，OpenAI曾宣布与AWS达成380亿美元的算力协议，如今其计划将这一协议在未来8年再追加1000亿美元合作规模。

最新的合作将覆盖Trainium3芯片及下一代Trainium4芯片，为各类高阶人工智能工作负载提供算力支撑。Trainium4预计于2027年开始交付，将提供更高的FP4计算性能、更宽的内存带宽、更大的高带宽内存容量。

**OpenAI和亚马逊将开发定制模型，以驱动亚马逊面向客户的应用。**

亚马逊团队将能够针对直接服务客户的AI产品和Agent定制OpenAI模型。这些功能将补充亚马逊开发者已有的模型，包括亚马逊的Nova系列，为团队提供另一种大规模构建和交付的工具。

  

**_02_****_._**

**重申与微软合作关系**

**双方合作依然稳固**

OpenAI还发布了和微软的联合声明。

其中提到的条款包括：

**双方合作依然稳固且处于核心地位**：微软与OpenAI将在多年深度协作与共同成果的基础上，继续在研究、工程及产品开发领域紧密合作。

**双方的知识产权合作关系保持不变**：微软继续持有对OpenAI各类模型与产品知识产权的独家许可及使用权。OpenAI与亚马逊这类合作，早在双方协议框架内已有所规划，微软也期待看到他们共同取得的成果。

**双方的商业合作与收入分成关系保持不变**：现行的收入分成安排维持原状，且该机制始终包含对OpenAI与其他云服务商合作所产生收入的分成。

**Azure依旧是OpenAI无状态API的独家云服务商**：微软是提供OpenAI模型及知识产权访问的无状态API独家云服务商。这些API既可通过微软购买，也可直接从OpenAI购买。OpenAI与任何第三方的合作所产生的、对OpenAI模型的无状态API调用，都将托管在Azure上。

**OpenAI的第一方产品，包括Frontier，将继续托管在Azure上。**

**AGI的定义和流程保持不变**：AGI的合同定义及是否实现的程序保持不变。

**此次合作将助力OpenAI实现发展**：随着OpenAI不断扩大规模，其仍拥有灵活空间，可在其他平台追加算力投入，包括构建星际之门等大型基础设施计划。

其博客最后提到，双方合作的初衷，就是为微软与OpenAI保留独立开拓新机遇的空间，同时继续开展协作。目前两家公司也正是这样做的——既携手合作，也各自独立发展。

  

**_03_****_._**

**结语：豪华资本团押注**

**OpenAI面临上市、冲刺AGI压力**

2026年开年，硅谷的明星AI创企接连拿下大额融资。1月马斯克的AI独角兽xAI宣布完成200亿美元E轮融资，英伟达投了；1月，Anthropic被曝拟以3500亿美元的估值筹集100亿美元资金；2月，李飞飞创办的空间智能创企World Labs宣布完成新一轮10亿美元融资，英伟达、AMD参投，再加上最新的OpenAI融资，一系列千亿、百亿美元级融资正密集落地。

对比来看，OpenAI的吸金能力在AI界仍然处于顶尖水平，此次亚马逊、英伟达、软银联合的豪华投资阵容，再加上1100美元的金额，在产业界可谓少见。

下一步，其或许可依托充足资金与战略伙伴资源，加速基础设施建设、下一代模型研发与全球商业化扩张；同时也会带动数据中心、高速互联、定制芯片等产业链全线升级，重塑全球AI算力供应链与技术标准话语权。

但从多家外媒的爆料来看，多位投资方均希望OpenAI能够尽快实现上市，并加速推进通用人工智能目标，这一诉求或许会让OpenAI更多兼顾资本市场的回报预期与商业化压力。
