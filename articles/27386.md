# 下半年我的重点方向：OPC漫剧短剧出海

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/4KW1YfFgWQliaFTsXf2QXNj2qQrHMhRuM62s91TiaCjIHJemNqNSJ9XcHLJ1FwyuKC2b7MzF5xHEE6WHp1um78iaF6ISViacv1g7uefiaylecR9k/640?wx_fmt=other&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)

  

  

最近陆续看到一些数据。

2026年第一季度，全行业上线微短剧约12.8万部，其中AI微短剧约12.2万部，占比超过95%。

更夸张的是单月的数据。今年3月，抖音平台新增AI剧、漫剧接近5万部，这一单月上传量几乎追平了2025年全年的总量。

人民日报还发了一篇文章：《AI微短剧正在跨过“奇点”》。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQnuCHkjnu4KDqdpejzRvDO13hB4wZyJhtoX5sNb2AdMzWLrnM3aa4sfJPChkVZSqQnVOkPIHJcpORraAic091pZU7LnLD8pWZZU/640?wx_fmt=png&from=appmsg)

就像文章里说的：“作者即导演”“一人剧组”的创作潮流悄然而至。

一些网络作家自学AI技术，成功影视化改编自己的小说；大量普通网友娴熟运用AI工具制作短剧、短片，“雪山救狐狸”等成为大众喜闻乐见的文化热点。

很多亿级播放AI短剧，延伸出了更多的话题和讨论，也诞生了各种新的分类标准。

包括我自己以前爱看的沙雕动画、末日科幻等题材，不知不觉，已经有了很多高质量的AI新作品。

我采访了一批AI短剧、AI漫剧制作人、工作室，交流过程里，有一个高频词：LibTV。

LibTV之前介绍过不少，但今天，又有了新信号。

**01**
======

**亿万播放爆款，可以复制吗？**
=================

我好像不爱看真人短剧，就爱看AI做的。

关键是，很多AI短剧，细看下来，真的是完全可以复刻的。

带大家拆一部作品。

名字叫《不卷了回家种田开店》，24小时观看破亿。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQnFP9LmL0D2GDlAWrKvIXxQNGGPN7cUkUq8NfoKpibCCticI5m1Bic44Ll3YwUBXOrqjxDBQVFoOibcBdnG1OpDFB9CMrfRgyiaiboCo/640?wx_fmt=png&from=appmsg)

叙事很清晰：它捕捉了当下年轻人不愿再卷、渴望回归真实生活的情绪。

回家种田的剧情设定，很契合当下大众情绪化的表达。

它的爽点不是传统的升级打怪，更像是生活主导权的重新夺回，理解门槛极低。

观众不需要理解庞大世界观，完全可以带入小人物个人视角：职场压迫、生活失序、选择回乡。

情感上的共鸣，瞬间达成。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQniax1b6UojkWGMibIGEasENmkV6sWQTh02D0wXA3gM324TS41hk8IntpwpJMNKZ3pLLm5Ep4yJ5Dekr9bB301kdDKBNgQMEsdDg/640?wx_fmt=png&from=appmsg)

我尝试复刻了一下。

先给出一个简单的《不卷了回家种田开店》，在LibTV内，用「脚本生成器」，先拓展成剧情完整的脚本。

景别、光影、对白、音效、剧情……都会用表格形态完整呈现出来。

![](https://mmbiz.qpic.cn/mmbiz_gif/4KW1YfFgWQmUBTqBBUY8Jic8RjbjEibBULnQWBVU4hAZnDNBbUwkSPfm4KJkxyp6f7PgQXA60iaOjCENXUtS38wDgqhoeXicDyeKyTKadU9ffE8/640?wx_fmt=gif&from=appmsg)

再之后，可以一键生成相关内容资产，把纯文本进一步变成可视化的图片、分镜等等。

我发现，用AI做这类题材，难点不在特效，在于保持故事的连续性。

所以脚本生成部分，是我特别看重的。

种田、装修、烹饪、创业、邻里往来，场景很多很复杂。

在LibTV内，生成提示词、确定角色形象之后，接下来的人物和场景都可以复用。

![](https://mmbiz.qpic.cn/mmbiz_gif/4KW1YfFgWQkmZWy5ThibPShFFCoVxu8ZZcNJiaY1YpIxrBn3hdrrMafG6OzNVHibWXNaSdr7Fsic9EicuzQvicBckolPbsjNAj815ek9V0RYlwmb8/640?wx_fmt=gif&from=appmsg)

实际生成的结果，一致性就不错。

怎么用AI稳定、高效地生产出合格的内容，这是我一直在研究的。

给大家看看复刻出来的一个情节效果：

**02**
======

**从剧本到成片，一个人能完成吗**
==================

这里稍微展开说说，一个完整的片子，怎么用LibTV做出来。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQnN8RlfjPZicS01fGZrf9roOCczBuWRo7NYEt13IIkZEF8vJjoE2qkFWjeibewLZxnbeVAxdQSWcPRgPESJVlwWsLulicy7GKd09k/640?wx_fmt=png&from=appmsg)

首先是剧本，可以说出想法，借助AI生成完整文字。

然后借助LibTV的「脚本」功能，一键从剧本到专业的脚本，生成了我们熟悉的脚本视图表格。

在节点这里，点击“脚本（Beta）”功能，从基础的画面描述、角色描述到最后的分镜提示词、视频运动提示词。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQkNoIkHqdvzooNWUrPSia4qwlSddWaE1FYsBBoL0XciamkId5agEicYqRnqCLKVxT92b8poal3pfGc2EbblYFfzeicx6IviaXiaQuY3g/640?wx_fmt=png&from=appmsg)

LibTV在脚本视图的基础上，还有创意视图，不同小伙伴一起做视频，都可以找到自己喜欢的阅读方式。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/4KW1YfFgWQkVNAdhicZpU60c3R71HVicjDadTRNWMKxIuia6Cd2iacBxucfyCHibmeYTqQLek5l8ianqg8SAzWHLflbav1icpVLy6NSP8iaO8QWicdWg/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQnkE77Z6H4SiaSjWkEgVZr32Ic9HEicO9BSvnYK0C1Icicsd19jNHHlcsF8ibRqENeiac7iaNytFJagMo2fJHibdtibnjLaqAp5kVicf2ww/640?wx_fmt=png&from=appmsg)

再之后，LibTV就会基于已经生成的脚本视图，开始创作生成。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQk4q1a19ZAph1kkeAMu8on2icicUHnzynLxhESYF8EnYRsibufPPZApibovWlRJlepMdcu1ovC6bdu3x8licD5W8Op9IreNH0nZFPBw/640?wx_fmt=png&from=appmsg)

在创作完分镜图之后，更有意思的是，有个【全景】功能，一张场景图，能生成720°全景图，可以根据需要选择更适合的场景图作为接下来的分镜图。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/4KW1YfFgWQlUTBzLCaILAPicAtgagrf5Fj0chgOJjSCNXcEeevNR8lM7u6baNlWSd6xV9dUc3dod3Hte62iaRYzPrIIz5Dj8sNEDPHTOf01ZE/640?wx_fmt=png&from=appmsg)

点击生成图片的节点，开始图片创作。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQlgEEGnthhUMexSSsppsFFagnwzibRx4qCWXrxT9t4ZKWlH9lPCN7xe3H2CVAyaLKozckVCiavL1QJanNTicFGZADEaANeBluiaOug/640?wx_fmt=png&from=appmsg)

选中所有的分镜图以后，开始批量生成视频。

每张图都会有根据设定好的脚本来生成视频，极大释放了生产力。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQkYXRTo7r8Ioe4LiccJcP2EfPEqCiaib3Yu4oga5rYGTP49DIvWF4Ed7o2RCXguTYtSAV9yXY7TcPQdORgKZsoObnGDAbPRqicvI6I/640?wx_fmt=png&from=appmsg)

更让制作视频这件事儿夯到极致的，是LibTV的视频合成功能。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQkAH0ZcKyqiajmCw2hjlAGBekM9sqZWichf88IXcnKPF7nF9C6Fs6eJTagRbTjQiaa89dOAbd8Ftj2DaqnKcAjszmR1qTQlyKnymQ/640?wx_fmt=png&from=appmsg)

传统的手搓视频，需要逐个下载，然后在剪辑软件里合成。LibTV把这件事儿做成了一键合成的功能。

使用节点把需要合并的视频串起来，接下来点一下【视频合成】。

![](https://mmbiz.qpic.cn/mmbiz_png/4KW1YfFgWQmpvnfDRzmrulcYBLKaicGFC02odtaBgNFNIulUuicx6bibyCfH6ZRI2oulCxKXWPzId4s4ut5FdhrtQQ4EjE9UapYkxT0GWvZQ24/640?wx_fmt=png&from=appmsg)

省去了下载再进入其他软件的麻烦，10个分镜就在一起了。

这里可以进行基础功能剪辑，能让创作者们快速出片。

效率的提升，往往就在这些看不到的小地方，而LibTV就是这么在意创作者的每个细节。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/4KW1YfFgWQnNYMibm1tnBMF7XsWMzEwJ7Q0eXMd8X0LLKyz2rEyLIeOicWeNcvJkNPabWPhKkUhfHibPO8mINubX4HjEicZFD3V6OaZ2PicIzjms/640?wx_fmt=png&from=appmsg)

影视工作室的创作周期长，人员流动频繁，这是行业常态。

主创资产交接这件事，很多团队都头疼。一个好用的交接文档，比一个好的AI模型还难找。

工作室用上LibTV后，主创团队可以在画布里把角色设定、场景风格、灯光方案全部节点化存储。

后续加入的成员打开画布就能看懂所有设定，不需要导演一遍遍解释。

像我这个小片子，虽然中断过几次，而且是和同伴分头协作完成的，还跨了个假期，但是大家任何时候上手做，都还接的上。

创意方向的变化也记录在画布里，不会因为某个人离职或者换项目就把前期积累丢了。

可能有朋友会说，做成这样一个片子，团队里积分的用量，会不会是笔糊涂账、要花多少钱呢？

这就说到「积分预算管理」了。

LibTV的团队版里有积分消耗监控，可以实时看到每个成员用了多少积分，哪个环节消耗最大。

月底对账的时候，直接拉数据就知道预算花在哪里了。

漫剧短剧、影视娱乐，一直是LibTV的优势主场。而现在，这样的优势正在被团队协作能力放大。

到这里，我只想说：超过300家短剧公司和影视工作室、10万多创作者用LibTV，是有理由的。

**03**
======

**接下来的几个信号**
============

说几个值得关注的点。

首先一个，LibTV与火山引擎保持长期官方合作，Seedance系列模型已支持原厂核验。

这意味着创作者可以查询模型来源、参数信息及任务ID，模型能力与官方版本保持同步。

模型稳定靠谱，这是长期做内容的前提。

第二个我关心的，是LibTV的AI短剧生态建设。

今年5月，LibTV联手优酷和北京民艺非遗研究院，推出精卫计划AI漫剧原创扶持活动。

一边拿出千万级算力资源兜底创作，一边给到优酷站内S+项目最高200万现金保底加分账。

从创作工具、流量发行到IP合规、商业变现，全方位覆盖创作者的刚需。

第三，也是最重要的一点：越来越多过亿播放的AI短剧由LibTV创作者完成。

《不卷了了回家种田》、《荒年开局：我靠空间当大佬》、《圣体震长生》等等，这些作品背后的创作者，都是在用LibTV搭建他们的内容生产线。

![](https://mmbiz.qpic.cn/mmbiz_gif/4KW1YfFgWQkt4soyIbMYreKjrloNY33QetKN5fZVBthxYlibANsWYGKpbf6bqnibicMh3taV5FTCj9sZJsLib4CUsd3LYxyZEDCoU5pejiaPr4Og/640?wx_fmt=gif&from=appmsg)

真正的竞争，越来越取决于内容判断和工业化生产能力。

谁能更快地判断什么题材能爆，谁能更高效地把一个好点子变成一部完整的作品，谁就能在这个市场上活下来。

LibTV的价值恰恰在这里。

它提供的不是某个单一功能的优势，是一套完整的工业化生产流程。

从剧本到分镜到角色到镜头到成片，所有环节被组织在同一张画布上，可以被反复修改、复用、优化。

对于创作者来说，现在确实是最好的时候。

AI正在降低内容生产门槛，让更多个人创作者、小工作室拥有进入短剧行业的机会。

如果你手里有一本喜欢的网文，有一个想讲的故事，或者有一个一直想实现的世界观，现在就是开始创作最好的时候。
