# 真正的Jarvis要来了？涂鸦智能押注Physical AI

![](https://mmbiz.qpic.cn/mmbiz_png/ZBjVrHIdkOlmJxibib7DicsdiaplbHLrFPlcIe2EnQNZcNoEsAI2l2SdLXBu0LoaJJsx3icibapjpMbicJLlZGEJsZg6g/640?wx_fmt=png&from=appmsg)

作者 | 陈姚戈

在智能家居的爆发期，许多硬件品牌依托涂鸦的能力快速迭代、生长，走向全球市场。其中不乏许多明星产品，如曾在 Kikstarter 爆火的智能观鸟器 Birdty。

敏锐的先行者已经发现，基于涂鸦的能力，以细分场景切入的 AI 硬件可以快速迭代并跑出成功案例。但在过去，涂鸦的名字时常隐藏在爆款产品背后。

当 Agent 进入物理世界，新一代硬件产品仍然会选择涂鸦。但这一次，涂鸦不再只站在幕后。

最近，涂鸦发布了能协调各类 Agent 和硬件的 AI 生活助手“Hey Tuya”，让智能硬件可以思考、决策、共享知识，主动协助用户处理日常事务。涂鸦离 C 端用户越来越近，并且成为 Physical AI 在家庭落地的平台，而不仅仅是技术供应商。

一个能够协作的家庭智能管家

过去十年，全屋智能完成的，并不是“智能化”，而是设备的联网。通过手机 App，用户可以远程开灯、定时关空调、查看摄像头画面；通过自动化规则，设备能够在预设时间或触发条件下执行动作。

这一阶段的全屋智能，本质上是将人从控制的时间和距离限制中解放出来，但并未触及智能的核心。

一方面，系统的逻辑始终围绕单个设备展开。灯是灯，门锁是门锁，摄像头是摄像头，设备之间缺乏对同一家庭状态的共同理解。

另一方面，智能的实现高度依赖静态规则。用户需要提前设定大量 if–then 条件，而一旦生活节律发生变化，规则很快就会失效，系统无法自行适应，更谈不上进化。

更重要的是，这一代全屋智能几乎不具备持续理解与长期记忆能力。每一次交互都是孤立的，系统不会因为长期使用而真正更懂一个家庭。

这也是为什么，尽管设备越来越多、功能越来越丰富，用户仍然很难感受到一个真正的“智能管家”。

这正是 Hey Tuya 试图改变的体验边界。

在 Hey Tuya 出现之前，涂鸦智能已经长期服务了大量异构设备，形成了规模化设备接入、状态管理与执行调度能力。无论是安防、节能还是健康场景，涂鸦都能够将复杂的物理设备行为，拆解为可复用、可工程化的智能能力，供硬件厂商快速调用并落地。

而现在，涂鸦通过 Hey Tuya，将多年积累的感知、判断与执行能力，纳入同一套长期运行的系统之中——让设备不再只是被控制的对象，而成为能够共享状态、参与决策的 Agent 节点。

如果从一个典型的 Tuya 深度用户家庭的一天来看，这种变化会变得非常直观。

清晨，当智能手环检测到用户睡眠状态由深转浅，就可以调动预设的起床模式：窗帘自动拉开，引入自然光；音箱开始播放用户偏好的晨间新闻或轻音乐。

出门后，智能门锁确认家庭成员离家，全屋进入离家模式：灯光关闭，非必要电器通过智能插座断电，减少待机能耗；门锁自动上锁，摄像头切换为巡查模式，门窗与人体传感器进入警戒状态。节能与安全，不再依赖用户的持续操作，而是成为系统的默认行为。

白天用户离家工作时，Hey Tuya 仍在运行。宠物、孩子或老人的异常状态，可通过 Tuya App 主动推送。

![](https://mmbiz.qpic.cn/mmbiz_png/ZBjVrHIdkOlt5AvsU0tOGjQWJaDWHPhAmIbOuzuS2QHRr47Q9OXX5z6VFjOvCKQOtru2skgSqMbXYFsHJoxvcQ/640?wx_fmt=png&from=appmsg)

注：Hey Tuya 具有 AI 宠物看护功能，除了自动喂食等常见功能外，还可以记录宠物饮食数据，生成报告和建议。

这次的 Hey Tuya 升级，从全屋智能的场景延伸到学习、远程办公与协同之中。新增一系列 AI 效率功能。

![](https://mmbiz.qpic.cn/mmbiz_png/ZBjVrHIdkOlt5AvsU0tOGjQWJaDWHPhAOgs2Iu4HpZeqy1RVrO0poTVAer1zzFZVSDBpqgYNvfPdW77Glo99yw/640?wx_fmt=png&from=appmsg)

日常的碎片需求也将被更智能地承接。用户可在备忘录中记录的事项，例如提醒父亲下午两点服药，家庭中的智能音箱就可以恰当时间完成提醒。

更重要的是，随着用户与 Hey Tuya 的日常互动不断积累，未来系统也会形成对家庭成员长期习惯的理解。当用户通过 APP 询问“今晚吃什么”，HeyTuya 可以结合近期的健康数据、作息节律与过往偏好，给出更具针对性的晚餐建议。

从单个设备的“智能”，到家庭层面的“协作”，Hey Tuya 带来了家庭智能从“响应式”迈向“主动式”的拐点。

而这，恰恰也解释了一个更大的问题：为什么“Jarvis”，直到现在才开始具备初步落地的可能。

Jarvis 来到之前

过去几年，智能硬件和大模型快速演进，但一个能够理解家庭状态、持续协作并完成闭环执行的“Jarvis”，始终未能出现。

问题并不在于设备数量或模型能力，而在于 Agent 一直缺乏进入物理世界所必需的系统条件。家庭并非单一终端，而是由大量异构设备构成的动态空间；如果没有稳定的实时通信、持续的状态理解以及统一的执行调度，智能助手就只能停留在给出指令。

首先，传统系统并未为人机的长期协作设计。在弱网和移动网络环境下，延迟不可控、状态不同步，一旦交互中断便难以恢复，AI 无法在真实家庭中承担持续决策的角色。

其次，系统缺乏真正意义上的长期记忆。大多数智能助手只能理解当下指令，无法判断哪些信息值得保留、如何随时间演化认知，因而无法形成对家庭成员习惯与节律的持续理解。

更关键的是，AI 与物理世界之间缺少统一调度机制。理解意图并不等于完成任务，真正的执行意味着跨设备协同、异常处理与结果验证。过去，能力分散在不同设备与云服务中，控制链路冗长且不可回溯，Agent 难以对结果负责。

这三道门槛，使得绝大多数“智能助手”最终被困在屏幕和单一设备中。

Hey Tuya 从系统层面补齐了 Agent 进入物理世界所缺失的关键能力。其底层支撑，是涂鸦自研的 Physical AI Engine（PAE）——一套专门面向真实物理空间运行的 Agent 基础架构。

![](https://mmbiz.qpic.cn/mmbiz_png/ZBjVrHIdkOlt5AvsU0tOGjQWJaDWHPhAAcZ6k4G4VuTOkufDOyvenyeTnhkkIANhYsVNqDiagjjT6VhVft5ibkog/640?wx_fmt=png&from=appmsg)

在记忆层面，Hey Tuya 正在逐步通过 PAE 内置的 OmniMem 个体长记忆引擎，不断引入对短期与长期记忆的统一管理。OmniMem 在保障实时响应的前提下，对高价值信息进行长期留存，并根据使用频率与相关性进行动态更新与遗忘，使系统能够在连续交互中逐步形成稳定认知。其长记忆召回延迟被控制在毫秒级，避免了开源方案中常见的秒级延迟问题，使长期记忆可以被安全地用于实时对话场景；在多轮对话与长期交互中，也有效降低了记忆混淆的发生概率。

在通信层面，Hey Tuya 依托涂鸦自研的 Tuya Real-Time Communication（T-RTC）系统，实现 AI 与设备、人与 AI 之间的持续实时连接。T-RTC 针对家庭常见的弱网与移动网络环境进行设计，即便在较高丢包率条件下，也能够保持交互的连续性。结合涂鸦覆盖全球的数据中心布局、实时媒体加速网络以及大规模边缘节点调度体系，HeyTuya 得以在跨区域场景中维持较低时延与稳定可用性，为长期在线的 Agent 协作提供通信基础。

在执行与调度层面，PAE 引入了智能体编排引擎 Dynamic Orchestration Agent（DOA），将语音识别、意图理解、模型推理、知识检索与设备控制等能力纳入统一的编排逻辑之中，使复杂的多 Agent 协作流程可以被快速构建、验证和迭代。通过可视化方式，开发者能够在不显著增加系统复杂度的情况下，完成跨能力、跨设备的业务逻辑组合，从而缩短方案验证与产品迭代周期。

在更开放的系统扩展层面，HeyTuya 通过 MCP 与 A2A 等标准协议，支持开发者将自有模型、知识库、云端能力与本地硬件能力接入 PAE 架构，实现端云一体的能力协同。同时，涂鸦自研的 Adaptive Expert System（AES）对 Agent 能力的调用与组合进行动态约束与调度，在提升系统协同效率的同时，降低复杂场景下的不可控风险。

围绕这一架构，PAE 进一步整合了覆盖交互全链路的 AI 能力，以适配不同芯片平台、硬件形态与应用场景。

其中，Conversational AI Engine 负责自然语言与语音交互的稳定性与流畅性，支持多语言识别、低延迟打断与多样化语音合成；Vision AI Engine 提供面向 AI 硬件的视觉理解能力，使摄像头与陪伴类设备能够进行基础的场景与内容识别；IoT Intelligence Engine 则作为 AI 与物理设备之间的控制中枢，统一管理设备信息、通信协议与行为执行，确保指令能够被可靠落地。

Hey Tuya：涂鸦向 Physical AI 平台的

关键一跃

伴随 Hey Tuya 的推出，涂鸦的角色也在发生转变。

过去，它是幕后技术供应商，提供模组、云服务和设备接入能力；今天，这些积累被整合成面向 Physical AI 的底层平台。对开发者来说，它是构建家庭级智能产品的统一系统；对用户而言，它让 AI 从单纯的工具，变成真正参与日常生活的家庭助手。

Hey Tuya 的核心价值在于，它让跨设备、跨场景的家庭级 Agent 真正具备工程可落地性。原本分散在不同设备和功能里的智能能力、长期记忆与调度机制，现在被整合到同一系统中。家庭助手不再只是被动响应指令，而是能够理解用户需求、制定策略，在得到用户许可后跨设备执行。这意味着智能从一次性操作升级为持续、主动的服务，也为多 Agent 协作提供了可控、稳定的技术基础。

从生态和平台角度看，HeyTuya 不仅是一个 AI 生活助手，未来更有机会成为一个开放的家庭级 Physical AI 平台。开发者可以在同一系统里调动多个 Agent，实现跨设备、跨场景的智能协作。

这意味着涂鸦从幕后赋能者升级为生活 AI 的基础设施：统一接入、统一管理，降低开发成本，提高落地效率。创业者和厂商无需重复解决底层技术问题，可以把精力集中在产品体验和服务创新上。

Hey Tuya 的出现，意味着 AI 生活正在从零散功能走向整体智能体验。系统能够持续学习、预测需求，并在多设备、多场景间主动服务，让“Jarvis 级体验”开始有机会成为现实场景。

未来，随着更多设备和服务接入 HeyTuya，将形成可持续迭代的 AI 生活生态，AI 不仅理解生活，还能主动参与管理和优化家庭事务。

对于涂鸦而言，这是一场 AI 时代的战略关键布局：用 Physical AI 构建家庭生活的新基础，让家庭助手从工具，真正变成长期伙伴。
