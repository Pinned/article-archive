# DeepSeek秘密造芯！专攻推理，一年前已启动，招聘全程不公开

##### 梦晨 发自 凹非寺  
量子位 | 公众号 QbitAI

DeepSeek自己造芯片了！

目标：降低对英伟达的依赖。

路透社消息，这家凭借模型算法震动硅谷的中国AI公司，正在秘密开发一款自研AI芯片。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHjo9HbkPa5qFhz0FFSJ1h5OkXibnL1UzUKm9dFFL5Ohj5Ad0vibKj3NsIOsb5REkv1V1d7iaomeSmzozdR7ksT2NIkMicsCic3Dw6s/640?wx_fmt=png&from=appmsg)

芯片定位是推理而非训练，项目大概在一年前已经启动。

当前，该项目还处于早期阶段。消息透露，DeepSeek已与芯片设计公司、晶圆代工厂和存储器供应商展开接洽。

从算法公司到硬件玩家
----------

DeepSeek一直以来的标签是对算法的极致优化。

从2025年1月引爆全球的R1推理模型，到后来适配华为昇腾的V4系列，这家公司展示的始终是用更少的算力做更多的事。

但现在，它要亲自下场做硬件了。

知情人士透露，DeepSeek近几个月一直在招聘芯片设计工程师，但招聘方式极为低调，没有在任何公开招聘平台发布职位信息。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEERY89Yw0nm13GOHeL2YgicgRJcFopLDVIYUYMibOS6SJsTpHW7lK6ZGu9YkAmcfauJG8Co67XuhcOyrM0xcDXVoibDqicAOicl8WU/640?wx_fmt=png&from=appmsg)

模型公司自研芯片并非孤例。

放眼全球，AI模型公司自研芯片已成趋势。OpenAI上个月刚刚公布了与博通合作开发的首款定制推理芯片Jalapeño，Anthropic也在今年4月被报道正考虑自研AI芯片。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEknL18K8X9fITFqDZ2cqekEOHLgHUbTe7u9Fmic1ehZUf268MxN6dMzMB4meoeaJkB3EjRmmTZlDJQWg2cC5nicDyyrTkkH10HM/640?wx_fmt=png&from=appmsg)

DeepSeek创始人梁文锋在2024年一次罕见的媒体采访中曾透露，芯片不足对公司构成了挑战。

R1模型的底座是在英伟达H800上训练的，此后DeepSeek转向华为昇腾，今年4月发布的V4模型已适配昇腾芯片，华为也确认其处理器参与了V4-Flash的部分训练。

但鸡蛋放在两个篮子里，DeepSeek也觉得不够。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHic6t2sFiaF6XC8205oSxibicg02ab0Zb3jnjwAlHwCkHaLulC6SW7P8KJxBq2kZHPxGbvqT4xicTYl64QBtRHhLreu5bIksa6HdCk/640?wx_fmt=png&from=appmsg)

DeepSeek这款自研芯片专为推理设计。

随着AI应用大规模铺开，行业算力需求的重心正从训练向推理快速倾斜。训练是一次性投入，而推理是持续性消耗，每一个用户的每一次提问，都需要推理算力来响应。

针对推理场景定制的专用芯片，能够比通用GPU实现更低的功耗和更低的单位成本，这对于一家正在走向商业化的AI公司来说至关重要。

DeepSeek在模型设计层面早已为硬件协同做了铺垫。

DeepSeek-V3.1引入的UE8M0 FP8数据格式，被认为是专门为下一代国产芯片的硬件特性而设计的，算法团队在写模型的时候就在想芯片的事。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGibmhicmxPsOP7xzhXL0xajv5N5Ut5go2JxXqX1bSUqvZbHicryIzNI2Qq5MVBAF6dbURYIMZZLaTkxjmiaCNRiaibOEgb8Qp9eAhBo/640?wx_fmt=png&from=appmsg)

不过挑战同样巨大，设计一款有竞争力的AI芯片通常需要数年时间和巨额资金投入，目前尚无法保证成功。

510亿融资落袋，算力基建同步铺开
-----------------

支撑这一计划的是真金白银。

2026年6月，DeepSeek完成了成立以来的首轮外部融资，筹集约510亿元人民币（约74亿美元），投后估值在520亿至590亿美元之间。这家公司此前多年一直拒绝外部投资，此次开放融资本身就是重大战略转向。

资金用途已经明确：扩建以国产芯片为主的算力中心、自研AI芯片、扩充全球顶尖人才团队。

基础设施层面的动作同样在推进。DeepSeek已发布IDC设计规划工程师的招聘，计划参与从兆瓦级到吉瓦级数据中心的规划与建设，招聘信息中明确提到了内蒙古乌兰察布等建设地点。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFDbvayO5DxN8ibvS6oR6BauSb0X6cOVEsSHHFYLbftJGEdwvENibFaqvuZbNoD67Vdiacs7UicxUvq4TEWgicjCk454RpoO10pTIaA/640?wx_fmt=png&from=appmsg)

DeepSeek本身依然保持着一贯的低调。三位知情人士均要求匿名，公司层面也未做公开回应。

但无论如何，这家改变了全球AI竞争节奏的公司，现在要改变的是它脚下的硬件底座。
