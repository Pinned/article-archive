# 怀旧党狂喜！有人把Windows 8 UI复刻到了Linux上

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

翻译 | 苏宓  

出品 | CSDN（ID：CSDNnews）

2012 年，Windows 用户迎来了一个“分水岭”。微软推出了全新的桌面界面——Windows 8，但它糟糕的设计几乎让人忘记了微软曾经的辉煌。这个界面的核心是大而方的图标，专为触屏笔记本和平板电脑设计，而经典的鼠标键盘桌面几乎被彻底抛弃。当时微软的逻辑很简单：未来所有设备都会以相同方式操作。然而十多年过去，这个预测依然未能成真。

更糟糕的是，微软同时推出了类似苹果的应用商店，并建立了一套严格的应用限制机制。玩家和开发者立刻感受到了压力——彼时 Valve 联合创始人加布·纽维尔（G 胖）甚至直言：“Windows 8 对整个 PC 世界来说是一场灾难。”于是，他开始加大对 Linux 的投入，为玩家提供新的选择。

可以说，这段时间是 Windows 发展史上的低谷。

戏剧性的是，时隔多年后的今天，有人把 Windows 8 的桌面风格”复活“了，不过不是在 Windows 上，而是在 Linux 系统上。这也证明了一个道理：再“失败”的设计，也总有人会怀念它。

Win8DE：在 Linux 上重现 Windows 8 界面

日前，一位名为 er bharat 的开发者推出了开源项目 Win8DE。这是一款能在 Linux 上运行的桌面环境，几乎完整复刻了 Windows 8 的用户界面。

更准确地说，它是一个适用于 Wayland 窗口管理器（如 Labwc、Hyprland 等）的桌面 shell，整体效果非常接近 Windows 8 的 tiled 界面。

Win8DE 重现了 Windows 8 最具标志性的体验，包括：

*   色彩丰富、像 Windows 8 动态磁贴一样的方形和矩形图标
    
*   支持水平滚动的主屏幕，看起来像基于磁贴的桌面
    
*   可以自由排列的应用页面
    
*   类似应用抽屉的紧凑视图，用于查看已安装的所有应用
    

![](https://mmbiz.qpic.cn/sz_mmbiz_png/q7kM1wTg0eiaRGLGic5UJLjicIpMFw1sNKElFpGXWVfMLHM7fIGxicocnCP4G4zVKjhmy2oGVEwgtdDQMZPL95pfDg/640?wx_fmt=png&from=appmsg)

开发者 er bharat 直言：“如果你曾经喜欢 Windows 8，怀念它流畅的动画效果，但后来转向 Linux，又无法回到 Windows 8——因为应用都无法使用，也接受它无法安装在新硬件上的事实——那么 Win8DE 就是为你准备的。”

有趣的是，开发者也承认 Windows 8 中有些设计不值得复刻。例如臭名昭著的 Charms 菜单——那个隐藏在侧边、许多人觉得混乱且无用的功能——在 Win8DE 中被刻意省略。他表示，这东西从来没用过，也不值得再回归。

Win8DE 的价值

从实用性来说，Win8DE 不太可能取代像 KDE Plasma、GNOME、Cinnamon 这种成熟的 Linux 桌面环境。这些主流 DE 有多年的打磨，更加稳定、可定制性也更强。Win8DE 更像是一个兴趣项目、一种有趣的尝试。  

但这个项目仍然有它的意义：  

*   它显示了 Linux 桌面极大的灵活性，你几乎可以把任何界面概念搬到上面。
    
*   它也揭示了开放平台与 PC 游戏生态之间的关系。Windows 8 的封闭尝试曾推动像 Valve 这样的公司去支持 Linux，进而催生出 Proton、Steam Deck 等技术。Win8DE 是这段历史中一个有趣的文化现象。  
    
*   对一部分人来说，这像是 UI 的时光胶囊，让他们在现代系统上重温那段经历。  
    

er bharat 透露，项目仍在开发中，因此体验可能不够完善，也缺少一些功能。这也正是它“有趣”的地方。如果你是 Linux 资深玩家、怀旧爱好者，或者只是喜欢搞怪技术玩意儿，可以去项目的 GitHub 仓库看看安装说明和源码（https://github.com/er-bharat/Win8DE）。
