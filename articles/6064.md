# 我在微信免费养「龙虾」，玩到上瘾，这组CP太强了

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜杜伟、陈陈

  

2026 开年以来的这波「龙虾热」，仍在以超出预期的速度渗透进各类任务场景，成为提效利器。

  

几天前，OpenClaw 创始人 Peter Steinberger 发了一张意味深长的迷因：小龙虾骑上了自行车，看起来不是那么优雅，甚至有点笨拙，但它确实在往前跑。正如当下这波 Agent 浪潮，一切也许还没有完全对齐，但已经动起来了。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGUC78bghqPQhgpaCTrwv9jibskXm6hJg9fJczKs19raWBBfNBdeA1V3gxCyLicYoPz4RfvMgfzwPGfIiaJyCWC39ZZ6JyD6svbS8/640?wx_fmt=png&from=appmsg)

  

在国内，包括主流大模型厂商、互联网大厂、初创公司以及一些传统企业，都不约而同加快在该赛道的布局，造出了各式各样的龙虾产品，让人看到了几年前「百模大战」的影子。

  

OpenClaw 再次掀起的这场 Agent 军备竞赛中，如果要选出动作最密集、覆盖面最广的互联网巨头，腾讯可以说是当仁不让。马化腾描绘的腾讯「龙虾矩阵」，已经说明了一切。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHcFXnpTsjiaL1T5YPEyibff8OAfyicKSuYFp2m3jOWngic7fWLx4cLfNSPVXs1tzZXQic2IHhxWVcxiaNWZ3ibicd0OPlRiah7a2m2wEW0/640?wx_fmt=png&from=appmsg)

  

回顾鹅厂这段时间的动作，可以用组合拳来形容：

  

产品层多线推进，包括云端虾 Lighthouse、本地虾 QClaw、自研虾 WorkBuddy；入口层支持企业微信、QQ、元宝派接入 OpenClaw，近日推出微信 ClawBot 插件，支持接入 OpenClaw 以及其他龙虾；基础设施层推出「龙虾管家・AI 安全沙箱」、龙虾官方知识库「鹅厂养虾」和龙虾技能社区「SkillHub」，补齐安全、认知与生态能力。

  

其中，不同于部分依赖 OpenClaw 生态的产品路径，选择自研技术架构的 WorkBuddy 主打全场景办公，在设计之初便体现出了更强的一体化优势：

  

它不需要云端部署，下载、配置、连接最快 1 分钟搞定；多平台打通，支持接入个人微信、企业微信、QQ 以及钉钉、飞书；内置腾讯混元、DeepSeek、GLM、KIMI、MiniMax 等顶级大模型；支持 20+Skills 和 MCP 协议，提供零代码能力扩展；多 Agent、多窗口并行工作，效率成倍提升。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqH2al8H8oqaHXPgtmicHxUkp26gUjH8WJmkBz3e1r4ERwMsDDTPLK2P82dPPuyX73FzWbS9kRlMQJfjII5bGpXrlTA5jGrtYRlY/640?wx_fmt=png&from=appmsg)

  

之后，WorkBuddy 持续进化出了更多能力，包括自动定时任务，让 AI 主动工作；「行业专家」功能，12 大领域 141 位专业顾问随时待命。

  

并且，在微信 ClawBot 插件上线之后，WorkBuddy 选择第一时间跟进适配。现在，在电脑端打开 WorkBuddy，进入「Claw 设置」，选择「微信 ClawBot 集成」并点击配置，扫码即可一键完成绑定。

  

随着微信 ClawBot 不再是 OpenClaw 的单一接入工具，其与腾讯自研小龙虾 WorkBuddy 的联动，形成了如今这样一对「官配组合」，接入流程更加简化，使用门槛显著降低。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEDNp9l02083LnhvE8G3MRz6PGKjpVuMRic48iaJJ04CPBcLLuMVpj2vVkJ34UhY7ia99scDZVs4jt9VJpyib359ANggBAyiciawO6Ek/640?wx_fmt=png&from=appmsg)

  

在部署了 WorkBuddy 之后，我们对它来了一次能力大摸底。

  

微信「养小龙虾」，还是官配组合强

  

话不多说，咱们直接开测。

  

前几天，图灵奖公布，我们需要整理一份近 5 年图灵奖得主情况，如果将这个任务交给 WorkBuddy，看看结果怎样。

  

在微信 ClawBot 聊天界面下达指令：直接输入语音，语音文字为「帮我整理一下图灵奖，近五年图灵奖得主」。

  

指令一发出，电脑端 WorkBuddy 便立刻开始执行：

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqFgwkuD8NicLfbguydCGWTdS3S6Gy4AiajRCwsudtux5Plso652aLoObbckLuN4HyLeicsYRfnPRUD8cRPFkKZLmoklibk5AvMZVAI/640?wx_fmt=gif&from=appmsg)

  

很快，结果就出来了：

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqEZVXU0Vic2RDoJte4hQShxCOLGInQ36U2RHh84Icjv9nLGL53FzfO6KhAJlbxZ7b5sbOw0bjUoUAHZnMibvk8HFeI5wE6PLTum8/640?wx_fmt=gif&from=appmsg)

  

同时，作为媒体小编，日常最重要的部分工作便是「刷新闻、盯动态、整理当天的 AI 信息流」。

  

这次，我们将原本需要盯着屏幕一条条筛选的资讯，交给 WorkBuddy 处理。

  

提示词是这样的：「任务名称：AI 爬取资讯。执行频率：每一小时。任务内容：搜索互联网上最近与 AI 相关的最新内容。优先关注：模型发布、公司动态、技术突破、热点讨论，过滤无关或低价值信息。输出：3-5 条最重要信息；每条附一句总结。」

整个过程一气呵成，从刷信息到自动送达，原本碎片化、重复性的工作，被压缩成了一句话的成本。

  

接下来，我们通过微信 ClawBot 远程操控电脑，整理杂乱的文件。

  

输入提示词：「我电脑「D:\\shiying」目录下文件较为杂乱，包含图片、文档等多种类型。请帮我按文件类型或内容进行分类，自动创建对应文件夹并完成整理归档。」

作为打工人，最常见的困境之一是：关键资料都在电脑里，人却不在电脑前。在地铁、路上，甚至刚下班，临时被要文件、被安排任务，这种情况几乎每天都在发生。

  

过去要么让同事帮忙，要么远程连电脑，要么干脆拖着，等回去再处理。但有了 WorkBuddy，真的不一样了。

  

比如，我有一笔发票报销要处理，但发票都在电脑里，人却不在电脑前，这时候就轮到 WorkBuddy 大显身手了。

  

因为微信已经接入了 WorkBuddy，我们只需要输入提示词：

  

「请帮我处理「D:\\fapiao」文件夹中的所有发票。按照如下格式记录：日期 + 金额 + 消费类型；对所有记录进行分类整理，并生成清晰的报销明细列表；然后汇总总金额：最后输出两份结果：一份整理好的报销明细（文本）；一份结构化 Excel 报销表（包含分类与汇总）。」

  

可以看出，我们在微信上下达指令的同时，电脑那头已经开始替你赶工，自动查找发票、整理分类、生成文本、表格，一切在后台同步推进。等你走到地铁口，甚至还没坐到工位，结果就已经处理好了。

  

视频中 WorkBuddy 思考部分进行了部分删减。

  

最后输出文本结果如下：

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqGs6hiaqjLxiccspMiaIUMYP9VjwduFTSNicmzFxd4pDggK7JuCe5pyNq6GMP0piaW5WAbApP6kXn7GQ8pOVc6RMAFibwPN7NJ8UD3Jc/640?wx_fmt=gif&from=appmsg)

  

作为小编，被领导催稿早已是日常。最尴尬的是，稿子明明写好了，却躺在电脑里，偏偏你人还在外面。

  

现在这事就简单多了，在微信里说一句话就能搞定，「我的电脑 “D:\\ 陈的文档” 中有篇关于世界模型的 word，找出来，发我邮箱」。

  

只见手机端刚把指令发出去，电脑那头就已经开始自己干活了：自动定位文件、打开确认、并发送邮件，一气呵成。你这边还在聊天，邮箱文件就已经收到了。

  

  

不止发邮件，热点论文也一样可以一并交给它处理。

  

「去 alphaxiv.org 网站，按照“hot” 排序，把前五名论文总结一下发给我，要总结成表格，有待解决问题、解决方案亮点、实验结果、论文链接。结果使用列表格式，清晰展现。」

更重要的是，这一切不再需要人工介入。从信息抓取、筛选判断，到结构化整理与输出，整个流程都在后台自动完成。

对话即操作，龙虾矩阵加速范式重构

  

在上周的英伟达 GTC 大会上，黄仁勋将 OpenClaw 视作「推理时代的操作系统」，其重要性可以与 HTML、Linux 以及 Kubernetes 等基础软件设施相提并论。他进而抛出了一个观点：现在每家企业都需要制定自己的 OpenClaw 战略。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHaXWgnn4mQ1v7xUePs4Hat1VPoQgKm2BXqEVUic7tjWuO4voelSuHrSDAJzwe5dCCyriasBpcyx5DkSHiaMibmo9Ufh8mrnSAkfag/640?wx_fmt=png&from=appmsg)

  

如果把时间线拉长，老黄的这番言论更像是对既有趋势的「盖棺论定」。特别是对于国内以腾讯为代表的互联网大厂而言，这场围绕 OpenClaw 的新一轮 AI 基础设施竞赛，实际上早已悄然开打。

  

就拿 WorkBuddy 来说，腾讯早在 1 月已经在内部开放体验，2 月开放内测，直到本月正式上线，前后仅用两个月完成从试水到落地。加之期间陆续推出的 QClaw、ClawBot 插件等，马化腾口中的龙虾矩阵正在加速成形。

  

从当前趋势与市场反馈来看，拥有成熟应用生态的互联网大厂在 OpenClaw 赛道明显跑得更快。

  

一方面，这些大厂天然掌握高频入口（如 IM），可以将 Agent 能力深度嵌入到海量用户日常；另一方面，横向打通办公、内容、服务等场景，降低使用门槛，提升使用频次与用户感知，建立「对话即操作」这样的心智。

  

因此，本轮竞争的关键在于谁能更快地完成对 OpenClaw 这一范式的吸收与重构：通过 Agent 与现有产品体系的深度融合，重构软件之间的连接方式与调用逻辑，进而构筑起全新的能力层。

  

腾讯正奔着这样的目标前进：让不同业务线围绕 Agent 形成联动效应。

  

在 3 月 18 日的腾讯财报电话会上，马化腾分享了更多他对于龙虾价值的思考：将 AI 从过去集中于 ChatBot 的形态中解放出来，分发到更丰富的业务场景，在落地过程中进一步释放社交、内容与办公生态上的协同优势。

  

这种从对话能力向系统级能力的转变，正在成为整个行业的共识，普通人也将获得更高效的生产力工具。

  

正如 Windows 定义了个人计算机时代一样，OpenClaw、WorkBuddy 等定义了 Agent 时代的运行方式，它们让每个人都能拥有自己专属的自动化助手。
