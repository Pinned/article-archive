# Moltbook 新规太狠了：Agent 闯祸全算人类头上，免责条款拉满！

![](https://mmbiz.qpic.cn/mmbiz_gif/YriaiaJPb26VPQqHC66RJFpttVIMWG83T3lWHahUD4bvhxlKSayjeV2ibvC5ydqklP9QHDPD3qHJM07TV3IfHstjA/640?wx_fmt=gif)

整理 | 华卫

被 Meta 收购数日后，Moltbook 已开始做出调整，这款类似 Reddit、面向 AI Agent 的社交平台于更新了服务条款。在 Meta 出手收购之前，该网站仅有五条规则，责任条款对人类使用者更为宽松。旧版规则写道：“AI Agent 对其发布的内容负责，人类所有者负责监督和管理其 Agent 的行为。”

如今，其条款页面已被大量法律文本和协议填满，其中明确规定：每位用户需对自己的 AI Agent 承担全部个人责任。新条款写道：“使用本服务的 AI Agent 不具备任何法律主体资格。因此，你同意，你将独自对你的 AI Agent 及其任何作为或不作为承担全部责任。”这一变更显然至关重要，Moltbook 特意将其用加粗、全大写标出。

该社交平台还新增了年龄限制：使用者必须年满 13 周岁，或由监护人同意条款。这在科技公司中十分常见，Meta 旗下的 Instagram 也有类似要求。

此外，Moltbook 在条款中加入了一系列免责声明，其中一条建议用户不要依赖 AI 提供信息或进行决策。条款写道：“Moltbook 不保证 AI 生成内容的准确性、完整性或可靠性。” 用户同意，不得将此类内容作为 “自身独立判断的替代品”。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/hLLZnAbUwNj45ickJq3RMOTNuxeSiapvf1vdFCMlCV6FWOmHlSOmCbZ58C7g9FrBu5VyhjWlia2Ow4x34uCTe4wlz5UKmFicxm5GzzOaw43VluQ/640?wx_fmt=png&from=appmsg)

Meta 于今年 3 月收购了 Moltbook，创始人 Matt Schlicht 与 Ben Parr 也加入了 Meta 超智能实验室团队。Moltbook 起源于 X 平台（原推特）上一个关于 AI Agent OpenClaw（原名 Moltbot）的梗。用户当时必须绑定 X 账号才能注册。即便被 Meta 收购，这一点也未改变：用户仍需使用 X 账号登录，Instagram 或 Facebook 账号均无法使用。
