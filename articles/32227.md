# 估值4800亿，DeepSeek火速开启新一轮融资！最快明年IPO

##### 鹭羽 发自 凹非寺  
量子位 | 公众号 QbitAI

马不停蹄，趁热打铁！刚完成第一轮融资后，DeepSeek被曝第二轮融资已经启动。

据《金融时报》消息，目前DeepSeek正在与新投资者接触，希望开启新一轮融资，融资前估值约**710亿美元**_（折合人民币4800亿元）_，相比首轮融资后估值520亿美元，上涨**37%**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGFEeLOdprb0FgkibfYDvia4TEL1VR6OkRJBVibY26ZuRKfqrL9hkqhvuIst46N4cDhlHice7lKsFV3azknUKezlyygGicUAUOCK2hUo/640?wx_fmt=png&from=appmsg)

与此同时，更进一步的爆料说，DeepSeek已经开始为**2027年IPO**做准备，最快将在今年年底递交上市申请。

要知道距离上一轮融资结束，仅仅过去六周。

首轮融资额超500亿元人民币，梁文锋个人出资200亿元，腾讯、宁德时代等多家产业龙头与国家人工智能产业基金共同参投。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGGHOnU9TY6u1wXdOyFrxYsfmLvX89VcQq0Q16C80Wg3oz1O7icgYtyfAjTZY7uDbgBohoqhZqCVEX9RyeYmnLYjhEb7Uficpb2cI/640?wx_fmt=webp&from=appmsg)

对于一家成立不过三年的AI公司来说，这个速度几乎是史无前例。DeepSeek也将再次刷新国产大模型公司最高估值记录。

迅速开启新一轮融资
---------

很多人第一反应可能会觉得奇怪：两个月不到融资两次，DeepSeek很“缺钱”？

毕竟这家公司一直以来给外界的印象都是**“不融资、不商业化、不路演”**。

甚至梁文峰过去也一直强调，希望把更多精力放在AGI研究，而不是资本市场。

为什么现在突然变了？答案其实只有两个字：**算力。**

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGIS4OibtlZEEKBKn66tKZXjd2GHCxCgnS42fkIFLwJ1Co5CopVDcLjKN1zmmCDF8mGWaodncg0IyAgRWK51Ao7y197n9wLibZwA/640?wx_fmt=png&from=appmsg)

如果说去年的大模型竞争，比的还是谁的模型更聪明。那么从今年开始，竞争已经变成了谁拥有更多算力。

Agent需要持续推理，长上下文需要更多显存……几乎所有AI的新方向，都在疯狂吞噬算力。

DeepSeek想要加快基础设施扩张，就需要借助资本力量，提前锁定**未来几年的计算资源**。

报道称，DeepSeek的新融资，将主要用于三个方向：

*   建设自己的数据中心；
    
*   采购更多AI芯片；
    
*   研发Agent等下一代AI系统。
    

最近DeepSeek尤为密集的招聘动作也印证了这一点。

先是官网上线**“IDC设计规划工程师”**一职，根据岗位JD，DeepSeek正在计划自建GW级数据中心，团队将负责数据中心从前期选址、方案、布局到施工图、落地配套的全流程规划设计。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGEd5Nb5oPG21NPafmuicJneGUmH3YGMIfz4em2lj9V8JWdbfcK35nnxh5YPO4CaPmA8wXURDJSh909hPzd4xms1bf0xSTVw88Vc/640?wx_fmt=webp&from=appmsg)

再是**Harness**部门负责人崔添翼在线直聘，一口气放出研究员、工程师、产品经理三个岗位，实习全职都要。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGEY4NIDRXAc6Fo6gVibJFBGyG7VLqBia6Licf0K2w8c3dBDXAOibd80f1ficicDJL1DfuFdBWgyK6U8GuxKLgtzWf7CR7tpx0WqFS6ZY/640?wx_fmt=webp&from=appmsg)

**Infra**这块也招了不少。

大模型训练和推理框架工程师负责训练和推理的底层框架，高性能算子和通信编译器工程师负责榨干GPU的每一分算力，超算集群研发工程师负责让几千张卡协同工作不出问题。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGHunshEttWiatfWibUuJQPPegFibw7GCObic28ibv7q0b6GuHpUFj2RLYDWbCIic21Jzn5jx1wHyxBg4Qh6WUbD3IEbzwZyzzyUrsb3s/640?wx_fmt=webp&from=appmsg)

同时路透社消息，DeepSeek已经开始秘密研发一款**自研AI推理芯片**，目前已与芯片设计公司、晶圆代工厂和存储器供应商展开接洽。

多个方向集中发力，DeepSeek现在缺人缺钱，自然需要真金白银来填。

计划2027年IPO
----------

另一个耐人寻味的信息是，DeepSeek已经开始筹备**IPO**，目标是在2027年上市，最快今年底启动上市申请。

相关讨论仍在进行中，具体IPO时间安排和融资计划可能会有所变化，最终将取决于市场状况和公司业绩。

**梁文锋**个人也在首轮融资后，净资产翻了一番多，目前是全球人工智能模型研发者中身家最高的人之一。

据彭博亿万富翁指数显示，他的身价已经从原先的167亿美元增至360亿美元，远超Anthropic联合创始人Dario Amodei和OpenAI的Greg Brockman。

不过对于梁文锋而言，打造DeepSeek本身就不是为了个人身家，在DeepSeek投融资中他都是率先放鸡蛋稳定其他投资者的创始人。

如此发心和行动，世所罕见了。

而梁文锋如此这般，也不为别的，就是希望DeepSeek保持对Anthropic和OpenAI的冲击。

先进的AI，不该被闭源玩家垄断。

正道光明，一以贯之，**DeepSeek Go Go Go！**
