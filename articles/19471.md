# 黑灯社区 ——人工智能时代开源范式转变的思考

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

巴黎时间 5 月 5 日上午，2026  GOSIM 全球开源峰会在巴黎正式启幕。这座数百年来不断为人类文明注入新精神准则的城市，迎来了全球开源领域的顶尖从业者与思想者。华为首席开源联络官、CNCF 董事会成员任旭东出席峰会，并发表了题为《黑灯社区 ——人工智能时代开源范式转变的思考》的主题演讲。

不是工具迭代，而是时代跃迁：我们正进入 Software 3.0 新纪元

演讲开篇，任旭东便抛出了一个贯穿始终的核心问题：我们当下究竟在经历什么？

![](http://mmbiz.qpic.cn/sz_mmbiz_png/0uSFKjFCOxwwwfoKQmoY91G4roibxiar5MR9V4UDkQCXxPTzeD1evHshUNg9kNedH0vget5ib9wjBopF6F64MjhZYuMwNPUvPSLA5icQTYBliaMs/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)

他指出，行业内存在两种截然不同的答案。令人心安的普遍认知，是将 AI 视作又一次工具升级—— 如同从编辑器到集成开发环境（IDE）、再到 AI 编程的演进，只是生产效率的提升，劳动分工、组织架构与行业底层逻辑并未改变，“不过是一匹跑得更快的马”。

而真实的行业变革，早已突破了工具升级的范畴。任旭东强调，当下人类协作正演变为人工智能协同调度，传统的“碳时间”正被 “硅时间”全面取代，角色边界被重新划定，企业赖以生存的基础假设正在被彻底颠覆。“我们并非在加速旧时代的进程，而是在步入一个新时代。

一旦接受了这一点，所有关于招聘、流程、开源以及治理方面的决策，都必须被重新审视。”

这一判断并非孤论。任旭东援引 OpenAI 联合创始人、特斯拉前人工智能总监 Andrej Karpathy 提出的 Software 3.0 理论，进一步印证行业的时代跃迁：软件 1.0 时代，人类编写明确的命令式代码；软件 2.0 时代，人类负责数据集管理，神经网络完成权重学习；而当下的 Software 3.0 时代，大语言模型成为解释者，语境转化为记忆，提示词成为新的编程语言，规范、工具、模型协调平台与Agent，成为这个时代的核心基础模块。

他同时引用 Karpathy 的核心观点强调，在这个新时代，“你可以将思考过程外包，但无法外包理解能力”。智能体可以完成输入、搜索、重构与测试，却无法代替人类判断哪些事物值得追求，判断力、品味与道德准则，始终是人类不可替代的核心价值。

时代变革的四大维度：从领导力到协作模式的全面重构

基于新时代的底层逻辑，任旭东详细拆解了 AI 时代行业变革的四大核心维度，完整勾勒出开源范式转变的核心轮廓。

第一维度是领导力的反转：从“人参与决策” 到 “人主导决策”。 过去三十年，软件开发始终由人类主导全流程，AI 只是辅助工具；当下，AI 已深度融入工作流闭环，人类仅需监督关键节点；而即将到来的新阶段，人类将彻底凌驾于决策循环之上，转向意图层与仲裁层，核心工作从 “指挥人类” 转变为 “协调智能体”。在任旭东看来，这绝非工具层面的改变，而是领导力层面的根本转变。

第二维度是流程定义的重构：流程从叙述性描述变为基准事实。在“碳时代”，流程通过报告、会议、状态更新被口头叙述，结果可见但过程无形，无法被机器读取与复现，问责依赖记忆与解释，并行协作依赖信任；而在 “硅时代”，每一个决策都转化为账本数据，智能体的决策日志包含意图、推理过程、约束条件与最终结果，全量可被机器读取、可复现，代码提交同时兼具状态报告、审计追踪和任务交接的功能。任旭东直言，没有明确的流程，就没有可信的并行协作，这是实现可靠并行计算的核心前提。

第三维度是生产力衡量标准的更迭：并行处理取代人员数量统计。他用一组直观的数据展现了行业的剧变：传统软件时代，微软、JetBrains 等企业打造一款优质 IDE 需要数千名工程师，企业估值达万亿级别；AI 原生时代，Cursor 项目仅数十名工程师，便实现 500 亿美元估值；而 Agent 时代，Claude code核心团队仅十余名工程师，对应项目总价值达 3800 亿美元。

“人员数量已不再是衡量能力的单位，计算能力才是关键。” 任旭东指出，一个人通过 Agent、模型与并行工作流所能调动的计算资源量，决定了其能力上限，如今十个人协同利用计算资源，就能完成过去需要一万人才能完成的任务。

第四维度，“黑灯社区（Lights-Out Community）” 的到来。任旭东解释，在制造业中，高度自动化、无需人工值守、全天候运行的工厂，被称为“黑灯工厂”，而类似的现象，正以超出行业预期的速度在开源社区发生。

2026 年 4 月，成为开源 AI 历史上活跃度最高的月份，DeepSeek-V4、Qwen 3.6-Plus、Gemma 4、GLM-5.1、Llama 等五款基础模型在一个月内完成多次迭代，模型发布频率从每月一次变为每日一次，行业竞争的焦点从 “下一次发布” 变成 “接下来的 24 小时”。

而推动这一进程的项目，早已与传统开源项目截然不同：它们是 1-3 名核心人员搭配 N 个 Agent 的微型组织架构，Agent 成为真正的同事；采用去中心化模式，没有项目经理，仅保留意图收集与决策仲裁职能；项目增长速度从按月计量变为按日更新，开发的软件也愈发具备一次性使用的特性。

任旭东点出了其中最根本的转变：三十年来，源代码始终是为人类读者而写，供人审阅、理解与维护；而从现在起，源代码越来越多地为模型而写，供智能体解析、执行并生成新的代码。“这就是彻底黑灯的社区。开源不会消失，办公室里的灯光正在熄灭。”

支付粒度革命与开源价值剪刀差

在范式转变的表象之下，任旭东进一步拆解了支撑这场变革的经济底层逻辑，以及开源价值体系的根本性重估。

他首先提出了支付粒度革命的“100 倍法则”：过去七十年，商业支付的最小粒度每提升 10 倍，就能解锁规模扩大约 100 倍的市场。1950 年信用卡时代，最低消费金额约 3 美元，解锁了百万级市场；1999 年互联网支付时代，PayPal、支付宝、微信支付将最低支付金额降至约 3 美分，下降百倍，解锁了数万亿美元的市场；2025 年至今，AI 微量化计数、代币化技术将最低交易金额降至每笔约二十万分之一美分，精度较互联网时代提升至 10 的四次方级别，催生出全新的 Agent 经济体系。

“每一次技术粒度的革新都会催生全新的基础设施架构。” 任旭东强调，信用卡催生了消费金融，互联网支付推动了移动商务，而如今的 Token 与微支付技术，正引领着 AI 经济的发展，开源的经济基础已然改变，忽视这一变化的项目，其价值将只能以昨日的货币标准来衡量。

与此同时，软件消费的定义正在被改写，Token 需求迎来爆发式增长。互联网时代，普通用户每月宽带支出约 15 美元；流媒体时代，用户每月仅需 3-5 美元即可享受订阅服务；而在 Token 经济时代，单个开发者的编程套餐月费高达约 150 美元，顶级平台的编程套餐往往数秒内便售罄。Token 已成为新一代开发者不可或缺的生产工具，软件消费从 “购买使用权”，升级为 “使用权 + 智能使用权”（模型调用、计算资源使用、Agent 协调的权利）的双重需求。

任旭东援引杰文斯悖论指出，单个 Token 的成本正在快速下降，但这并不意味着 Token 总支出会减少，反而会推动总需求的激增，如同蒸汽机提升了煤炭利用效率，却最终推动煤炭总消耗量千倍增长，这一规律将在 Token 经济中重演。

在此基础上，开源的价值正呈现出显著的“剪刀差” 现象：一边价值逐渐趋近于零，另一边则不断攀升、没有上限。

*   趋近于零的一侧，是 AI 已能高效完成的工作，包括依赖关系升级、格式修正、常见问题解答、重复性维护、模板化代码生成、文本翻译与转录等，聚焦于这些领域的贡献，其经济价值正被 AI 逐步取代；
    
*   无限攀升的一侧，是 AI 在很长一段时间内都无法实现的能力，包括架构定义、价值对齐、安全审计、跨领域创新以及伦理决策，这些能力的核心是品味、判断力、设计思维与道德准则，也是未来开源生态最核心的价值锚点。
    

“未来五年中能够存活下来的维护者，并非是打字最多的那位；而是当系统管理员在凌晨三点提出修改建议时，社区最信任其判断力的人。” 任旭东直言，开源技术并未消亡，只是经历了重新定价，新的定价机制对重复执行者并不友好，却对架构师极为优厚。

针对行业对 Token 的诸多误解，他也提出了核心观点：Token 并非货币，而是一种新的计量单位。如同 19 世纪 80 年代千瓦时的发明，同时承担了成本计量与价值计量的双重职能，让电力产业成为可能；如今的 Token，同时承担着 GPU 运行时间的成本计量与智能输出的价值计量职能，服务于整个智能应用产业。

他进一步预判，行业定价基准将迎来三次迁移：2024-2026 年，核心基准是 GPU/NPU/TPU等，对应材料稀缺性；2027-2029 年，核心议题转向能源，对应物质资源稀缺性；2030 年后，核心竞争力将回归人才，对应智力资源的稀缺性，而品味、判断力、设计能力与道德素养，将成为最核心的稀缺资源。

华为的开源实践：做Agent时代的共建者，而非参与者

在完整阐述 AI 时代开源范式的变革逻辑后，任旭东也具体分享了华为基于这一行业趋势的开源实践，强调华为并非全球开源生态的 “参与者”，而是 “共同建设者”。

在开源治理层面，华为已深度融入全球开源生态体系：已加入 20 家以上的国际基金会，深度参与 Agentic AI 基金会与 LF AI & Data 两大 AI 时代核心生态枢纽，持续投入人力、代码与资金；同时积极推动 MCP（Model Context Protocol）、agents.md 等通用协议的落地与采用。

任旭东表示，Agent 时代需要一个全球性的数字宪章，它不能成为任何一家公司的专属标准，也不能成为单一国家的国家标准，这正是华为选择基金会模式的核心原因。

在技术架构层面，华为打造了自下而上的全栈 AI 开源开放架构，实现从芯片到 Agent 的全链路开源开放。

作为华为 Agent 时代技术布局的核心，任旭东重点介绍了 openJiuwen ，以Agent 应用负载为核心服务对象，南向亲和模型和算力基础设施，北向使能各类智能体，其生态定位即为AgentOS，并提出三大设计理念：Skill as New Library，让 Agent 开发者可复用已沉淀的智能能力；Agent as New Service，让智能体实现分布式协同；CLI as New POSIX， 定义了智能体时代全新系统原语，以此为指引构建了完整的总体架构，分为Agent System Service层、Agent Distributed Runtime层、Agent Framework层。

4月底首发“Coordination Engineering”（协同工程）全栈技术体系，让智能体从“单兵作战”到“精锐团队”。目前已完成三项生产级部署：华为云 OfficeClaw，企业级 Claw 应用，覆盖各类办公场景，为企业员工打造具备特定人设和专业 SKil技能的 Agent 专家团队，助理企业办公效率跃升；中国邮政储蓄银行“龙虾PSBC-Claw”，以自主可控/授权可信/安全可靠/集中管控/多模态/高并发等为核心优势，在大模型集群的支撑下，将重塑金融智能服务的新范式；中国科学技术大学“灵境造物”智能科研工具，标志着“人工智能驱动的科学研究”正走向工程化、平台化和开放共享。

此外，任旭东还分享了 Asend 芯片、DeekSeek 模型与 AtomGit 托管平台的三重开放式创新循环，“在智能时代，没有任何单一层面能够单独取胜，只有所有层面都实现开放，这个闭环才能真正闭合。”

另外，在演讲中任旭东也直面 AI 时代行业面临的核心挑战，以及传统组织与企业的转型困境。

他指出，二十世纪最伟大的发明之一“流程”，在当下正成为企业最重的 “增长税”。碳时代的流程设计，核心是减少混乱、错误与返工，将专家经验固化为标准化步骤；而硅时代，行业最核心的风险不再是出错，而是错失机遇，僵化的流程、层级架构、偏离的 KPI 与冗长的会议，正在吞噬企业的创造力与发展速度。

任旭东表示：“昨日最优的流程最大限度地减少了错误，而当今最优的流程则必须缩短收敛时间。” 

这一困境也直接推动了行业人才流动的剧变。过去六个月，多家知名企业的 CTO 纷纷离开高管岗位，以独立贡献者的身份加入 Anthropic 等前沿 AI 实验室。在任旭东看来，这并非 “降级”，而是行业杠杆的倒置：碳经济时代，CTO 的权力取决于管理的人数、应对博弈的能力；而硅经济时代，权力取决于与模型的契合程度，一位资深工程师搭配前沿模型，产出可超过百人团队。顶尖人才的选择，已从 “晋升管理更多人”，变为 “迁往模型所在之处，把握关键杠杆”。

面对全行业的系统性变革，以及企业、组织、价值体系的全面重构，任旭东在峰会现场向全球行业参与者提出了全球协作倡议——“数字 20（Digital 20，简称 D20）”。

他表示，正如 G20 成为全球协调工业与金融范式的核心平台，行业也需要一个 “数字 20”，来协调全新的数字文明范式，汇聚政府、标准组织、开源基金会、高校、社会学家、经济学家、人类学家、非政府组织与企业的跨领域领导者，共同解答四大核心问题：

*   当一个人雇佣上百名 Agent 时，组织应如何进行重组？
    
*   碳经济时代的流程、KPI 与会议，应如何适应 “硅时代” 的节奏？
    
*   在代币经济中，贡献、知识产权与收入应如何定义？
    
*   在 AI 驱动的协作模式下，人类价值观应如何实现协调与平衡？
    

“数字 20” ，是制定人工智能时代的数字宪法。任旭东用一句话点明了这一倡议的核心内核：“当代码主宰世界时，世界也必须主宰代码。”

演讲的最后，任旭东呼吁全球开源从业者共同参与这场变革。他表示，在当下的时代，代码已成为新的语言，开放代码的边界，决定了人类共享世界的边界。“文明的下一部宪法将是代码 —— 而代码必须是开放的。碳纪元的时代假设已经过时，硅纪元的首个标志性特征仍有待书写。我们正在共同构建一个共享命运的数字共同体—— 由人工智能引领、人类参与仲裁、全球共同治理，并建立在开放基础之上。”
