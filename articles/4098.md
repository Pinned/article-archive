# 第二家2万亿AI独角兽即将诞生！微软英伟达红杉押注

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgxrpgnvRNm7wJ2N6XB72icia5mTMTQR0BMlTPBdrSMhuF959SHMFIz1kAtRtUK9YWX0ZA6zAMI5F4A/640?from=appmsg)

**本轮融资将在未来几周内完成。**

**作者 |**  陈骏达  

**编辑 |**  李水青  

智东西1月19日消息，昨天，据英国《金融时报》报道，美国AI超级独角兽Anthropic正在进行一轮总额超过**250亿美元（约合人民币1742.1亿元）**的融资。本轮融资预计将使其估值提升至约3500亿美元（约合人民币24389.0亿元），较去年9月的1700亿美元（约合人民币11846.1亿元）估值实现翻倍。

此轮融资的重要部分将来自**[微软与英伟达此前已宣布的投资](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652792813&idx=2&sn=cfc047e23f3fb2baa5962a4951dd158a&scene=21#wechat_redirect)，**总融资规模则比上次宣布时高了100亿美元。去年11月，微软分别承诺向Anthropic投资**50亿美元（约合人民币348.4亿元）和100亿美元（约合人民币696.8亿元）**。

此外，美国知名风险投资机构红杉资本计划首次投资Anthropic，并且很有可能会“下重注”，与新加坡主权财富基金GIC和美国投资机构Coatue领投Anthropic的此轮融资。

红杉此时入局Anthropic，代表着其在AI领域投资逻辑的变化。此前，风投并不会同时投资同一领域的竞争对手，但是AI领域的巨大财务机会正在改变这一做法。

此轮融资中，风险投资机构及其他投资者将向Anthropic投入**至少100亿美元**。其中，GIC、Coatue两家将会分别投入15亿美元，红杉的投资规模尚未披露。

目前，这笔融资的谈判仍在进行中，最终金额仍可能发生变化。Anthropic尚未决定引入哪些投资者，**但预计本轮融资将在未来几周内完成。**

Anthropic进行最新一轮融资之际，也迎来一段高速增长期，**其年化收入（ARR）已从一年前的10亿美元，增长至如今的100亿美元**。

过去1年中，Anthropic在模型和产品侧都做出了广受认可的成绩。

Anthropic的Claude系列模型持续迭代，由3.7版本升级至最新的4.5版本，编程能力、Agent能力都有明显提升，能在生产力场景更为**端到端、长时间**地完成复杂任务。

在其专精的编程领域，Claude虽然并不总是位居榜单第一，但在开发者群体里的口碑较好。

产品侧，Anthropic做出了**爆款产品Claude Code**。2025年2月，Claude Code随着Claude 3.7 Sonnet一同发布，此后迅速成为开发者圈内的爆款。

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgxrpgnvRNm7wJ2N6XB72iciaw4A1icicbO9CxLoia5PqibliaEjHSZdKCCWV9p6iaxVuicPYPEMnicePAThjCg/640?from=appmsg)

作为一款能自动执行工程任务、读写代码库、运行测试的AI编程助手，Claude Code采用了开发者熟悉的命令行界面，还在上下文持久化、Agent能力上持续进步，体验不断优化。

今年1月，Anthropic还推出了Claude Code的图形化UI版本——**Cowork**。如今，更多非技术背景的人员也可以使用到Claude Code的Agent能力，完成整理文件、抓取网页等非编程类任务。

此外，Anthropic也试图定义Agent这一赛道的开放生态。2025年10月，Anthropic推出了**Agent Skills**概念，并在同年12月将其作为开放标准正式发布，试图让智能体具备类似“技能包”的模块能力，这些技能可以被封装、复用、共享和组合。

Agent Skills本质上是由指令、资源和可选脚本组成的文件夹式结构，Claude会根据当前任务自动识别并按需加载相关技能，从而实现可重复、可组合的专业化任务执行能力，而不是每次都靠提示词驱动。

**Anthropic正在为IPO做准备**，最早可能在今年进行。Anthropic已经聘请律师事务所开展前期准备工作，并与多家投行就上市事宜进行了初步接触。

  

**结语：Anthropic差异化路线获市场认可**

从本轮融资的规模、参与方阵容以及估值跃升幅度来看，资本市场显然认可Anthropic的战略价值。

与OpenAI更强调通用对话、C端流量的路径不同，Anthropic的重心更多放在to B市场，在企业级生产力工具、编程与代码生成、安全可控的模型能力等方向。

100亿美元的年化收入，也从侧面印证了这一选择的商业可行性。由此可见，AI并不只是爆款产品的竞赛，也可以是嵌入企业工作流、提升效率、创造长期价值的较量。
