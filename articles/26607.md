Claude Code迎来重大升级！

**@一下，Claude就成了你同事**

  

Claude Tag的新，不在它更聪明，在它换了一个「身份」。

  

你不用打开任何新窗口，不用切到终端，不用离开你和团队天天泡着的Slack。

  

在那个频道里敲一个 @Claude，它就出现了——不是来接话的，是来干活的。

  

![](https://mmbiz.qpic.cn/mmbiz_gif/Rvq8Ow69CYWicDNPp0ea5Ayibyt7AX83ZXIIia6uV6NWBxqhH9mXCO5fkOJzGFUSGibyG0ubLatQaAPvAVf7l0n2CzumrS37dgNibQY8JiczCYF44/640?wx_fmt=gif&from=appmsg)

  

它带着整个组织的上下文，认得你频道里的同事，知道你们在推进哪个项目，能调起公司批给它用的工具：

  

文档、数据库、代码仓库、第三方服务。它不再从一张白纸开始听你说话。

  

最关键的是它自己动手。

  

你 @ 它一句「昨天那个bug修了，PR开好」，它真的去读代码、定位、改、跑测试、提合并请求。期间不需要你盯。

  

Anthropic给这个身份起了个名字——「共享身份」（shared identity）。

  

同一个Claude Tag，跨频道、跨工具、带着记忆，连贯地待在你的组织里。

  

你今天 @ 它的，和明天 @ 它的，它都记得是同一个团队、同一件事。

  

![](https://mmbiz.qpic.cn/mmbiz_gif/Rvq8Ow69CYUZsicxicmIvtfrhemZmm4BPnyhHs6aTdhZUQahxWjsaTMCOvFR2rpx2lRrMqX52JM7iaJueO647tQuv9eXkySWl5xTWb6Xr2eg9A/640?wx_fmt=gif&from=appmsg)

  

这是AI第一次坐到人和人之间那张虚拟的办公桌旁，而不是被关在某个对话框里等召唤。

  

跑在底下的是Claude Opus 4.8，Anthropic今年5月底发布的旗舰模型。

  

Claude Tag目前以beta形式面向Team和Enterprise工作区开放，免费内置于Claude Cowork，没有额外标价。

  

旧的Claude Code in Slack，已经被它取代。

**前两代AI，都得你亲自去请**

  

说它新，得有旧的做参照。在Claude Tag之前，AI和人打交道，有过两代。

  

第一代是对话框。

  

ChatGPT爆红那两年，所有人跟AI的交道都一样：打开网页，开个新对话，把问题喂进去，等它吐答案。

  

第二代是命令行里的agent。

  

Claude Code把盒子搬进了你的电脑，它终于不只动嘴，还能动手——读写代码、跑测试、提PR。但你仍得切到它的环境，主动派活儿。

  

两代的共同点就一句话：是人去找AI。

  

它不认识你的同事，看不到你挤满消息的Slack频道，更记不住你昨天干到哪。

  

它很强，但它是座孤岛——你得离开自己的工作流，走进它的世界。

  

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**Karpathy点破的底层工程**

  

正如Karpathy所说，想要让它跑起来，你就得在台面下做掉所有那些工程活儿——跨工具、跨集成、跨算力环境、跨记忆。

  

光 @ 一下让Claude去改一段代码，它就得先知道去哪个仓库、用什么权限、怎么连上你的CI、改完怎么测、PR往哪开、谁来审。

  

这些每一步，过去都是一个工程师坐在工位上手搓的事。Claude Tag要把它们全部接通，封装成一个「叮」的瞬间。

  

Karpathy敢用「范式」这个词，正是因为这些台面下的工程拼到一起，量变引起了质变。

  

@ 一下就有一个能干的同事出现——这事成立，才配叫新东西。不然它只是Slack里多了一个更聪明的bot。

  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**Anthropic自己已经离不开它**

  

最能给Claude Tag作证的，是Anthropic自己。

  

官方的原话是：「@ Claude现在是我们办成事的主要方式之一。我们产品团队65%的代码，来自内部版的Claude Tag。」

  

65%，是这家公司每天真金白银往生产环境里提交的代码里，将近三分之二来自一个AI队友。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYVZia1swxkDOABRSeZTIp8VG2O3V7Og43dVysPicyCpaazwWz4mJZm9ocXanU4LfyervxvdyzSZvYlRD1Ux1zgnfHA7T1Tz6ae9o/640?from=appmsg)

  

厉害的地方有两层。

  

第一层：它证明Claude Tag扛得住真实工程的重活，不是能发推的玩具。

  

能在Anthropic这种体量、这种代码质量要求的公司里挑起65% 的大梁，说明agent已经跨过了「能用」到「主力」的门槛。

  

第二层：造它的人，自己已经离不开它了。

  

如今，Anthropic把自己的工程团队变成了Claude Tag的头号用户和头号试验场。

  

某种意义上，Claude Tag是用Anthropic自己的血肉长出来的。

  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**三家在赌不同的未来**

  

2026年这场agent之争，三家头部公司都在下注，赌的却不是同一个未来。

OpenAI的牌叫Workspace Agents，今年4月上线，走无代码路线。不用懂技术，拖拖拽拽就能搭一个接Slack、Salesforce、HubSpot的智能体。

它赌的是「把门槛打到最低，让会点点鼠标的人都能造agent」，借着ChatGPT近10亿用户的盘子往下灌。

谷歌走的是「生态内嵌」。Gemini直接长进Workspace全家桶，Gmail、日历、Drive、Docs里到处都是它。

它押的不是某个产品，而是几十年攒下的数据和分发优势——你的邮件、日程、文件，本来就是喂它最好的养料。

Anthropic的Claude Tag，押的是「嵌入式队友」。它不抢办公软件的饭碗，而是把自己变成那个能在所有工具间穿梭、把活儿串起来的同事，还格外强调安全叙事。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYWWDEYwM7DCre9rQdN7PzofJiaVeMCtDe8vdwqmQWJbPibmvC9ZL5NPMvRIBc9YrJYiaEgW3HEZQgmXU7m6B90WcAOot8Vzda7AWE/640?from=appmsg)

  

虽然打法不同，但三家指向同一句结论，而且这句已经没人在争——

AI不再是你用一下的工具，是要在你工作里长期占一个位置的人。

  

谁是同事、谁是工具，这条分界正在被重新书写。
