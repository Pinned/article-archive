**泄露**

  

本月，瑞士黑客 maia arson crimew 收到匿名线报，Dialog 官网 dialog.org 的页面源代码中藏着一个完整的数据库目录。

  

这位曾曝光美国政府禁飞名单的黑客活动人士将发现转交给 WIRED，后者独立核实了内容。

  

泄露的数据存储在商业数据库 Airtable 中，包含近 200 份详细档案，记录了住址、私人电话、出生日期、紧急联系人、食物过敏信息，以及成员自愿填写的政治倾向。

  

一份独立获取的注册名单进一步列出 222 名 2026 年聚会报名者的姓名、会员状态和参会历史。聚会定于 8 月 12-16 日，地点在爱尔兰都柏林附近。

  

Dialog 花了 20 年拒绝公开任何成员信息。

  

但它的登录页面 app.dialog.org 没有使用条款，没有标注「仅限会员」，甚至没有提示需要邀请，任何人都可以用邮箱或 Google 账号尝试登录。

  

20 年的沉默，被一次最基础的安全疏忽击穿。

**圈子**

  

Dialog 由亿万富翁科技投资人 Peter Thiel 和数据经纪商 Auren Hoffman 于 2006 年联合创办，自称「跨党派邀请制社区」。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUtlu59Kqu7rF4YEtfDZGo9dGfZicvNicSomajaDXU6srrR44hhaCqo8iae3lH2XibJ3ALA63TuhgQvCNZBW2GuyEaPhXlGNN9HJqI/640?from=appmsg)

根据一份由过往参与者提供给 WIRED 的内部文件，该组织拥有超过 1000 名付费会员，累计 2500 余人参加过年度聚会。

  

Dialog 每年至少搞一次闭门聚会，地点选在亚利桑那丽思卡尔顿鸽山度假村（Dove Mountain）、加州圣芭芭拉丽思卡尔顿酒店、威尼斯圣克莱门特宫（San Clemente Palace）这类地方，注册费超过 16000 美元。

议题横跨一整个光谱，从「重启核能」「应对第三次世界大战」「战场技术」，到「你的性生活怎么样？」和「如何建立邪教」，后者由基督教社交平台 Pray.com 创始人主持。

  

泄露的名单覆盖了美国权力结构的几乎每一个关键节点。

**评分**

  

比成员名单更具爆炸性的，是 Dialog 对这些成员做的事。

  

WIRED 从一个独立信源获取的内部数据显示，Dialog 在每个人加入之前就给他们打分。

  

**等级分 A、B、C 三档，C 是最高，留给最有名最有钱的人**，192 份档案中只有约七分之一拿到；141 人拿到 B；A 级主要分配给年纪较大、知名度已下降的老会员。

  

**评分标准几乎只围绕两个变量，财富和名气。**

  

Dialog 使用 AI 工具为至少 26 人编写档案，算法反复衡量同一个问题，「普通人」能不能认出这个人。

  

演员 Josh Brolin 从未参加过 Dialog 任何一次聚会，但档案将他列为 VIP，依据是出演灭霸的《复仇者联盟》系列（其中《复仇者联盟 4》全球票房 27.9 亿美元），以及他 340 万的 Instagram 粉丝。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUGRrBRKUv9T55ZHT51ibcCjn5sQgsCPXXyskqr7v0GB1mZxs28ibyEkQ1eXwcUjibHdsgO6NKbzjtUHyR1lJEQvVJW690740PkkE/640?from=appmsg)

  

Brolin 的经纪人后来告诉《好莱坞报道》，他想知道「自己到底被卷进了什么鬼东西（what the fuck he got himself into）」。

  

经济学家 Tyler Cowen 被 AI 工具拒绝授予 VIP，理由是他虽然「在自己的领域广受认可」，但他领导的机构「对普通人来说不算家喻户晓」，工作人员随后手动覆盖了 AI 的判断。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYUvqGUWNo8UmIWG8m4rl4apqUiaB95VjWeCTNvICQVriam3Ny2pibiaeciaAib5oMIHia4r2YWH2picyogKQrc0xnVUSdDxZP6T8k0AGDY/640?from=appmsg)

  

曼哈顿研究所所长 Reihan Salam 拿到 B，原因类似，「曼哈顿研究所对普通人来说不如财富 500 强公司知名」。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUJ718TZOrTK4A9sLwNwj2kzeWcS8eOxKwiaFicKC68BLiaZn5D16qmC7VNq97qz8QmLOENRz6L1qrBChhPT5hwITyU3K0Tl8yceE/640?from=appmsg)

  

一位投资人的档案用他管理的 300 亿美元资产作为评分依据；另一位投资人被两个单词打发，「Small AUM」，管理规模太小。

  

**等级直接决定三件事。**

  

**座位。**

  

一条工作人员笔记写道，「给她打这个分，这样她就不会被安排跟 C 级坐在一起」。

  

**收费。**

  

低等级成员约 70% 的概率被安排全价，VIP 只有约四分之一。

  

工作人员手动定价时也会纠结，一位畅销书作者的费用被拒绝上调，理由是「不能仅仅因为她男朋友有钱就涨价」。

  

**去留。**

  

成员可因「附加价值太低」「文化不匹配」「等级下降」被踢出局。

  

一位量子计算创业者在一次聚会后被标记清退，「没有足够的影响力，附加价值不够高」。

  

每次聚会结束，工作人员会进行内部「代码评审」（post-retreat code review），重新审视每个人的等级。

  

在这台机器里，每个人都是一个待估值的资产，每年至少重新定价一次。

  

性别数据同样刺眼。

  

女性占被评分者的约三分之一，但只拿到 18% 的顶级评分。

  

Dialog 收集的远不止评分。

  

注册表要求成员填写政治倾向，组织承诺这些信息「绝不会与其他参与者分享」。

  

泄露让这一承诺化为废纸。

  

工作人员在成员自报之外另有一套独立评估，11 名从未披露政治立场的成员被主动贴上标签，15 名成员的自我描述被覆盖，其中一位全球最大保育组织负责人自称左翼，被工作人员标为右翼。

  

在 165 名披露政治立场的成员中，超过一半认同左翼，但右翼成员获得 C 级评价的概率是左翼的两倍多。

  

一个自称「非党派」的组织，在分配最高等级时，对右翼表现出明显的倾斜。

  

**评分之外，Dialog 还运营着一套匹配系统。**

  

dating.dialog.org 打出的口号是「为杰出人士打造有意义的连接」。

  

**约 10% 的成员加入了单身匹配池**，算法推荐候选人后由工作人员人工微调，配对理由具体到「你们都在纽约，都在政府工作」。

  

数据库中还有一份「禁止配对」名单，标注了前配偶、商业伙伴、家族关系，以及一些没有给出任何理由的禁配组合。

  

名单中最大的一组，是成员不得与 Dialog 自己的工作人员配对。

**表演**

  

Dialog 的一份内部文件中有这样一段自我描述：

  

我们没有意识形态议程。

  

Dialog 是非党派、非政治的。

  

我们相信，当各领域最优秀、最开放的人聚在一起，他们会学到新东西。

  

**泄露的数据讲了一个截然不同的故事。**

  

一个自称「非政治」的组织，内部按政治倾向给成员贴标签，覆盖他们的自报结果，且右倾者拿到最高评级的概率远高于左倾者。

  

一个追求「最优秀、最开放」的社区，用 Instagram 粉丝数和资产管理规模来判断一个人是否「值得认识」，学术声望不如票房数字管用。

  

一个号称让成员「学到新东西」的平台，用 AI 给人建档打分，而这个 AI 反复追问的唯一问题是，「普通人认不认识你？」

  

Dialog 成员在注册表上预测未来时，反复指向同一件事，AI 将在几年内重塑工作、战争、教育和信仰。

  

有人写下「社会退化将继续加速」。

  

他们用来给彼此打分的那个 AI 工具，倒是率先演示了一种退化的样本，当判断一个人是否值得交往的全部标准，被压缩进财富和名气两个数字。

  

Dialog 的主持人指南要求参会者自我介绍时「避免地位信号」。

  

但这整个组织，从 ABC 评级到按身价排座到按名气收费到按「附加价值」决定去留，就是一台精密运转的地位量化机器。

  

它精密到会纠结「她男朋友有钱」算不算她自己的等级。

  

这台机器唯一没有精密处理的，是自己的网站安全。

  

也或许，20 年来他们从未觉得这是个问题。

  

毕竟在这个圈子里，谁会去看网页源代码呢？

  

Dialog 的全部故事最终指向一个简单的事实，在这个房间里，**外界看起来水火不容的左翼参议员和右翼亿万富翁之间的距离，远没有他们与门外所有人之间的距离大。**
