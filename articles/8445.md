# 智赋算力 领航AI——上海人工智能实验室发布《超节点技术体系白皮书》

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acuo5rRdUJLic5Gf3ckOY2hq1apRelRtmqJv6w7kM2U8xXmAnHubgZlKcAsBVbd0svrYXKrBibNfgEYOE8xp1bk6jxJfxicahkQUEuc/640?from=appmsg)

**为超节点规模化落地解决异构协同难、跨域调度效率低等核心痛点。**

2026年3月29日，由上海人工智能实验室（上海AI实验室）发起并主办的第二届浦江AI学术年会开幕。浦江AI学术年会期间，**由上海AI实验室****DeepLink****团队发布了《超节点技术体系白皮书》（以下简称****“****白皮书****”****），旨在为超节点的规模化落地，解决异构协同难、跨域调度效率低、工程化部署复杂等核心痛点，从产业实践侧提供了理论方向上的指导。**

DeepLink团队联合8所顶尖高校、16家核心产业伙伴达成了阶段性产业共识，覆盖芯片研发、芯片产业链、算力部署、软件适配、学术研究、行业应用等全产业链环节，实现了“全行业发声、全维度共建”。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuovXLBibCjmCibvxKACfRJsRDSUpIcRcphATDFOtSNV6E73zCyo9g0DLGR1E4pG6ptEwUibuxb3DnSicPKWjFVthASJfHyn51P9a3A/640?from=appmsg)

  

**_01_****_._**

**超节点之后：算力竞争**

**从芯片级转向系统级**

2025年，Deepseek、华为CloudMatrix 384成为业内热点，**超节点成为行业争相研究的技术焦点，超高带宽、超低延迟的芯片间互联，将数十甚至上百颗计算芯片构建成一个逻辑上高度协同的****“****超级计算单元****”****。**在芯片制程工艺逼近物理极限的今天，超节点技术提供了突破算力瓶颈的新路径。对正处于人工智能发展关键期的我国而言，超节点的战略价值不言而喻。

在此背景下，众多厂商都陆续推出了超节点方案。中科曙光推出单机柜640卡scaleX640he和全球首款无线缆箱式超节点scaleX40；沐曦科技推出耀龙3D Mesh超节点、Shanghai Cube等；阿里云发布磐久 AI Infra2.0 AL128 超节点服务器；新华三集团推出H3C UniPoD S80000超节点产品。

超节点的出现，也对算力行业的发展有了新的启示。**算力竞争从单一变量（芯片）变成了多变量协同****——****互联、软件、整机、****RAS** **共同决定系统能交付多少有效算力。**如今大模型的算力需求被三条指数曲线同时驱动——训练侧的 Scaling Law、推理侧的 test-time scaling、以及 Agent 应用带来的调用量爆发。这些需求叠加起来，远不是单芯片能力提升能独自承接的，真正决定格局的越来越落在互联、整机、软件和系统协同上。

  

**_02_****_._**

**产品百花齐放**

**一场关于技术生态的竞赛**

国内厂商们在超节点市场上演的“竞赛”，除了是产品层面的竞争，但深究其本质，也是一场关于技术生态主导权的战略博弈。**超节点正处于技术路线尚未收敛的窗口期，谁能率先构建完善的软硬件生态，谁就有可能成为事实标准。而且超节点不是孤立的硬件产品，而是****“****芯片****+****互联****+****软件****”****的系统级方案。**客户一旦采用某家的超节点，从上层应用到底层调度都会深度绑定，形成极强的客户粘性。这种“锁定效应”意味着，谁先拿下大规模部署，谁就能在商业上建立长期优势。

倘若各家的软件栈无法兼容，开发者就需要为不同平台重复开发。这不仅增加了开发成本，也阻碍了AI应用的跨平台部署和生态共享，不利于形成统一的国产AI软件生态。

多家厂商不约而同走向超节点，说明产业界已经意识到：单芯片增速撑不住系统需求增速，中间出现了结构性缺口。这个缺口必须靠全产业链协同来弥合，任何一家独自闭环都会面临系统性风险——技术栈越封闭，生态越脆弱，代际迭代的负担越重。主要体现在以下三个方面：首先是访存语义缺失，互联层如果不能原生支持 Load/Store 语义，软件生态就无法跨代积累；其次是整机工程约束，带宽域扩展到机柜级后，互联距离、供电、液冷、可维护性深度耦合，需要芯片侧和整机侧协同设计；最后则是缺少行业共认的评价框架，各家口径不同无法横向比较。

**_03_****_._**

**深度协同共创**

**打造高质量行业指南**

**上海****AI****实验室****DeepLink****负责人王辉在浦江****AI****学术年会****AGI4S Infra****分论坛中提出，****“****国内产业界在各方向已有大量先进探索，但相对分散，缺少一个跨路线的公共分析框架。亟需把分散的探索连接起来，形成可以共同推进的技术底座。****”**

正是基于上述判断，上海AI实验室联合AI产业伙伴，共同撰写并发布了《超节点技术体系白皮书》。据了解，白皮书采用“学术主导编辑、产业分域贡献”机制。框架设计和审校由实验室与国内优势高校承担；沐曦科技、天数智芯和曦望科技等在芯片方向提供架构数据，奇异摩尔、盛科通信等贡献互联协议经验，阶跃星辰等提供大模型训练的系统需求反馈。所有素材经编委会审校纳入，确保白皮书是行业公共知识产品，不是某一家的技术主张。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuqPhE3n2F5Xo7YAqeibZaz8kAA65srSVjebTYStFFmHH2N7Zv4980f6reoTUw8e9icbbnJzBsZLS7qcMfOriaaibT6g6ZBl28lWibRA/640?from=appmsg)

**奇异摩尔作为国内****AI****网络互联领域的创新力量，首创了****Kiwi G2G IOD** **通用****IO****芯粒方案，专为超节点网络设计，用于应对****Scale Up****互联挑战。**

Kiwi G2G IOD支持2TB级带宽与高并发数据传输，给大模型的训练和推理提供持续稳定的高效互联支撑；支持多网络拓扑，可有效应对拓展或任务切换时重新设计网络的需求。奇异摩尔深度参与了白皮书的撰写与研讨，将自身在超节点网络互联工程的实践经验，以超高浓度干货素材的形式，携手上海AI实验室，共同编撰形成具有行业指导意义的《超节点技术体系白皮书》，为超节点和算力行业生态高质量发展注入产研协同新动能。

**奇异摩尔联合创始人****&****产品及解决方案副总裁祝俊东认为，《超节点技术体系白皮书》从根源出发，系统回答了我们为何要做超节点、为何设定这些指标、以及应当如何落地实施。它将工程实践与产业经验提炼升华，删繁就简，最终凝结为系统的产业理论。**

总体而言，它不仅是对超节点发展历程的总结，也与当下产业发展紧密融合，提出了富有洞察力的观点和理论，对行业未来具有重要的指导意义。与其说它是一册白皮书，不如说它是一部凝结了产业经验与理论思考的著作。

国产超节点生态的建设，是一场持久战，在长期演进的过程中，需要始终保有走向收敛的共识和耐心。**围绕超节点，上海****AI****实验室的规划沿着****"****认知****—****评价****—****验证****—****标准****"****四步展开。**

白皮书解决的是第一步：建立系统级的分析框架和共同语言。接下来，我们会推进超节点能力指数的落地，把白皮书提出的多维评价框架变成可度量、可比较的评测方法，组织行业进行试评估。在此基础上，建设仿真与建模平台，为不同技术路线提供系统级的量化比较和瓶颈定位能力，降低产业试错成本。更长期地看，我们希望以白皮书和超节点能力指数为基础，逐步沉淀出一组关键接口标准和协同规范，形成可持续推进的标准簇。整个过程保持开放共建，白皮书本身就在 GitHub 上开源维护，后续的评测和标准工作也会邀请更多高校和产业伙伴参与。核心目标不是给出一个统一答案，而是建设一套行业可以持续使用的公共基础设施“规范”。

  

**_04_****_._**

**破解算力碎片化难题**

**构建全国算力“一盘棋”**

值得一提的是，除撰写发布《超节点技术体系白皮书》，从行业理论侧推动国内超节点生态发展外，**上海AI实验室推出的****DeepLink****混训混推平台，正以其开放的技术体系和创新的架构设计，成为推动国内超节点行业生态建设与发展的关键力量。**

2025年，上海AI实验室发布DeepLink超大规模跨域混训技术方案，成功攻克超大规模跨域异构集群调度、高性能通信协议整合、高可靠容错机制设计等技术难题。而随着今年DeepLink混推方案的推出，**上海****AI****实验室逐步构建起混合****“****训练****+****推理****”****全方位算力赋能体系，为算力资源的高效调度、多元协同与生态融通筑牢技术底座。**在计算资源有限的场景下，赋能行业运用不同性能芯片的灵活组合，而无需依赖高算力芯片“扎堆”式部署，有效降低对特定硬件的路径依赖。DeepLink平台加速了国产异构芯片的大规模应用，基于繁荣的自主算力生态为更多企业拥抱“人工智能+”创造机会，为我国打造新质生产力增添新动能。

**在技术层面，****DeepLink****为超节点提供了软件层面的****“****操作系统****”**。通过统一通信协议和高效调度算法，DeepLink能够充分挖掘硬件互联的潜力。**在生态层面**，DeepLink通过标准化的算子接口和连接协议，实现了**“****一次适配，全生态可用****”**的目标。上下游厂商基于DeepLink一次适配即可接入广泛的算法生态，大幅缩短了从芯片流片到实际应用的落地周期。**在标准层面**，DeepLink的实践验证了跨域混训混推的实用价值，也为行业提供了全新启示。在计算资源有限的场景下，行业依旧可以实现不同性能芯片的灵活组合，**无需依赖高性能算力芯片****“****扎堆****”****式部署。**

超节点生态的建设，是推动我国AI算力自主可控、实现超智融合发展的关键举措，也是应对全球算力竞争的重要支撑。未来，上海AI实验室将持续深化技术创新，完善DeepLink技术体系，拓展跨域算力网络覆盖范围，推动“训练+推理”及超智融合全方位算力赋能体系升级，为盘活算力枢纽节点效能、优化全国算力资源配置提供技术及经验支撑，同时为我国数字经济和科技发展注入新的强劲动能。
