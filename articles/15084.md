# Claude 接入 Blender、Adobe、Ableton... Anthropic 一次放 9 个创作连接器

PRODUCT

  

今天，Anthropic 公布 Claude for Creative Work，然后放出 9 个连接器，覆盖 3D、设计、音乐、VJ 四类创作软件。合作方包括 **Blender**、**Autodesk**、**Adobe**、**Ableton**、**Splice**

公告原文：`anthropic.com/news/claude-for-creative-work`

**Claude 把自己，接到了别人家去，通过【连接器】**

这里先说下连接器，也就是 Connectors，可以理解为是第三方维护的 `MCP` 接口，然后被 Claude 认可并塞进去了，有点像远古时期的 ChatGPT Plugin。Blender 自己的 Blender Lab 团队做了官方连接器，Adobe 把 Creative Cloud 接了 **67** 个工具进 Claude

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdgrGHnYqQh4xCiat9yus6fA81AJUVHuicibCsPc8ibwbnNDxzENEqgswmLiaB8wFFnyeLlWbOdIrKibLV5YlBVFSzaWhoTsseOP3ZN88/640?wx_fmt=png&from=appmsg)

  

9 个连接器，分别是什么
------------

我把官方的信息梳理了下：

**Ableton Knowledge**  

8 个工具，本地搜索 Live、Push、Move、Note、Cloud、Link、Max for Live 的官方文档、知识库、视频字幕

**Adobe for creativity**  
**67 个工具**，Photoshop、Lightroom、Illustrator、Firefly、Premiere、Express、InDesign、Stock 全在里面，三大能力是图像编辑/转换、资产创建/动画化、文件搜索组织

**Affinity**  

本地 add-on，要 Claude Desktop 和 Affinity 同时开着才生效，能做批量图片调整、图层重命名，工作流可以保存到 Affinity 自带的 Scripting Panel 复用

**Autodesk Fusion**  

本地连 live session，跟 Claude 对话直接在 Fusion 工作区里看到结果，要在 Preferences 里启用 Fusion MCP Server

**Blender**  

用自然语言操作 Blender 的 Python API，能查文档、能理解复杂场景，要先装一个 add-on（lab.blender.org）

**Resolume Arena MCP Server**  

实时控制 Arena 的 compositions，加载素材、加减效果、增删 layer 和 column，要 Resolume Arena 7.26+

**Resolume Wire MCP Server**  

控制 Wire 的 node-based patches，要 Resolume Wire 7.26+

**Trimble SketchUp**  

3 个工具（build\_model、get\_docs、save\_model），跟 Claude 描述一个房间或建筑，模型在 SketchUp 里打开继续打磨

**Splice**  

5 个工具，里面有一个 Describe a Sound (Beta)，用自然语言搜 Splice 整个免版权采样库

这九个里面，搞 3D 占三家（Blender、Autodesk Fusion、SketchUp），设计占两家（Adobe、Affinity），音乐占两家（Ableton、Splice），VJ 占两家（Resolume Arena、Resolume Wire）

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdhTZp28r3icb83C7SvP4BelibPOw0HInPdovdL2cTsZ11snJ7AwRu5jpibfootcUMhibOzKBZicf0pWO0rJCedYKHiaWWd2oxGPUwvZk/640?from=appmsg)

四家是远程 SaaS 连接器（Adobe、SketchUp、Splice、Resolume），可以直接 Connect。另外几家是本地 MCP，要装额外的 add-on 或者本地 server，对操作软件有版本要求

  

Claude 在创作里能干什么
---------------

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdianZJhUbqnMUUyjCVEb6nqZmSdoh5ibspVe4GuqUxI6wnNjT5LfPojKrkYkKib9E5LjSn2JVKV2RKTzwLjq1V9qCP89fGIVT10ow/640?from=appmsg)

对于运用场景，A社在官方说明里给到了五种典型用法：

**学创作软件**Claude 当随身教练。问一个 modifier stack 是怎么回事、解释一种合成方法、演示一个不熟悉的功能，它把步骤走给你看

**写脚本和插件扩展软件**Claude Code 给你正在用的软件写脚本、插件、生成系统。custom shader、procedural animation、parametric model，产出带文档可复用的代码

**跨软件流水线打通**格式转换、数据重组、设计/3D/音频之间的资产同步，Claude 在中间做翻译，不用手动倒来倒去

**快速试错和交付**这一条直接指向 Claude Design，下面专门讲

**批处理重复活**批量处理素材、搭项目骨架、对场景做程序化修改

第一条对设计学校学生有价值，第二条对 technical artist 和 creative coder 有价值，第三条是真正能省时间的部分

  

Claude × Blender：从工具调用到基金赞助
---------------------------

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdiaGmrecb3ZBowUugkb5KtNwySxvk3aFnm9Blb7e5mzL3iahoLghvyibmSpgpBtPibGcmlURVcCtvRYYKdqCqAwmA3HP4NfoysUy4A/640?from=appmsg)

大名鼎鼎的 Blender 这次也进入了 Connector，这是个免费开源的 3D 创作套件，独立游戏、Motion Graphics、建筑可视化、电影制作都在用

这次 Blender Lab 团队做的 MCP 连接器走的是 Blender 自家的 Python API。Claude 拿到这个接口之后能做三件事：**分析和调试整个 Blender 场景**，**写脚本批量改场景里的对象**，**往 Blender 界面里直接加新工具**

跟连接器一起放出来的还有一件事。**Anthropic 加入 Blender Development Fund，作为赞助方**。今年 1 月 Netflix Animation Studios 刚以 Corporate Patron 身份加入，每年最低 24 万欧元。Anthropic 这次加入的具体级别官方没披露，但这是 Anthropic 第一次给一家开源创作工具基金会签长期赞助

MCP 在创作软件这一领域，越跑越远

连接器是 MCP 协议写的，并非 Claude 独占。对此，Blender 这边的说法是「对开源和互操作性的承诺」

  

Claude Design：另一边走的同步动作
-----------------------

![](https://mmbiz.qpic.cn/mmbiz_jpg/jXSGuwJvpdiasr516Q5Net5dVzz3oyia5yLESoBayTicSaa4cpLEAbiaKfN8eeHhxE7a7GA6YUt1b8WIYZtdysy9EVLZjujJlSFlkXJj0UiaD8uY/640?from=appmsg)

刚出说的连接器，是把 Claude 接到别人家软件里。Claude Design 是另一个方向：**在 Claude 里做出来的东西，能往别人家软件里送**

Claude Design 看名字就知道，也是 Anthropic 搞的，4 月 17 日推出，今天的 Creative Work 公告里再次串起来讲。它由 **Claude Opus 4.7** 驱动，研究预览开放给 Claude Pro、Max、Team、Enterprise 订阅用户，入口在 `claude.ai/design`

能做的事：

**真实可点的原型**把静态稿变成可分享、可点击的交互原型，不走 PR 不走 code review

**产品线框图和 Mockup**PM 画完功能流程交给 Claude Code 做实现，或者交给设计师继续打磨

**设计探索**一次铺多个方向，给设计师试错空间

**演示稿和 Pitch Deck**从粗大纲到符合品牌的完整 deck，导出 PPTX 或者送进 Canva

**营销素材**落地页、社媒物料、campaign 视觉

**前沿设计**把语音、视频、shader、3D、内置 AI 都接进来做原型

它还接 Claude Code 做交付：设计完成后打包成 handoff bundle，一句话指令交给 Claude Code 去实现

Canva CEO Melanie Perkins 在 Claude Design 公告里讲到，从 Claude Design 出来的草稿能直接进 Canva 编辑、协作、发布。Brilliant 的产品设计师 Olivia Xu 给了一个数字：他们的复杂页面在别的工具里要 20 多个 prompt，在 Claude Design 里 **2 个 prompt** 就能复现

  

跟艺术院校的合作
--------

Anthropic 同时宣布跟三所院校的艺术和设计项目合作：罗德岛设计学院（RISD）的 Art and Computation 课程、林林艺术与设计学院（Ringling College of Art and Design）的 Fundamentals of AI for Creatives 课程、伦敦大学金匠学院（Goldsmiths）的 MA/MFA Computational Arts 项目

学生和老师能拿到 Claude 和这批新连接器的访问权，反馈用来给后续连接器迭代。Anthropic 的话术是「之后会扩展到更多机构」

选的这三所学校都是计算与艺术交叉方向有自己课程体系的学校。RISD 是顶尖艺术院校，Ringling 是动画与游戏方向的老牌艺校，Goldsmiths 的 Computational Arts 在欧洲做了二十多年
