# Chrome再出重拳：经典广告拦截器或将走向终结

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

整理 | 苏宓  

出品 | CSDN（ID：CSDNnews）

对于很多 Chrome 用户来说，一个时代可能真的要结束了。

近日，Google 旗下浏览器 Chrome 被曝将在接下来的版本更新中，彻底移除对 Manifest V2 扩展框架的剩余支持。这意味着包括 uBlock Origin 在内的大量经典广告拦截器，将无法继续在新版 Chrome 中运行。 

消息传出后，迅速引发开发者和隐私社区关注。不少用户认为，这实际上是 Google 多年来推动 Manifest V3 计划的最后一步。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)

发生了什么？

简单来说，Google 正在彻底淘汰旧版浏览器扩展规范 Manifest V2（MV2）。

Chrome 扩展运行时都需要遵循一套规范文件（Manifest）。过去十多年，大部分广告拦截器都基于 Manifest V2 开发，因为它允许扩展深度访问和过滤网络请求，从而实现高效的广告屏蔽、反追踪和隐私保护。 

但从 2019 年开始，Google 推出了新一代扩展框架 Manifest V3（MV3），并逐步推动开发者迁移。

经过数次延期后，Google 已在 2024 年启动 MV2 退役计划，而如今最新的 Chromium 代码提交显示，Chrome 即将删除最后残留的 MV2 兼容代码和相关开关选项。 

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEwSqIjV5M86brNgf9aFRic0A5o27na3GNiaaI4C8hASKZB33EDFIkBDc1cZaTGOBdyxEVLVQLFoX7EBkm7MhIdMOiaX7auZeCokoY/640?wx_fmt=png&from=appmsg)

换句话说，以前用户还能通过一些隐藏设置、企业策略或兼容机制继续运行旧版扩展，但这些“后门”也将被彻底关闭。

哪些扩展会受到影响？

受影响最大的，无疑是广告拦截器。

其中最具代表性的就是：

*   uBlock Origin
    
*   部分旧版 AdBlock 扩展
    
*   部分依赖 MV2 的隐私保护工具
    
*   一些高级反追踪扩展
    

这些工具之所以广受欢迎，正是因为它们能够动态分析和拦截网络请求。

而 Manifest V3 对扩展权限进行了严格限制，很多过去可以实时执行的过滤操作，现在必须通过浏览器预设规则完成。 

因此，不少开发者认为，MV3 版本虽然还能拦截广告，但功能灵活性已经明显下降。

以 uBlock Origin 为例，其开发者后来专门推出了 MV3 版本的 uBlock Origin Lite。不过该版本缺少动态过滤、自定义规则、部分高级拦截功能等特性，本质上属于“功能缩水版”。 

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

Google 为什么坚持这么做？

Google 给出的理由主要有三个：

第一是安全性。Google 认为，MV2 允许扩展获得过高权限，存在被恶意扩展利用的风险。

第二是性能问题。旧框架需要扩展频繁介入浏览器网络请求流程，会增加资源消耗。

第三是维护成本。Chromium 工程师 Devlin Cronin 在代码提交讨论中表示，继续保留 MV2 会带来技术债务和额外安全漏洞，团队已经发现多个只存在于 MV2 架构中的 Bug，因此无法无限期维护下去。 

从 Google 的角度来看，MV3 是一次安全和架构升级。但也有人则认为，由于 Google 本身是全球最大的在线广告公司之一，因此限制广告拦截器能力难免会引发利益冲突的质疑。 

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q3SAo8dibFicWvibnB5u4gdHLgb2AbA6UJ8VImKnyW92hibqZefwpDbPYAQ/640?wx_fmt=png)

Chrome 用户现在会受到影响吗？

如果你已经升级到支持 MV3 的广告拦截器，那么影响可能并不大。

真正受到冲击的是仍然坚持使用经典 MV2 扩展的用户。据外媒 9to5google 曝光的时间线：

*   Chrome 150（预计于 2026 年 6 月 30 日发布） 将移除高级用户长期以来用于继续运行 MV2 扩展的主要技术绕过方案——ExtensionManifestV2Disabled。虽然届时仍会保留一种基于开发者工具（DevTools）的有限方法，但用户需要在每次浏览器会话中手动修改页面元素，几乎不具备日常使用价值。
    
*   Chrome 151（预计于 2026 年 7 月发布） 则会进一步移除剩余的相关开关，包括 ExtensionManifestV2Unsupported、ExtensionManifestV2Availability 和 AllowLegacyMV2Extensions。至此，Chrome 中与 MV2 兼容相关的最后遗留机制也将被彻底清除。
    

简单来看，Chrome 150 和 Chrome 151 版本将陆续删除剩余兼容代码，届时旧版扩展将彻底失效。 

那此举是否意味着广告拦截器彻底死亡？答案并非如此。

Manifest V3 并没有禁止广告拦截器存在，而是改变了它们工作的方式。

事实上，近期一项针对 MV2 与 MV3 广告拦截器效果的研究发现，在广告拦截和反追踪测试中，许多 MV3 版本工具与旧版相比并没有出现显著性能下降，部分场景甚至略有提升。 

不过对于重度用户来说，问题不在于“能不能拦广告”，而在于是否还能像过去那样拥有完全自定义、动态过滤和细粒度控制能力。这也是为什么许多隐私社区用户仍然坚持使用原版 uBlock Origin，而不愿迁移到 Lite 版本。 

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0jqAkXB2hEohZkOqYzXVsmJtnNRQncYh54ZQOpcC5ZGSicYFgtBZP4w/640?wx_fmt=png)

接下来会发生什么？

从目前情况来看，Google 已经进入 Manifest V3 迁移的最后阶段。

Chrome 将全面转向 MV3 已成定局，而旧版 Manifest V2 基本宣告退出历史舞台。 

对于普通用户来说，未来依然可以使用广告拦截器，只是选择会越来越集中在符合 MV3 规范的新产品上。

而对于那些依赖原版 uBlock Origin 等经典工具的用户来说，这次更新可能意味着：Chrome 上最强广告拦截时代，真的要结束了。
