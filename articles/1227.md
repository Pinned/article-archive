# C++ 之父亲临现场，2025 全球 C++ 及系统软件技术大会日程抢先看！

![](https://mmbiz.qpic.cn/mmbiz_gif/VeJKXItpwPVibOSMQeyaGwgUuVlcQvxkGXXWJd5ibVobOMbl2fuGGsbm5Ric4aJjgUiahJfqib9xbibDticDldfIziaT8g/640?wx_fmt=gif)

秉承“全球专家、卓越智慧”的理念，由 CSDN 与奇点智能研究院举办的「**2025全球 C++ 及系统软件技术大会**」将于 **12 月 12-13 日**在北京金隅喜来登大酒店正式举办。

届时，C++ 之父、美国国家工程院院士、ACM Fellow、IEEE Fellow Bjarne Stroustrup 将亲临现场，与全球顶尖的系统软件专家、高性能计算领军人物、AI 基础设施架构师及开源社区核心贡献者齐聚一堂，共同探讨 AI 原生时代下的 C++ 与系统软件的演进路径、工程实践与未来范式。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/q7kM1wTg0ejkE1K5uSeCvnn044xI4329Fibo9QiaTZ7E9xXNyfE0hv3ZtLBHr8Qh9EZ2jJvC5xLHq9f4GG85SA0w/640?wx_fmt=png&from=appmsg&wxfrom=5&wx_lazy=1&tp=webp#imgIndex=8)

官网：https://cpp-summit.org/

本次大会共设立现代 C++ 最佳实践、架构与设计演化、大模型驱动的软件开发、AI 算力与优化、异构计算、高性能与低时延、并发与并行、系统级软件、软件质量建设、安全与可靠、研发效能、嵌入式系统十二大主题，全面覆盖从语言标准到工程落地、从算法优化到系统创新的全链路议题。超过 40 位来自百度、阿里、腾讯、智元、快手、小米、京东、美团等企业技术专家及高校科研工作者带来深度分享。

目前大会报名正式进入倒计时，请大家立即报名，以免错过这一行业盛会！

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)

C++ 之父亲临现场，大会全日程抢先看

  

在 12 月 12 日上午，大会首日的主会场将迎来重量级技术嘉宾分享。C++ 之父、ACM 和 IEEE 院士 Bjarne Stroustrup、CSDN 高级副总裁、奇点智能研究院院长李建忠以及来自头部科技企业的系统软件领军人物将齐聚一堂，他们将分别从语言设计哲学、AI 原生基础设施演进、高性能系统工程实践等多角度，深入探讨 C++ 如何持续赋能 AI 时代的关键底层系统。

![](https://mmbiz.qpic.cn/mmbiz_jpg/VeJKXItpwPWEKoPAeoFAhiaLJ0GyMaZk4yT2lTdvLqZibfMG0aOEgX3WjuKP4nU2onwhl7Giaibn0ujicyJgeoziaJYw/640?wx_fmt=jpeg&from=appmsg)

此外，主论坛还将特别设置 “AI 原生时代的系统软件”高端圆桌讨论，Bjarne Stroustrup 将与来自头部一线的技术专家、科研工作者同台对话。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

共探系统软件新范式：一线专家论道前沿实践与技术趋势深度对话

  

在听完主会嘉宾精彩且干货满满的分享后，在首日的下午场，大会将围绕「现代 C++ 最佳实践」、「AI 算力与优化」、「并发与并行」等专题，邀请 Adobe、阿里、小米、奇点智能研究员、清华大学、北京智源人工智能等行业领军企业、顶尖高校及科研机构的技术专家、实战派工程师，带来深度技术拆解、实战案例剖析与前沿趋势探讨。
