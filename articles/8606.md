# 接入OpenClaw，一句话生成爆款视频：拆解小红书OpenStoryline背后的工程密码

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAugJkibeIkBwI7feN33tcTm1qPpuj2lvd72ibrAX8fmhZCzuPoUE3c3B4TPE2SZsSOicDkKSOSOQusEBQ/640?wx_fmt=gif)

“不用拖拽复杂的时间线，只需像聊天一样输入自然语言，AI 就能自主完成从搜集素材、挑出高光、撰写文案到智能卡点剪辑的全套工作流。”

小红书开源项目 FireRed-OpenStoryline 为下一代视频创作给出的终极解法。

也正是许多自媒体和 UGC 创作者梦寐以求的“嘴替剪辑师”。

在大模型的加持下，视频创作正在经历一场从“工具辅助”到“智能代工”的底层范式跃迁。

当 AI 已经能够看懂视频画面、听懂人类的情绪与指令，我们不再需要手动去一帧一帧地抠时间线，而是通过自然语言与系统对话，由 Agent 自主完成从素材理解到剪辑生成的全过程。

这种颠覆性的交互范式，在底层工程上究竟是如何实现的？

在 4 月 17-18 日，由 CSDN&奇点智能研究院联合举办的「2026 奇点智能技术大会」上，我们诚邀了 FireRed-OpenStoryline 开源项目负责人、小红书算法工程师许思杰亲临现场。他将结合真实业务 Workflow，深度拆解如何将复杂的视频制作流水线，转化为一个高可靠、可交互的 Agent 系统。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/S1iaf4GgGjEzn1rn8olliabYzRXC6hK6HF8F9voxmbZbJvBiaOkf67q9GjZQfb5J8UxPsOdkwnv5P1hfKe3pLYMrjMJfgnrxva8JtaTGCvB9wc/640?wx_fmt=jpeg&from=appmsg)

作为小红书 AIGC 领域的算法专家，许思杰的底色非常“硬核”。

在投身于对话式视频剪辑 Agent 之前，他长期深耕于计算机视觉（CV）与 AIGC 的底层研究，在图像/视频风格化、扩散模型（Diffusion Models）加速等方向积累了深厚的功底，其研究成果曾多次发表于 ICCV、ACM MM 等国际顶级学术会议。

目前，他正主导 FireRed-OpenStoryline 的设计与落地，试图用 Agent 架构彻底打破传统的视频生产逻辑。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QuwjkSP1G6wEJHaJCLTONqlcQexqRgJcIICxofIOJs6B6tWBfibb7now/640?wx_fmt=png)

从素材到成片：对话式剪辑的全链路硬核能力

这团被命名为 FireRed（红色的火苗，取自“星星之火，可以燎原”） 的开源项目，并非简单的大模型 API 调用套壳，而是一个精密设计的自主智能体系统。如下图所示，系统主要由 Agent Client（智能体大脑与中枢）、MCP Server（工具执行层） 以及 Resources & Input（数据与资源层） 构成。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/P9Hs04VFGRnEQQwytwfiaN5L7sJUmickJyqJcIv9hUWOGYyuYFichKictdcXA69yWgKeKdls91uJVKy8s2b9s5z1jozicmS58SZajzoMFbU2dALM/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

在 3 月份更新的版本中，FireRed-OpenStoryline 强势接入了当前爆火的OpenClaw，新增了 openstoryline-install 与 openstoryline-use 两个专属 Skills，覆盖了从安装首跑到实际使用的全流程。

此外，还添加了面向 Claude Code 的 Skill 使用说明，方便 Claude Code 依据仓库规范完成自动安装与调用。

翻开 OpenStoryline 的核心特性列表，你会发现，它精准地击中了创作者的核心痛点：

*   智能素材搜索与整理： 告别繁琐的找图找视频，系统能自动在线搜索并下载符合你需求的素材，并基于用户主题进行片段的智能拆分与内容理解；
    
*   智能文案生成与仿写：结合主题、画面理解与情绪识别，它不仅能自动构建故事线及旁白，还内置了强大的少样本（Few-shot）仿写能力。你只需输入参考文本（如“种草测评”或“日常碎碎念”），它就能实现语感、节奏与句式的精准复刻；
    
*   情绪化配乐与字体推荐：支持导入私有歌单，系统会根据视频内容和情绪自动推荐背景音乐并智能卡点。你甚至只需用自然语言描述“克制一点”、“偏情绪化”、“像纪录片旁白”，它就能完美匹配相应的配音与字体，保证整体风格的协调统一；
    
*   对话式精修，即改即得： 想删减或重组片段？想修改字幕文案？想调整文字的颜色、字体、描边甚至位置？所有操作均通过自然语言对话完成，所见即所得；
    
*   剪辑技能（Skill）沉淀： 这也许是生产力质变的核武器——你可以将调教好的完整剪辑逻辑，一键保存为专属的剪辑 Skill。下次只需更换素材并选择对应 Skill，即可快速复刻同款风格，实现高效的批量生产。
    

从底层多模态理解，到上层对话式交互，从单一视频生成，到沉淀为可复用的企业级 Skill 资产。
