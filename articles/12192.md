# 「Claude Code之父」其实是野路子来的......

##### 鹭羽 发自 凹非寺  
量子位 | 公众号 QbitAI

谁能想到，用**Claude Code**砸了程序员饭碗的人，还是个自学成才的野路子。

这下真是端起碗吃饭，放下碗砸锅_（bushi）_

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGFhiaTwDibdtjPJC9iceK5utmpIlZUTk6In6dwZ4icDvib4fCfTrcHnKCOicT34IlpDqs6XTKO9yZtfMhEsJURCazkNuicSicFF93wPRUs/640?wx_fmt=jpeg&from=appmsg)

说起「Claude Code之父」**Boris Cherny**，职业生涯主打的就是浪～

经济学专业出身，从零开始自学编程，一路杀到Meta首席，还顺手写了本TypeScript圈的顶级“圣经”……

去年他还一度出逃Anthropic，结果又被立马请了回来，好一出抓马至极的「C妃回宫」。

而他回归后的第一件事，就是宣布手搓编码时代的ending。Claude Code在他手上彻底起飞。

他本人还透露，从去年年底开始，他就再也**没手写过一行代码**……

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGG9Oc2Gg1KQ168FBWq8LaRIZ9GShNsaU65gZPLdZqVjKrsvOxX8arK7gI44RsFRQutX92EpfJCblSmwY8VfaxaDBCBPu6Dziaog/640?wx_fmt=jpeg&from=appmsg)

也是，当顶级程序员都不再亲自写代码，这届开发者的路，怕是要被这位不爱走寻常路的前辈彻底堵死了。

哥们自学成才，真·狠人
-----------

要说为啥最近Boris Cherny的关注度直线飙升，离不开他一手缔造的Claude Code。

**CC：我超能赚！**

Claude Code的年化收入已经从去年12月的10亿美元一路暴涨到今年2月的25亿美元，连带着Anthropic总收入突破300亿美元。

诚然，A社目前最赚钱的还当属大模型，但Claude Code已经隐隐约约有了印钞机潜质。

另一件围绕在Boris身上的事，则有点乐子人属性。_（doge）_

**Claude连夜封杀OpenClaw**，想必大家都已清楚，这里就不加以赘述了。

但在公告刚发布之际，Boris就火速空降，立刻化身品牌公关降温：在𝕏上一边安抚愤怒的龙虾用户，一边答疑解惑。

一直干到晚上11点，就这敬业程度，龙虾之父**Peter Steinberger**也出面给了个夯！

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHjZSibcsq8DqPUoibkvsWt5ic3tTqcNnxFOUnSw4h0gHqFaEE92NpJn1dgy8h2dVxowrzP86wC065DhFB2RQO9icRj4wPFUUibuniaU/640?wx_fmt=png&from=appmsg)

话已至此，网友们开始纷纷关注起另一件事——**Boris Cherny**究竟是谁？

在此之前，他在Anthropic算是技术流扫地僧的存在。

细扒他的社交媒体，清一色的全是各种硬核干货，这也让他在大众圈层知名度不高。

但实则这哥们，才是Claude Code的灵魂人物。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGGBkGicmfFCbhdbty9hWVb0EjUNnicRVfQnabx0WiaqMKOwUauQtZ1M5jImNSpJDNGBfKa6QTMNZREwC8AMWvf386skcn2iaBTG6yA/640?wx_fmt=jpeg&from=appmsg)

出乎意料的是，他并非专业计算机科班出身，曾在加州大学圣地亚哥分校就读经济学专业，然后辍学，几经波折才拿到学位。

不过他的编程天赋在中学时就早有显露，彼时自学编程后就开始为eBay商品列表编写网页代码，还为图形计算器编写过程序。

本科毕业后，他就开始尝试创业互联网公司，后来又加入了**Meta**，这一干就是将近7年。

在Meta里，他负责领导Instagram服务器架构和开发基础设施，以及提升Meta的代码质量。

在此期间，他还顺便写了本TypeScript畅销书《Programming TypeScript: Making Your JavaScript Applications Scale》，被公认为TypeScript的必读书目。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGGgvr6icKGXSH0HalDb1XPAry4cHwjGV4S8wHPrD6X9QDRhTog12YSeGh7dD4Pd4JuGatzjo8OyLInj4LJIo84vArpbYVqCm2iag/640?wx_fmt=jpeg&from=appmsg)

然后作为Meta的首席软件工程师，跳槽到了Anthropic。

小扎也是扎心了，现在的他还在到处高薪挖人，实则自家早就飞出了金凤凰。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGGq37IicwsSRAqwfXSbqKabiaEVZUpGfvTCn7uyEiclBpSzn7Tdicjk304VJicBTXk86MiboQQmPJ2PtGecf0suGgXlrGz1DXo0G6C38/640?wx_fmt=png&from=appmsg)

到了A社的Boris可谓如鱼得水，一年不到，就整出了**Claude Code**。

据Boris自己所说，CC的诞生其实很意外很偶然。最初也不过是因为他作为新员工，想要学会使用公司的API，所以编写了一个极简的终端_（CLI）_应用。

他先是给模型加了批处理工具，又把Python示例改成了TypeScript，结果CC的表现完全出乎意料：让CC读文件，它用cat命令做到了；问CC自己在读什么歌，它直接写了一段Apple Script，去了Boris电脑里的音乐播放器查。

Boris形容那一刻，仿佛是**AGI之神降临**：

> 模型就是想用工具，这是它的本能。

事实上，OpenClaw的诞生，也是龙虾之父的灵光一现……果然高手过招，都是有共通之处的。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGFxFqWLEyiagFV3MdtIO3FzSW6pp4SRUhBDbTGUhZhb8uMjqeQzLGTdk9M77PdS5MDcEd4Yuk8D5mFCX5tTyGF6d0Jg1rHSOSHE/640?wx_fmt=jpeg&from=appmsg)

_（咳咳）_回到Boris身上，Claude Code原型一出，就迅速在A社内部流行起来。然后经过多次迭代发展，最后端给了用户。

但Claude Code刚上的时候，反响并不热烈。直到去年5月，Anthropic的首届开发者大会后台，Boris再次来了灵感。

他直接把同天公司发布的两款新模型**Opus 4**和**Sonnet 4**拿来打磨Claude Code，结果Claude代码生成比例从10%涨到了将近30%。

路线对了，但有趣的是，人跑了。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGEGMxia8qqRAOLx3fgvjJv61vwznP2M6kN1CXTYGJUK7dFNbdZgMbHELmIxVB5Ch0n27QhGAUFldSESllyNApF5eVzBytG94J5w/640?wx_fmt=jpeg&from=appmsg)

刚过去没多久，Boris就和Claude Code的产品负责人Catherine Wu双双跳槽到了对家**Cursor**。更反转的是，半个月后，Anthropic就三顾茅庐，又把两位请了回来。

原因无它，还是Claude Code立大功，直接让A社那段时间的营收暴涨**5.5倍**。而Boris此番回来也没有辜负A社的期望，Claude Code彻底商业化起飞，开始稳坐AI编程头把交椅。

现在得益于Claude Code，Anthropic的年化收入已经**赶超OpenAI**，Claude Opus 4.5的发布又让Claude Code更上层楼。

其实除了Claude Code，Boris还主导了今年年初发布的**Claude Cowork**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGF9WEczUR6mukjJeg6Dry52FibXFcd16Q5I6PrkHbxqGM73w6qibictQBd6M33LTsD7bruC0rN07w7n5cnWyhIfnIcecnoGJLVSB0/640?wx_fmt=png&from=appmsg)

Claude Cowork的底层逻辑和Claude Code一脉相承，因为Claude Cowork就是团队用Claude Code 10天时间生成的。但与之不同的是，目标用户从程序员变成了普通人。

它能实现真正全自动干活，在经过用户授权后，可直接操作用户电脑桌面、本地文件夹和各种办公软件。

就以Boris自己为例，他现在直接让Cowork接管了他生活工作的方方面面。

Boris只管喝喝咖啡，自有Cowork帮他逐条审核同事的每周工作总结，以及顺着Slack网线去催那些还没填写的同事。

另外，他还会让Cowork来预定航班、支付停车罚单，甚至办挖蛤蜊的许可证……

好好好，Boris才是**将dogfooding贯彻到了极致**。

One More Thing
--------------

但最近另有一件全网围观的事，则是让人有些啼笑皆非了。

Claude Code源码被开盒，结果被扒出一堆屎山代码……

连“始作俑者”Chaofan Shou都忍不住吐槽：**没意思。**
