# Sora 死了，OpenAI 正在变成另一家公司

就在昨天，OpenAI 官方账号发了一条简短的告别声明："我们要和 Sora 说再见了。"App 要关，API 要关，连本来计划整合进 ChatGPT 的视频功能也不做了。

六个月前，Sora App 上线五天就突破一百万下载，比 ChatGPT 还快。迪士尼砸了十亿美元投资 OpenAI，把米老鼠、漫威、皮克斯、星球大战超过两百个角色授权给 Sora 用户做视频。那时候所有人都觉得，AI 视频的时代来了，OpenAI 又赢了。

现在迪士尼也退出了。迪士尼发言人很客气地说："我们尊重 OpenAI 退出视频生成业务的决定。"据 Reuters 报道，迪士尼的团队前一天晚上还在和 Sora 团队一起工作，第二天就被通知对方"战略转向"了。

先看数据。Sora App 去年九月底上线后确实火过一阵：**首日十万下载，五天破百万，一度冲上美国 App Store 总榜第一**。但热度消退得很快。

_图：第三方商店数据平台上的 Sora 下载概览截图，显示 2025 年 11 月到 2026 年 1 月全球累计下载约 458 万。_

Appfigures 的数据显示，**去年十二月下载量环比下降 32%，今年一月又暴跌 45%，只剩 120 万**。用户消费也从十二月峰值的 54 万美元跌到一月的 36.7 万美元。**App 累计收入只有 140 万美元**。在美国 App Store 排名掉到一百名开外。

![](https://mmbiz.qpic.cn/mmbiz_png/ZlwY8rlDcFN1tcK0sEGQQZp4YHLhPzPNEsFJFSiar3ticBVrTwsyEHCdicF3bqsgXp8AeyJXj9ZPEXTU1dttYpFI3nwR40tKwqKhFLcNBXPd80/640?from=appmsg)

_图：Appfigures 下载量数据图，经 TechCrunch 发布。_

作为对比，TikTok 去年十二月全球下载量超过 1800 万次。Sora 想做"AI 版 TikTok"，但事实证明，AI 生成视频目前还撑不起一个社交平台的日常消费场景。用户玩了几天觉得新鲜，然后就放下了。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/ZlwY8rlDcFP4qVfOtzICx0uadIltRf0yXUqFccrlthHXP0UPcgkK0dia5aRNBsXol1IZiaw4VNo02A902d24LaWeHbBnbO5aBNCZCmFNGvtiac/640?from=appmsg)

_图：Appfigures 消费额数据图，经 TechCrunch 发布。_

版权问题也在不断消耗产品的生命力。一开始 Sora 采用"退出制"，也就是版权方要主动申请排除自己的作品才行。结果用户疯狂用海绵宝宝、皮卡丘这些角色做视频，好莱坞炸了锅。美国电影协会、SAG-AFTRA 工会、Bryan Cranston 等一线演员纷纷抗议。OpenAI 被迫改成"许可制"，限制变多了，用户兴趣也跟着跌了。

和迪士尼的合作本来是个转机，让用户合法使用迪士尼角色，但数据表明这个合作并没有拉回用户增长。

不只是砍产品，整个公司在转向
--------------

Sora 的关停不是孤立事件。过去两周，OpenAI 做了一连串动作：

三月十九日，OpenAI 宣布把 ChatGPT 桌面端、代码工具 Codex、和浏览器 Atlas 合并成一个"超级应用"。应用负责人 Fidji Simo 在内部信里说："我们把精力分散在太多 App 和技术栈上了，这种碎片化拖慢了我们的速度，让我们达不到想要的质量标准。"

同一天之前，Simo 在全体员工大会上更直接："我们不能因为被支线任务分心而错过这个时刻。我们必须在生产力方面、特别是企业端做到极致。"

然后是更大的信号。The Information 记者 Stephanie Palazzolo 爆料：**OpenAI 已经完成了下一代大模型的预训练，代号"Spud"，对应 GPT-5.5 或 6.0 级别**。Altman 在内部称 Spud 是一个"非常非常强的模型"，团队相信它"能真正加速经济运转"，预计几周内就能推出。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ZlwY8rlDcFPTlNkNEYHksy5o7LQX9ovdfhWQEoSlPcRtfKib1y6SSmaQFzCWRjaqAx7WazqcE5RrxDiaIVpvPlgrGUiajIwMaDTLKIyv295ndk/640?from=appmsg)

_图：The Information 报道截图。报道明确把 Spud 的推进和 Sora 被视为算力负担放在同一个判断里。_

与此同时，Altman 放下了两个关键团队的直接管理权：AI 安全（Safety）团队移交给首席研究官 Mark Chen，信息安全（Security）团队移交给总裁 Greg Brockman。他自己的精力转向了什么？**"融资、供应链、以前所未有的规模建设数据中心。"**

Sora 团队也没有解散，而是转向了长期的"世界模拟"研究，聚焦机器人方向。砍掉的是面向消费者的产品，留下的是底层研究能力。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ZlwY8rlDcFNnHeTOe0zmGEeVwrNtfBDoMMZiaPoibRtvfVrmVteVHSvgsZ1yKicabtpD2lYibZsiauQxddnq7CLStRWDLoQ0IhP7Gv8kqDMXvGcw/640?from=appmsg)

_图：The Information 报道截图。Sora 研究没有消失，而是转向高保真的世界模拟，并更明确地对准机器人。_

最耐人寻味的一个细节：**Altman 宣布把 Fidji Simo 领导的产品组织改名为「AGI 部署」（AGI Deployment）**。AGI，通用人工智能，过去一直是 OpenAI 的终极目标，一个遥远的愿景。现在它被放进了一个具体部门的名字里，成了可执行的动作。

这些信息拼在一起，画面就很清楚了：OpenAI 正在把算力和人才从视频这个"烧钱大户"上撤走，全部押注到下一代模型和企业级工具上。Sora 不是被放弃了，是被让路了。

为什么是现在
------

三个压力同时收紧。

**第一是竞争。** Anthropic 在企业市场上的增长速度让 OpenAI 很不舒服。据 Ramp 的数据，一年前每 25 家企业客户里只有 1 家用 Anthropic，现在跳到了接近四分之一。**在新客户争夺中，Anthropic 拿走了大约 70% 的首次采购份额**。Claude 在今年三月的美国下载量超过了 ChatGPT。Claude Code 在开发者群体中的增长势头非常猛。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/ZlwY8rlDcFMOdWw56zAYibjtKDroN1Jtmk7W7Sic3M53J3OJcd5mz8lIg5p4CqL5ZbQ2QlHwJ2DASrD7O9pm4NPJ0V68KfTYRkJHxPucGFuak/640?from=appmsg)

_图：Ramp AI Index 模型采用率。来源：Ramp AI Index March 2026 update。_

Anthropic 的策略很有意思：不做图片生成，不做视频生成，把所有算力集中在文本和代码上。NBC News 在报道 Sora 关停时特意提到了这一点。OpenAI 现在的转向，某种程度上是在回应这个策略：与其把算力花在一个用户留不住的视频 App 上，不如集中火力守住代码和企业市场。

![](https://mmbiz.qpic.cn/mmbiz_png/ZlwY8rlDcFMWbYKBZSwicTxM57zsGSphRENiahTrQP2UK8Y43wqs2wvic4P9DKvbc7D8AzIlqjOnNtiaYiaor0DohKXXK2C5iaIkI6BLPEicqmTia1I/640?from=appmsg)

_图：首次采购 AI 服务时的 Anthropic 与 OpenAI 份额变化。来源：Ramp AI Index March 2026 update。_

**第二是 IPO。** 多家媒体报道 OpenAI 最早可能在今年第四季度 IPO。上个月刚完成一轮 **1100 亿美元的融资，估值 7300 亿美元**，投资方包括软银 300 亿、英伟达 300 亿、亚马逊 500 亿。在这个节点上，产品线必须讲一个清晰的故事：核心收入来源是什么，增长飞轮在哪里。一个下载量持续下滑、版权风险高企、变现乏力的视频 App 显然不是投资人想听的叙事。

**第三是算力。** 视频生成是出了名的烧算力。OpenAI 自己在技术报告里就说过，Sora 是一个大规模扩散 Transformer（Diffusion Transformer）模型，可以生成长达一分钟的视频，而且性能提升的路径就是更大的模型、更多的训练算力。现在 Spud 已经完成预训练，接下来的推理部署同样需要海量算力。Altman 亲自去抓数据中心建设，说明算力已经成了 OpenAI 最紧迫的瓶颈。在这个局面下，Sora 占用的每一块 GPU 都是从下一代模型嘴里抢的。

对创作者和开发者意味着什么
-------------

对用 Sora 创作的用户来说，最紧迫的问题是：我的作品怎么办？OpenAI 在声明里承诺会给出具体的时间线和数据保全方案，但目前还没有细节。如果你在 Sora 上有重要作品，建议尽早备份。

对开发者的冲击可能更大。Sora API 一旦关闭，意味着所有基于 Sora 搭建的第三方创意工具、广告制作流程都要找替代方案。Google 的视频生成能力目前是这个领域里最有规模的替代选项，但 Google 自己也在被版权方起诉。

对整个行业来说，Sora 的故事给出了一个信号：**AI 视频在技术上已经能做到很炫的效果了，但"技术上能做"和"商业上该做"之间还有很大一段距离**。版权、审核、算力成本、用户留存，每一个环节都可以让一个好产品变成一个坏生意。

OpenAI 变了，还是回来了
---------------

回过头看，Sora 从来就不是一个纯粹的消费者产品。它一开始是作为研究里程碑亮相的，OpenAI 反复把它定位成"世界模拟器"的雏形，强调的是 3D 一致性、物体持久性、物理模拟这些科研属性。后来做成消费者 App，做社交 feed，和迪士尼签约，这些都是在研究成果上嫁接商业化尝试。

现在 OpenAI 的选择是：把"世界模拟"留在实验室里做长线研究（转向机器人方向），把面向消费者的产品砍掉，把产品组织改名叫「AGI 部署」。这个改名信息量很大。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/ZlwY8rlDcFMHckGt3rM5DCvtuS0zxVYTR2cq0Wu0fmjzkSviamCYoTwzx4o9H0jibjoODDWicokLA6DHtUER79sDF8ILicxLBZ9q8MRD5dGIp3M/640?from=appmsg)

_图：The Information 报道截图。把产品组织直接改名为「AGI Deployment」，说明 OpenAI 连部门叙事都一起换了。_

把产品部门叫「AGI 部署」，只有两种可能：要么 OpenAI 内部认为他们已经接近甚至达到了 AGI，需要一个新组织来负责把它推向市场；要么这是在 IPO 之前精心构造的叙事，用「AGI」这个词给估值加码。考虑到 Altman 同时在说 Spud「能加速经济运转」，又在不遗余力地融资建数据中心，这两种可能并不矛盾。

有人在社交媒体上总结得很直接：OpenAI 在 IPO 之前宣布 AGI，把最后一个烧钱的消费者产品砍掉，CEO 全力去搞基建和融资。一切都指向同一个方向。

但这种收缩的代价是，OpenAI 在消费者多模态领域暂时让出了位置。在竞争对手都在追求多模态和智能体（Agent）的时候，OpenAI 选择收缩战线、集中突破。这是一个赌注：赌 Spud 够强，赌企业市场够大，赌算力集中投放能换来足够的竞争优势。

Altman 说了一句话：「事情发展得比很多人预期的要快。」

这句话既是在说 AI 的进步速度，也像是在为这次急转弯找一个理由。

**Sora 的关停是 OpenAI 在 IPO 之前做了一次残酷但理性的取舍。放弃一个烧钱且增长乏力的消费者产品，把资源押注在下一代模型和企业市场上。但代价也很明确——那些信任 Sora、在上面投入创作精力的用户和开发者，成了战略调整的牺牲品。**

你觉得 OpenAI 砍掉 Sora 是明智的断臂求生，还是他们真的觉得 AGI 来了、一切都该为它让路？
