# 解密龙虾，探索新一代AI Agent范式！OpenClaw最强轻量平替负责人黄超深入分享

4月21-22日，**GenAICon 2026 | 2026中国生成式AI大会（北京站）**将在北京富力万丽酒店正式举行。中国生成式AI大会已成功举办四届，现已成为国内人工智能领域最具影响力的产业峰会之一。

  

本次大会由智一科技旗下**智东西**联合**智猩猩**发起主办，为期两天，由**开幕式+专题论坛+研讨会+交流晚宴+展览区**组成，将以**“奔赴AGI 重塑未来”**为主题，邀请**70+**位重量级嘉宾与会带来致辞、报告、演讲和对话。

  

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurGq7ZLgUpiaYhD8gGzBH8D60YHTzdB2ab4ugiaOgJzPiayOBX0SUeLxOurPibCxk4lSPwJoMfSHjGYcUxhxYgwIBzDqQzo5ykcjog/640?wx_fmt=png&from=appmsg)

  

目前，**香港大学助理教授&博士生导师、Nanobot团队负责人黄超**已确认出席，将在4月22日分会场二上午的「OpenClaw技术研讨会」带来报告，主题为**《解密“龙虾”：下一代AI Agent挑战与机遇》**。

  

  

 **Part.1**

**嘉宾介绍**

![](https://mmbiz.qpic.cn/mmbiz_png/rk3H224bjtuqggvfO4na9Z8VgE6cxcI4uIyGjy1icozaEDpBlVaYRV3xlI4aGSrtmTjSm4ByswZzib2hNNmE4on3tal9Mt4MToh8uicn72VVoo/640?wx_fmt=png&from=appmsg)

  

黄超现任香港大学数据科学研究所及计算与数据科学学院助理教授、博士生导师。他的研究方向涵盖大型AI智能体（AI Agents）、语言模型（LLMs）、及图机器学习（Graph Machine Learning），其研究成果在Google Scholar上被引用超过16,000次。团队已推出多个影响力开源项目，包括Nanobot、LightRAG、DeepCode、RAG-Anything、DeepTutor、AutoAgent、AI-Researcher、AI-Trader、MiniRAG、VideoRAG、ViMax等。作为团队负责人打造的HKUDS GitHub开源平台累计获得超过150,000 GitHub Stars，位列全球Top-100，并超过90次登上GitHub Trending榜单。因其在人工智能开源领域的贡献，黄超荣获世界人工智能大会（WAIC）2024云帆奖「璀璨明星」、国际基础科学大会（ICBS）2024「前沿科学奖」，并入选「2025 AI100青年先锋」以及入选「2025 AI 2000全球最具影响力学者」。其研究成果多次入选KDD、WWW、SIGIR、AAAI等国际顶级会议的最具影响力论文之一，并获得ACM MM 2024、WSDM 2022、WWW 2019等会议的最佳论文提名奖。

**Part.2**

**报告主题**

《解密“龙虾”：下一代AI Agent挑战与机遇》

**Part.3**

**报告概要**  

随着人工智能技术的深度演进，“龙虾”正在开创新一代AI Agent范式，引领智能代理从单一任务执行向自主决策与协作的范式转变。本次分享将剖析这一技术革命背后的核心驱动力与创新逻辑。我们将从范式层面探讨传统AI系统向自主Agent的演化路径，揭示龙虾如何重新定义人机交互与智能协作模式，以及基于龙虾Agent生态系统的涌现特性与网络效应
