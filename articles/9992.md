# 会挖0-day漏洞！凶残版Claude官宣却不让用，微软苹果齐下场“看守”，Anthropic到底在图什么？

  

****整个行业最近都在往“安全、算力、基础设施、国家能力”这条线收紧。****

整理 | 王启隆

出品丨AI 科技大本营（ID：rgznai100）

Anthropic 今天官宣了 Mythos Preview，这家公司又做出了一个更强的 Claude，但反常识的是，他们这次根本没打算先把这东西放给普通人用。

模型刚亮相，Mythos 就先被塞进了一个由 AWS、Apple、Google、Microsoft、NVIDIA、Cisco、CrowdStrike、Palo Alto Networks、Linux Foundation、JPMorganChase 等机构组成的防御联盟——Glasswing 里。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/iaM6L8b5B1GnReYiaJqRt5uNGNQlvQBBsYdewBxyz8LK2dkaFyS5mwwdHX7a3f8ejg8lpBJTpggp48DhBbBVxaZxOxNWaXKPF8gJlc1icyru7c/640?wx_fmt=png&from=appmsg)  

官宣了，但你用不上。

不是一次常规的产品更新，而是一种很少见的发布姿势，先限用，再解释为什么要限用；先拉联盟，再谈后面怎么放；先把风险写出来，再去谈能力有多强。

围绕 Mythos，Anthropic 同时放出了三份彼此咬得很紧的材料，Project Glasswing 主发布页、红队技术页 Claude Mythos Preview，以及一份 system card。这家公司在公开演示，面对一种可能改变网络安全攻防平衡的能力，它准备怎么把这东西先关起来，再慢慢处理。

如果一家前沿模型公司真做出了一个“史上最强”的新模型，大家熟悉的剧本通常是什么？开放 API，晒 benchmark，放 demo，拉开发者试用，让市场先跑起来。

Mythos 没有先被推给开发者，而是先被塞进了 Glasswing。

![](https://mmbiz.qpic.cn/mmbiz_png/iaM6L8b5B1GnvC9ebLlY71YxnpKgzjTibr4nCmDhpf7DjRgvsUx6gmFEUpTGAJib0dlfariccAWaeF1bhO3ImtebibqnByrT7na2AYibLjDy9rtvM/640?wx_fmt=png&from=appmsg)

Anthropic 对 Mythos 的描述非常直接，它在漏洞发现和漏洞利用上，已经超过绝大多数人类，只比最顶尖的安全研究员略逊一筹；它已经在主流操作系统、主流浏览器和其他关键软件里找到成千上万处高危漏洞，其中一些问题在经历多年人工审查和数百万次自动化测试之后，依然没有被发现。

但 Anthropic 同样觉得，AI 网络安全能力已经越过了一条不能再按普通产品逻辑处理的线。既然这样，它第一次亮相就不是市场开放，而是先被装进一个限制性框架里。

Glasswing 干的第一件事，不是帮 Anthropic 做市场，而是帮它把风险先锁进防守体系里。Mythos 没有先流向普通开发者，而是先被交给了一群负责补洞、守城、加固关键系统的人。

Anthropic 最怕外界把 Mythos 理解成一个普通的“更强代码模型”，所以它专门放出了一篇红队技术页，把问题摊得很开。

![](https://mmbiz.qpic.cn/mmbiz_png/iaM6L8b5B1GmHN1lqHrNJK9WPuf9Zvd3yDdxwGE0JlMNf1qsMUXvjCPJGfibQduhxqDmsxJTt0K8AV3VW2ia1F91PaxKRhlmTNeg4sAWD89OUw/640?wx_fmt=png&from=appmsg)

Mythos Preview 的跃迁，不只是代码能力更强，而是它在漏洞识别、漏洞理解和漏洞利用这条链路上出现了明显变化。它能在主流操作系统和浏览器中识别并利用 zero-day；有些漏洞已经潜伏十几年，甚至长达 27 年；它不是把 bug 找出来就结束，而是还能继续往 exploit 方向推进。

它开始会自己找洞、自己打洞了。

这也是 Anthropic 这次不敢直接大规模放手的根本原因。过去一年，行业对“代码模型”的理解，大多还停留在工程效率、AI coding、agent 助手这些层面。Mythos 暴露出来的，却是另一条能力带，它不只是帮工程师更快写程序，而是在逼近安全研究员、甚至攻击者的工作方式。

一旦模型进入这个区间，发布的性质就跟着变了。你发出去的不再只是一个更顺手的编程助手，而是一种可能同时增强防守方和进攻方的能力。问题不在 Mythos 更强，而在它强得已经很难按旧规则发布。

如果 Anthropic 只是单独官宣 Mythos，再附一篇技术博客，外界很容易把这理解成一次带安全包装的秀肌肉，公司做出了更强的 Claude，然后顺手提醒一句风险不小。

Glasswing 没把 Mythos 变得更好卖，反而先把它围了起来。

Anthropic 一上来不是把 Mythos 丢给普通开发者，也不是先把它做成一款大规模可试用的明星产品，而是把它先交给云厂商、安全公司、基础设施玩家和关键机构，让这些人先拿它去扫描代码、补漏洞、加固关键系统。Anthropic 还承诺 usage credits 以及对开源安全组织的捐助，说白了，它不是先把这东西推向市场，而是先把它组织进防守体系。

先围起来，再使用。

这一步把原本容易引发不安的“危险能力”，改写成了“联合防守的必要性”。Anthropic 想让外界接受的，不只是 Mythos 有多强，而是这种强能力该先被谁用、该先落在哪里、又该由谁先看住。

从传播角度看，Glasswing 还做了另一件事，它把 Mythos 从单家公司的产品新闻，抬成了一件需要产业共同处理的事。你看到的不只是一个模型发布页，而是一家公司在说，我们可能已经碰到一种会改变网络安全攻防平衡的能力，所以这东西不能再按普通新品节奏来处理。

![](https://mmbiz.qpic.cn/mmbiz_png/iaM6L8b5B1Gm8Nb4iciccjscd0At9PvYP6K9ticwzHiaicIaTzcoIG4zAAT63iah6ktt1icCAMRxO8icyXviaTeY62cnaCrdKJBBPdqMTBKCmica2vbb3Y/640?wx_fmt=png&from=appmsg)

很多模型公司也会附 system card，但 Anthropic 这次的 system card 明显不是例行公事。它承担的是同一套叙事里的最后一层，把“模型很强，也很危险”这件事，从传播语言和技术事实，推进到制度化、责任化的表达中。

先限用，再谈怎么发。

过去几个月，AI 舆论越来越少只谈聊天机器人、办公助手和榜单排名，越来越多开始谈算力、数据中心、电力、基础设施、国家安全和网络攻防。Anthropic 自己前几天刚被广泛报道与 Google、Broadcom 达成大规模算力合作，未来的多吉瓦级 TPU 产能会成为其前沿 Claude 模型的基础。这些信息不断强化一个背景，前沿模型公司已经不再只是软件公司，它们越来越像同时牵动基础设施、资本、国家能力和安全系统的节点。

在这种背景下，Anthropic 宣布自己造出了一个已经会自己找洞、打洞的模型，然后第一时间把它关进防御联盟里，听上去就不再像一个为了吸睛而做出的夸张叙事，而像是顺着行业逻辑往前迈了一步。

还有一层现实的对比红利。过去一段时间，OpenAI 和 Sam Altman 周围的舆论一直笼罩着组织稳定性、领导风格和信誉问题的阴影。无论这些争议最终如何沉淀，它们都已经让“谁更可靠、谁更像知道如何处理强能力风险的公司”变成 AI 行业里一条越来越明显的暗线。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/iaM6L8b5B1GkKicVBbLd4lkUicWgGKm8BgNN9GGbNNVkib9OlQ1YdKTWFyz1IXfG5KqfRbeINwIQfibEmWcfaLnDiagJzQNhp1KPOWv0VHS8maZlU/640?wx_fmt=png&from=appmsg)

Anthropic 这次做的，恰恰就是把这条暗线公开化。它没有只说“我们也有最强模型”，而是在更大声地说另一件事，最强模型不该按普通产品那样乱发。

当模型强到足以改变网络安全攻防平衡时，应该由谁来决定它怎么被发布、谁先能用、什么样的限制才算合理？过去，模型公司的竞争主要围绕参数规模、性能、产品渗透、训练算力和推理成本展开。但当模型能力开始进入网络安全、基础设施、防守与攻击平衡这些高风险区间之后，新的竞争维度也开始出现，谁能先定义边界，谁能先定义风险，谁又能把自己的发布方式讲成未来行业的默认做法。

Anthropic 显然已经在抢这个位置。今天这件事是一场话语权争夺。它想让外界接受一种新的常识，未来最强的模型，不一定先开放给所有人使用。这和过往开源与闭源之争不同，因为过去的闭源 SOTA，大家花钱都能测。

如果这种叙事站住了，Anthropic 获得的就不只是一次新闻热度，而是一个更深的东西，它在提前占据“负责任的 frontier lab”这个位置，并试图把这种位置变成自己的长期品牌资产。
