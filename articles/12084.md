# 梁文锋首次开口融资，DeepSeek只估值680亿，融资20亿？

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜泽南、杨文

  

本周五，AI 圈曝出重磅消息。

  

据 The Information、路透社等媒体援引知情人士的话称，国内人工智能初创公司 DeepSeek 正与投资者进行洽谈，拟以 100 亿美元的估值筹集至少 3 亿美元资金。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFk32RqiaoXiaddxFbdXN2C7Msicnia7icyia8Bz349iabzu4wL9MGCJCLUsEBtfqQftLZ6MtOIZ1SYic7XzebLu4nblpugWVYmthH5x2I/640?wx_fmt=png&from=appmsg)

  

据机器之心了解，Deepseek 确实是在接触一些机构，但关于此次融资金额和估值金额，是不是有点少？有待核实。

  

在去年，DeepSeek 凭借其高性能、低成本的推理模型 DeepSeek R1 席卷了整个 AI 行业，并引发了股市震荡，引发了一轮对于大模型技术、AI 基础设施的重估。但作为一家初创公司，DeepSeek 从未接受过外部风险投资，该公司脱胎于国内顶尖量化对冲基金幻方量化，其早期的庞大算力储备和研发资金完全由母公司及内部资源支持。

  

此前，DeepSeek 曾拒绝了来自中国顶级风险投资机构及科技巨头的多份融资邀约。

  

此次潜在的融资行动突显出，在 AI 领域高级推理能力和智能体（Agent）兴起的背景下，开发并运行顶尖大模型对资金有着极其庞大的需求。

  

近期信息显示，DeepSeek 正在内蒙古乌兰察布招聘数据中心运维工程师，这是 DeepSeek 首次招聘直接负责计算基础设施运营的人才，表明其正转向自建数据中心基础设施。这种「重资产」模式需要巨额的资金储备来降低长期的算力成本。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFgjhBhkSN7ByGzGHkuGqpc7tjxMo4JQnearbGNRyf2bV62GsrjLXziaVdVy5VBLx5UpdsOWpwZLZJotZic5cKmqlbY4VCWsoJCo/640?wx_fmt=png&from=appmsg)

  

与此同时，整个科技圈都在关注预计于近期推出的下一代基础大模型 DeepSeek-V4，研发和运行这种顶尖 AI 模型对资金链提出了更高的要求。

  

除此以外，DeepSeek 此前的大模型以极高性价比席卷全球，其 API 定价通常只有美国同行（如 OpenAI 和 Anthropic）的几分之一。充足的外部资金能够帮助他们在这场全球 AI 价格战中保持进攻姿态。

  

当然我们也要注意到，若 DeepSeek 本次能以 100 亿美元估值融资 3 亿美元，意味着 DeepSeek 出让的股权极少。母公司幻方量化和创始团队会依然保持绝对的控股权和战略主导权。因此未来 DeepSeek 走向闭源的概率应该不大。

  

DeepSeek 的成功在于其极其专注、克制的战略：死磕算力效率和核心基座模型（如 MoE 架构、强化学习 R1）。然而在最近一段时间，我们经常看到该公司旗下大牛跳槽的消息。

  

人才出走

  

DeepSeek 新一代模型迟迟未露面，两位核心研究人员却已相继出走。

  

DeepSeek V3 的重要贡献者罗福莉已加入小米，主导 AI 业务。

  

曾参与 DeepSeekMath、DeepSeek-V3、DeepSeek-R1 等多个重磅项目的 95 后研究员郭达雅，则近期转投字节跳动旗下 Seed 团队，担任 Agent 负责人之一，职级为 L8。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqF9QrkiaSJsLtQclLZuORNsKagcdl7AvK9OPiaETy4cXSVyfu3ic2puLQQfiayWDDlvxatK0s8EBQicCyLibFQFkJcTrkZOOvz8XBb4g/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHeT5icjicE1KiaUnhyMuUSef65jYpyDia8a1ibCLcziaqiax8yXBPzj8CVAkauCWJLI5TwEamwF7uQTPBmQCgrvdv9wRJgdDvXOGNaaw/640?wx_fmt=png&from=appmsg)

  

围绕郭达雅的薪资，外界颇多猜测，此前有报道称其年薪（含期权收益估算）接近亿元。

  

对此，抖音集团副总裁李亮公开回应称，近期没有招聘到什么近亿元年薪的员工，Seed 团队所有技术人员薪资体系统一，由现金、字节期权与豆包期权构成，期权是四年期全部归属，不存在「需要满足一定条件才能拿全」的情况。他同时坦言，若业务发展顺利，部分技术人员四年后的期权收益确有可能达到数亿元量级。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/5L8bhP5dIqFSVXj683cDkDKvAtHTTJibdaBDeJP5l9ofwF1X5umVtF5micBYBSb01CVBJoS3dM31OiccyHNCSFAtQicQNVgwCNtXa1Fcv8zjcOo/640?wx_fmt=jpeg)

  

大厂对于 AI 人才的强势吸引，一方面是对研究方向的认同，一方面也印证了 AI 行业正在从早期的大规模技术探索，逐渐走向工程验证与生态构建的新阶段。

  

未来可能的融资，或许会让 DeepSeek 有更多的资源进行商业落地的部署。
