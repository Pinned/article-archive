# 融资超10亿！AI眼镜企业开年官宣：移动联通都投了

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgzh4CaNCMxPYapEFQTex0uMmOrmUtZvgP5pKo1qwkkBUyj1skRv03BZ2IcZPHSYnx4HzVkuDf8wQ/640?from=appmsg)

**中国移动现持有雷鸟创新7.92%的股份。**

**作者 |**  陈骏达  

**编辑 |**  李水青  

智东西1月5日报道，今天上午，国内AR智能眼镜企业雷鸟创新宣布，已于近期完成新一轮**超10亿元**融资。

本轮融资由中国移动链长基金与中信金石（旗下茅台金石、金石成长等基金）共同领投，中国联通旗下联创创新基金等机构参与。这标志着**国内运营商首次以战略投资者身份进入智能眼镜赛道**。

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgzh4CaNCMxPYapEFQTex0u2Q1IrX1jS1TwYWwPjohB7nZUd4DeGa9mwftssRUCHGlZZVuaKNczcg/640?from=appmsg)

▲雷鸟创新官宣融资海报（图源：雷鸟创新）

值得注意的是，雷鸟创新曾在2025年11月7日宣布完成由中信金石领投的一轮融资，并称其**刷新了2025年国内AI+AR眼镜赛道单笔融资纪录**。今天官宣的消息，正是对那笔融资进行了正式披露，**明确了其规模“超10亿元”，并公布了更完整的投资阵容。**

企查查信息显示，这轮融资在今年12月24日完成交割。交割后，中国移动通过两家公司总计持有雷鸟创新**7.92%**的股份，中国联通通过一家公司持有雷鸟创新**1.188%**的股份

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgzh4CaNCMxPYapEFQTex0urOxIRYZFIgXsLGMEUsx6iaKKGNyjskZibxENsVkMhKOwiaJJP3Ux9DLtQ/640?from=appmsg)

▲雷鸟创新本轮融资部分投资方（图源：企查查）

融资之外，雷鸟创新还宣布，将与运营商建立**战略合作关系**，后续双方将围绕市场渠道共建、商业模式创新与核心技术生态展开合作。雷鸟创新计划借助运营商的线下服务网点和数以亿计的用户基数，让雷鸟AR智能眼镜将触达更广泛的消费群体。

此外，雷鸟创新称，**“充话费送眼镜”、“合约机”**等运营商擅长的服务模式，有望大幅降低用户对AR智能眼镜的尝试门槛。运营商在**eSIM、5G等关键通讯技术**，AI服务及内容生态等方面的资源，也将为雷鸟创新的产品演进提供支撑。

即将在CES 2026亮相的**雷鸟X3 Pro Project eSIM**，正是双方合作的成果。该产品采用双目全彩MicroLED显示方案、集成了eSIM通信模块与4G协议支持，无需手机联网，即可实现包括拨打和接听电话、多模态AI对话、AI实时翻译、在线流媒体音乐等功能。

雷鸟创新称，这是**首款真正具备独立通信能力的消费级AR智能眼镜**，此前，智能眼镜始终被诟病为“手机的附属品”，eSIM的加入将使眼镜进化为“完全体”。

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgzh4CaNCMxPYapEFQTex0ucpicZNkAKXqfuEbuGxo5ONd3AcluORVVrbuLz6ZIErgrnsK35BEqwlg/640?from=appmsg)

▲雷鸟X3 Pro Project eSIM概念图（图源：雷鸟创新）

雷鸟创新成立于2021年10月，在近眼显示光学设计、自研AI算法模型，以及多模态人机交互等领域有技术积累，有核心光学方案全链路自研及量产能力。

市场占有率方面，Counterpoint Research的数据显示，2025年第三季度，雷鸟创新在全球AR智能眼镜市场中的份额达到24%，**连续两个季度实现出货量全球第一。**

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgzh4CaNCMxPYapEFQTex0upicItZJvCWPtvzKjgaibruR83cRuTz2w8458ibvqlOzF2Bo8DI9TZ5Xcg/640?from=appmsg)

▲图源：Counterpoint Research

雷鸟创新称，截至2025年上半年，其产品已覆盖全球超过25个国家和地区，累计用户量突破50万，其中海外市场销量同比增长3.8倍。

  

**结语：AI眼镜热度持续攀升**

**运营商押注或成新变量**

最近1年，AI眼镜赛道的热度持续攀升。一方面，国内AI眼镜赛道频现大额融资，雷鸟创新、XREAL、李未可、影目科技、闪极科技等企业都曾官宣数千万到数亿元不等的融资。另一方面，小米、联想、百度、阿里、理想等大厂也扎堆入局这一赛道。

雷鸟创新的此次融资，及其与运营商的技术和商业合作模式的探索，或许会给这一赛道带来新变量。
