大聪明老师的雄文我们已拜读：[Token，应该翻译成「新智元」](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247514440&idx=1&sn=a2386d0274b3ec515aa428a3e281130d&scene=21#wechat_redirect)。

认真思考和讨论过后，我们认为，Token 翻译成「智元」，看起来确实是一个「信达雅」的翻译，我们也乐于顺势而为，为之站台！

智元（Token）已成为 Agent 时代的主角之一，开始从晦涩的专业术语走进寻常百姓家。

如今消耗着全球最多智元（Token）的中国，理应给它一个统一的中文名。

**OpenClaw 带火了智元（Token）**

刚刚闭幕的英伟达 GTC 2026，吸引了全球目光。

除了爆火的 DLSS 5，黄仁勋用了相当大的篇幅，强调现在已从 AI 训练时代步入 AI 推理时代，数据中心成了智元（Token）中心，也将英伟达冠以智元之王（Token King）的名号。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYXykEUVCHq6HoJLxJg4xVTtu5W7sW2NnBdFfck0FeSItTEUKXTB8eCZbrdBVPpwITNpvKVZ0zRXtaibmU3DibeVm700OPaTicprdQ/640?wx_fmt=png&from=appmsg)

促进产业格局大转变的最大变数，莫过于 OpenClaw。

今年 1 月份横空出世带火 Mac mini 的这只小龙虾，成为今年 3 月份中国绝对的 AI 舆论场主角。

有人觉得是革命，有人觉得是炒作。

除了安全问题外，最大的争议之一，是其背后调用的 API 以惊人的速度消耗着的智元（Token），真正让人第一次体会到什么叫花钱如流水。

有多少智元（Token），就有多少智能；

有多少美元（或人民币），就有多少智元（Token）。

2024 年 12 月，ChatGPT 刚推出 200 美元 / 月的 Pro 会员时，大家没怎么见过哪个 To C 的软件的订阅费如此高昂，都在观望到底有哪些大冤种会捐这么多钱给 OpenAI。

到了今天，如果你想要用 OpenClaw 体验到真正的最强生产力革命，昂贵的 Claude Opus 4.6 会让你在历尽千辛万苦终于配置好自己的最强小龙虾准备起飞时，用直观的金钱成本让你浅尝辄止：随便说了几句话，几百人民币就烧完了。

Agent 没有在今天引发大规模失业，主要原因之一是它们的工资比很多人高。

但潮水已经越来越近、越来越高了：英伟达的最新旗舰 GPU 架构 Rubin 的推理成本，仅是上一代 Blackwell 架构的 10%。

**黄仁勋的秘密闭门会**

GTC 刚闭幕后的英伟达总部，黄仁勋举办了闭门会议。

PPIO 派欧云 CEO 姚欣，参与了本次会议，分享了以下会议内容。

黄仁勋将近年 AI 发展划分为三个阶段：生成式 AI 革命（ChatGPT 开启）、推理革命（OpenAI o1 开启），以及现在的 Agent 时代。

他认为，OpenClaw 的出现是 Agent 时代真正走向大众的标志性节点——只需一行代码即可部署一个会自主学习、使用工具、不断尝试直到完成任务的 Agent。

作为 OpenClaw 风靡全球的最大受益者，黄仁勋毫不掩饰地为其摇旗呐喊，甚至称之为第二个 ChatGPT 时刻。

**Token 的翻译问题**

之所以 Token 难以翻译，有两大原因：

其一，单就计算机科学领域内部，Token 在不同子领域，既有截然不同的含义：

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYVN2kwy2DOPHyhBn1paicYaib6uX2oTmnUeemIt3YvicLrjduMYTibUjoL0loODLiaqu9MB8xtqRSDoiakduXjZXoiceywcLBsuYnTmJU/640?wx_fmt=png&from=appmsg)

图源：[token为什么今天才叫词元？](https://mp.weixin.qq.com/s?__biz=MzI2NDYzMjIzOQ==&mid=2247486575&idx=1&sn=c8e9b6e562ff730b556238c0cd367dae&scene=21#wechat_redirect)

其二，虽然 AI 领域早在 ChatGPT 诞生前的 2021 年，就为 Token 定下了一个译名「词元」，但这个源自自然语言处理领域的译名，随着 AI 的多模态能力迅速增强，在非文本模态中，「词元」已经脱离了本意，需要一个能够更恰当地反映其含义的本质的统一术语。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYXnFDXibJRyDau37x9IDJ9PslFBLNJVxBDePnLADxteCRAGfkdvKCic1tS8IPIPQT8MqVl6AG9BXDoW1gjvIMk3Pz5rELgryjIPE/640?wx_fmt=png&from=appmsg)

图源：[token为什么今天才叫词元？](https://mp.weixin.qq.com/s?__biz=MzI2NDYzMjIzOQ==&mid=2247486575&idx=1&sn=c8e9b6e562ff730b556238c0cd367dae&scene=21#wechat_redirect)

刚刚结束的英伟达 GTC 大会期间，Token 也因老黄不遗余力的捧场，再次成为热词；Token 经济学也成为进入公共讨论的又一全新概念。

这两天迅速引发广泛讨论的，是昨晚腾讯研究院发表的文章：[杨斌：确定token的中文译名，已经迫在眉睫了](https://mp.weixin.qq.com/s?__biz=MjM5OTE0ODA2MQ==&mid=2650996175&idx=1&sn=af7c222e2ddd35a213fcff16e510c451&scene=21#wechat_redirect)，为 Token 提出了「模元」这个新译名。

各路 AI 大佬中意的新名字，也在 AI 交流群中互相碰撞。

很快，「智元」这个名字浮出水面。

**为什么「智元」是 Token 的好译名**

以下内容主要引自赛博禅心。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**智——对应 Token 的领域属性**

它服务于智能计算，是 AI 系统处理信息的基本载体。

这个字把术语的适用范围锁定在了正确的领域。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**元——对应 Token 的原子性**

中文里「元」的语义谱系非常清晰：元素、单元、元数据、元认知，每一个都指向不可再分的基础构件。

这个字也是目前大家确定下来的最主流共识。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Rvq8Ow69CYUJmElOmHC0N4sibbWqKMEPS17icJfxD1PT4ficoedRj0l0LovmcAqtibq0ib1IMGk5Soibd5SKibv949ZuUPXibnqmj7u65Pzpf6TfeKc/640?wx_fmt=jpeg&from=appmsg)

我们建议，将 Token 译作行业新共识：「智元」，把「新」字留给我们。
