# DeepSeek V4真要来了？万亿参数模型匿名开测，免费跑龙虾

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

机器之心编辑部

  

昨天晚上，推特博主「青龍聖者」发了一张截图，似乎「预示」了 DeepSeek V4 马上要来。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEibdsJsbmlyLuwFVxyUrlvC92vKQSWdnF6u6ibBYhRCDZZ2fnIkJjgeshvSp4u3HHljsXU2pDkqD9kYyowicS055GXsAEa4ufwpE/640?wx_fmt=png&from=appmsg)

  

不过紧接着，这位博主进行了澄清，表示自己不确定上述截图信息是否真实，并判定另外一张显示 DeepSeek V4 更详细参数的截图为假消息。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGsr83I736nuqTUbez4xmVQibJPt6IKU1lnWjCwauXsfjtHSSvuFNTsbLSSJ9anjLtWekZErOBzNsqXfr7sjjvRXCrD6OJBZ3DU/640?wx_fmt=png&from=appmsg)

  

而就在今天早上，全球知名大模型聚合平台 OpenRouter 突然上新了两个神秘模型，分别是「Hunter Alpha」和「Healer Alpha」。

  

前者为 1  万亿（1T+）参数和 100 万 token 上下文的大模型，专为智能体工作流、长周期复杂任务和深度工具调用设计；

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFgApOVQG4pn5o6WyN0fiaclwqHkOhzzbECzoeebUVjlIfs6xL6zWOHDiazx9EicbN5dq7AR9ZkParPYaSoa9bSVmdpk8WDpjZp5k/640?wx_fmt=png&from=appmsg)

  

后者为全模态交互模型，上下文窗口为 26 万 tokens，能够原生感知视觉和音频输入，跨模态进行推理，并以精确和可靠的方式执行复杂的多步任务。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqF5szNZYt9e3Bt5FHR5D4foFb0I4Mr7iayjBgMzD4RofuLA5TUFNvSZg8eS6kSiagFhGQ6T2opv3Is2UxYxdicJ5NV2qGwzMliagib8/640?wx_fmt=png&from=appmsg)

  

其中，「Hunter Alpha」还专门强调，它具有 OpenClaw 等「龙虾」框架所需的可靠性和指令遵循精度。OpenRouter 的统计也显示，不少用户已经用这两个模型在跑「龙虾」了。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqH6icnB57UIPGgMMiaEmFyUTxeYB6K3oyU5L4FRwIuhhVWnpgUNfZNsWsvYGEI9J86705YGia0YNIYUYrGQoqYebKuLsGotgEUh8Q/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHsRq6RFjibg001AtPjUuaiaKF9UcNuxINljAsjzcFyHYrGBrA7t989AibsUV9u10p3lv8M0ZEKFSYPOy4licUo3GZicniaxS3ticGEuQ/640?wx_fmt=png&from=appmsg)

  

现在，和所有其他「隐身模型」一样，这两个模型都是免费的，并且模型提供商会记录它们的所有提示词与生成结果，这些数据可能会被用于改进模型。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHXqzSzoAFAx5KfhtjVqwiajXtgVa3SjItlEbU4NiaQMicFTY0GO2dC6rupY9pg0WwciccullkZhhicgvWsoficMADiazEHGPo1iaVGcC4/640?wx_fmt=png&from=appmsg)

  

它们到底是谁家的？

  

这两个神秘模型登场之后，社区里关于它们「到底是谁」的猜测便蔓延开来。

  

有人从对话风格入手，说 Healer Alpha 用起来有 Gemini 的「Google 味」，但也有人说它某些方面像 Claude。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqF2zAKCW69jOmQqXdQO8Tic6fGricBSR0HOOWFAgWTBuawSh1SQoeyIBS144CG8XnWiaWa2Z0H7FbsicsdAC9WN4VgVnPGNC5P8JYQ/640?wx_fmt=png&from=appmsg)

  

还有人猜测它们是两款中国模型，其中 Healer Alpha 很可能是 DeepSeek V4 或 V4 Lite。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqH2DeUEnRCKNNJBibw15FRMPzOlqjTNNum1edTn0avFQy0pEhQa1wEtbDbgibgENoveDYIKqLlX6XtA4tickw3rlqCkibatJmydGRQ/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGVfnpI3j9l2xkFG7wH3Xz832dqY4HOw3y33jk2LiaRW4dOmAgtwHGt7UXeEPFU63lLEJWCiceEameD7CrViaSouaKGibPn3Tq33wg/640?wx_fmt=png&from=appmsg)

  

对于 1T 参数版本的 Hunter Alpha，猜测对象更多了，除了 DeepSeek V4，还有 Kimi K3、Grok 4.2、Claude 以及 GPT 5o 或 5.5 instant。毕竟，这么大的参数量和上下文组合，目前看来只有几家大厂能做到。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqF7EoFG1ibnfseqvlWhFwvHGibOpI6u6BBDYNcYTvVz70Ql4uWzVs2Jiab7ruVLwvdEEibNWW1wNmhEicD16gH8IE0tSE46ibghiacdSc/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFcNaPHib4YEibE8JYiaULGViam8ViaqNSMlFCw1QRCg2IumYoXkSBib3yknXzFdG9ndRz7sltibQE7QLHxzKjxvo8uLtOLDk01E4ukBQ/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEdVGqvufGueGMtYlZic8qicT4nXqDSo3fWicvsD41GxOsnYdGUR1UhTHW940yN8ibNmxBvredalfKgGXxiacsdcKOvPIrHQIt4ibicQs/640?wx_fmt=png&from=appmsg)
