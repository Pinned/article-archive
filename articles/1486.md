# Anthropic一篇博客，IBM大跌13%，创25年来最大单日跌幅

##### 听雨 发自 凹非寺  
量子位 | 公众号 QbitAI

Anthropic又搞事情了。

就在昨天，IBM股价**下跌13.15%**，创**2000年以来最大单日跌幅。**当天市值从2408亿美元跌至约2087亿美元，蒸发了约310亿美元，成为人工智能技术飞速发展下的“最新受害者”。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGkRc9Q6gVzEuPhgvEdj00XMxQChqbGvgk5pN619RxvomJqOGq2jrAAonDD5YzjAhhWwKUtfia4zibJiaM1XojibbbeUupv6r06soo/640?wx_fmt=png&from=appmsg)

原因是啥？仅仅因为Anthropic发了一篇新的博客文章，介绍旗下的Claude Code工具可用于对运行COBOL语言的老旧系统进行现代化改造。而COBOL相关业务正是IBM的核心业务之一。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHwWnda9MNRUI0wicHh6icptl8iaP1MWmib9O1Wy3pMyRibYdDv6to44oarmBub4H1FBneiaAQPRU2icEnHSo90CkjyTmTXR0UHQhqaFs/640?wx_fmt=png&from=appmsg)

长期以来，IBM一直销售针对大规模交易处理优化的大型主机系统，这类系统普遍使用**COBOL语言**——这种编程语言比目前科技行业普遍使用的语言要古老得多。

Anthropic一出手，IBM股价抖三抖
---------------------

COBOL是“通用面向商业语言”（Common Business-Oriented Language）的缩写，是上世纪50年代末开发的主流代码系统，广泛用于商业数据处理，例如支付处理和零售交易系统。

Anthropic在其播客文章中表示，**美国约95%的ATM交易都依托COBOL运行**，每天有数千亿行COBOL代码在生产环境中运行，支撑着金融、航空和政府部门的关键系统。这也使其成为最具备成本优势的AI技术冲击的主要目标。

然而，每年了解它的人数却在减少。

开发这些系统的开发人员多年前就退休了，他们积累的专业知识也随之消失。生产代码几十年来不断修改，但文档却未能跟上。与此同时，新的文档并没有得到补充——COBOL语言只有少数几所大学开设，而且每个季度寻找能够阅读 COBOL的工程师都变得越来越难。

COBOL系统的现代化改造也是一个难题。遗留代码的现代化改造之所以停滞多年，是因为理解旧代码的成本比重写代码还要高：需要大量顾问花费数年时间绘制工作流程图，这导致工期漫长、成本高昂，很少有人愿意承担。

但现在，**AI扭转了这一局面**。

据Anthropic介绍，Claude Code可以帮助实现COBOL代码库现代化，自动完成其中最耗费精力的探索和分析阶段，例如：

*   绘制跨越数千行代码的依赖关系图；
    
*   没人记得的文档工作流程；
    
*   识别出那些需要人工分析师花费数月才能发现的风险；
    
*   为团队提供做出明智决策所需的深刻见解。
    

总之，借助Claude Code，团队可以在几个季度内而非几年内就完成COBOL代码库的现代化改造。

好巧不巧，运行COBOL的大多数大型计算机都是IBM制造的，IBM的大部分收入仍然与其大型机业务息息相关。对可靠性要求高的客户，例如金融或政府机构，仍然会购买大型机。

IBM的护城河不在于技术，而在于其他公司根本无法理解它。正是因为COBOL代码过于老旧、过于复杂、过于关键，所以才催生了庞大的咨询帝国。企业需要向IBM支付数十亿美元进行系统维护，否则系统可能面临灾难性的崩溃。

当Anthropic宣布Claude Code可以简化COBOL代码工作时，市场反应相当直接：IBM股价大跳水，**暴跌超13%**，创下自2000年10月以来最大的单日跌幅。

彭博社汇编的数据显示，受此影响，IBM股价在2月份已下跌27%，有望创下至少自1968年以来最大的单月跌幅。

有网友在X上表示：之前每年要花210万美元请IBM来升级COBOL系统，现在用Claude Code只需9个小时就完成了，原有的团队也解雇了。

市场反应是否过度了？
----------

惊叹之余，也有人质疑，Anthropic一篇博客文章就引发如此大的影响，**投资者的反应是否过度了**？

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGFmJJbuKVx9dpeSpGX0a6LVibBno32jXqO3aORE4rAur8fibxOk7zpsLBShUvdGbvSWdqicsTcO992GCMWhbiau6tYialLXvC4efCK8/640?wx_fmt=png&from=appmsg)

**IBM的护城河，或许没那么容易被跨过。**

事实上，用AI来简化COBOL编程工作并不是一件新鲜的事情。早在2023年，IBM自己就推出了一款名为“watsonx Code Assistant for Z”的AI工具，用于将COBOL代码转换为Java代码，作为应用程序现代化工作的一部分。

这款工具旨在帮助开发人员评估和确定最需要现代化改造的代码，从而使他们能够更快地更新大型应用程序并专注于关键任务。

但IBM的目的在于**希望改造后的代码仍运行在自己的软硬件体系上**。吸引并留住客户使用IBM大型机，是IBM带动其他软件与服务收入增长的核心方式。

上个月，IBM首席财务官James J. Kavanaugh在公布最新财报后表示：“我们Z系列大型机的装机优势，为集团整体带来了极具吸引力的3到4倍的产品乘数效应，驱动增长飞轮。”

此外，AWS、微软及IBM的前基础设施部门Kyndryl、NTT也都推出过大型机迁移计划。例如AWS在2022年推出大型机现代化计划，提供了一套完整的开发和运行时环境，帮助企业将其基于大型机的工作负载迁移到云端。

还有从IBM中剥离出来的前基础设施部门Kyndryl，也曾于2022年与微软合作，以实现大型机系统（包括Kyndryl的zCloud平台）与微软Azure云之间的数据管道，旨在让客户更容易将存储在其大型机上的数据迁移到云环境中进行分析。

但所有这些，**都没有动摇IBM的大型机业务**。上个月，IBM公布了2025年全年业绩，显示**大型机销售额大幅增长**。

IBM的第四季度持续经营业务营收达197亿美元，同比增长12%；净利润达56亿美元，同比增长91%。全年来看，营收增长8%至675亿美元，净利润飙升76%至106亿美元。

IBM首席执行官Arvind Krishna表示，GenAI业务规模目前已超过125亿美元，其中软件业务超过20亿美元，咨询业务超过105亿美元，这两项业务的季度增幅均创历史新高。

Krishna将部分原因归功于IBM在2023年大力推广的AI代码转换工具。他还表示，对于某些工作负载而言，大型机仍然提供最低的运营成本。

此外，也有网友提出：阅读COBOL代码对AI来说很简单，真正的难点在于**理解业务逻辑**。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFibtmkubUKibzIoib0QNuLmXw61FZibtH86vSfQ8rF9Cum6vMzEU6F2z2t6onch5VoQx6TnlxFFQX814v1R4FvHvrTOCRBao1zibibM/640?wx_fmt=png&from=appmsg)

由于COBOL已经存在了60多年，许多编写这些应用程序的开发者如今已经退休或去世了。其中隐藏着很多老旧的业务逻辑，比如欺诈规则、费率计算、特殊情况处理等等，要理解它们的真正含义，还是一个需要**人为解释**的问题。

IBM方面也曾表示，他们多年来一直致力于代码现代化，watsonx代码助手就是他们借助AI能力的一次实践。但**AI无法改变大规模运行关键任务工作负载这一根本性的工程挑战**：

> COBOL代码的转换只是第一步，真正的挑战在于数据架构的重新设计、运行时环境的替换、事务处理的完整性以及基于数十年来紧密软硬件耦合而构建的硬件加速性能。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGF29pPy0yYdpicHzSdWSeN51eicricJjCQL74I7aXYK0XB9YVWxhNdLFryAnh8nZLR1eJ4jOhdN6K5hFAOHfT4C4hWiayIgvQmNIYg/640?wx_fmt=png&from=appmsg)

搅动股价的事，Anthropic已经干了三回
----------------------

无论Claude Code到底能不能动摇IBM的大型机业务，搅动美股股价这档子事，**Anthropic已经干了三回**。

2026年，我们已经多次看到这样的故事：Anthropic发布了一款新的AI工具，Claude在编码和工作流程自动化方面有了显著的进步，**几个小时内，目标行业的市场就崩溃了**。

细数一下——

1月30日，Anthropic发布Claude Cowork生产力工具的11款行业插件，覆盖法律、金融、销售等领域。AI直接接管合同审查、财务分析等核心工作流，传统软件“按席位收费”模式遭遇挑战，引发全球软件股剧烈震荡。

软件股单日蒸发约2850-3000亿美元市值，汤森路透（-16%）、Salesforce（-44.40%）、Adobe（-42.27%）等企业股价跌幅达15%-61%，高盛软件指数创自去年4月以来最大单日跌幅。

当时的市场恐慌还历历在目：网上铺天盖地流传**“Saas已死”**言论，华尔街对软件股的悲观情绪，已经彻底演变成了**末日论调**。

紧接着，就在上周五，Anthropic发布了其Claude Code软件新增功能**Claude Code Security**，称该功能可以扫描代码库中的安全漏洞，并找出软件漏洞供人工审查。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGFv3BbHfw1nriaYxgv10K2Ghf7JlP78oOfetRZwpKzzlNjrYruQdefcuh0iaFWxfmXJZ4RoQHu8j0NaxTRYPq2OKwHibBibExBWvuc/640?wx_fmt=png&from=appmsg)

受此消息影响，**多家网络安全公司股价暴跌**：

*   周一，CrowdStrike和Zscaler的股价分别下跌约10%；
    
*   Netskope和Tenable暴跌约12%。
    
*   SailPoint下跌 9%，Okta跌幅超过6%；
    
*   SentinelOne和Fortinet的跌幅均超过4%；
    
*   Palo Alto Networks最新下跌3%；
    
*   Cloudflare（此前受益于近期Moltbot热潮）股价下跌超过 9%；
    
*   iShares网络安全与科技ETF下跌约 5%；Global X网络安全ETF跌至 2023年11月以来的最低水平。
    

从传统Saas领域，到网络安全，再到软件咨询，股价纷纷遭受Anthropic的重创。

现在，一个问题摆在了所有人面前：AI还会重构多少所有代码、系统和流程？
