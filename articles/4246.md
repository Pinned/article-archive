传魅族手机业务实质性停摆，3 月正式退市，李楠谈 2 年前重振魅族失败

多位知情人士表示，魅族手机业务已经实质性停摆，将于 2026 年 3 月正式退市。此外，魅族旗下 FlymeAuto 车机业务将独立运营，魅族品牌或将继续保留在吉利体系内。星纪魅族的飞书大群目前还剩 1000 多人，但有不少员工已提出离职，少量员工已转岗至吉利体系内的极氪汽车。

2026 年 1 月，星纪魅族集团中国区 CMO 万志强在一场活动上表示，内存的大幅上涨对手机商业计划是巨大的冲击，魅族 22 Air 将取消上市计划。他在当时透露，魅族 23 仍将在 2026 年发布。但据了解，该项目已经不再实质性推进。有接近魅族的人士透露，从去年 4 月开始，魅族已经有很多供应商的款项无法正常结算，至今仍有大量欠款已成为坏账，“魅族的结局大概率是申请破产。”截至发稿，星纪魅族方面尚未对此作出回应。

不过，前魅蓝总裁、怒喵科技 CEO 李楠最新发文表示：“两年多以前我是做过一个秘密计划，关于怎么重振魅族的。但是后来并没有被彻底执行，最主要的原因是第一步就是非常激进的裁员......只留下几百人。我知道没有深刻的2c市场竞争教训的领导层是不会执行这么激进的计划的。但是我也知道这是拯救这个品牌的唯一正确的道路，如果你没有精锐、敢干，并且团结的选锋为精锐，那么什么战略战术方向都无从谈起。” (界面)

刘强东要造游艇了：像汽车一样走进千家万户

刘强东个人投资 50 亿元创立游艇品牌 Sea Expandary，2026 年初宣布布局新能源游艇全产业链。品牌已与深圳、珠海政府签约，将在珠海建制造基地、深圳设总部，并参与码头及配套建设。Sea Expandary 计划推出 10 万元级新能源小艇，利用风光自然充电，搭载 AI 与机器人技术，取消船长厨师，仅保留服务与安全人员，目标让游艇“像汽车一样普及”。目前已获 5 艘 72 米双体船订单，单船售价 6000 万欧元。（快科技）

小米起诉自媒体账号赔偿 500 万，实控人为蔚来李斌？相关人士：假的

小米法务部发布信息，自媒体账号“AutoReport 汽车产经”因侵权，被判赔小米 500 万元。该账号认证主体为北京易车互动广告有限公司，有网络信息称，其实际控制人是蔚来创始人李斌。相关知情人士表示，涉诉账号由北京易车信息科技有限公司 100% 控股，李斌 2020 年已将相关股份全部转让，与涉事账号主体无任何关联。相关知情人士表示，目前易车的实控人已经不是李斌，网络上流传的股权信息，是 VIE 架构下股权变更信息没有更新。（新浪科技）

网友称苹果手机深夜自动外拨电话，官方建议升级至 iOS 26.3

iPhone 自动打电话话题登上微博热搜。有网友称自己的 iPhone 凌晨两点自动拨出白天联系过的快递员电话，当时手机处于桌面充电、锁屏、关闭网络且开启睡眠模式的状态，系统版本为 iOS 26.1。部分用户建议升级系统解决，Apple 专家也建议将系统升级至 iOS 26.3 以修复异常。早在去年 11 月及 2024 年就有网友反映类似问题，苹果客服曾称问题可能与多设备登录同一 Apple ID、开启「同步通话记录」有关，添加通话相关小组件、开启语音拨号等功能也可能导致该情况。

OpenAI 星际之门搁浅，一年过去未动工

OpenAI 星际之门项目遭遇滑铁卢。2025 年 1 月，OpenAI 宣布与软银、甲骨文组建合资公司，计划 4 年投入 5000 亿美元兴建数据中心，但推进遇麻烦，三方在分工和架构安排上分歧不断，OpenAI 自建数据中心计划也搁浅，至今项目仍未开工，团队未配齐，且仍是单方面合作。

而马斯克的 xAI 进展迅速，第一厂 Colossus I 建设高效，Colossus 2 已成为全球首个 GW 级超算集群，还布局太空算力概念，收购 xAI 并考虑租用特斯拉闲置算力。不过，数据中心建设过快也带来问题，xAI 数据中心因污染和噪音遭投诉，美国部分地区可能因数据中心用电需求激增而停电。Sam Altman 在活动上回应数据中心能耗问题，认为应对比模型训练完成后回答问题与人类回答同样问题的能耗，按此口径 AI 能效或已追平甚至超过人类。

Anthropic 舍弃标志性安全护栏承诺

以高度重视安全著称的人工智能公司Anthropic，正对其安全承诺作出重大调整。该公司当地时间周二表示，为在与其他 AI 实验室的竞争中保持优势，已对核心安全政策进行“软化”。此前，若其模型被判定为“具有危险性”，Anthropic 将暂停相关开发工作，多年来该公司高层反复强调这一承诺。但Anthropic 如今表示，一旦竞争对手发布了“同类或更先进”的模型，将不再执行这一暂停机制。

OpenClaw 之父给开发者的建议：保持玩乐心，去构建梦寐以求的东西

2 月 24 日，OpenClaw 之父彼得·斯坦伯格在 OpenAI 的节目中表示，建议开发者保持玩乐心，去构建梦寐以求的东西，“如果你具备高行动力且够聪明，市场对你的需求将比任何时候都大”。

OpenAI 与 Anthropic 毛利目标双双落空，非付费用户与算力成主要负担

AI 两大头部企业 OpenAI 与 Anthropic 盈利路径面临严峻考验，均未达成毛利率目标，推理成本超预期增长是主要拖累因素，这在用户规模扩张背景下尤为突出，引发外界对其本十年末实现 60% 以上毛利率目标的质疑。OpenAI 去年毛利率降至 33%，低于预测。Anthropic 预计 2025 年毛利率为 40%，仍低于目标。推理成本是毛利率承压核心原因，OpenAI 去年推理成本同比增约四倍，Anthropic 2025 年预计增长逾三倍。此外，OpenAI 毛利率压力还来自庞大非付费用户群体和产品结构因素。不过，其服务付费用户效率已有提升，还计划优化收入结构。长期来看，OpenAI 预计到 2030 年付费用户推理成本比例升至约 94%，毛利率目标约为 67%，但如何在成本攀升时实现目标仍是核心挑战。

Unity 考虑出售中国业务，估值或超 10 亿美元

据彭博社报道，知情人士透露，Unity Software 正在评估其中国业务的多种战略选项，其中包括潜在出售的可能。报道称，这家总部位于旧金山的实时 3D 技术公司已聘请顾问评估 Unity 中国的市场。相关人士表示，Unity Software 对该业务的目标估值可能超过 10 亿美元，约合 68 亿元人民币。由于事项尚未公开，消息人士均要求匿名。目前，Unity 尚未就中国业务相关战略选项作出正式公开回应。

苹果股东会：库克 5.1 亿年薪获批，两超龄董事罕见留任

苹果公司当地时间 24 日召开 2026 年度股东大会，批准公司高管薪酬方案，CEO 蒂姆・库克 2025 财年实际总薪酬 7430 万美元，处于近三年区间中位。董事会成员均获连任，包括超 75 岁的核心成员。续聘安永为 2026 财年独立注册会计师事务所。批准修订后的《非员工董事股票计划》。此外，未宣布库克接任董事长，也未释出首款苹果折叠手机更多信息。

**程序员专区**

阿里云上线 Qwen3.5 等四大开源模型

阿里云百炼推出包含 Qwen3.5、GLM-5、MiniMax M2.5、Kimi K2.5 四大开源模型 API 服务的 Coding Plan。用户订阅套餐后不再受限于单一模型，可实现多模型自由切换，享受更稳定、Tokens 额度更高的模型服务。目前，全球云厂商中仅阿里云提供这一服务。

GPT-5.3-Codex 编程开发模型已全面开放，开发者可以通过 API 调用

OpenAI 于 2026 年 2 月 5 日推出编程开发模型 GPT-5.3-Codex，初始阶段仅限 ChatGPT 付费订阅用户在部分平台使用，次日部分开发者可通过 API 调用。目前该模型已全面可用，所有开发者均可通过 OpenAI API 等平台调用。其亮点包括速度提升、支持中途交互引导、多语言测试、通用自证、大上下文窗口、全新参数等。官方平台 Responses API 定价为每百万输入 1.75 美元、每百万输出 14 美元、每百万缓存输入 0.175 美元。

Ladybird 浏览器项目将在 AI 帮助下使用 Rust 语言

Ladybird 浏览器项目宣布将在 AI 帮助下使用 Rust 语言。Ladybird 是非盈利组织 Ladybird Browser Initiative 开发的开源浏览器，计划在今年内发布一个 alpha 版本，2028 年发布正式版本，它最初使用的语言是 C++，开发者表示他们一直在寻找一种内存安全语言替代 C++，他们在 2024 年评估过 Rust，但因为它在 C++ 风格的面向对象编程(OOP)上表现不佳而放弃，但一年之后它还是决定采用 Rust，而 Firefox 和 Chromium 都已开始在其代码库中引入 Rust。Ladybird 将首先用 Rust 重写部分代码，第一个目标是 JavaScript 引擎 LibJS，开发者在 AI 辅助编程工具 Claude Code 和 Codex 帮助下完成了 2.5 万行的代码。Rust 将主要用于开发子系统，浏览器引擎仍然继续使用 C++ 开发。

Perplexity Computer 上线：多模型协同自动完成工作流，月费 200 美元

Perplexity 推出了全新聊天界面“Perplexity Computer”，该界面将多款具备自主执行能力的 AI 模型整合为一套系统。其与 Claude Cowork 类似，但基于浏览器运行，且可调用来自不同厂商的模型，能够自主完成完整工作流程。用户只需描述想要达成的结果，系统便会自动创建子智能体，执行网络搜索、文档生成、数据处理或 API 调用等任务。Perplexity 表示，AI 模型正变得越来越专业化，因此一套完整的工作流程需要调用各类模型。

Perplexity Computer 目前以 Opus 4.6 为核心模型，同时搭配 Gemini、Grok、ChatGPT 5.2、用于图像生成的 Nano Banana 以及用于视频处理的 Veo 3.1。每项任务都会拥有独立的安全环境，包含浏览器、文件系统和工具接口。该功能已纳入 Max 订阅套餐，每月费用为 200 美元。
