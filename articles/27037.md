# 科技大厂，超2万人没工作了

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuqbXTWNoAvnhpwe8oDwbu7K6R7nVmdnv8dNo0UL5I6rgTia5ShZx5QcKuBJOxL6Y0oK2Z5icM54KuYZuia5sAjolyEh9NEyrDDLYI/640?from=appmsg)

**大裁员背后是为筹措AI数据中心建设资金而采取的财务操作。**

**作者 |**  陈佳  

**编辑 |**  云鹏  

智东西6月23日消息，美国数据库与云计算巨头甲骨文（Oracle）6月22日向美国证券交易委员会（SEC）提交的2026年报（10-K文件）披露，截至2026年5月31日，甲骨文**全球员工总数**约为14.1万人，较上一财年同期的约16.2万人**减少约2.1万人**，**降幅达13%**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acup9pBmJFOwaXNCeT80pbhZcHQyEupOXWxjcPSkISOxylwlYaPoLrnUb6gmjibKdJBtmjPO6huCW9IsicuA1jotYiaEPuS14j0EMGk/640?from=appmsg)

▲甲骨文公司2026年10-K年报申报文件首页（图源：美国证券交易委员会）

甲骨文在2026财年为重组活动支付的**遣散费**和其他人员**安置费用达到18.4亿美元**（约合人民币124.8亿元），是上一财年3.74亿美元（约合人民币25.4亿元）的**近5倍**。

甲骨文称，此次人员调整由多重因素共同促成，涵盖管理层与产品架构调整、绩效不达标、战略转型及并购整合等。

多家外媒报道，甲骨文于今年3月31日**凌晨6时通过一封邮件，一次性解雇了约2万名员工**。

另据TechCrunch 5月9日报道，曾有一批员工联合起来和甲骨文谈判，至少**90人签署公开请愿书**，敦促甲骨文**对标其他以AI为由大规模裁员的科技大厂**，给出同等**离职补偿待遇**。

遣散方案的争议之处不仅在于补偿金额偏低，还牵涉**限制性股权激励**（RSU）的**强制没收**。对部分资深员工来说，股权补偿构成其总薪酬的绝大部分，被裁时损失动辄数十万乃至数百万美元。

据美国《时代周刊》4月31日报道，一名在甲骨文工作30年的技术培训师Jill称，裁员让她**一夜之间失去了约30万美元**（约合人民币204万元）原本即将到手的股票奖励，原定退休计划也因此被打乱。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcupvWuAqvWNRK6TUsRoJwNaHuJL65xA7AGQRAPkq8PWkHpPaicBFjJTHxSPqQM8gn298BZzHzjhVRQ3xmVsQydIPlUMXKI2xXEI0/640?from=appmsg)

▲2007年6月26日，位于加利福尼亚州雷德伍德城的甲骨文公司总部大楼外景（图源：美联社）

据路透社报道，甲骨文长期在云计算行业处于第二梯队，近数月接连与OpenAI、Meta签订大额数据中心合作协议，以此强化与亚马逊、微软等竞品的竞争实力。

但与上述依靠充沛经营性现金流支撑大额投入的科技巨头不同，甲骨文只能依靠消耗自有现金、增发债务筹措资金，其股价年内累计下跌约10%。

甲骨文本月初在财报电话会上称，2027财年净资本开支预计约为700亿美元（约合人民币4746亿元）。为筹集这笔资金，甲骨文将通过债务和股权融资再募集约400亿美元（约合人民币2712亿元），其中包括此前已宣布的200亿美元（约合人民币1356亿元）股票发行计划。

这场大裁员背后，是甲骨文**为筹措AI数据中心建设资金而采取的财务操作**。彭博社2026年3月31日报道，甲骨文在年初新增了约580亿美元（约合人民币3944亿元）债务，并计划在本财年投入500亿美元（约合人民币3400亿元）建设AI数据中心。

华尔街投行TD Cowen估计，甲骨文此轮**裁员每年可释放80亿至100亿美元**（约合人民币544亿至680亿元）**现金流**，直接用于支撑上述基础设施投入。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuoUTBtFCXC9OCnL400YZcF5ODBZiagqEDauLteO2SRHsdsahic5P2CB7fv9cMeh35CTxfnIfBDT3GadACbibykoV9MdSnERAmtl1I/640?from=appmsg)

▲OpenAI和甲骨文计划在威斯康星州150亿美元的数据中心站点（图源：Vantage Data Centers）

大幅裁员发生的同时，甲骨文在2026财年**录得创纪录的财务业绩**——总营收674亿美元（约合人民币4570亿元），同比增长17%；云服务收入340亿美元（约合人民币2305亿元），同比增长39%，云收入占总营收比例从上一财年的43%跃升至51%。

行业裁员追踪网站Layoffs.fyi数据显示，今年**美国已有196家科技企业累计裁员超11.98万人**，AI技术冲击引发的失业担忧持续升温。

  

**结语：AI资本开支失控后，**

**科技巨头开始向员工“要现金流”**

随着科技巨头持续加码AI数据中心、算力集群与云基础设施建设，企业资本开支正快速攀升，越来越多公司开始通过裁员、压缩福利与收紧股权激励等方式，为AI投入腾挪现金流。

甲骨文、Meta、微软、Cloudflare四家公司裁员条款的核心差异，集中体现在股权处理方式上。Cloudflare和微软均选择在解雇员工的同时，为其保留一段时间的股权归属；Meta的方案也包含现有归属周期内的股票连续。甲骨文是四家中唯一一家在被裁当日即强制没收全部未归属股权的公司。

这场裁员揭示出一个矛盾：在科技行业人才紧俏时，高薪与股权承诺是吸引员工的核心筹码；而当公司占据主动地位时，员工几乎没有任何制度性保障来兑现这些承诺。
