**国内要闻**

脉脉发布 80 家「隐形大厂」：平均月薪最高近 7 万

4 月 8 日，职场社区平台脉脉发布 80 家“隐形大厂”名单，涵盖智谱、SHEIN、地平线、Momenta 等企业，覆盖 AI、智能驾驶、机器人、芯片等热门赛道。名单显示，高薪岗位正加速从传统大厂向一批高速发展的“隐形大厂”扩展。报告指出，当前，人才市场的高质量机会正在发生变化。一批在招聘规模、薪酬水平、雇主口碑和行业影响力上已具备“大厂”实力、但在大众认知中相对低调的企业，正成为中高端人才流动的重要目的地。脉脉将这类企业定义为“隐形大厂”。

「隐形大厂」薪酬水平不输传统大厂，如 Momenta 新发岗位平均月薪达 69815 元，还有企业年薪 1 亿招聘「具身智能首席科学家」，非技术岗位也有竞争力。从城市分布看，北京集中于 AI 等领域，上海聚焦芯片制造与游戏，深圳以机器人等为特色，杭州在机器人、脑机接口方向突出。

阿里百度腾讯集体加入 AI 算力涨价潮，云计算终结 20 年降价史

AI 时代，算力将从普惠资源转为稀缺性「战略物资」。继阿里云和百度智能云之后，腾讯云也加入 AI 算力涨价队列，将于 2026 年 5 月 9 日起对 AI 算力等相关产品刊例价上调 5%。此前，阿里云、百度智能云宣布将于 2026 年 4 月 18 日起对相关产品调价，海外亚马逊 AWS、谷歌云也有提价动作，国内科大讯飞、商汤科技、智谱等也有价格调整。

涨价潮背后是供需失衡与产业转型双重推力，需求端 AI 智能体爆发式增长引爆算力消耗，供给端高端 GPU 芯片产能受限加剧市场紧张。云厂商正从「卖算力资源」转向「卖智能服务」，头部科技公司或提升盈利，中型企业面临两难，初创和中小企业面临更高「算力门槛」，普通用户长期将面临 AI 工具免费功能缩减、付费订阅价格上调。

抖音集团发布 2025 年全年反舞弊通报：50 余人因受贿罪被判刑

抖音集团发布 2025 年反舞弊通报，五十余人因涉嫌违法犯罪被移送司法机关处理。通报列举了多起员工利用职务便利进行商业贿赂、舞弊或职务侵占等案件，涉及 Data、电商、抖音、番茄小说等多个业务板块，相关员工行为违反员工手册，集团已作出纪律处罚并移送公安机关。此外，抖音集团将 23 家合作方列入永不合作主体清单，并附新增清单。

曝红帽 Red Hat 裁撤中国区研发团队，涉 500 人

4 月 9 日，据《云头条》获得的内部邮件显示，开源巨头红帽（Red Hat）通知中国区员工：停止在华全部工程活动，相关研发工作将转移至亚太其他工程枢纽。受影响员工从即日起不再承担日常工作职责，雇佣关系统一于 2026 年 7 月 31 日终止。  

另有消息显示，裁员方案为 N + 3 到 N + 6，预计裁掉 419 人。报道提到，红帽大中华区（含中国大陆、中国香港和中国台湾地区）目前拥有 700 多名员工，设有多个分公司和办事处，研发和服务团队 500 多人，其中北京研发中心拥有 300 多名软件开发与科研人员。这次调整，几乎等于将整个中国区的核心技术力量连根拔起。

**国际要闻**

OpenAI 推出 100 美元 ChatGPT Pro 方案，Codex 使用量提升 5 倍

OpenAI 周四宣布，推出每月 100 美元的 ChatGPT Pro 订阅方案，提高其人工智能编程助手 Codex 的使用上限，以加强与 Anthropic 旗下 Claude Code 的竞争。公司于社交平台 X 表示，新 Pro 方案的 Codex 使用量为现有每月 20 美元 Plus 方案的五倍，适合进行「更长时间及高强度」的编程任务。OpenAI 指出，Plus 方案仍适合日常稳定使用，而 100 美元 Pro 则为重度用户提供更易升级的选择。

推出新方案后，ChatGPT 个人订阅层级增至五个，包括 Free、Go、Plus，以及两个 Pro 级别。OpenAI 早前已提供每月 200 美元的高阶 Pro 方案。

拒绝成为 AI 训练集：Google 力推 Gmail 集成 Gemini 并承诺保护用户隐私

Google 宣布将生成式人工智能模型 Gemini 深度整合至 Gmail，以重塑邮件处理效率，承诺不利用用户个人邮件内容训练底层 AI 模型。Gmail 设计坚持安全第一，调用 Gemini 处理邮件在严格隔离环境完成，采用「私人房间」模式，任务结束后 Gemini 失去收件箱访问权，阻断数据外泄可能。集成后的 Gemini 能实现邮件润色等多项辅助功能，Google 借此吸引用户，强化「不训练、不留存」底线，为 Gmail 用户构建数字信任防线。

马斯克对 xAI 工程团队进行大规模重组

在 SpaceX 历史性 IPO 前夕，马斯克的人工智能公司 xAI 与 SpaceX 进行更紧密整合，并对工程团队进行大规模重组。SpaceX 星链项目高级副总裁迈克尔・尼科尔斯称 xAI「明显落后」，他已出任 xAI 总裁。xAI 在与竞争对手的激烈竞争中经历组织架构调整，失去多位联合创始人及高级领导，马斯克效仿特斯拉模式重建公司。多位研究员加入 xAI 负责不同工作，产品团队、物理和计算基础设施等也有相应负责人。尼科尔斯表示 xAI 计算团队训练性能低，计划在两个月内提高。自 2 月 xAI 被收购后，马斯克首次重组公司，多位联合创始人离职，组织架构变动频繁，还裁减了数十名员工。马斯克称 xAI 最初构建不完善，正在从头重建并重新筛选候选人吸纳新人才。

5 天暴涨 30%，英特尔市值突破 3000 亿美元创 5 年新高

4 月 9 日消息，整体市场势头强劲，再加上英特尔宣布将扩大与谷歌的合作、加入马斯克 TeraFab 项目，推动英特尔市值达到五年来最高水平。受一系列利好消息刺激，英特尔股价今日延续涨势，在过去五个交易日内涨超 30%，六个交易日上涨了 43.1%，总市值已突破 3000 亿美元。早些时候，英特尔与谷歌宣布将围绕至强 CPU 与定制 IPU 展开深入的多年期合作，推进下一代人工智能和云基础设施的发展。英特尔与谷歌将扩大 IPU 这类网络 ASIC 上的开发合作，提高 CPU 利用率、提升效率，并在超大规模 AI 环境中实现更可预测的性能。

微软高管变动仍在继续，开发者部门负责人宣布辞职

微软开发者部门（DevDiv）负责人潘正磊在任职 34 年后即将辞职，她执掌开发者业务 12 年，期间微软加大开源投入并收购 GitHub。潘正磊将任职至 6 月底，之后转为「顾问职务」向微软 CoreAI 负责人汇报。目前不清楚接替人选，也不确定团队汇报情况。潘正磊称已考虑此事一段时间，认为现在是合适时机。她离职距前 GitHub 首席执行官辞职不足一年，此前她统筹 GitHub 相关工作。潘正磊是近几个月微软接连离职的高管之一，此前 Xbox、体验与设备事业部等负责人也已离职，微软还进行了高层管理架构调整及人员任命。

OpenAI 拟分阶段推出新模型，以防范网络安全风险

一位知情人士称，OpenAI 正在完善一个有先进网络安全能力的模型，计划仅向少数公司发布，类似于 Anthropic 对 Mythos 的有限发布。人工智能能力已达临界点，模型制造商担心工具造成破坏而不愿公开发布。Anthropic 周二称因担心先进黑客能力，将新的 Mythos 预览版模型限于一组挑选的科技和网络安全公司使用，是首家对新模型采取此做法的 AI 公司，OpenAI 也计划采取类似做法。

Anthropic 上诉被驳回，未能暂时阻止美国防部将其列入黑名单

美国华盛顿特区一家联邦上诉法院周三驳回人工智能初创公司 Anthropic 暂时阻止美国国防部将其列入黑名单的请求，该公司正就此制裁提起诉讼。此前旧金山联邦法院一名法官曾批准 Anthropic 的初步禁令，禁止特朗普政府强制执行对其 Claude 人工智能技术的使用禁令。美国国防部 3 月初宣布 Anthropic 存在供应链风险，认定使用其技术可能威胁美国国家安全。Anthropic 上诉称这是报复行为，违反宪法等。法院承认不中止诉讼该公司可能遭受不可弥补的损害，但认为其利益主要是经济方面，且未证明言论受压制，同时表示有必要加快处理速度。

**程序员专区**

MiniMax 发布面向 AI Agent 的命令行工具「MMX-CLI」

MiniMax 近日正式推出 MMX-CLI，这是一款专为 AI 智能体设计的命令行工具，旨在解决传统工具在自动化流程中的交互障碍。该工具支持文本、图像、视频、语音及音乐等全模态能力调用，使智能体能够在Claude Code、OpenClaw 等环境中无缝集成，独立完成从资料搜集到多媒体生成的完整工作流。

微信支付已支持 AI 原生接入，推出打包 Skill

微信支付官宣发布面向 AI 的支付接入 Skill，包含 Skill 技能包、AI 友好文档、AI 友好 API 三大工具，开发者通过「唤起 AI— 加载 Skill— 说出需求」可速通接入。开发者可在 AI 开发工具 IDE 输入指令或在 Skillhub 等平台下载技能包加载至工具中开始对话。微信支付为 AI Agent 提供专业技能包，每个 Skill 对应一个产品，包含业务知识、代码示例、交互规范。此次支持 AI 友好接入，工具箱涵盖基础支付 Skill 和商品券 Skill 等，方便商户和开发者一键调用。

阿里云百炼上线 Agent 记忆库，让「龙虾」应用更懂用户

阿里云百炼推出「记忆库」功能，限时免费向所有用户开放，用户可通过 API 直接调用或通过 OpenClaw 等 Agent 产品一键安装。当前主流 AI Agent 缺乏长期记忆系统，存在记忆断层问题。阿里云百炼记忆库内置「提取-存储-检索-注入」四大模块，基于特殊算法优化检索，提升了检索性能和内容相关性。用户可通过 OpenClaw 插件配置安装快速获取服务，企业用户还能通过 API 在其他 Agent 应用上调用。目前，百炼已为多个应用场景提供企业级 Agent Memory 基础设施，且在今年 1-3 月创下历史最高增速。

腾讯云 QClaw V2 发布，开启多 Agent 协同新时代

腾讯云宣布 AI 助手 QClaw 正式上线 V2 大版本。此次升级打破 AI 单兵作战局限，引入多 Agent 协同和跨应用直连技术，构建「数字工作团队」，提升复杂任务处理能力。新版本支持最多 3 个 Agent 并行工作，解决 AI 处理复杂逻辑的问题。引入连接器功能，打通办公软件壁垒。同时，业内首发「龙虾管家」防护系统，保障数据安全。目前 QClaw V2 已接入多项高频办公场景，腾讯云旨在保障用户数据主权，让 AI 成为可靠生产力伙伴。
