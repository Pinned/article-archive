# 刚刚，OpenAI拿下1220亿美元创纪录融资，首次向散户敞开大门

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜Panda

  

昨晚，Anthropic 意外在一个 npm 包更新中泄漏了其核心产品 Claude Code 高达 51.2 万行的底层源代码，参阅《[全网疯传fork！刚刚，Claude Code源代码泄露被开源了](https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651024890&idx=1&sn=ef03a969ea07774bbe6935c89a1370e2&scene=21#wechat_redirect)》。此事过去还不到一天，刚刚，Anthropic 的主要竞争对手 OpenAI 官方宣布已完成最新一轮融资，本轮获取承诺资本达 1220 亿美元，投后估值飙升至 8520 亿美元！

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqEDaEJPV6XLB2eurOIG0r88KMAibicUPm3jGXMvAlLNfHibJic13KEkXTN7J5F2EVLSyzxH6FuvVFicA1aU10iaWaTlNTrrVdofEIhr8/640?wx_fmt=png&from=appmsg)

  

这笔交易正式创下了硅谷历史上规模最大的融资纪录。

  

据悉，OpenAI 预计将于今年年底进行 IPO；而借助此轮融资，该公司将进一步巩固了其作为全球最具价值私有公司之一的地位。

  

早在今年 2 月份，OpenAI 就曾宣布了 1100 亿美元的融资承诺。周二宣布的最新消息显示，他们在之前的基础上又追加了 120 亿美元的投资。

  

据《纽约时报》报道，本轮融资由软银集团、Andreessen Horowitz、D. E. Shaw Ventures、MGX、TPG 以及 T. Rowe Price 共同领投。其中，亚马逊作为新投资者注入了 500 亿美元，英伟达和软银则各自贡献了 300 亿美元。微软也作为长期合作伙伴继续参与了本轮投资。

  

值得注意的是，OpenAI 首次向个人投资者敞开了大门。

  

通过银行渠道，OpenAI 从高净值个人投资者手中筹集了超过 30 亿美元资金。同时 OpenAI 还将被纳入由 Cathie Wood 领导的 ARK Invest 旗下的几只交易所交易基金中，让更多普通投资者有机会分享 AI 时代的发展红利。

  

在惊人的估值背后，OpenAI 展现出了同样相当惊人的营收增长速度。

  

目前该公司每月的营收高达 20 亿美元。在刚刚过去的 2025 年，其总营收达到了 131 亿美元。ChatGPT 作为其核心产品，目前每周活跃用户超过 9 亿，并拥有超过 5000 万名订阅用户。企业端业务同样增长迅猛，目前已占其总收入的 40% 以上。

  

尽管如此，由于需要支付庞大的计算能力和 AI 人才成本，OpenAI 目前尚未实现盈利，并且仍在大规模消耗现金。

  

为了让公司财务状况对华尔街投资者更具吸引力，OpenAI 近几个月来正在努力控制成本，甚至关闭了包括[短视频应用 Sora](https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651023578&idx=2&sn=d4822b55bc70dc39cb6fb0cc3f0967fd&scene=21#wechat_redirect) 在内的部分功能和产品。

在产品和技术愿景方面，OpenAI 明确提出了构建统一的「AI 超级应用（AI superapp）」计划。随着模型变得更加强大，用户期待一个能够理解意图并跨应用程序执行操作的单一系统。

  

OpenAI 计划将 ChatGPT、Codex、网络浏览以及更广泛的智能体能力整合到一个以智能体为先的统一体验中。

  

正如 OpenAI 在其官方博客中所言：「推广 AI 益处的最快方式是尽早将有用的智能交到人们手中，并让这种访问在全球范围内产生复利。」 

  

这笔巨额融资不仅关乎 OpenAI 一家公司的未来，也标志着整个科技行业的底层逻辑正在发生改变。

  

OpenAI 在博客中总结道：「这样的时刻并不常有。在过去的世代中，资本市场帮助构建了定义现代经济的系统，从电力到高速公路再到互联网。现在又是这样的时刻。今天部署的资本正在帮助构建智能本身的基础设施层。」
