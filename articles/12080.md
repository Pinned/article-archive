# 终于，Claude革了设计行业的命：Figma、Adobe股价重挫

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1#imgIndex=0)

本周五，美股多家软件公司迎来股价巨震。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqEYssGyl6axG349D3PLF2ZntibVka80ZHBj9YrdibUtCib7pgLiaefNMgOjvytibynqFtxeiaV1UibZC26797tuia4Xic9Nx4gaU4RjMVuU/640?wx_fmt=png&from=appmsg)

  

震源又是那个 Claude。

  

Anthropic 发布了全新 AI 产品 Claude Design。Anthropic 称，这款新工具能让你与 Claude 协作，创建精美的视觉作品，如设计稿、原型、幻灯片、单页文档等等。

  

Claude Design 由最近刚刚发布，具备强大图像处理能力的 [Claude Opus 4.7](https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651028195&idx=1&sn=4ddb302944a43baa4aebf9d792401975&scene=21#wechat_redirect) 模型提供支持，目前面向 Claude Pro、Max、Team 和 Enterprise 订阅用户提供研究预览版。Anthropic 将逐步向所有用户推出该功能。

  

  

Claude Design 为设计师提供了广阔的探索空间，也为其他人提供了一种创作视觉作品的方式。作为一款生产力工具，它能够直接将自然语言、代码库、PPT 文档甚至直接截取的网页元素，转化为高保真、可交互的产品原型、设计或营销物料。

  

现在你只需要用自然语言描述需求，提供补充资料用作参考（如代码、图片或文档），AI 即可构建出一个初始版本的设计。之后，你可以通过对话、内联评论、直接编辑或自定义滑块（由 Claude 自动生成）继续进行完善，直至获得理想的设计效果。

  

在获得权限允许后，Claude 还可以自动将你团队的设计系统应用于每个项目，从而确保 AI 的输出与公司其他设计风格保持一致。

  

Anthropic 表示，此前在内测过程中，已有团队使用 Claude Design 来进行以下一些工作：

  

*   逼真的原型：设计师可以将静态模型转换为易于共享的交互式原型，以便收集反馈和用户测试，而无需代码审查或 PR。
    
*   产品线框图和模型：产品经理可以绘制功能流程草图，并将其交给 Claude Code 进行实现，或者与设计师分享以进一步完善。
    
*   设计探索：设计师可以快速创建各种各样的探索方向。
    
*   推介演示文稿：创始人及客户经理可在几分钟内从粗略的提纲制作出完整的、符合品牌形象的演示文稿，然后导出为 PPTX 或发送到 Canva。
    
*   营销材料：营销人员可以创建着陆页、社交媒体素材和活动视觉素材，然后让设计师进行润色。
    
*   前沿设计：任何人都可以构建带有语音、视频、着色器、3D 和内置 AI 的代码驱动原型。
    

  

Claude Design 遵循自然的创作流程。在初始训练期间，Claude 会读取你的代码库和设计文件，为你的团队构建一套设计系统。之后的每个项目都会自动使用你的颜色、字体和组件。你可以随着时间的推移不断完善该系统，团队也可以维护多个系统。

  

你可以从任何位置导入：从文本提示开始，上传图像和文档（DOCX、PPTX、XLSX），或者让 Claude 直接访问你的代码库。你还可以使用网页捕获工具直接从你的网站抓取元素，使原型看起来更接近现有的真实产品。

  

你也可以使用精细的控制功能进行微调：直接对特定元素添加注释、编辑文本，或使用调整旋钮实时调整间距、颜色和布局，然后让 Claude 将你的更改应用到整个设计中。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEnOfP3a8gmaW4mViblMuDXEXrg65Nic4CkrGAticuErn7RzQzvEXzB2YA3aKAdxhQGsEsSb8XAUbsZibhmVJvz5RpgUpmiajL2xMAU/640?wx_fmt=png&from=appmsg)

  

在协作方面，Claude Design 设计稿支持组织范围内的共享。你可以将文档设为私有，也可以共享链接，让组织内任何拥有链接的人都能查看，或者授予编辑权限，以便同事可以修改设计稿，并与 Claude 进行群组对话。

  

在完成设计后，你可以将设计以内部 URL 的形式在组织内部共享，保存为文件夹，或导出为 Canva、PDF、PPTX 或独立的 HTML 文件。

  

Claude Design 也与 Claude Code 实现了联动。当设计准备就绪可以构建时，Claude 会将所有内容打包到一个移交包中，你只需一条指令即可将其传递给 Claude Code。

  

Anthropic 表示，在接下来的几周里，Anthropic 将简化与 Claude Design 的集成，以便用户将新工具连接到人们原有的工作栈中。

  

Claude Design 上线后，开发社区纷纷前来尝试，有不少的好评和效果展示。人们表示，这回 Anthropic 要干翻设计了，而且该说不说，Claude 的「审美水平」还挺高。

不论如何，这回设计的门槛是被打下来了。

  

Anthropic 自己的设计师 Ryan Mather 也出来现身说法。他表示自己目前一个人负责七个产品线的设计，对于怎样用好 Claude Design 有七条建议。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGEyV7AEXvgEsIs60ynz2JMOvkiaHDjpJ10vbia0ChYic4f3ytUSibGC2UCvK7tvLdUp05AiaFY7C3HOgWJ84DuUht5Hxd4GP4yZK9k/640?wx_fmt=png&from=appmsg)

  

1、设置好你的设计系统和核心屏幕。这里值得花上一个小时进行设置和优化。

2、与你的工程师实时迭代。最好能在单次会议上与工程师一起设计新功能。Claude 在制作原型方面速度极快，设计师与工程师可以在对话中持续迭代，围绕概念和约束即兴发挥，看着想法逐步成形。

3、使用评论工具进行快速精准的编辑。在粗略的初稿之后，你可能会有数十个细节想要调整。用言语描述想要的所有更改很棘手 —— 所以别那么做！直接指出并批评。

4、让 Claude 为你的想法制作视频演示。一般来说，Claude Design 可以实现你能想到的几乎任何事情。老实说，它更像是 Claude Code，而非基于画布的设计工具。

5、使用连接器（尤其是 docs/slack）。设置好后，你可以发送这样的提示：「请阅读产品 roast 的会议笔记，并为所有出现的问题创建一份探索不同设计解决方案的演示文稿」。然后你就可以出门放松一下，然后带着全新的视角回来看结果了。

6、让 Claude 创建定制的即时工具。一般来说，不要试图以使用基于画布的工具的方式来使用 Claude Design。它是一个不同的物种，具有不同的能力。去尝试吧，玩得开心点！你会发现自己设计得比以前的方式更胜一筹。

7、学会在哪里可以放慢脚步，手动完成任务。新图标、现场插图、命名。有些细节总是能产生超乎预期的冲击力。你很容易就会被智能体式设计的超高速度所吸引。知道何时放慢脚步，本身就是一种艺术形式。

  

Ryan Mather 认为，Claude Design 对工作带来的最大提升在于能让设计变得更加「开心」，让人们可以轻松尝试更多想法。
