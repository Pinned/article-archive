# 为人类做软件，还是为Agent？李建忠×邓侃再聚首：从OpenClaw看Agent时代的软件新生态！

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

AI 领域标志性人物、OpenAI 创始成员 Andrej Karpathy，昨日在其个人 X 平台发布了他的最新观察：

> 从服务器配置到最终部署上线，一套完整的流程，你需要多久？一天？还是三天？
> 
> Karpathy 给出的答案是：30分钟，并且全程只需一句英文指令。

这不是魔法，而是 AI Agent 带来的现实。Karpathy 指出，AI Agent 的能力在去年底已实现质的飞跃，它们已经能自主完成过去需要一个工程师团队数天才能完成的复杂任务。

AI 的角色正在发生根本性转变：从一个知识渊博的“军师”进化为一个能独立领兵打仗、攻城略地的“将军”。

这恰恰印证了奇点智能研究院院长、CSDN 高级副总裁李建忠在 2024 年全球产品经理大会上提出的那个前瞻性断言：“未来，智能体会把我们今天的互联网从‘信息网络’重构为‘行动网络’。”

如今，伴随着爆火的 OpenClaw 等 Agent 的涌现，这张庞大的“行动网络”正在被迅速织就，这个预言正以前所未有的速度成为现实。

这，就是 Agent 时代的开端。

当 AI 开始自主“行动”，作为身处一线的开发者与从业者，我们是会沦为那个被 AI Agent “优化”掉的成本，还是学会驾驭这股力量，成为创造十倍价值的时代新贵？

带着诸多时代叩问，本周六（2 月 28 日）晚 20:00，我们荣幸地邀请到了奇点智能研究院院长、CSDN 高级副总裁李建忠，与北京大数医达创始人 & CEO、CMU CS/Robotics 博士邓侃，做客 CSDN 直播间《AI 进化论》栏目。

两位身处风暴中心的顶尖专家，将为所有开发者和技术从业者，带来一场关于未来的深度解码与硬核对谈！

💡 OpenClaw 在 AI 生态版图中最大的变革意义何在?

*   它仅仅是又一个开源项目，还是引爆 Agent 生态的“核武器”？
    

💸 Agent 经济学: 为人类做软件，还是为 Agent 做软件?

但这仅是冰山一角。

围绕当前处于风口浪尖的 OpenClaw 以及背后折射出的 Agent 应用层大爆发，我们还为两位大咖准备“十道灵魂之问”！

它们不仅涉及了技术底层的路线之争（MCP 协议、系统操作），更直击了商业模式（流量分发、大厂绞杀、护城河）和开发者个人的生死存亡（代码质量、核心壁垒）。

为什么你绝对不能错过这两位大咖的对谈？

把时钟拨回一年前。2025 年初，当 DeepSeek 刚刚以极低的成本震撼全球、引发行业剧震时，无数人陷入了“Scaling Law 是否失效”、“闭源是否彻底死亡”、“程序员是否马上失业”的集体焦虑。

在那场堪称经典的开年直播中，李建忠与邓侃前瞻性地抛出了诸多“神级预言”。一年后的今天回头看，这些洞察不仅字字珠玑，更已成为现实：

*   精准预判了“推理计算”的爆发： 当大厂还在执迷于囤卡“大力出奇迹”时，他们果断指出 DeepSeek 引领了从预训练到推理计算的范式转移。预言了未来算力需求将更加多元（MoE 架构、国产算力生态的崛起），打破了单一巨头的算力垄断。
    
*   看透了 AI 生态的终局： 邓侃大胆预测 DeepSeek 的开源路线将成为“AI 时代的 Android（安卓）”，团结全球开发者构建跨设备生态；而李建忠则用经典的“5D理论”一针见血地指出，大模型真正的护城河不在于技术壁垒，而在于未来的 Development（开发平台/Agent生态） 和 Distribution（分发渠道）。
    
*   最早触及了“Harness Engineering”的灵魂： 面对甚嚣尘上的“程序员失业论”，两人出奇一致地断言：程序员绝不会大规模失业，但职业路径将被彻底重构。 开发者将从单纯手撸代码的“代码工人”，进化为拥有全局架构能力、驾驭 AI 工具的“软件构建师”。
    

这，不正是我们今天所面临的“Agent 时代”和“驾驭工程 (Harness Engineering)”的最早萌芽吗？！
