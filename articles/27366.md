# 扣子3.0，让我不再担心网络ip，遥控家里的 Claude Code 干活

本文完全主观

  

扣子的这波更新，我必须给到夯：我终于在户外，手机 PUA 我的 Claude Code & Codex 干活了

扣子3.0，来了波大的：[扣子 3.0 正式上线：新一代 AI 团队，从扣子开始](https://mp.weixin.qq.com/s?__biz=Mzk0MzY0MTMwNA==&mid=2247488612&idx=1&sn=b80e6d443905b32add9522d481c9f5cf&scene=21#wechat_redirect)

这里面解决了我的大问题：无论在手机、网页还是桌面端，我都能远程摇操我的、接入原版服务的 Claude Code

注意，这里的 Claude Code，可以是我电脑里面的，也可以是云端的

简直夯爆了

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdg1icIIckpBxNRUiagl2p3b6nMtjZ7Yo6DTRDn70AGelMKEMSUYMUv07rRMQcYmOdgddaiagnSUEiaoOO2bK0o7gDYUXEgfLaZtd1s/640?wx_fmt=png&from=appmsg)

  

**本文极不客观，**

**没错，本文极不客观**

**充满着我的惊叹和赞美，毕竟我是「中文互联网第一扣吹」**

跟扣子这个团队认识快三年了，过去三年中，扣子做了很多产品探索，有些产品实打实的是标杆，比如 Workflow，但也有些方向并不令人兴奋，但我坚定的认为：扣子一定能做大做强

这次 3.0 出来（其实更早的时候，就拿到内测了，毕竟太熟了），试玩了之后，就他妈有种热泪盈眶的感觉：他妈的，牛逼！

  

更新的内容有很多，比如这些是还算 ok，但对我来说没啥感觉，比如：多人多 Agent 协作、行业技能商店、视频项目搭载 Seedance 2.0 这些

但是，他妈的能让我用手机 APP，无论环境，都能远程管理 Claude Code 和 Codex CLI，这事儿必须得夯

别跟我说什么前段时间 Codex 支持了手机版，那玩意儿我手机也得科学上网啊，很多时候由于各种情况，就是网络不稳定；

还有可能会说：Claude Code 本身就支持 Remote 方法啥的，那玩意儿你得推上云端，或者是用 Claude 自己的云服务。我特么很多文件都保存在自己的电脑里，压根就推不上去

我之前还尝试过魔改 Claude 网页版，然后在里面做了很多诡异的 skill，让他给我的服务器发 api call，当中台去用（很邪修就是了）。但这里也有个问题，就是：特喵的手机里骚扰电话太多，我特么 iphone 用户，天天进程被打断

我还试过很多其他方案，比如在服务器上常驻一个 Claude Code，从任何设备都能连上去。改了很多次，没有一个足够丝滑

  

御三家的会员，我是都买了：Claude Max、ChatGPT Pro、Gemini Ultra，每个月 600 美金

其中 Gemini 纯靠信仰续费（哈基米你自己看看，好意思么）而干活来说，日常的主力是 Claude Code + Codex

这里我就有个需求了，很多场景下我的工作台很碎，没办法，鬼知道谁家更新了什么新玩意儿，然后两边的很多信息都不同，还都得在本身的 cli 下才好用，所以好多时候上下文我是手搬的，就很傻逼

  

然后扣子 3.0 出来了，完整切合我的需求

  

不是钱的事儿
------

我已经完全离不开 AI 了，我让 Agent 保存了我的各种材料：身份信息、日程记录、API 权限、社会上下文...反正就是「量俺之物力，结硅老爷之欢心」

我的自我认知是：在硅老爷面前，我是小five。至少是 Claude Code + Opus 4.5 的时候，在我看来就已经是 Coding AGI 了（纯个人暴论，没有任何解释），时代面前，反抗是没有用的

于是，我给各家先充了 200美金/月的会员，然后配了 20000美金的 extra credits，就供着

  

在家里，我是有电脑，配好了稳定的网络环境，持续跑着东西的

但一出门，我就傻了眼：我手机得连上。很多时候就很麻烦，比如进电梯、到车库，ip 断网啥的，总会抖动

然后我是 Claude Code 和 Codex 混用，这俩东西能力边界不一样，总需要我人力搬运上下文，也很烦

  

我的这些卡点，也算是在当前环境下的无奈，又能咋办呢？直到看到了扣子3，是在是特么太开心了：孩子有救了

出门也能干活
------

我之前的工作流是这样的：把 Claude 的网页版通过一些方法魔改了，让它每次能跑半小时的长程任务，还可以作为中台发请求

很多时候不在电脑前，就用手机去操纵 Claude 网页版，再通过各种魔改方式去控制我的 Agent，Claude 网页版就是我的中台

没办法，在外面不方便

  

但这套流程有个极其讨厌的问题：在国内，手机总有接不完的骚扰电话

电话一来，Claude 的网页版或者 App 就会被切出去，正在跑的任务可能就挂了，就特么很无奈

  

但扣子的遥控功能就没这个问题了，完全不受影响

安装扣子桌面端并连接电脑后，我在手机上打开扣子 App，@ 我的 Agent，它就在我电脑上干活。**这是我最看好的功能，没有之一**。它是一个真正的中台，比起我用魔改 Claude 网页版当中台要强太多：一定能稳稳的把我接住

  

举个例子，我打开手机扣子，使用 Claude Code（实际上是连的我的电脑，已科学上网），让它帮我写一个斐波那契数列的交互式网页，完美～在这个过程中，我手机就是普通网络
