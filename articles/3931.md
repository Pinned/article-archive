# 我，18岁高中生，靠15个龙虾员工开公司“干翻”行业，16万网友围观

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurY44iaOBbvY3OVlAfibtRZgl9ogPna2K3DgTbTMwbdIpQ8C4joIibUygAo1IBibicklfEVfDq2dj0n4Jfp5ARsQYZe0C0IicxZ0bibmA/640?from=appmsg)

**16万海外网友围观。**

**作者 |**  李水青  

**编辑 |**  心缘  

一位18岁小伙用OpenClaw做AI外包，在外网火了。

他没上过大学，零编程基础，就靠OpenClaw+Claude+GLM，养了**15个“龙虾”员工**24小时给他干活，创办公司接单做网站、自动化任务等外包。月成本不到**400美元**，目前已精准**获客450+**，今年已由此获得**上万美元**收入。

智东西3月6日报道，昨日，加拿大无代码开发机构联合创始人兼 CEO Jacob Klug在社交平台X发文，分享了与这个年轻“龙虾”创业者的对话视频，迅速在社交媒体X引起了**超16万人**的围观。这位去年才从高中毕业的小伙名为Vadim，创办的公司叫Vugola，也是一家Agent公司。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurQFx2WlTcsZueBuQSOy3kT0sI01ql4esKSxMoxa43ia0LRXrNGyibYsqclaBZoc0zmLeDEfy3blWIkrPtI8MYcfNp4DO5kx5pkU/640?from=appmsg)

▲Jacob Klug的推文（图源社交平台X）

Vadim公司的“龙虾”员工负责处理**研究报告、社交媒体帖子、YouTube视频、图片/视频、编程、Bug修复**等，以及**支持运营和推广其一款视频剪辑Agent产品Vugola**。在零广告投入的情况下，仅通过“龙虾”运营团队，Vugola已吸引了450多位自然用户。

据Vadim分享，用户可以用Vugola自动获取Youtube视频的热门片段，下载视频并剪辑成片段、添加字幕，最后自动定时发布到X等社交媒体账户。只一键按下，用户可以用Vugola自动运营多个社交媒体。Vadim称：“**社交媒体代理机构将在2026年消亡。**”

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurJOSPkk2nOmsJkklCmVrdiaJIWAz2xYGdfdeyQpCm7ibFFwVLpQbQMFv5wKNNqkYKqhwbia3HCtYe8kwk03GUOXgNoITVibeFM9gY/640?from=appmsg)

▲Vadim的推文（图源社交平台X）

Vadim公布了“龙虾”员工公司的架构，架构包括：1个CEO代理、1个深度研究员、1个文案撰稿人、1个热点监测员、1个图像设计师、2个视频制作人、2个动态设计师、2个高级开发者、2个代码审查者、1个剪辑师、1个助理。

拆解其成本可以看到，其中**6个智能体运行在Claude模型上**，其余使用API更便宜的**GLM、Higgs Field、Brok Imagine**等。Jacob Klug称：“这是我见过的个人搭建的**最完整的OpenClaw系统。**”

Vadim将具体的架构提示词公布了出来，用户只要将其完整粘贴到OpenClaw里，就能够搭建一个和Vadim类似的公司架构：

**1、核心层：**

**贾维斯（Jarvis）—— 大脑中枢**

模型：通过Claude Max OAuth调用Opus 4.6

自动将所有任务分配给对应子智能体：

收到YouTube链接→转给剪辑师

收到调研报告→ 转给撰稿人

所有任务分配逻辑都写在结构化Markdown文件中，由智能体自动读取执行。

**2、研究层：**

**阿特拉斯（Atlas）——深度研究员**

模型：通过OAuth调用Claude

API：Brave搜索、XAPI、FireCrawl

定时任务：每1小时执行一次

7×24小时在X、Reddit与全网做深度研究。

训练数据包括： MrBeast所有关于YouTube数据分析播客里的爆款逻辑，以及Dan Koe的爆款文章结构。

输出：研究报告+通用爆款策略手册（MD文件），供内容团队直接使用。

**3、内容层：**

**写手（Scribe）—— 文案撰稿人**

模型：GLM5

定时任务：每3小时一次

接收阿特拉斯的研究成果，按创始人语气与风格撰写初稿。

**趋势猎手（Trendy）—— 热点监测员**

模型：GLM 4.7

API：X API

定时任务：每2小时一次

扫描X、Reddit热点话题与爆款规律，实时反馈给写手，确保内容紧跟当下流量趋势。

**4、设计层：**

**图像设计师**

模型：Nano Banana Pro（谷歌API）

按需生成图片。

**视频制作人**

模型：Higgs Field API + Brok Imagine API

制作AI原生UGC视频与视频内容。

**动态设计师**

模型：Claude Code（OAuth）+ Remotion

制作动态图形与动画内容。

**5、开发层：**

**爪爪（Clawed）—— 高级开发者**

模型：Claude Code（OAuth）+ Codex 5.3（API）

定时任务：每晚 11 点

自动审查整个代码库，识别缺失功能，次日早上提交合并请求。

它自主实现的第一个功能：发现首页需要 FAQ 并直接开发完成。

在Claude Code内部启动多智能体并行工作：

一个负责审查、一个负责开发、一个负责安全。

**哨兵（Sentinel）——代码审查+Bug监控**

模型：独立大模型（第二层审查）

定时任务：每2小时一次

在代码合并到GitHub前，审查所有Clawed提交的内容；

同时监控线上环境的用户反馈Bug与报错。

**6、增长层：**

**阿特拉斯+写手协同工作**

阿特拉斯挖掘Reddit上用户吐槽竞品、求剪辑工具推荐的帖子；

写手生成回复文案；

创始人直接复制发布。

仅靠这套流程，零广告预算其就为这款SaaS带来了超450位精准用户。

**7、运营层：**

**剪辑师（Clipper）—— 剪辑智能体**

API：Poster API

触发方式：贾维斯收到YouTube链接时即时启动

接收YouTube链接 →剪辑→加字幕→自动定时发布到社交平台。

**莱德（Ryder）—— 朝九晚五助理**

按需启动

帮创始人处理日常主业工作：写文章、做研究、日常办公支持。
