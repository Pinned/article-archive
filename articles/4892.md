# 市值511亿，1月8日IPO，大模型第一股花落智谱

今天早上，Manus 被 Meta 以数十亿美金收购：[始末｜Manus 这一年](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247510753&idx=1&sn=9e071258cc0e1b913d6dc73eb189af96&scene=21#wechat_redirect)

现在，智谱将于1月8日正式 IPO，市值 511 亿港元，股票代码`2513`，多的话不用说了 ![](https://res.wx.qq.com/t/wx_fed/we-emoji/res/assets/newemoji/Yellowdog.png)![](https://res.wx.qq.com/t/wx_fed/we-emoji/res/assets/newemoji/Yellowdog.png)![](https://res.wx.qq.com/t/wx_fed/we-emoji/res/assets/newemoji/Yellowdog.png) 

**1月8日**，智谱将以这个代码在港交所主板挂牌，市值**511亿港元**，成为**全球第一家以 AGI 基座模型为核心业务的上市公司**

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYojCPxFBicKSziaFyh3PNcqCqTvZ4ciaZtHSib1ibibkM2DKCugcziap18EjfeOaENUmXOuOM96EngTmib5sA/640?wx_fmt=png&from=appmsg)

  

发行结构
----

发行价定在每股`116.20`港元，发行市值**511亿港元**

本次发行`3741.95`万股`H`股，其中香港公开发售`187.1`万股，国际发售`3554.85`万股

募资总额约`43.5`亿港元，扣除费用后，本次募资规模预计将达`41.7`亿港元

![](https://mmbiz.qpic.cn/mmbiz_jpg/2icSMc1VBIYojCPxFBicKSziaFyh3PNcqCqYYUlL1uyzcxrotIrcqsQ4vbNgqwNMFibQxgHiaFEv4eChEg5cksk6L8A/640?wx_fmt=jpeg&from=appmsg)

基石投资者
-----

智谱作为中国最早研发大模型的企业，也是中国收入体量最大的独立大模型厂商，一直备受市场青睐

在 IPO 之前，智谱已完成`8`轮融资，融资规模超`83`亿元，云集一批主流明星投资机构

本次上市，智谱引入 `JSC International Investment Fund SPC`、`JinYi Capital Multi-Strategy Fund SPC`、`PerseverancAsset Management`、`上海高毅`、`WT Asset Management`、`泰康人寿`、`广发基金`等`11`家投资机构为基石投资者，共拟认购`29.8`亿港元

![](https://mmbiz.qpic.cn/mmbiz_jpg/2icSMc1VBIYojCPxFBicKSziaFyh3PNcqCqEYInOoqYJabBicmaDFcdibuia63XhM22VA0zicXYvApNwLrde7tW642NLw/640?wx_fmt=jpeg&from=appmsg)

本次募资的`70`%（约`29`亿港元）将继续投入通用 AI 大模型研发，`10`%（约`4.2`亿港元）用于优化 MaaS 平台

商业化：MaaS 跑通了
------------

智谱的收入主要来自于大模型，采用 MaaS 模式，即通过 API 调用向开发者和企业提供服务

在商业化逻辑上，与 OpenAI、Anthropic 相似，即：**向客户交付的是通用智能**

智谱从`2021`年开始布局 MaaS。在战略方向上，智谱从战略方向上，在稳固本地化部署收入基本盘的前提下，持续拉升MaaS平台的收入占比

目前，MaaS 已产生规模收入，API 平台的企业和开发者用户数超`290`万，`2025`年 API 收入增长超过十倍，短期内有望实现 API 收入与私有化模型部署收入打平

在 OpenRouter 上，GLM-4.5/4.6 自上线以来调用量稳居`全球前10`，**付费 API 收入，超过所有国产模型之和**

![](https://mmbiz.qpic.cn/mmbiz_jpg/2icSMc1VBIYojCPxFBicKSziaFyh3PNcqCqMHF3XlVlSK29MibnubOZ0NC5pCGIPEr9LMmYkheT4UxPRNBWvCjgepg/640?wx_fmt=jpeg&from=appmsg)

截至目前，智谱大模型已赋能`12000`家企业客户、超`8000`万台终端设备、超`4500`万名开发者，是**中国赋能终端设备最多的独立通用大模型厂商**

技术现状
----

智谱的 GLM 系列模型，每`3-6`个月完成一次基座迭代

最新发布的 GLM-4.7，强化了编码能力、长程任务规划与工具协同，同时提升聊天、写作、角色扮演的综合性能

在 Artificial Analysis Intelligence Index 中，GLM-4.7 以`68`分拿下开源模型与国产模型双料第一

![](https://mmbiz.qpic.cn/mmbiz_png/2icSMc1VBIYojCPxFBicKSziaFyh3PNcqCqSNJc5yBOHh39QPJwxKW4QPKOUzibkw21YRFTEhN2PbBmbRYDCPGmg9w/640?wx_fmt=png&from=appmsg)

在 Code Arena 编码评估中，GLM-4.7 位列开源第一、国产第一，超过 GPT-5.2
