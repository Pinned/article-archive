# 花99万，买个机器人男友谈恋爱

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4Acuq8w1cJnvqpOnGvGO9PRSy679sUuiau1zsroqicIy65vhz9OWKJdpTiccVkT0ibmGVF8xsLoo3OKXYmIQuf85NLe5fnStGlMg2aOMk/640?from=appmsg)

**优必选正式发布了首款全尺寸超仿生人形机器人U1。**

**作者 |**  许丽思  

**编辑 |**  漠影  

智东西6月30日深圳现场报道，今天，优必选正式发布了首款全尺寸超仿生人形机器人U1系列。其首批产品包括**女版机器人小优Una**和**男版机器人凌夜Nix。**

U1 最出彩的地方，一是面部表情非常逼真，二是全身动作也挺流畅丝滑，能和人类舞者搭档跳《爱乐之城》，整套舞蹈的动作、节奏都把握得不错。

▲舞台上，女版U1Ultra与人类共舞

U1系列共有三款产品：**U1Lite**是轻量化半身版，方便携带、搬运，价格为11.98万元；高配全身版**U1Pro**为16.98万元；高动态全身版**U1Ultra**女版为88万元，男版贵一些，为99万元。

U1 Ultra 男女款差价高达**11 万，**差不多都能买一辆车了。

U1系列主打高颜值，仅限成年人购买，它可以情绪陪伴、日常互动，还能在陪伴过程中越用越懂你。不过，对于不少用户好奇的能不能买回家顺便完成做饭、打扫卫生、收纳等家务活，优必选明确表示：那还不行。

优必选创始人、董事会主席兼CEO周剑现场说，上台前，他刚得知订单**突破11000台**；而在半个小时后，优必选首席品牌官、机器人大消费创新事业部总裁、优世界总经理谭旻提到，这个数据又刷新到**超13361台。**

用户支付3000定金即可订购，在7月15日预售结束前可退。截至目前，在京东平台上，可以看到U1已定5467件，天猫平台已定80件。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuryGmd7cAibaWyTAw3vsqLCGv1gB8PjxvqS9fqxw0FWn321lMJQFs0p4AmGQd5leYzVXHbzLKJch1h7WyAnGEcwyuYXHwTYkkFI/640?from=appmsg)

按照计划，优必选将于9月16日启动量产交付，2026年全年目标**交付1万台以上。**

发布会现场打造了一个4000㎡沉浸式人机共生艺术空间，展示了U1的多个不同应用场景，包括**医康养评估接待，家庭情感陪伴、青少年心理疗愈、前台迎宾接待等。**

比如在前台迎宾接待的场景中，化身接待员的U1可以给观众自我介绍、讲讲优必选的发展、不同的产品线，可以说是有问必答。

不过，从现场体验来看，U1有时需要等待片刻才能作出回答，语音与唇形也没能完全同步。

优必选展出了多款不同造型设计的U1，他们都有着不同的角色设定：废土生存者、真实世界觉醒者、星际开拓者、海洋生命守护者、秩序维护与谈判者......

**_01_****_._**

**能模仿人类大多数动作、表情，**

**还具备长期记忆能力**

U1采用接近真人比例的全尺寸设计，男性版本身高约1.83米，女性版本身高约1.68米。女版更注重贴近东方审美的三庭五眼、骨相，男版则五官更挺拔些，轮廓清晰。

整机搭载**24个**主动自由度和**64个**被动自由度，共计88个自由度，覆盖人体14个核心关节，可完成坐下、倚靠、躺卧、拥抱等人类90%基础动作。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acup2Ik9dg3uooUGgpiad6Grlyvyeuu6Wmy1JyLJctic8QKftBQeUlJVZeibewZcyOBmN1k5aendVRiabx7NF1yGrPKiaIVs5xQMm00IQ/640?from=appmsg)

在面部交互方面，U1拥有33个面部自由度，包括19个主动自由度和14个被动自由度，可以实现眨眼、挑眉、嘴角上扬等**30多种复合微表情。**

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4Acup8VMRnLURYokQJqPuy5atEOPRLk3AMAkxSygicuHtfqnxvrP69es6TSF9chWnjwuNrb9PtSMv7egkpF7YlbgTA7jA5oZ7RUoJ0/640?from=appmsg)

在颈部，U1采用**双支点与四连杆**结合的仿生结构，可根据双目视觉信息跟随用户转头，并通过全时自锁技术，在断电后保持头部姿态。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcupusJoslA0JzgtRXXLbB95QKDiajybyGbI3prItPzUDUCRJdTj3xyy4cMM04w3etaIiaQia6yFeGjj9IEPfnbEtqAIG2JMzMHM8bk/640?from=appmsg)

为减少语音、口型和表情不同步带来的机械感，优必选开发了自研控制器，将语音与唇形之间的响应延迟控制在**20毫秒以内。**

U1表面采用仿生材料与凝胶结构，以模拟皮肤纹理、弹性和温润触感。机器人全身搭载多维柔性电子皮肤及传感器，可感知用户的触摸和拥抱。

U1搭载**超过200TOPS**的端侧算力，可同时运行视觉跟踪、情绪识别、语音处理、音色处理和运动控制等多个模型。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acuo8xK2WYYP3b7K2PiahfhCy8fWQ3xD6hulJib5LltoO03Crg4R03icYm3xMVpJpOdHzD88iaXee0bBa8ichGljTI9ibk1HRYtWYBYxicQ/640?from=appmsg)

系统采用“快慢脑”架构，端侧“快脑”负责低于500毫秒的即时反馈，云端“慢脑”负责知识查询和复杂推理。

U1还搭载面向长期陪伴的**Resonance-LM情感共鸣大模型，**可以根据用户的面部表情、肢体语言、语音、方言和神态，识别20多种细颗粒度情绪，包括开心、尴尬、委屈和疲惫等状态。

**跨时空记忆库Agent Memory OS构建数字生命系统**，主动关怀引擎基于环境感知实现无唤醒词互动，让U1学会“看眼色”。比如，在看到你加班回家后，他会主动贴心询问是否需要调整空调或播放音乐。

随着互动增加，机器人的性格和行为还会逐步适配各自用户。

  

**_02_****_._**

**9月开始发货，年底要交付超万台**

在正式交付前，首批用户可先通过手机上的**UWORLD  App**与所订购的机器人互动。

App预计于7月中旬开放，用户可提前录入称呼、偏好和生活习惯。9月16日机器人送达后，将继承此前在App中形成的记忆，仿佛一个认识了很久的老朋友。

隐私方面，U1采用“本地优先、云端加密、物理可控”的三层数据隐私架构，用户个人数据主要在端侧封闭运行，优必选称，不会将用户本地数据用于模型训练或其他研发。

优必选还提出**「万相镜」个性化角色打造系统**，用户可以通过AI捏人、捏脸，设置专属的机器人性格和音色。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuqVagTbgyq5GpfcqIvA7SOvhYINaGW616Ypib53bPrna4pBl2xibEwfm1EAlWCQMaSDuKuibmuEJfc0mpECaibAVq0NmdgRSx5ao4c/640?from=appmsg)

发布会现场还落地多项生态合作，全球知名的的电子音乐制作人及DJ**Alan Walker**出任优世界全球娱乐IP宣传大使，推出“WALKER×WALKER”联名IP，他也将成为世界首位拥有专属仿生机器人的音乐人。

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4AcuqXVDBLkZJGITkQpUHtGu3YDTd0A4rvicE3Kv78n5j6F4sgUMiaRoYckpsnKm5EvSYh0zGia3gbeqOB4c3jcZP6IkCfAcPRH3DOgU/640?from=appmsg)

针对超过大批量的U1的订单，优必选已在深圳建设一座全新的超仿真机器人生产工厂。由于U1涉及仿生皮肤、面部肌腱结构、触觉系统和复杂装配工艺，其生产流程与工业人形机器人不同，优必选重新制定了生产节奏、标准作业流程和质量标准。

在交付节奏上，优必选称，2026年下半年将优先完成国内市场交付，**9月16日开始发货，力争在12月31日前完成2026年度订单交付；**计划于2027年完成相关国际认证，并进一步拓展海外市场。

  

**_03_****_._**

**结语：从陪伴机器人出发，**

**打造家庭超级入口**

周剑提到，2025年至2036年，中国超仿生机器人市场规模有望从百亿元级增长至万亿元级。

U1的推出，正是其抢占家庭端入口的一次关键落子。相比单纯售卖硬件，优必选更希望将U1打造为连接情感陪伴、家庭服务、内容生态和开发者生态的“家庭超级入口”。

不过，从新品发布到真正进入千家万户，超仿生机器人仍要跨过量产交付、交互体验、安全隐私和长期服务等多重门槛。U1能否从一款高关注度新品，成长为家庭AI终端，将取决于优必选能否持续把技术能力转化为稳定、可用且可规模化的产品体验。
