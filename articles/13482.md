# OpenAI最强对手，被亚马逊追投2249亿

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqeGjO8MF69ibwKHtciaYiaM2rAdEf7icjFCdFjwiaNOFibGvk2rUu0ecGYibGicT72x4vudO9PMUqO7s8OVTEQFibuwFPLJAd2P2ibS5Cdk/640?from=appmsg)

**Claude年化收入已超过300亿美元。**

**作者 |**  ZeR0  

**编辑 |**  漠影  

智东西4月21日报道，今日，亚马逊宣布将向美国AI大模型独角兽Anthropic投资**50亿美元（约合人民币341亿元）**，未来还将根据一些商业里程碑的达成情况，追加投资至多**200亿美元（约合人民币1364亿元）**。

这笔投资是在亚马逊此前已向Anthropic投资**80亿美元（约合人民币545亿元）**的基础上追加的。

也就是说，加上追加投资后，亚马逊确认将向Anthropic投资的总计金额达到至多**330亿美元（约合人民币2249亿元）**。

双方宣布深化合作，Anthropic承诺未来十年将在AWS（亚马逊云科技）技术方面投入超过**1000亿美元（约合人民币6817亿元）**，用于购置当前及未来几代的亚马逊AI芯片Trainium以及数千万颗亚马逊CPU芯片Graviton核心。

Anthropic将获得高达**5GW**的亚马逊当前和未来几代Trainium AI芯片，用于训练和驱动先进的AI模型，其中包括预计将于今年上线的大量Trainium3容量。

此次合作还包括大幅扩展在亚洲和欧洲的国际推理能力，以更好地服务于Claude不断增长的国际客户群。

总体而言，该承诺涵盖Trainium2、Trainium3和Trainium4芯片，并支持Anthropic在未来几代Trainium推出时进行采购。

Anthropic由OpenAI前高管丹妮拉·阿莫代（Daniela Amodei）和达里奥·阿莫代（Dario Amodei）于2021年创立，已发展成全球估值最高的大模型企业之一。

今年2月，Anthropic宣布完成**30亿美元（约合人民币205亿元）**G轮融资，投后估值达到**3800亿美元（约合人民币2.59万亿元）**，产业投资方包括**微软、英伟达**等。

亚马逊并非Anthropic的唯一AI芯片供应方。

Anthropic在**AWS Trainium**、**谷歌TPU**和**英伟达GPU**等多种AI芯片上训练和运行Claude。

4月初，Anthropic宣布已与谷歌和博通签署了一项新协议，将获得数GW的下一代TPU容量，预计将于2027年开始上线。

此时Claude年化收入已超过**300亿美元（约合人民币2045亿元）**，远高于2025年底的约**90亿美元（约合人民币614亿元）**，也超过了OpenAI最新年化收入250亿美元（约合人民币1704亿元）。

Anthropic和OpenAI两家美国大模型龙头正竞相争取在今年完成IPO。据彭博社3月报道，Anthropic PBC考虑最早在10月份上市。

另据彭博社4月15日报道，知情人士透露，Anthropic PBC已收到几位投资者的新一轮融资提议，这些提议可能使其估值达到**8000亿美元（约合人民币5.45万亿元）**或更高，但截至当时，Anthropic一直拒绝这些提议，尚不清楚该公司是否会接受投资者的条件，也不确定其是否会以8000亿美元的估值进行融资。

不过，亚马逊仍是Anthropic主要云服务提供商和训练合作伙伴。双方自2023年开始合作，双方工程团队密切沟通，合作持续深化。

**在定制芯片合作方面**，Anthropic使用AWS Trainium芯片构建、训练和部署其AI模型。

Anthropic与亚马逊芯片团队Annapurna Labs紧密合作，共同开发和优化未来的Trainium芯片，并利用Claude训练工作负载的直接反馈来塑造下一代芯片设计，以支持前沿AI模型，同时也惠及其他AWS客户。

**在AI计算集群方面**，两家公司共同打造了全球最大的AI计算集群之一Project Rainier，不断突破大规模基础设施的极限。

Project Rainier拥有近50万张Trainium2芯片。Anthropic正积极利用该集群为全球客户训练和部署Claude模型。Project Rainier还用于构建和部署未来版本的Claude模型。

**在访问Claude方面**，超过10万家客户在AWS上运行Anthropic Claude模型，Claude成为亚马逊高性能推理服务Amazon Bedrock上最受欢迎的模型系列之一。

此外，AWS客户将能够直接从AWS访问Anthropic原生的Claude控制台，客户可通过现有AWS账户访问。

**在训练和云服务方面**，Anthropic继续选择AWS作为其关键任务工作负载的主要训练和云服务提供商。亚马逊的开发人员和工程师可使用Claude模型进行构建，以改善亚马逊旗下各业务的客户体验。
