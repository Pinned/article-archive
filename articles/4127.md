# 22美元AI便签走红CES 2026！这家中国公司让外媒狂追，24小时曝光近10亿

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6v5P1ib5ptZzgqChNic7eiaHAcQ6gauvkzPoiatyPBuWNPNOWyJ4oEEd6WbQ/640?from=appmsg)

**CES 2026直击大模型硬件化元年：用Agent管住大模型，AI走进普通人的口袋。**

**作者 |**  李水青  

**编辑 |**  漠影  

**2026年的拉斯维加斯消费电子展（CES），正在悄然完成一次主角更替。**

展馆里，AI依然无处不在，但与前两年“模型参数竞赛”、“算力军备展示”不同，今年最密集出现的是一大批**围绕大模型设计的终端硬件**。它们不再只是“会说话的音箱”、“加了AI的旧设备”，而是走进生活场景的AI原生新硬件。

过去一年，AI新硬件赛道刚经历了剧烈的“去泡沫化”。Humane AI Pin、Rabbit R1等第一代探索者，用高定价、复杂交互和模糊定位为行业验证——**单纯把大模型塞进硬件，并不会自动诞生一个新物种。**

也正是在这样的背景下，**今年CES现场，一家来自中国深圳的公司，试图给出另一种明显更“落地”的解法。**

首次亮相CES的**玄源科技（X-Origin）**，携两大品牌**AIPi与Yonbo**登场：从22美元起售的**口袋型AI助手**，到覆盖家庭与户外的**儿童AI伙伴系统**，这家公司用极低门槛、清晰边界、可规模销售的产品，迅速成为展会中的高频讨论对象。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6vL1mdGwCNOjvlN7J8h45W2BjDUEiaZxZp43JLqQe4N4ch9DH7CGp1icWA/640?from=appmsg)

▲玄源科技展台吸引了来自全球的观众参观

AIPi与Yonbo获得了多项CES媒体奖项提名与背书。其接连接受包括福布斯、路透社、BBC、CGTN在内的多家国际媒体现场采访，在24小时内曝光达**近10亿**。其中，福布斯（Forbes）记者在现场对其品牌理念与产品路径给予高度评价，并在随后发布的报道中，将Yonbo形容为“专为有孩子家庭打造、通过AI互动帮助儿童学习成长的陪伴型机器人”。

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6viaaAsQddWen7txNIkA0Kw02iaU0vkgTnXYrSlMseqvYwyDqIicwDEhribQ/640?from=appmsg)

▲福布斯报道截图

玄源科技在本届CES期间还获得了多项“Best of CES 2026”奖项，并成为**NVIDIA Inception成员之一**——在本届参展的中国大陆及港澳企业中，**仅有玄源科技、宇树科技等极少数公司获得这一标识。**

**这背后，或许正指向AI硬件进入下半场的一个关键共识。**

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6viajYvjVM2AXzq6X23GTRxCSwaXuFDLn8kK7lhHOJKza1wtfhXgzQHEA/640?from=appmsg)

▲玄源科技入选NVIDIA Inception成员

  

**_01_****_._**

**仅22美元起**

**四款AI硬件在CES 2026引围观**

在CES 2026的威尼斯人会展中心Lifestyle展区，玄源科技的展位前，你很难不被一块“像便签一样大小”方形设备——AIPi Lite引注意力。

这是一款口袋型AI助手，它厚度不足8毫米，可轻松握于掌心或置于口袋，售价仅22美元。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6vH7QqEeLhFVV1SSibdqicL7JQR9wsJuu3bzGTu7y5S0vAA8MGpqHh04HQ/640?from=appmsg)

▲玄源科技AIPi Lite

**1、AIPi Lite：把“用不用得起”这件事，先解决掉**

在AI硬件普遍动辄数百美元的背景下，22美元几乎是一个“反行业共识”的定价。

AIPi Lite的思路非常直接：先把AI的使用门槛降到足够低。这款设备没有App、没有复杂界面，核心交互只有一件事——按键，对话。用户无需学习教程，也无需理解模型概念，插电即用，几乎是“肌肉记忆级”的操作。

更关键的是，AIPi Lite并非功能单一。设备内部预置了上百种适配Agent，覆盖娱乐、教育、创作、工具等常见场景，把大模型能力直接“封装”为可即刻调用的能力模块。这种“口袋型Agent”的设计，让大模型具备了随身属性。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6vnEmic1bJnxzQoneedj6gLdgR0ic8vBk0FkAicia4wxnu5SqtkjI19iaUZ5A/640?from=appmsg)

▲玄源科技工作人员在为观众讲解AIPi Lite

**2、AIPi Pro：让AI的交互变得可控、可预期**

如果说AIPi Lite解决的是“普及”，那么定价99美元的AIPi Pro回答的，是AI硬件下一步的关键问题：如何让交互变得稳定、可预期。

在AIPi Pro上，玄源引入了NFC实体卡片切换Agent的设计。用户可通过触碰不同卡片，瞬间将设备转换为“英语陪练”“健身教练”或“创意伙伴”，解决语音唤醒的不确定性。相比纯语音指令，这种物理交互显著降低了误触与不确定性，也让“切换角色”这件事变得可见、可控。

同时，AIPi Pro引入了更完整的智能记忆系统与成长机制。设备会通过向量化方式记录用户偏好，形成长期记忆；交互频率与内容，也会反过来影响AI的反馈方式，使其呈现出明显的“养成感”。

此外，AIPi Pro加入小型触摸屏，实现“语音为主、视觉为辅”的多模态交互，进一步打磨体验细节。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6vzKvibjGsoeia2Qyum0ya3QdBTKxE3EsibgdlS57uibq5Own8EP6bGzIFaA/640?from=appmsg)

▲AIPi Pro加入小型触摸屏并且融入3D打印生态

**3、Yonbo双机体系：一个灵魂，两种形态**

相比AIPi的“随身助手”定位，Yonbo则是玄源在垂直场景中的一次深度实践。

Yonbo看起来像一款儿童故事机，实际上并非这样简单，它是一套“双机一体”的伴随式成长体系——由Yonbo X1家庭陪伴机器人与Yonbo Camera户外探索相机组成。两者最大的创新点在于：共享同一套AI记忆系统。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6v5hnoguCYxvCLJ0YoaDcqFGHwgTv9GDNx5NHO7zEsYC4oSFWMvzFJXQ/640?from=appmsg)

▲Yonbo“双机一体”

具体来看，Yonbo X1作为家庭陪伴机器人，侧重听觉与深度对话，提供情感支持与互动学习；Yonbo Camera则是户外探索相机，鼓励孩子将所见转化为所学。它们通过共享AI记忆系统（Shared AI Memory System） 打通数据与情感。这种跨设备、跨场景的连续记忆，使AI不再是“每次重置的工具”，而更像一个持续在线的伙伴。

在玄源科技的设计中，孩子白天用Yonbo Camera在户外拍下的内容，会成为晚间Yonbo X1对话的一部分。比如，孩子白天用Camera拍下的蝴蝶，晚上回家后，X1会主动提起并延伸讲述科普故事。

在儿童场景中，Yonbo通过年龄分层算法与内容过滤机制，强调“有边界的情感计算”：既提供情绪价值，又确保内容安全与教育导向。

在外媒眼中，Yonbo的价值并不止于“会说话的机器人”。Forbes在CES现场报道中特别指出，Yonbo通过可视化情绪表达、拟人化动作以及长期记忆机制，构建了一种更贴近儿童心理节奏的互动方式。

可以看到，AIPi Lite、AIPi Pro、Yonbo X1、Yonbo Camera以上这四款产品并不是孤立设备。相比强调单点功能，玄源科技更关注的是：跨设备、跨场景的AI记忆连续性。这一点，也构成了其与多数“单体AI硬件”最本质的差异。

  

**_02_****_._**

**用Agent管住大模型：**

**AI开始有边界、有性格**

如果说2024-2025年是AI硬件的“概念探索期”，那么2026年，行业已明显进入下半场。消费者关心的问题正在发生变化：不再是模型有多大，而是是否真的好用、是否能长期留下来。

这背后，暴露出通用大模型直接嵌入硬件的两大结构性问题：

**1、幻觉问题。**我们越追求AI的全能，越容易在专业场景输出不靠谱信息；

**2、平庸化问题。**AI什么都能聊，但缺乏性格、记不住人，难以建立长期情感连接。

玄源科技在CES上抛出的核心观点，正是针对这两大痛点：“未来的AI伙伴，不该是‘无所不能’的工具，而应是‘有边界、有性格、有记忆’的实体。” 其解法颇为反直觉，可概括为——“用Agent限制大语言模型”。

在玄源的产品架构中，大模型并不是“直接暴露给用户”的存在，而是被Agent体系所约束。Agent负责定义边界、角色与能力范围，让AI在特定领域（如儿童教育、日常助手）中更专业、更可信。这种“限制”，反而提升了用户对AI行为的预期稳定性。

“在大模型追求‘全能’的时代，我们提出‘限制LLM’。只有在边界内，大模型才能更好地植入情感模型和成长体系，让AI从‘工具’进化为‘有性格、有生命感的实体伙伴’。”玄源相关负责人称。

这一理念在产品中得到了充分体现。在AIPi系列中，上百种预置Agent就是上百个“边界清晰的数字角色”；在Yonbo体系中，边界则体现为“有边界的情感计算”与严格的内容过滤——通过MBTI性格分析与年龄分层算法，Yonbo会严格过滤不适龄信息，确保教育安全，同时基于共享记忆理解孩子情绪与兴趣，实现个性化成长陪伴。

![](https://mmbiz.qpic.cn/mmbiz_jpg/z7ZD1WagSLia1TaaTGOcEXGjSlZqdKia6vORAElXLB8QibQLaSP0BibvicHo48NFnoDwb7tq05E7Ghh3D16ZB6uNSNA/640?from=appmsg)

▲Yonbo“双机一体”

在Agent之上，玄源构建了持续记忆系统与情感计算逻辑。AI不只是响应问题，而是在长期交互中形成对用户的理解与反馈风格。这也是其“AI伙伴”概念的核心——不是无所不能，而是有性格、有成长路径的实体。

也正是这种“先划边界、再谈情感与成长”的思路，让Yonbo在CES现场并未被简单归类为“AI玩具”。多家国际媒体在报道中，更倾向将其视为一个服务于儿童成长与家庭教育场景的AI伙伴，而非单一功能设备。

从行业层面看，中国AI硬件厂商正在推动一次明显的交互革命：从“人适应机器（App模式）”，转向“机器适应人（Agent模式）”。在玄源科技的实践中，AIPi与Yonbo都在极力减少用户的学习成本，把交互压缩回最自然的语音与实体操作。

  

**_03_****_._**

**从“概念机”到全球可售的成熟产品**

AI硬件最容易被质疑的一点，是“只能展，不能卖”。

从爆火的Humane AI Pin到Rabbit R1，AI硬件赛道从不缺乏惊艳的概念，缺的是能规模交付、持续盈利的成熟产品。玄源科技在CES上备受关注的另一原因，或许是其证明了“AI+硬件”已具备商业落地能力，而非停留在PPT阶段。

量产与渠道能力是其关键佐证。据悉，AIPi Lite与Yonbo X1目前已在Shopify、Amazon、TikTok Shop等主流电商平台现货销售，全球用户可直接下单购买。尤其是Yonbo X1，自2025年第四季度上市后，已热销全球25个国家及地区，验证了其供应链的交付稳定性与商业模式的初步跑通。这意味着玄源科技的产品，已跨越“工程样机”阶段，进入真正的消费电子市场循环。

支撑这种快速落地能力的，是玄源科技背后的技术平台化架构。公司底层技术平台支持接入ChatGPT、Gemini等多种主流大模型，并能通过Agent机制进行深度定制与“裁剪”，兼顾前沿AI的通用能力与垂直场景的专属优化。以Yonbo为例，其“共享AI记忆系统”不仅是一个功能亮点，更是一套可在儿童教育场景中深度落地的技术方案，实现了数据本地化存储（保障隐私）、多模态信息融合与长期记忆索引。

玄源科技的相关负责人此前在采访中透露，公司技术架构历经三年打磨，已实现记忆、视觉、听觉、语音与软硬件控制的全面打通，这使得产品研发周期大幅缩短。例如Yonbo Camera从0到1的开发仅耗时约100天，而“内容更新最快一天即可完成”。

正是“平台化能力+快速迭代”的模式，让玄源科技能在AI硬件尚未定型的市场中灵活试错、快速响应。

  

**_04_****_._**

**结语：AI硬件的真正分水岭已经出现**

CES 2026，或许会被记住为大模型硬件化真正开始落地的一年。

当行业集体反思第一代产品的教训，玄源科技试图以其“低门槛、强场景、有情感、可量产” 的产品矩阵，证明AI硬件不应盲目堆砌算力或追求通用，而应关注降低使用门槛、专注真实场景、实现情感化陪伴。

以玄源科技为代表的中国深圳创业天团，正扮演“AI硬件平权”的推动者角色。22美元的AIPi Lite、799美元的Yonbo X1，以及连接家庭与户外的陪伴体系，都在诠释技术普惠的愿景。

展望未来，随着更多企业加入这场以Agent为牵引、以用户体验为核心的硬件创新，AI硬件将逐步褪去“概念机”的光环，真正融入大众生活的细枝末节。

也许不久后，人们不再争论“哪个模型更强”，而是习惯身边有一个“有边界、有记忆、有温度”的AI伙伴，无缝参与自己的学习、娱乐与成长。
