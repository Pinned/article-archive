# 马斯克冲刺机器人量产，果断停产特斯拉豪华车型！2026年资本支出将“非常大”

##### 鱼羊 发自 凹非寺  
量子位 | 公众号 QbitAI

让特斯拉转型成「机器人」公司，马斯克来真的了：

在最新财报电话会议上，马斯克宣布，特斯拉将在2026年第二季度**停产豪华车型Model S和Model X**。

目的很明确，就是要给特斯拉机器人擎天柱（Optimus）让出生产线。

> 现在是时候让Model S和Model X体面地退出舞台了，因为我们正在迈向一个以自动驾驶为基础的未来。

擎天柱官推则更为明确地放出了这样一个视频，配文：

> Model S和X将通过我继续存在。

马斯克透露，在把特斯拉加州弗里蒙特工厂的Model S/X生产线改造成擎天柱生产线后，其机器人的产量将达到**每年一百万台**。

特斯拉「情怀」车型走向终结
-------------

Model S/X的退役，倒也不是无迹可寻。

毕竟马斯克2019年就说过，这两款分别诞生于2012年和2015年的车型纯粹是因为“情感原因”才被继续生产的。

数据很直接：在过去几年里，特斯拉每年能卖出差不多170万辆车，其中Model S/X也就占几个百分点，并且销量还在持续下降。

以2025年为例，特斯拉全年全球累计交付1636129辆电动车，其中包含Model S/X和Cybertruck的「其他车型」的销量为50850辆，只占到约3%。与2024年相比，「其他车型」的销量还下降了40.2%。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtCj0jucpq4bgUhrE6HrKEg3cYbQ2YPUS6agP4txpys7iaLgd9PdSJ1OCR9Cjv3wUabUDks56icaXBkg/640?wx_fmt=png&from=appmsg)

一边是不挣钱的「情怀」_（Model S是特斯拉第一款自主量产车）_，另一边是马斯克眼中「经济增长的重大推动力」，在叠加上特斯拉从「汽车公司」向「自动驾驶汽车和人形机器人」公司转型过程中的利润下降，马斯克弃情怀为擎天柱机器人让路，也就是情理中事了。

财报显示，特斯拉第四季度净利润为8.40亿美元，较上年同期下降了61%。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtCj0jucpq4bgUhrE6HrKEg3508U1GaAIE0WKlCGZicauOpiamk05yYx1q6BgBRvpwTSVx7Gq3PO7LBg/640?wx_fmt=png&from=appmsg)

但怎么说呢，情怀这种事嘛，主打一个精神不死（doge）。

擎天柱官推在短短28秒的致敬小短片里是这样明示的：从Model S/X诞生起，特斯拉在电动汽车上不断积累的核心技术，将被擎天柱所继承。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtCj0jucpq4bgUhrE6HrKEg3vZIak6wiaPNtMwlY1QpgCzJ1dCeibO7aerh6KpA8prlxpF4Ud1KN9V3Q/640?wx_fmt=png&from=appmsg)

包括执行器、电力系统、电池、制造工艺和自动化产线经验、数据通信、音频系统、摄像头、特斯拉自研芯片、训练集群以及仿真系统和AI算法等。

特斯拉20亿美元投资xAI
-------------

不过特斯拉利润下滑这事儿，看上去并不妨碍马斯克继续传统艺能「左手倒右手」。

特斯拉2025Q4财报还披露，特斯拉已经在2026年1月16日签署协议，将在xAI最新一轮融资中向其投资20亿美元。这笔投资预计将在2026年第一季度完成。

特斯拉表示，这是《宏图计划4》的一部分，两家公司的合作旨在增强特斯拉大规模开发并将人工智能产品和服务部署到物理世界的能力。

2025年7月，马斯克旗下的SpaceX也给xAI投了一大笔钱，金额同样是20亿美元。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtCj0jucpq4bgUhrE6HrKEg3lJxE65ibickngQMLqe7miaMEptKKzTIo1a2sgwnxKHpWvoCSibOMtJ1UKg/640?wx_fmt=png&from=appmsg)

另外，马斯克还在财报会议上表示，2026年特斯拉资本支出将“规模空前”。

特斯拉首席财务官瓦伊巴夫·塔内贾透露，包括加紧生产人形机器人在内的工厂投资计划，将使特斯拉今年的**资本支出超过200亿美元**，是2025年85亿美元的2倍多。

One More Thing
--------------

特斯拉之外，马斯克还带来了旗下脑机接口公司Neuralink的最新消息：

下一代Neuralink「赛博增强装置」将实现3倍性能提升。

另外，Neuralink已经在准备一个帮助盲人恢复光明的新项目，能使完全失明的人以低分辨率“看见”世界，并逐步提升这种新视觉的分辨率。
