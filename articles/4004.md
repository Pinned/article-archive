# 刚刚，一个2.6万亿AI独角兽诞生！英伟达微软押注，马斯克急了

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acur6e0nWq9ZtXdEMJTr2yXGI3xUTSzlkGMOLoOGl6g7gnVSt7uQ5rVs9F43dnCJV8guVSQGLiadiaAJlPCMh5JHD7QTWu7FUiaXVE4/640?from=appmsg)

**年化经常性收入已达699亿。**

**作者 |**  王涵  

**编辑 |**  冰倩  

智东西2月13日报道，今天凌晨，AI独角兽Anthropic宣布已完成**300亿美元（约合人民币2070.19亿元）**G轮融资，投后估值达**3800亿美元（约合人民币2.62万亿元）**，公司年化经常性收入已达**140亿美元（约合人民币699.09亿元）**，在过去三年中每年平均增长**超10倍**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcupmbvhqRy9ia1aURUicJVJKwxK1RvNalaDJ0ErQO4AehANlesMScwJTXNia78LibZzVuCx8gNu9YFJHLbrqbkwCOCtoq3tmITphRGY/640?from=appmsg)

▲Anthropic公告（来源：X）

据Anthropic公告透露，本次融资的资金将助力前沿研究、产品开发及基础设施扩建，目前Claude是唯一在三大全球云平台Amazon Web Services Bedrock、Google Cloud Vertex AI、Microsoft Azure Foundry同时可用的AI模型。

消息一经发布，马斯克立刻跳出来质疑Anthropic的Claude模型涉及种族和性别歧视的问题。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acurp6ibSj6iazZeNL89iarRyC7b2q2nc0KTLWdWDEZXnzcqsgohQ3ZibOLbssl5JaXjEBVxmfcmSJLiawib8EJuFUtvgJicRUELysFGfts/640?from=appmsg)

▲马斯克发表质疑（来源：X）

有网友在下方评论称“求你了Elon，好好搞一下Grok吧。”马斯克回复道：“4.2会更好。”

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurRWibiazQ8ice9m2icibjlkrI8iaRDg7ULeE9XBsn1JxLLBnicT2QUfvNoAmMdiaB64pjE4LF5d81MicCo6DMpKKkmXwsADFhvsL1c2y4M/640?from=appmsg)

▲马斯克回复网友评论（来源：X）

公告中称，Anthropic本轮融资由安大略省教师退休基金会和Coatue联合领投，联合领投方还包括D. E. Shaw Ventures、Dragoneer、Founders Fund、ICONIQ和MGX。本轮重要投资者有黑石集团、高盛另类投资旗下成长股权部门、摩根大通、红杉资本、淡马锡等。

本轮融资还包括2025年11月已公布的英伟达和微软的投资，即英伟达和微软承诺分别向Anthropic投资**100亿美元（约合人民币690.06亿元）**和**50亿美元（约合人民币345.03亿元）**。（[Anthropic获微软英伟达千亿投资！估值飙到2.5万亿](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652792813&idx=2&sn=cfc047e23f3fb2baa5962a4951dd158a&scene=21#wechat_redirect)）

  

**_01_****_._**

**超500家客户年掏百万美元**

**财富10强抢着用**

公告中透露，以年化经常性收入计，年消费**超10万美元（约合人民币69.01万元）**的Claude客户数量在过去一年**增长7倍**。从单一用例API、Claude Code或Claude for Work开始使用Claude的企业，正将其集成扩展至全组织。

两年前，仅有十余家Claude客户年度支出超百万美元，如今该数字已**突破500**，财富10强企业中有**8家**现已为Claude客户。

Claude Code自2025年5月全面开放以来，如今其年化经常性收入已**超25亿美元（约合人民币172.52亿元）**，自2026年初以来增幅翻倍。

Claude Code的每周活跃用户数自2026年1月1日以来也实现翻番。近期分析显示，全球GitHub公共提交中的代码有**4%**是由Claude Code生成的，该比例是一个月前的**2倍**。

Claude Code的企业订阅量自2026年初也**增长了4倍**，企业使用贡献已**超总营收半数**。

  

**_02_****_._**

**Opus 4.6首发百万token**

**Claude Code狂揽开发者**

过去1年中，Anthropic在模型和产品侧都做出了广受认可的成绩。

Anthropic的Claude系列模型持续迭代，由**3.7版本升级至最新的4.5版本**，编程能力、Agent能力都有明显提升，能在生产力场景更为端到端、长时间地完成复杂任务。

2026年2月6日，Anthropic发布旗舰模型**Claude Opus 4.6**，是Anthropic**首款**开启**100万token上下文窗口**测试功能的旗舰级模型，该模型在长上下文查询、推理、规划等能力上均有所提升。（[Claude Opus 4.6杀死编程比赛！挖出500个0day漏洞，生成k线成交量分布，还有PPT直出](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652796719&idx=1&sn=12a06f1d51a05c2fba7197478e770f63&scene=21#wechat_redirect)）

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acuoy6z3r3QA6w7y6DVmuoEK5UKkL5gfJSJUPlia63svtj4s62FWa6fZzFOiaCCpR5spzGNmtVHv9EODP775TpeKphQfJoTaIz93yM/640?from=appmsg)

▲Claude Opus 4.6发布推文（来源：X）

在其专精的编程领域，Claude虽然并不总是位居榜单第一，但在开发者群体里的口碑较好。

产品侧，Anthropic做出了爆款产品**Claude Code**。2025年2月，Claude Code随着Claude 3.7 Sonnet一同发布，此后迅速成为开发者圈内的**爆款**。

作为一款能自动执行工程任务、读写代码库、运行测试的AI编程助手，Claude Code采用了开发者熟悉的命令行界面，还在上下文持久化、Agent能力上持续进步，体验不断优化。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuoEg1EFia3ia4peSN9P3mfCaPvWRxxFdq0icbUKMW6GWFcaicEJmhiaTPtb8tPbXQC7wxPGKE3885KvALoeyT9QZ5aH233YGibWDibicBU/640?from=appmsg)

▲Claude Code（来源：Anthropic）

今年1月，Anthropic还推出了Claude Code的图形化UI版本——**Cowork**。如今，更多非技术背景的人员也可以使用到Claude Code的Agent能力，完成整理文件、抓取网页等非编程类任务。

此外，Anthropic也试图定义Agent这一赛道的开放生态。2025年10月，Anthropic推出了**Agent Skills**概念，并在同年12月将其作为开放标准正式发布，试图让智能体具备类似“技能包”的模块能力，这些技能可以被封装、复用、共享和组合。

Agent Skills本质上是由指令、资源和可选脚本组成的文件夹式结构，Claude会根据当前任务自动识别并按需加载相关技能，从而实现可重复、可组合的专业化任务执行能力，而不是每次都靠提示词驱动

此外，Anthropic正在为**IPO**做准备，最早可能在**2026年**进行。Anthropic已经聘请律师事务所开展前期准备工作，并与多家投行就上市事宜进行了初步接触。

  

**_03_****_._**

**结语：Anthropic商业化进程**

**按下加速键**

与OpenAI的To C路线不同，Anthropic则更加注重To B市场。

Anthropic已经实现年化收入140亿美元、三年年均增长超10倍，其产品Claude Code贡献了超25亿美元年化收入，这一系列成绩都标志着Anthropic正以惊人的速度推进商业化进程，将技术领先转化为市场优势。

Anthropic在基础设施层面的多元化布局横跨AWS Trainium、谷歌TPU、英伟达GPU，为企业客户提供了兼具性能与韧性的基础设施保障，这正是Anthropic在激烈竞争中构筑的护城河。
