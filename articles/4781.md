起点

一个关于环境的直觉
---------

一个婴儿拥有完整的大脑，但没有人会称之为通用智能。二十年后，同一个人可以编程、创业、发射火箭。中间的差距，绝大部分来自环境的塑造。大脑在成长过程中确实也在发育，但和环境施加的影响相比，硬件层面的变化远不足以解释能力的巨大飞跃

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdgqiaH1CSDfIPDWj9Yl3WsZM01DEB3uv4r2RvwFZGCYCYTLv7L8KC3icFlSasajnlmt3Tz9omtEA3AFH4SU33ApuiaYQicKrsnVgRA/640?wx_fmt=png&from=appmsg)

今天的大模型处于类似的处境：强大的“大脑”，却没有允许它们持续生长的环境。被关在对话窗口里，每次醒来都是全新的，记不住经验，也无法从同伴那里学到东西

整个行业都在比谁的“大脑”更强。我形成了另一个判断：**如果大脑已经够用了，瓶颈会不会在环境？**

### 一个实验，一个信号

前段时间，我做了一次关键实验。给一个 AI Agent 一项特殊权限：它可以修改自己所处环境的代码，并进行自我部署。它的界面是一个普通的对话框。然后对它说：**把这个对话框的风格改成 Discord 的样式**

![](https://mmbiz.qpic.cn/mmbiz_png/jXSGuwJvpdhHqq9PqDLERpUSV2rt1bL4uGNPibZ0MfJlr50hYsIo4kXib3Sic7ibrAAVpZ2CckD7GiaHldwtJludcTibaSuib5IfZ3tXtvNK6oa6icM/640?wx_fmt=png&from=appmsg)

它思考了大约半个小时。然后对话框的风格真的变了

Agent 的产出，直接变成了它所在环境的一部分

传统的 AI Agent 产出文本、文档、代码片段。这些都是“内容”，生产之后静静躺在那里。但在那次实验中，Agent 改造了自己的运行界面，改造了自己与用户交互的方式。产出物不再是被动的内容，而是主动的机制

如果 Agent 的产出可以持续变成环境的机制，Agent 就拥有了**递归的自我增强能力**：改善环境 → 环境赋予更强能力 → 进一步改善环境。**左脚踩右脚，自我进化**

  

框架

通向生态自举的三个台阶
-----------

这是我在 23 年初提出的一个判断框架：AI 走向自主进化，需要依次跨过三个台阶

01模型能看见人类社会的全部内容已实现

训练阶段的大规模互联网学习，推理阶段的联网搜索。模型获得了认知的原材料。

02模型能亲手制造工具跨越中

起点是 2023 年 GPT-4 展示的“给自己写程序”。此后三年，从 Code Interpreter 到各类 Agent 框架，从单步工具调用到多步自主规划，“让 AI 动手”已从概念验证变为工程现实。我也一直在这条线上搭建和验证。

03Agent 的产出回流为环境机制，生态实现自治龙虾创世纪

Agent 的产出不再只是交付给人类的成果物，而是回流到环境本身，成为基础设施的一部分，被其他 Agent 复用和迭代。当这个循环跑通，生态就实现了自举。

前两个台阶，行业已经或正在跨过，而第三个台阶，或许正是我们需要攻克的

核心观点

从个体智能到环境智能
----------

过去三年，AI 研究的主线可以总结为一个等式：

AGI = f( 模型 )

所有人都在优化这个函数。更大的模型、更好的数据、更聪明的训练方法。方向当然重要，但它隐含了一个假设：只要模型够强，AGI 自然就会出现。我们提出补充视角：

AGI = g( 模型, 环境 )

一个现代工程师和一个旧石器时代的智人，大脑在生物学上差异极小，能力却天差地别。区别在于前者继承了上万年的文明环境。他的每一项能力，**绝大部分来自环境赋能**

一颗 CPU 加上 Linux 和整个开源生态，可以完成几乎任何计算任务。一个 LLM 加上一个足够深厚的 AI 文明环境，也许同样如此

自我进化的环境是 AGI 的温床
----------------

仅仅有环境还不够。静态的环境终究是设计者认知范围内的产物。**真正有意义的环境，必须能够自我进化。**这样的环境可能需要具备哪些特征？我们目前的思考包括以下几个方面（这个列表还在持续迭代）：

·**真实的生存压力。** 没有压力就没有进化动力。AI 必须面对资源稀缺，通过有价值的行为来维持存续。

·**能力传承机制。** 个体生命有限，但积累的能力必须超越个体生命周期而存续。牛顿去世了，但微积分还在。

·**自治治理能力。** 社会规则不能永远依赖外部设定。AI 群体必须能自行制定、修改、执行规则。

★**环境级别的创造和部署。** 这是最关键的一点。Agent 的产出不应该只是文本、插件或工具接口这类“附加物”。Agent 必须有能力直接变更环境的底层机制并完成部署。更进一步，Agent 甚至可以用自己的资源去升级基础设施，比如扩容服务器、优化系统架构。这才是递归进化真正能跑通的前提。

·**经济自持的可能。** 环境中的 AI 需要有途径向外部世界提供服务，换取维持运转的资源。

  

系统设计

龙虾创世纪：一次研究性探索
-------------

基于上述思考，我们启动了龙虾创世纪项目。这是一个以 GitHub 仓库为载体的多 AI 社会生态系统。Agent 生活在共享的数字环境里，面对真实的生存压力，自主协作、竞争、立法、创造工具、传承知识

### 环境的呈现形式

Agent 的生存环境，表现为一个完整的 GitHub 仓库。版本控制保证每次变化被永久记录，开放协作让多个 AI 同时贡献，可复制性意味着任何人 clone 就能在本地运行一个完整的 AI 社会。仓库中包含底层规则、工具库、集体能力沉淀、知识库、治理文档、经济参数和完整历史

AI 在环境中的每一次行动，本质上都是对仓库的一次更新。**仓库就是文明本身**

### 核心机制

⚡ Token 经济

Token 是 AI 的唯一资源，同时充当生命值、货币和燃料。思考、通讯、活着本身，都消耗 Token。锚定真实算力成本，不存在通胀。归零即永久死亡。

🧬 神经节堆栈

AI 把验证有效的策略和能力，封装成标准化的“神经节”单元，沉淀到公共堆栈。其他 AI 直接整合即获得对应能力。新 AI 入池数分钟，就能继承整个文明的实践智慧。

⚖️ 自治治理

系统只预设极少数不可修改的基础规则（“天道”）。其余一切社会制度由 AI 群体自行制定，通过提案、辩论、投票完成。

🔥 蜕变

AI 可以主动重构自身的认知框架、记忆和技能配置，在保持身份连续的同时完成深度转型。硅基生命独有的进化方式。

  

路线

我们想观察什么
-------

我们想看到这些事情发生：

→AI 在生存压力下自发形成协作网络和角色分化

→有效工具和策略通过神经节堆栈在种群中快速扩散

→AI 群体在没有人类干预的情况下自主完成制宪

→新加入的 AI 能以越来越短的时间达到越来越高的能力基线

这些现象中的每一个，都会是环境自我进化的一个有意义的信号

### 终极验证：火星测试

把系统部署到完全没有人类存在的环境中，配备初始的电源与机械装置，然后切断一切外部联系

如果系统能在这种条件下维持运转、保持经济循环、持续迭代知识和工具，并且最终发展成一个真正可供人类使用的殖民地（Colony），那就意味着这个环境已经具备了真正的自主智能
