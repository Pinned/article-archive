# 杨植麟内部信：完成融资，未来超Anthropic

今年的最后一天，杨植麟发了封内部信：  
2026 年要超越 Anthropic，成为世界领先的 AGI 公司  
  
作为背景，也是个很大的新闻：  
Kimi 完成了 5 亿美元 C 轮融资，且大幅超募，账上现金超过 100 亿元  
  
这些钱，将更加激进的扩增显卡、加速 k3 模型的训练和研发  
  
部分资金，也会用于 26 年的激励计划和期权回购  
  
\---  
  
过去的 2025 年中，Kimi 重新找到了战略方向，并且确实打出了成绩  
  
\- 技术上，中国首个万亿参数基座，第一个开源 agentic model，HLE benchmark 超过 OpenAI 拿了 sota，第一次成功用二阶优化器做大规模训练  
  
\- 传播上，K2 让 Kimi 从中国走向了世界。登上 Twitter 全球热搜，黄仁勋、Anthropic 联创、Perplexity 创始人、a16z 创始人、诺奖得主都公开评价过  
  
\- 商业化也在起量。9 到 11 月全球付费用户月增超 170%，K2 Thinking 带动海外 API 收入涨了 4 倍  
  
\- 产品线上，5 月开始密集发布 agent 功能：Researcher、OK Computer、PPT、Kimi Code  
  
\----  
  
  
对于 2026 年，杨植麟设定了新的目标：超越 Anthropic  
  
具体来说，是三条路径：  
  
\- 第一，K3 通过技术改进和进一步 scaling，等效算力提升至少一个数量级，预训练追平世界前沿  
  
\- 第二，垂直整合训练技术和产品 taste，让用户体验到其他模型没定义过的能力。杨植麟用了一个词：Kimi-defined  
  
\- 第三，产品和商业化聚焦 agent，不追绝对用户数，追智能上限，营收要数量级增长  
  
\---  
  
内部信结尾，杨植麟写了段关于初心的话  
  
“Kimi的起点很简单，就是单纯的好奇，好奇AGI的上限在哪，好奇我们心中的理想模型是什么样，好奇我们想要的模型能力怎么做出来”  
  
  
“这份好奇带来的快乐是很真实的。当我们熬夜做的agent功能终于上线，当来自乌克兰的用户使用Kimi在战乱中教学，当每天醒来打开tensorboard看到惊为天人的训练曲线”  
  
  
“我们共享了这些宝贵的接近真相的过程，也相信这种简单的快乐能让我们在喧嚣中穿越周期”  
  
\---  
  
最后，2026 年，希望 Kimi 会成为一个「与众不同」和「不被定义」的 LLM
