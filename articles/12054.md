# 谢尔盖·布林重启「创始人模式」？谷歌组建「突击队」，重押「AI编程」

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

机器之心编辑部

  

被 Anthropic 「逼急」了，谢尔盖・布林「发飙」了。

  

近日有媒体报道，谷歌内部已组建一支由研究人员和工程师组成的「突击队」（Strike Team），专门提升其 AI 编程模型能力，最终目的是让公司内部更多的编码工作实现自动化，甚至推动 AI 自主完成 AI 研究本身，即 AI 的自我优化。

  

据悉，「突击队」由 DeepMind 工程师 Sebastian Borgeaud 领导，此前曾负责 DeepMind 的 Gemini 预训练工作。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGXneoyCh6DDVVHG1HK82jbUQGUOufXKiaQhexR8N5vLibia6icw4bxDEIPmSR68CgfcKzYoL5s0BbJBTOm23l7zSMictT44RFE6W7U/640?wx_fmt=png&from=appmsg)

  

而谷歌创始人谢尔盖・布林和谷歌首席 AI 架构师、DeepMind CTO Koray Kavukcuoglu 也都直接参与「突击队」工作。甚至在一份最新的备忘录中，谢尔盖・布林直接言明，要求团队「激进转向」，以追赶谷歌在 AI 编程，以及 Agent 领域的差距。

  

「要赢下最后的冲刺，我们必须迅速弥合在 Agent 执行能力上的差距，并让我们的模型成为代码开发的核心主体。」

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGKHDWm0a4U9iciaeHacZXhjQfGRxlt4pukcmbrTRRuS1QHvJOQwDwaQsY4plPNBATQeyJxjibhWp6IrwIUbBp0ZaGJU3HAZ6CKzs/640?wx_fmt=png&from=appmsg)

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGrZA1QTa5yPIIe4OAoXibIC3rp4mkGxgzqicMic6bxroiavQLS2B5PavG3I15KwkI7SlhiaC6SVzJcRIviaangaolEqWRxbd6mic5tkM/640?wx_fmt=png&from=appmsg)

  

可见，谷歌意识到当前的竞争形势不容乐观，开始着急了。

  

据知情人士透露，谷歌之所以组建「突击队」，一部分原因来自于 Anthropic 近期发布模型的直接刺激，这无疑是指 4 月 16 日发布的 Claude Opus 4.7。

  

从数据上来看，Claude Opus 4.7 编码基准提升了 13%，生产级任务的自主完成率提升了 3 倍，Anthropic 更是将其描述为「软件工程领域的显著提升」，业界也是对其称赞声一片。

  

而在谷歌内部，DeepMind 的研究人员也认为，Anthropic 的编程工具在代码编写能力上已经超过了 Google 的 Gemini 模型。关键是这种「超过」，甚至不是一星半点的领先……

  

今年 2 月初，谷歌在发布 Gemini 3 以及智能体平台「Antigravity」时，谷歌 CFO 阿纳特在财报电话会上透露，「谷歌约 50% 的代码已经由 Coding Agents 编写」，之后再由工程师进行审查。

  

而对于 Anthropic 来说，这个数字是「100%」。

  

今年 1 月，Anthropic 的 Claude Code 负责人 Boris Cherny 发文称，他已经两个多月没有编写任何代码了，「100% 的代码都由 Claude Code 和 Opus 4.5 编写。」

  

他还表示，甚至公司其余部门也「几乎 100%」的代码都是由 AI 生成的……

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEwb6FRKL7OujPWZvWpB3ASib0fYvvfSPibUqskbrv30icna6jpzgIQXrXXvgmokqz9S4Kdn6SIUBE3LDGKqNBhXXXyc6qBWBHwI0/640?wx_fmt=png&from=appmsg)

  

这一对比下的巨大落差，也难怪谷歌「急了」。

  

其实不仅是谷歌，外界也震惊于这一差距。

  

X 博主 Yuchen Jin 表示，「令我惊讶的是，谷歌拥有全球最大的内部代码库（>2BLOC），但在编码 + Agent 数量上却落后于 Anthropic 和 OpenAI。

  

当然，这或许是因为谷歌工程师主要被限制在内部模型和工具（如 Gemini CLI 和 Antigravity）上，因此出现这种局面也不足为奇。」

  

更重要的是，如今谷歌已经开始采取行动，尤其是谢尔盖・布林亲自坐镇，他认为，「这次处于创始人模式的谢尔盖・布林，依然能够再次扭转局面。」

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGFKN5zreia9T8yKrwkl8G1TIicyg8SBvb91nojdEFLUyHOm9tem7uQDKQk1p4f8xK6cusibkoekg3lJNKfGdCZ4tpSdzKklKfJdw/640?wx_fmt=png&from=appmsg)

  

为什么说「再次」？

  

其实这就不得不提，彼时面对 OpenAI 接连发布 ChatGPT 之类产品带来的巨大压力，谷歌无力招架，「生死存亡」之际，[谢尔盖・布林以「创始人模式」回归，](https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651003553&idx=1&sn=d1316bd38e8317876603ade5041c2de6&scene=21#wechat_redirect)亲自上阵写代码，并在内部进行大刀阔斧的整合，将 Google Brain 与 DeepMind 合并为 Google DeepMind，在短短一年内，让 Gemini 系列从 AI 领域远远落后到轻松站在行业顶端。

  

而此次，同样面对来自竞争对手的「围追堵截」，谢尔盖・布林再次重返一线了……

  

据媒体披露，此次「突击队」将重点优化模型在「长上下文编码任务」中的表现，例如从零编写完整软件，而这类任务往往复杂，需要模型理解多文件结构和用户意图。另外，将在 Google DeepMind 的私有代码库上训练模型，这些代码与公开数据差异很大，对模型性能至关重要。不过，据了解，这类模型无法直接对外发布，但可以反哺更强的公开模型。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFiahzgjv1NfnicDPAqTDdoTMxDA02ov6Tm6IMdmfq2o1pEICChAurWWOjusbnmOqZlKNPz6f7wy0HCdRfnA5yk9GJiaR4EicSkxiaQ/640?wx_fmt=png&from=appmsg)

  

消息一出，网友也是议论纷纷。

  

一位网友表示，当时谷歌发布 Bard 时许多人都在嘲笑，可现在大家也看到 Gemini 的表现了。「我猜再过 1–2 年，Google 在编程方面的模型不会比 Anthropic 差，让他们继续打磨吧。」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGIpfJFfUdekpaY1817bzIkclaskEINKHKlLl8CuiaKAY1rQ3eAlbCGcRsuF4OoGm7s5SZBtJKgICWpibzRiazuXbt8b9SkKqZs1M/640?wx_fmt=png&from=appmsg)

  

也有网友表示，此次「突击队」由谢尔盖・布林亲自参与，这本身就是一个非常明确的信号，可谷歌的问题从来不是人才或资源，而是能不能在窗口关闭之前，快到足以产生决定性影响。」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHOmlYJQaWtORFJlJQwNdS4jchm29ZvjrkIq2jrkqU2H8juPA2HPdsoicjPRLpGd9IRtvtHlVhaibAg8JhXTqP17uJLCCp2dZlwg/640?wx_fmt=png&from=appmsg)

  

诚然，今年以来，编程能力已经成为业界竞争的核心所在，尤其是 Anthropic 的表现一路突飞猛进，在代码生成方面开启长期领先局面。因此不仅是谷歌，包括老对手 OpenAI 也在追赶，包括前段时间，OpenAI 关闭了视频生成项目 Sora，将资源集中倾斜到编程和企业业务上。

  

因此，确实可以说，留给谷歌的时间「不多了」。

  

一个有意思的点是，彼时谢尔盖・布林出山，以「创始人模式」回归后，谷歌开始了绝地反击，被外界看来，之所以成功除了各种因素外，但最重要的一点是：来自 OpenAI 的竞争唤醒了这个沉睡的巨兽。

  

而这一次，谷歌似乎又拿到了相似的剧本：来自 Anthropic 的强力竞争、谢尔盖・布林坐镇、迅速组建「突击队」迎敌……

  

那么，问题来了：这一次，结果会如何呢？谷歌还会继续上演「绝地反击」成功的戏码吗？
