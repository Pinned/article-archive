##### **【新智元导读】OpenClaw 连更两版，直击业务痛点 ！首发 Gemini 多模态记忆，图音皆可作上下文；强化故障转移与 Agent 调度， API 报错自动切备胎。搭配 V2 级 UI 翻新与安全补丁，自动化稳定性大幅跃升！**

这两天 OpenClaw 的代码库相当热闹，昨天刚推了 2026.3.11，今天下午直接又刷了个 2026.3.12 出来。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYXLF1FyS6ptdFiaZ8Y57OYTiaIZvibD2iaJ1fxe03ARsKIxErbc9ic4W1SQ7f4FFJ3Yia5npzWNtib2znAbuv8Hho6RxA1uQsQNo1lqicw/640?wx_fmt=png&from=appmsg)

连看两版几百个 commit，信息量挺大。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYXPuhI1OPQDx9F4U8CJqkhwGaj3plnceBvbMRefeOdgLbByD1BMzqkmIQpXCDtlvUhtYVOfjgyGn58oIibvWicRLGrqYDXj3Iq0c/640?wx_fmt=png&from=appmsg)

简单过了一下，这次更新全在解决实际业务里的痛点。

普通玩家用起来会更顺手，而对那些手里管着成百上千只「龙虾」跑自动化的朋友来说，这版大幅提升了干活的稳定性。

挑对大家影响最大的点过一遍。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**UI 翻新与开箱即用**

Control UI 迎来了 V2 级别的翻新。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUjqleW7TbzWsJrbtPaRmDmGEpTsWbC4p0ibeW9vDr72z69LbCBBgXibFic57wfyTdCTDhkVWtAWqJP8ECd98Vfju1ONZk7uOdmg4/640?wx_fmt=png&from=appmsg)

网关仪表盘现在是模块化视图，概览、聊天、配置、Agent 和会话全部分开。

底层的聊天工具链也肥了一圈，斜杠命令、消息搜索、导出和置顶都加上了，还专门给移动端配了底部 Tab。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYXXmVLBrjcJh0le0BIKUxCG7M02BprB1y3iagBibE0tUMrokR9tc8ZUlJ6XhFKicliad7XOPgkTQ1VH6Cbv3Q4lsKHWY72e1qaCKAM/640?wx_fmt=png&from=appmsg)

界面清爽了不少。

Ollama 玩家这次待遇拉满。

之前配置本地模型还得对着文档敲配置，现在官方做了一套极其丝滑的 First-class Ollama 引导流程。

支持纯本地或者「云+端」混合模式，自带模型推荐，云端模型甚至能绕过本地无意义的拉取动作。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUvEz8TD5n6mncLJu3l9OqFbIUAm4Csq8bNSrzKpbia3KyM19B66YJFiaCpFaXPSghwvhsRVGWfahaBssfsKzdbnWhD1VNozeZM0/640?wx_fmt=png&from=appmsg)

新手配环境的门槛算是彻底踩平了。

苹果全家桶这边，3.11 里 iOS 端把 Home 界面换成了带 Dock 栏的新布局，重连和切回前台时也会及时刷新；

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUFoPPM67uzXfr66dDxttZP8bx4ibOnPOAACJJJaC9ibr5zAgtG3dvPUYSttCPFzhNPXvywyUj2fZn94UVfchIvMuCZ1mp0F17KQ/640?wx_fmt=png&from=appmsg)

macOS 端的聊天 UI 加上了独立的模型选择器，且可以直接持久化保存你的 Thinking 级别偏好，不用每次重启都重新选。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**引擎升级：提速与多模态记忆**

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**API 提速通道**

**顺应 OpenAI 最近刚发的 GPT-5.4，3.12 版本火速在会话层加了 Fast Mode 开关。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYWB5D7xjEE9e2YpoNkGicJ7MrW88Yl3B66Kpv88zPcNYgdfCJYqGl3eic7qgCvk3Bof288ZK9o07b71BTeOHR1oSfiaDrUYNzIONs/640?wx_fmt=png&from=appmsg)

Claude 这边也同步跟进，将底层逻辑直接映射到 Anthropic 的高级服务层级请求上。

花钱买速度的通道直接打通，/fast、TUI、Control UI 和 ACP 都能无缝切入。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**Gemini 加持的多模态记忆**

这绝对是给高阶玩法准备的神器。

新版引入了基于 `gemini-embedding-2-preview` 的多模态图像和音频索引功能。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYUfTsM2To4gdAFlMibrBrJ1Ayk1YMhmHG4nMB4751te6HHiaK19VUXFNCoOptbUABOclcP5SIrfxiaoVs8UwGRxibibj2vXOuhVgU5E/640?wx_fmt=png&from=appmsg)

我们也在前两天详细介绍了 `gemini-embedding-2-preview`：

[谷歌AGI底座降临！首个原生全模态嵌入模型上线，已实现全模态SOTA](https://mp.weixin.qq.com/s?__biz=MzI3MTA0MTk1MA==&mid=2652682199&idx=2&sn=8814b5016e65442e91a125cc893cf707&scene=21#wechat_redirect)

这意味着你的龙虾现在不仅能检索长文本库，连你扔过去的图纸、录音都能直接塞进 Memory 里做上下文。

那些靠丢截图来驱动电脑操作的场景，准确度能再拔高一截。

顺带一提，OpenRouter 的白嫖玩家注意，内置目录临时加了 Hunter Alpha 和 Healer Alpha 两个隐身模型，大概只有一周的尝鲜窗口，抓紧薅。

**Agent 编排与容灾：增强稳定性**

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**更暴力的 Subagent 路由**

3.12 加了个叫 `sessions_yield` 的硬核特性。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYWnaTJXNPoE2M4cN20W1ZOxD6PQk5YbVARyA0Zq4H7G853fibmhuScagTx2egX6BDOB3WlSnN1texAIRYCuzK8B9OD5qRdqPJE0/640?wx_fmt=png&from=appmsg)

之前 Agent 之间的回传有时候会卡在工具队列里，现在编排节点（Orchestrator）可以强行掐断当前回合，跳过所有排队的工具，带着一个隐身的 payload 直接砸进下一个回合。

做复杂多 Agent 协同的开发者，调度逻辑可以写得更加独裁和高效。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**自动恢复与备胎机制**

这块简直是挂机党的救星。

跑自动化最怕半夜跑着跑着，API 额度耗尽（402 错误）或者模型侧返回异常响应。

这两版继续补强了 Failover（故障转移）逻辑：遇到 Venice 或 Poe 欠费、Gemini 返回畸形响应，或者部分 Provider 的网络错误时，网关会更稳地切到你配置的备用模型上。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYVtU9KBuezYiaPOW3TEgTAZPYmZcqtzFDfJwVHh6m2rD9h0qYMicrcsw3kN9RMWuF1iaahI2pwED54xCwqWw1Z2A9b499FsLVWN0o/640?wx_fmt=png&from=appmsg)

它甚至自带冷却时间探测，充完钱自动恢复。

你的龙虾终于学会了在 API 宕机时自己找活干，而不是在原地挂起报错。

**![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb351381bTy5MO2IN89mV41M88GEiaCCibDxJoaQjYV6HfRtafnmEmfM3R1p0tmkHgBOVuXBD6UJKpsQ/640?wx_fmt=png&from=appmsg)**

**ACP（Agent Control Protocol）体验加固**

用 IDE 连 OpenClaw 跑代码的老哥会感觉更跟手。

ACP 桥接模式现在支持带文本提示的工具流，恢复会话时能精准重放终端文本，而且明确拦截了跨端不支持的模式修改，避免 IDE 客户端出现诡异的静默失败。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYWotqHuRULFErZXsmEul9GeXuY0CdKqtlkqIib7Xon3ia4uT6v1ndyTHuzUyZt85AlVLLJp6ZUXhx2uOJBy7ZqskicLpXrUV6WdIY/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/UicQ7HgWiaUb3uEdSPKrwGNmZEOaaGyzVvZ8dTtE9jU1rFsda3llYbCZpmWfiazUYjWBLTGvlPpXucH8Q0lEUJN3Q/640?wx_fmt=png&from=appmsg)

**安全漏洞修复**

这两版一口气打上了十几个安全问题（GHSA）的补丁，也是所有 OpenClaw 用户为了自己的数据安全，最应该更新的主要理由。

挑两个要命的说：

一个是彻底封堵了 WebSocket 的跨站劫持路径；

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYXNY0GT0vrzdx7VeG1NoR2XSY06yTQL17bndLpPick0icv2emFCcKOnRjicvDqcibI520ibusxiavLQIxB1ibmJz9bTnOTfROoPkVwbc8/640?wx_fmt=png&from=appmsg)

另一个是干掉了不可见 Unicode 字符对审批命令的伪装（之前有人用零宽字符绕过执行检测）。

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYWDzgQz6CYlPZaUOHzAAABpjPfLdXVYZP4XMsBN1HibZXmibnQZIw7ColKChTzK18nPSVxv0UQN5TZRcEWsaiaMtUiaD5PfMPGqkJw/640?wx_fmt=png&from=appmsg)

尤其是搞公网部署和集群部署的朋友，别犹豫，马上更新。
