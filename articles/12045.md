# 天塌了，Pro用户用不了Claude Code，除非100美元买Max

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜Sia

> 大家做好心理准备，大涨价迟早会来。

  

知名科技博主 George Pu 发文称，Anthropic 悄悄修改了价格页面，将其核心代码助手 Claude Code 从 20 美元的 Pro 套餐中剔除，变为 100 美元起步的 Max 套餐专属权益。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHEY4gneNdN9Jb4CAHWy1vXPGVc8SGzaUpUXgJc18WMGusZAQpkH8uz8UM2ibWOvicHF8a1kKNlZkJdU6Dkehk1f0tehvU4Dd57M/640?wx_fmt=png&from=appmsg)

  

这意味着，开发者想要维持原有功能，现在必须升级到 Max 套餐——最低月费 100 美元，价格翻了 5 倍。

  

没有官方公告，只是价格页面被修改了。

  

而他本人用的是最贵的套餐，暂不受影响。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFfFsnPOBGtbYBjOeqs7Xiad4Ta5MRt7OphKqEjOYmov67GWWACPEdst2EiamjLjeuqbeaabl5SmoKdE0rbFIOboe0OMqkjwpEw8/640?wx_fmt=png&from=appmsg)

  

一石激起千层浪，开发者反应激烈。

  

首当其冲的，是那些预算有限、资源紧张、以小项目为主的小团队和个体开发者。

  

当前的额度一旦用完，很多 Pro 用户根本无法继续开展工作，整个开发进度被迫停滞。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGS5lTNBiaeoicvBWbv4c9XUWNyQzAtdGytFkzYnChksVkibeic9vekDI31ohPA8rz6OP8e6vqVfy4Q2TTicyDe0t7MkCQq3OfziaRVQ/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHRTvHFs82ySNsG172Y5anJicEEbRQeVdYl5aubDpAYBXYticYVjm2bgGLtdHupzsTsnCPLQvutribicJtrDF4Tpa3k8KtgkVhIcdw/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGQUOFsiaPoZ32icIsvbDBWjRmKpeLwEgglwWZtRXEp1UHVyKSQCjZXudmWYJwPYtnhvTnwxzicl18Y42Fo6HsozT2ibaujcysXUqk/640?wx_fmt=png&from=appmsg)

  

面对突如其来的限制和费用上涨，Hacker news 上也一片惶恐。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGG3sS5VCxhTNtjHErkuJgkoDJQY8vGqknuWt1WfVYDFfhlQ0FmvvpvZl8iaSUKSwWiaD97IZ7EFMQibUGtsslhfhfia5rPZMCjrc4/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGibITNsOTFoU52Aq6qldPaJiaUO3OxM5K049JZaW9IOaPRwTBsJxZjIEuSZwO83jrrHWg0p1IRY5niaBK46icu9pu9lY5YPuQrA7g/640?wx_fmt=png&from=appmsg)

  

在用户习惯养成后突然大幅改变计费规则，被直指，吃相难看。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEpM445DT4GxOE4b7YcruRqZYpZibOI6jSr2STowHp08SJTWYflFMYkGpyvEFCl83u4qHNiakIDKlbyu9eoH8tLCyvgRl1MvGLVg/640?wx_fmt=png&from=appmsg)

  

而在与 OpenAI 的竞争中，这种反复横跳、折腾用户的行为，无异于自掘坟墓。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGOxF636OklrtibKhFAWicmjGet724nJdm8KAgOtqBcAn0GvG2OCuB8FR43KypKH1ZE23KLQP9PTQBFhwMKUQoDBWcl7e1Pu8vts/640?wx_fmt=png&from=appmsg)

  

有人调侃，从没见过哪家公司比 Anthropic 更加恨自家的客户和合作伙伴。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqH17S5gEDBZ8hOLBoBQ9lGV7cApHE1t2QRJqaBjiaaDQ6OldblFjN4eAWxLibDNeayExvPibEyS0qIcToajR3ibPmOp5klG9B4iaRz0/640?wx_fmt=png&from=appmsg)

  

不过，不少网友也发现，自己在官网上看不到这些变动，怀疑这是虚假消息或者 Anthropic 正在进行的小范围灰度测试。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqG1K0xMoAJJhECFzuelQicQKtGjE5ibF9870adBjNX3iczVUGMaNt4jpCCHAZgc2ZoPlycNH6rGzJ8kkXDgyWDNXs2uYdbs3vQ18M/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGR47zWJl6Gd1l9uBhyfFcakVo5G83NGgsQ4eFDrrFllDVSYV0LXhGqHoLo9PEF083Vlcmy6WXd5xmcibWd2zjmyNQ19jgfqMLM/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGUPBkTFic225BDexQtWnPCdaqJjib6ClOrbdxia0QnLrIgkek8VPIC1xjc5vYtwzbfRdYrHE0zpSnic8QVf0sY9UoIun07wqstVxc/640?wx_fmt=png&from=appmsg)

  

很快，Anthropic 负责人 Amol Avasare 赶来澄清。

  

目前，所谓的方案变动仅是针对 2% 新用户进行的定价测试，现有 Pro 和 Max 订阅者的权益完全不受影响。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFDSr8radL6iceNDYfzNYZgzCfG7vSJI29jwuHwLv25ZuTDQic9BC6jGnYk3Yh2bDtSNuHjV5LBpMr9tsLdlS4B91L5gaDhD3Xak/640?wx_fmt=png&from=appmsg)

  

然后，就是一段官方诉苦与预告。

  

兄弟们，不是我们想割韭菜，是你们用得太狠了！Opus 4 太强，你们拿它跑几小时的代码和 Agent，原先那点订阅费我们连电费/算力费都快覆盖不住了。

  

所以，我们要改规则（涨价或按量计费）了。

  

他解释说，自一年前推出 Max 套餐以来，用户的行为模式已发生根本性改变——从单纯的「聊天」转向了使用 Claude Code 和长时异步 Agent 等极度消耗算力的重度任务。

  

这导致，原有的套餐架构已无法承载这种爆发式增长的使用强度，所以，公司正通过灰度测试探索能平衡成本与体验的新方案。

  

他承诺，未来若有任何正式调整，必会提前给予充足通知，绝不会让用户只能通过社交媒体的截图来获悉政策变动。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEwgaG2jBoRDB2OSXJXdA43G2M5L1tSNUl5pG3pRT6G2PibdaqbLWq3AOcFa9qlha1HYwn9s9IHx9KcRhZVrFaq3Wp5ZUZGdDAc/640?wx_fmt=png&from=appmsg)

  

然而，面对官方所谓灰度测试解释，OpenAI 赶来献上一场拉踩式公关。

  

奥特曼亲自下场，回复了一句，「得嘞，老登。」

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGicrhu93sbkv5dJsicy4BNX7JzZNc4As4FaRMIJPahE4Ckp6PkZfasic5icdSgqkqzGduN5KCmibuE9CVdvbrpZsLjD97j8jZjT7Hc/640?wx_fmt=png&from=appmsg)

  

他的兵也来了。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqE3uq6vhGGu2mGicpheRyGZOdTpYB1EtGuLgp0WB93SEULyaZA5XbXI5XtCS1iabAmTjKQ4D6IUQNPOP4erMf3YTFrFxaXJawic3Y/640?wx_fmt=png&from=appmsg)

  

Tibo 直接贴脸开大——

  

嘿，隔壁 Anthropic 已经开始嫌弃你们（开发者）用得太多、算力太贵，甚至要涨价到 100 刀了；但我们这里管够，而且不加钱。

  

他特别强调， Codex 将继续保留在免费版和 20 美刀的 PLUS 版中。这精准打击了那些因为 Claude 可能取消 Pro 版代码权限而焦虑的开发者。

  

秀完财大气粗，还不忘把一次普通的商业选择升华为道德站队。

  

如果你支持公平和透明，就该来用 OpenAI。您的订阅就是在为您希望在这个世界看到的价值观投票。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGfdB9CiasUHYehdlicRmwBGAL9oXzJia2bJnHCFiaXyJ6oLMnUIBybxk6jsM1VzrmCOp3US8dsrKbmpG5yLxUbEnsXfnibnuZDEwWs/640?wx_fmt=png&from=appmsg)

  

网友们也没有消气，反而开启了更加猛烈的炮轰。

  

知名科技评论员 Ed Zitron 开门见山：

  

如果真的是针对 2% 新用户的临时测试，为什么要大费周章地修改官方支持文档、帮助材料，甚至连官网的定价对比表都改了？

  

别信他们的测试鬼话，他们本来就是想全员涨价，只不过被大家发现后认怂了。大家做好心理准备，大涨价迟早会来。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFnPU0tbuEIxMMtka9oQIZyCzWSIhibJQo6OzVhK9icePiaCEKWFNQjkL8ibt4uvZibicQBurNFBGGqSmvGurG1duqHZIGVQMp83aRMs/640?wx_fmt=png&from=appmsg)

  

其实，Anthropic 和 OpenAI 的唯一护城河就是资金。

  

这两家公司正利用资金优势，以低于成本的价格提供产品，尽可能坚持到最后，争取成为行业中硕果仅存的那一家。

  

过去两个月，Anthropic 的一系列让广大开发者苦不堪言的操作，大多可归结为算力撑不住了。

  

比如，限制了第三方工具的订阅额度（OpenClaw 的遭遇，真是冤大头），还在高峰期大幅削减配额。 

  

更离谱的是，Opus 4.7 的 Token 消耗比前代高出了不少，导致很多 Pro 用户在周三就把一周的额度用光了。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGV0sAgjZQgo7Cd2hiaAON4pt8FRbQXHzq2icHOIEdDMWXRsHeCcicBPzky6icXe2QE04xVB1ia5N4IiaicG4fE6W10qgVDOhMJQWnCKI/640?wx_fmt=png&from=appmsg)

  

为了应对算力瓶颈，它们可能会将有限的计算资源集中到高付费用户和企业客户，不断让普通用户的体验骨折。

  

所以，也就有了这次名曰灰度测试的骚操作。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHibicTIjB9UEISnr1drlbxTv2Xib3JtxCthQeWmCVEQPq4mNnX8VmrGOsTHofbK7BIXvpEA7vn8Ole32rBTDNMoNM66DRxicz5OZc/640?wx_fmt=png&from=appmsg)

更有人预测，小玩家们的时代，结束了。

甚至有用户准备尝试中国的大模型，价格更低，额度更多。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFy2oRWlXZAT2gGP5jwIIVNUuoBaqhq3gfH5fNhKA1jCX6WRnTN4XOHakEwqicTc5UB4BaAicjeRNKIsHj37mFZX2XFJUTJabZO0/640?wx_fmt=png&from=appmsg)

随着 agent 的兴起，Token 消耗越来越大，未来的 coding plan 价格恐怕还是得涨。

  

其实，OpenAI 现在的慷慨未必持久，Codex 在 20 美元套餐上的「烧钱模式」同样不可持续。毕竟，Anthropic 现在的做法，OpenAI 以后未必不会跟进。

  

这也提醒开发者们，AI SaaS 的定价仍然不稳定，供应商会在生产性消费者（prosumer）和企业/高级（Max/enterprise）之间不断实验定价模型，特别是当 GPU 和基础设施成本波动时。

  

大家应该做好预期，这些定价目标会发生变化，在设计工作流程时也要考虑到这一点，以免单一供应商悄悄调整定价页面，导致整个技术栈崩塌。

  

对此，你怎么看？是不是也受到波及了呢？有没有找到更稳定的应对方式？
