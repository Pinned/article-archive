# GPT-5.4一周狂赚10亿美元ARR！一句嗨烧掉80刀，效率却飙升32倍

##### 听雨 发自 凹非寺  
量子位 | 公众号 QbitAI

GPT-5.4破纪录了！

据OpenAI总裁Greg最新披露，GPT-5.4仅上线一周，**每天处理约5万亿token，带来10亿美元年化净新增收入**。

其日均流量已经超过OpenAI**一年前整个API的总量**。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGF8BtMk9zp2VO6ju3DXnNziaCCmK9NeibSvAQnvXGNGDTWCjMvlNj8YjClwQf2mWzAPaF88xC5sIJeNt8RPcMWOyhctt6vbTzDjw/640?wx_fmt=png&from=appmsg)

啥概念？一本《红楼梦》约73万字_（约合110万token）_，相当于GPT-5.4每天处理了超过4500万本完整著作。

靠着新模型，OpenAI这波又是赚得盆满钵满了。

一句Hi烧掉80刀
---------

为啥GPT-5.4的收入增长那么快？成本是很重要的原因。

早在模型刚上线的时候，知名博主YuChen Jin仅仅对着GPT-5.4 Pro说了一句“Hi，俺是Anthropic创始人”，**就花掉了整整80刀**。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFCjX011opoiaAlu4nAgialPicDCI4MibdrXdITbgxbtSUz0d6NQwrAXdIhnrO06s5eibicZnYr841icnIItia9O4Vib2KMF0ibnGtqib43Qw/640?wx_fmt=png&from=appmsg)

咱就是说，不知不觉烧掉一大笔钱，还是挺肉痛的。

作为首个“原生大一统”模型，拥有超高智能水平的同时，GPT-5.4的成本和token消耗量也相当恐怖。

根据Artificial Analysis的数据，GPT-5.4 (xhigh)与Gemini 3.1 Pro并列人工智能分析指数榜首。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A6fTew8FFGGPUvNACH34vq2SegdseMZibCUK6TMc5fiaqdKxSBSPh4AsUBibw2KMt7bKxQ0pcYyAev7IS7MQUfBuLiao4KQ0nnbYRicnFnkuDCZs/640?wx_fmt=jpeg&from=appmsg)

但是跑完整个智能指数测试，5.4的成本明显高出不少：约为2951美元，比 GPT-5.2 (xhigh)_（约2304美元）_还要高出约28%。

Gemini 3.1 Pro Preview的成本则为约892美元，也就是说，尽管两者在测试中表现相同，但GPT-5.4比它还要贵3倍。

成本差异的主要原因，来自于更高的输出token使用量。GPT-5.4花费了约1.2 亿个token，这个数字略少于GPT-5.2 _(xhigh, 1.3 亿)_，但比 GPT-5.3 Codex _(xhigh, 7700 万)_ 多约55%。

而Gemini 3.1 Pro Preview花费了5700万token，GPT-5.4是它的大约2倍。

说实话，GPT-5.4是真烧token。

并且单token价格也会更高：GPT-5.4的每百万token输入/输出价格为**2.5美元/15美元**。对比一下，GPT-5.2的定价为1.75美元/14美元，而Gemini 3.1 Pro Preview则为2美元/12美元。

不过大家可能还记得，官方宣传时，说的是GPT-5.4贵是贵了，但由于工作效率提升，所以整体用下来会更省钱。

这又该怎么理解呢？

根据ARC-AGI-2 上的准确率：

*   GPT-5.4得分74.0%，每项任务的成本为1.52美元；
    
*   GPT-5.4 Pro得分83.3%，每项任务的成本为16.41美元；
    
*   GPT-5.2 Pro（High）得分54.2%，每项任务的成本为15.72美元。
    

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEsbaAk3KYrmZuLoDFjAyPOEafiby7wZMeP7hjY38MSunRKMchSCKcpQ1muDViadKCuVyW2uoENjrZP0kWQBr0cCsmYc4UTfFTvI/640?wx_fmt=png&from=appmsg)

也就是说，要达到跟5.2相同的准确率，5.4可能只需要花费几美元甚至更低。

如果再看ARC-AGI-1，变化会更加夸张。

去年12月底，官方验证了 GPT-5.2 Pro (xhigh) 的最新 SOTA得分为90.5%，单项任务成本为11.64美元。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGH8YY4y1iaLR9CMYMV1t4m0NGzXVU17leYA6ziaY6G3W2IiclDKALwibzp0Wh7EiaFSqUAAXYK9L2YSYjlxt44cJb4JyJvjVWUCibWu8/640?wx_fmt=png&from=appmsg)

而如今的GPT-5.4 (High)，已经干到了90%的准确率，每个任务的成本仅为0.37美元。

相当于过去3个月，GPT-5.4的效率直接**提升了32倍**。

这也是GPT-5.4真正恐怖的地方：表面上看是token更贵，用得更多了，但在真实任务层面，**AI推理效率正在出现指数级提升**。

曾经几美元才能完成的事情，现在只需几美分就能做到。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHYNhEJZVJckWkeXfmprVx9FWgNibicjicKiaV4LP5916nAGrx5aSZOSkUicyp19SOIS3caXb9RdkxTC039sLZYgaKjhUibgCStxnzsk/640?wx_fmt=png&from=appmsg)

首个“大一统”模型，全面超越前代
----------------

贵归贵，GPT-5.4的智能水平还是万众瞩目的。

作为OpenAI的首个“大一统模型”，GPT 5.4融合了推理（Reasoning）、编程（Coding）、计算机原生交互（Computer Use）、深度网页搜索以及百万级Token上下文。

并且在以上所有领域中，GPT-5.4在多个关键基准测试中依然保持领先。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEkWSDLfvfR0SKcWib6icm2yf3Xia8um1bBdQQ3VLWRH0fNlIEFqd6Q2PpsNRMxbkEl3gAeTRn01oL4uiaXWmBJkibDXRVrCVunTK8k/640?wx_fmt=png&from=appmsg)

俗话说得好：你现在只有**六分之一**的几率在自己的工作中比GPT-5.4更出色。

GPT-5.4在44种不同的工作岗位上，有**83%的概率胜过人类**，包括律师、会计师、财务分析师、行政人员等等。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEO9RuoINGpGo0T0qYB8eZymdJibBII34443AQrJRib4PLiaQvtHLVUk8gMXBSp94Fnu4BxecscM0BfKiaFfDm5mibcI6U221icypKmw/640?wx_fmt=png&from=appmsg)

无论是写报告、做财务模型、制作演示文稿、分析商业数据，对于这类典型的知识型任务，GPT-5.4都进行了专门优化。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGmicEdU8AfHxBz9z7TfxNgdicpJorfnMWMWVOw5c83tahyotKhVBC29PiaREf9liavTxBkibKTsjaYTSfOebZibEkChFqSMK2YocNZY/640?wx_fmt=png&from=appmsg)

于是你会看到，它可以自己发邮件、安排日程：
