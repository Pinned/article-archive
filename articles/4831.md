# 世界是快的

中学时，移动互联网方兴，当时的我有幸看了本神书：Thomas Friedman 的《世界是平的》

眼界大开，受益匪浅。Friedman 核心在讲一件事：**由于互联网的存在，地理不再决定机会的分配**  
沃尔玛的供应链、班加罗尔的呼叫中心、UPS 的物流网络，把地球上的每一个角落拉到了同一个平面上

我是个行动力还可以的人，看完之后便干了件事：**我在中国和美国各租了一个仓库**，倒腾海淘和跨境电商。世界是平的，信息差和物流差就是钱。靠此，在大学前实现了青春版的财务自由

![](https://mmbiz.qpic.cn/sz_mmbiz_png/jXSGuwJvpdiaQQLwcNm2DDuQ2LKFVwOFBiaOFUhq6hPXbAYaORpQgHo6fnQufLOoGptDAuwS3jhVtMVxCwk0eYRvIe2J8nAEjboegQzPEE2IY/640?wx_fmt=png&from=appmsg)

Thomas Friedman《世界是平的》，当年看的就是这个封面

十几年过去了，互联网确实把空间压平了。但 2026 年正在发生的事情，跟空间没关系，AI 改变了每人可支配的思考时间

李继刚之前有一个总结，我深以为然：  
**互联网压缩了空间，AI 压缩了时间**

在 Agent 时代之前，AI 的使用方式，还停留在「一人一 AI」的阶段，你说一句，它回一句

之后有了 Claude Code，极大的提高了生产力，能够自主的去解决很多复杂问题，但给它一个复杂任务，它跑一个小时，你就得盯一个小时。这个进程跑在你的电脑上，你怕它瞎搞，得不断确认进展，一直盯着

**人被绑死在电脑前面**

之后龙虾来了，用它的钳子把这个绑定剪开了，你可以把它丢在一个独立机器上，手机遥操  
而本文由此展开，来讲三件事：

*   • **一只叫 OpenClaw 的龙虾**
    
*   • **龙虾，带来的时间分身**
    
*   • **人们开始加杠杆，成为逮虾户**
    

龙虾，十三香
======

2025 年 11 月，奥地利开发者 Peter Steinberger 写了个周末项目。一个跑在本地的 AI Agent，能连 WhatsApp、Telegram、iMessage 等十几个聊天平台，你通过发消息下指令，它在后台帮你干活

这个项目最早叫 WhatsApp Relay，后来改名 Clawdbot。Anthropic 的律师看到名字里有个「Clawd」，发音很类似「Claude」，便发了封商标警告信。Steinberger 连夜改名 Moltbot（龙虾蜕壳的意思），两天后又改成 OpenClaw

然后，这个项目在 GitHub 上积累了 17 万颗星，2 万个 fork，且还在上涨，堪称最快

Steinberger 自己跟 CNBC 说了句很诚实的话：「这是个免费的开源业余项目，需要仔细配置才能安全使用，不是给非技术用户设计的」

ClawHub 上 5705 个社区技能，Snyk 查了一轮，7.1% 存在密钥泄露。VirusTotal 扫出数百个恶意文件。1Password 直接发通告：「不要在公司设备上运行 OpenClaw」

跑向「一人多 AI」
==========

龙虾真正改变的事情，大多数文章没写到

过去的模式，无论是 ChatGPT 的对话模式，还是 Cursor、Claude Code 的结对编程模式，本质上都是一个人对一个 AI。除了少数多开狂魔，绝大多数人都是一对一地跟 AI 干活

龙虾的形态天然就不一样

它跑在云端或者一台独立的机器上，你通过 WhatsApp、Telegram 跟它通信。你在地铁上给它发消息下任务，它在后台干活，干完了把结果发回来

**人和电脑，终于可以分离了**

分离之后，一件更大的事情发生了：你可以同时管理很多只龙虾

一只帮你监控 ProductHunt 上的新项目，每天早上 10 点把摘要发到你的 Telegram。一只根据日报自动撰写月报。一只帮你做商品比价，你在聊天里问一句「AirPods Pro 2 哪里最便宜」，它自己去查，查完把结果发回来

每一只龙虾都是你的一个时间分身。你管理得越多，你的时间杠杆就越大

**过去是一人一 AI，龙虾之后，是一人多 AI**

世界已经被「人 + AI」疯狂加速了。OpenAI 最新的模型 GPT-5.3 是 AI 参与训练的，MiniMax 80% 的代码由 AI 生成，385 人撑起全球 2 亿用户的业务。这些数字已经说明了「人 + AI」能做到什么

但「人 + 很多 AI」，会让加速度再乘一个倍数，个人的时间被「影分身」了

新时代的「逮虾户」
=========

龙虾有个很实际的问题：**搭起来太麻烦了**

Steinberger 自己说了，这并不是给非技术用户设计的。你得有一台 Mac，得会用终端，得配 API key，得处理一堆依赖

所以龙虾火了之后，72 小时之内冒出来一堆公司，都在卖同一个东西：龙虾的云端托管服务

ShipClaw，49 美元/月，一键部署。MyClaw 卖隔离实例，Slogan 写着「Everyone wants an OpenClaw. Almost no one can set it up.」ClawNest 做 OpenClaw-as-a-service，5 分钟上线。Clowdbot 按 token 计费，0.5 美元起步
