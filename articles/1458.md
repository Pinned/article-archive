# OpenAI最新融资1100亿美元！英伟达亚马逊软银都抢到船票了

##### 听雨 发自 凹非寺  
量子位 | 公众号 QbitAI

一转头，OpenAI拿下了AI史上的最大一笔支票。

2月27日晚，OpenAI宣布获得新融资**1100亿美元**，其中300亿美元来自软银，300亿美元来自英伟达，500亿美元来自亚马逊，投前估值7300亿美元。

这也是AI领域迄今规模最大的单笔融资。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHeBibGH6M3SqQGFVqIWIz9cw87lDKPC1rIXteG308gTVYhicVaFQo7EJjJHWkJ6a38TSMcSvsrichIxKEEibzhlZxscZZM4ax8wQY/640?wx_fmt=png&from=appmsg)

官方说法是，这笔融资将用于扩大人工智能基础设施的建设，以加快推进AI的普惠。

奥特曼还在X上描绘了与三家的具体合作方向：

*   **英伟达**：负责底层算力供给，围绕 GPU 及新一代加速器，为OpenAI提供持续、可扩展的训练与推理能力；
    
*   亚马逊：提供云基础设施与全球部署能力，支持OpenAI在多区域、多行业场景中的模型交付与商业化落地；
    
*   软银：更多承担资本与生态角色，通过长期资金支持及产业资源整合，推动OpenAI在更广泛产业体系中的扩张。
    

另外，作为OpenAI的老股东，**微软**也没落下。

双方发布联合声明表示：之前的合作关系不受影响，微软Azure仍然是OpenAI API的独家云服务提供商，并且微软拥有OpenAI模型和产品中知识产权的独家许可和访问权限。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFdjx8KnWUtQOq8yCCNHoxZGZe9znZicpwUb1QDZacGApxE2U8icG02auVyDtsszOfJjNap655RSQUkHgMZIk9AFvbh6FN7TaNPE/640?wx_fmt=png&from=appmsg)

在同一时间，**算力上游**、**云基础设施**和**长期资本**三条最关键的AI资源线，被一次性拧到了一起。

一笔融资包圆三大巨头
----------

本轮融资中，**亚马逊**是出手最阔绰的投资方，投资**500亿美元**。

这笔投资将分阶段进行：亚马逊首先投资150亿美元，未来几个月内，当满足特定条件时，再追加350亿美元。

至于这个“特定条件”嘛，有知情人士称是OpenAI**“实现AGI”**或**成功上市**。

并且OpenAI和微软的合作中也有一项特殊条款：一旦AGI实现，微软将失去对其技术的访问权。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGiajb06pKibXA6Io4ibAL0OZTKmHZYf4dxcZj8CNpkepiayQb9ISpdJIcibfAtMXWPiazPxk2ABZITxEyWvUSVJ7ichYPPricNbPic9ZyI/640?wx_fmt=png&from=appmsg)

此次协议中，OpenAI计划开发一种新的“有状态运行时环境”，其模型将在该环境中运行于亚马逊的Bedrock平台上。

OpenAI承诺在AWS平台上消耗大规模云计算资源，包括约**2吉瓦的Trainium芯片算力**，用于支持Frontier、状态化环境及其他高级工作负载。

另外，双方将扩大去年11月签署的380亿美元协议，OpenAI将在未来八年内，在AWS上消耗累计约1000亿美元规模的云计算资源。

亚马逊首席执行官Andy Jassy还表示：“如今，全球最大的两家人工智能实验室都在大力使用Trainium。”除了OpenAI，另一家是Anthropic。

此前亚马逊已向Anthropic投入数十亿美元，并在印第安纳州建造了一座价值110亿美元的数据中心园区，专门用于训练和运行Anthropic的模型，名为“雷尼尔计划”。

不过亚马逊也表示，与OpenAI的合作不会改变其与Anthropic的关系。

第二位投资方，英伟达。

去年9月，曾有传言称英伟达将向OpenAI投资1000亿美元。但随后几个月又有报道称其投资额较小。

当时老黄还特意站出来辟谣了“英伟达将放弃OpenAI的说法”：

> 我们将投入大量资金，我相信 OpenAI。他们所做的工作令人难以置信。

这不，叮咚一声，300亿美元来了。

此次协议中，OpenAI承诺在英伟达的Vera Rubin系统上使用2吉瓦的训练容量，并额外使用3吉瓦的计算资源_（很可能以GPU的形式）_来运行特定的AI推理任务。

换句话说，英伟达是OpenAI的战略投资方，同时也是其芯片供应商，相当于自己掏钱让OpenAI买自己的芯片，这又是一套“循环投资”的逻辑。

第三位投资方，软银，投资300亿美元。

官方公告显示，软银的投资将在2026年的4月、7月、10月分三期到账。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGH0e2cY8vTISQwBicOzWMyJdoXptnOicrXOt4b3TAm07a0rAM7jVQvCMg4JiaYOMLUCNJl7epXGJBaicxOuicAZ0jawWnsmzhCKq7ck/640?wx_fmt=png&from=appmsg)

软银集团董事长兼CEO孙正义表示，软银对OpenAI的持续增长充满信心：

> 通过此次追加投资，我们将加速 OpenAI 的研究和生态系统拓展，同时推进我们自身的人工智能战略。

值得注意的是，有媒体称软银在其中不止扮演投资方，还扮演**“牵线人”**角色。

知情人士表示，预计OpenAI还将获得约100亿美元的一级股权融资，相关承诺将在未来一个月内最终确定，投资者包括主权财富基金和投资公司。据推测，这些投资方可能会通过软银搭桥。

与微软合作关系不变，但寻求多元化
----------------

接下来唠唠OpenAI的老股东，**微软**。

微软是OpenAI此前最大的支持者之一，也是其独家基础设施合作伙伴。此轮融资公布后，尽管外界多有猜测，OpenAI与微软在最新声明中依然极力维护双方的“核心地位”：

*   Azure的独家地位： 微软Azure依然是 OpenAI 无状态API的独家云服务商，OpenAI的第一方产品_（包括Frontier）_将继续托管在Azure上；
    
*   商业逻辑与收益分成关系不变： 现有的收益分成协议依然有效。即便 OpenAI 在亚马逊或甲骨文的云上产生了收益，微软依然能从中“抽成”；
    
*   知识产权独占： 微软继续拥有对OpenAI现有模型及未来产品知识产权的独家许可。
    

但此轮融资也是OpenAI超越其与微软长期合作关系、寻求业务多元化的一种证明。

亚马逊的加入就是最好的例子。亚马逊是微软在云计算领域的主要竞争对手，OpenAI承诺在未来8年内追加1000亿美元的AWS消费额度，并大规模采用亚马逊自研的Trainium 3/4芯片，也说明OpenAI并不甘心将所有的算力鸡蛋都放在Azure一个篮子里。

One More Thing
--------------

看到这里，大家可能也意识到了，这又是一套典型的“循环融资”逻辑。

英伟达既是股东，又是唯一能提供底层算力的“军火商”；而亚马逊既是新晋投资方，又是 OpenAI 未来的云服务提供商。

简单来说，供应商_（英伟达/亚马逊）_把钱投给客户_（OpenAI）_，客户再拿着这笔钱回来买供应商的芯片或云服务。对于供应商来说，这笔钱在报表上从“投资支出”变成了“营业收入”，不仅拉动了股价，还锁定了未来的市场份额。

也是美滋滋了。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGEKCR9c0hAeyuyA4mI2Hd9nXL9ed81PR1wAh1UkmhtNG7R0yHzl6sjXjkibbh5M1m6SFFZLOsPmHAXuBYdQREXvUY75lHvyzxc8/640?wx_fmt=png&from=appmsg)

尽管外界对这种“循环融资”的叙事多有担忧，但奥特曼表示：

> 我理解大家的担忧，只有当新的收入流入整个人工智能生态系统时，这一切才有意义。

OpenAI的上一轮融资于2025年3月完成，筹集了400亿美元，估值达到3000亿美元。当时，这是有史以来规模最大的私募融资。

本轮融资加上账上原有的约400亿美元现金，OpenAI的可用资金将增至**约1500亿美元**。该公司预计到2030年才能首次实现正向自由现金流。

OpenAI也确实需要这些资金和算力。2月最新数据显示，ChatGPT周活用户已经**突破9亿**，月访问量约为**57.2亿次**，个人订阅用户超5000万，活跃的企业客户数活超过1.5万家。

奥特曼也表示，自己正投入大量精力来争取更多的计算能力，以满足对ChatGPT和OpenAI其他产品的需求。

顶级人工智能实验室之间的竞争依然很激烈。隔壁Anthropic前不久从微软和英伟达等投资方处，拿下了**300亿美元**的资金，投后估值直指3800亿美元，增强了其与OpenAI抗衡的能力。

现在弹药都已填充完毕，接下来就看OpenAI会拿这笔巨额资金，给我们端出怎样惊喜的新模型吧。
