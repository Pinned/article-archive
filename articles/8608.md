消息称原特斯拉「王牌厂长」宋钢将入职小米汽车

4 月 2 日消息，据记者从知情人士处获悉，曾主导特斯拉上海超级工厂建设与产能爬坡的核心人物 —— 原特斯拉上海工厂生产制造副总裁宋钢，**将入职小米汽车**。针对上述消息，该记者联系小米方面求证，截至发稿，对方暂无回应。

据知情人士透露，宋钢此番加盟小米汽车，“或任职厂长，与厂长季国伟形成搭档。厂长其实是非常关键的经验岗位，小米汽车工厂员工人数众多，管理责任很重。”资料显示，宋钢曾在通用汽车和福特汽车任职，**2018 年加入特斯拉**，担任特斯拉上海超级工厂生产制造副总裁。（蓝鲸科技）  

“男子靠 AI 开一人公司年营收达 150 万”冲上热搜
============================

4月1日，“男子靠AI开一人公司年营收达150万”冲上热搜，杭州创业者武培文“单人+AI”模式引发全民热议。据悉今年3月3日，浙江省首个区级OPC（一人公司）专项政策在杭州市上城区发布，该区喊出要打造“OPC创业第一城”。随后，余杭区发布“OPC全要素护航包”；萧山区出台十条“养龙虾”举措，支持OpenClaw&OPC-STC发展……在中国数谷未来数智港，2025年底，园区布局了OPC一人独角兽社区，OPC其实就是One Person Company（一人公司），为个体加AI的创业模式。
======================================================================================================================================================================================================================================================

园区内一个工位的租金每月大概在700到800之间，而在扶持政策下，前六个月的租金、水电、网费、物业费全免。在这里的创业者，原来的职业五花八门：品牌设计、医疗器械销售……他们表示，公司的主要工作，除了“一人”之外，其他基本都靠AI来完成。
======================================================================================================================

那么，做传统公司的员工，和做OPC负责人哪个更累呢？创业者冉伟说，“我没有接触AI之前，我是晚上11点前能够睡觉的，接触AI之后是凌晨1点睡觉，现在全职出来成为OPC之后我是凌晨3点睡觉，痛并快乐着！”创业者比尔表示“身体上不累，心力上会更累。但是现在做OPC时间更自由了。”他告诉@钱江视频 记者，现在自己的“一人公司”年营收可达150万元。（凤凰网）

**字节 Seed 启动大模型校招，全球招募百位 AI 人才**

4月1日消息，字节跳动Seed正式启动大模型人才校招，下设2027届应届生招聘和在校实习生招聘。据悉，今年Seed将加大人才投入，本次在全球招募的2027届大模型人才将达到约100位。目前，字节为强化对大模型人才的长期吸引和激励还推出了专门的“虚拟股”。（格隆汇）

腾讯云全系「龙虾」产品，首批通过信通院「安全体检」

据腾讯云官微4月1日消息，腾讯云全系“龙虾”产品，首批通过中国信通院《云端OpenClaw基线能力评估》。今年2月，工信部NVDB（网络安全威胁和漏洞信息共享平台）发布预警：OpenClaw要是瞎部署、默认配置不改就直接上，很容易被黑客盯上，导致数据泄露。为了给这些Agent立规矩，中国信通院牵头发布《云端OpenClaw基线能力要求》标准，并以此展开了严苛的“体检”。

腾讯龙虾特攻队的WorkBuddy，QClaw，轻量云OpenClaw，云桌面云手机Claw，ClawPro首批通过这个重要“体检”，成功通过“云端OpenClaw基线能力评估”。（财闻）

小鹏副总裁魏斌已于近期离职，此前负责推动VLM大模型上车
============================

4月1日，据《一见Auto》报道，小鹏集团副总裁、智能座舱负责人魏斌于近期离职，此前因家庭原因处于休假状态。目前，小鹏智能座舱中心与自动驾驶中心合并成立通用智能中心，由刘先明管理。

离职前，魏斌主要负责的工作是推动VLM（Visual Language Model）大模型上智能座舱。魏斌曾任高德地图产品总监，负责导航与地图产品规划。2021年底加入小鹏集团之后，主要担任互联网中心负责人，主导智能座舱的研发与落地。（凤凰网）

QQ 成为国内首个被 OpenClaw 官方原生接入的社交平台

4月1日凌晨，OpenClaw发布v2026.3.31新版本，此次更新内置QQ Bot官方插件。这意味着，QQ成为国内首个被OpenClaw官方原生接入的社交平台。腾讯轻量云协同QQ团队贡献的QQ Bot代码也已合入OpenClaw主仓库。随即，OpenClaw创始人Peter也对这一消息进行公开点赞。（36氪）

湖南首例 AI 造谣刑事案宣判
===============

4 月 1 日消息，据耒阳市公安局公众号，近期湖南耒阳市公安局侦办了一起利用人工智能工具编造传播恶性谣言案件，**该案是湖南省首例因利用 AI 制造谣言被追究刑事责任的案件**。

根据通报获悉，阳某为博取关注、吸粉引流，使用 AI 剪辑软件制作“化工厂重大事故”“持刀抢劫珠宝店”“多人被杀抛尸案”等虚假视频，后在网络上发布，引发网民恐慌，严重扰乱网络和社会公共秩序。

耒阳网警迅速出击将其抓获归案，到案后，阳某对犯罪事实供认不讳。法院审理认为，阳某利用 AI 软件编造、传播虚假恶性事件信息，情节恶劣，其行为已构成寻衅滋事罪，依法判处有期徒刑六个月。（IT 之家）

**国际要闻**

Claude Code 开发团队回应源代码泄露：纯属人为失误

昨天 Claude Code 源代码泄露在开发者社区引起狂欢，不过随后 A 社也发送 DMCA 通知要求 GitHub 删除超过 8,100 个包含相关泄露源代码的仓库，理由是侵犯版权。不过无论发送多少 DMCA 通知也不可能将源代码彻底从网上清除，对 Claude Code 团队而言，事已至此只能想办法补救，例如检查源代码泄露的根源。

除了发送版权通知要求删除仓库外，A 社目前并未发布任何官方声明，但 Claude Code 主要开发者鲍里斯切尔尼 (Boris Cherny) 在 X/Twitter 上发布回应，承认这是人为失误。（蓝点网）

**鲍里斯切尔尼在社交媒体上表示：**

> 这是人为失误。我们的部署流程中包含几个手动步骤，而在其中一个操作步骤中，我们出现操作失误。现在我们已经进行了一些改进并深入研究，以增加更多的健全性检查。
> 
> 与其他任何事件一样，解决问题的反直觉方法是寻找更快的方法，而不是引入更多流程，在这种情况下就是要提高自动化程度，并且由 Claude Code 检查结果。

估值或超 1.75 万亿美元，SpaceX 据报已秘密提交 IPO 文件
====================================

4月2日消息，综合财联社等多家媒体报道，SpaceX已向美国证券交易委员会(SEC)秘密提交了首次公开募股(IPO)申请文件，正迈出可能成为史上最大IPO的重要一步。此前有报道称，SpaceX计划通过IPO融资约400亿至800亿美元，公司目标估值可能达1.75万亿美元。

此次申报使公司有望最早于今年7月完成上市，这一时间表与马斯克此前向内部人士透露的目标一致，届时将成为首个登场的超大型IPO，早于OpenAI和Anthropic PBC。另有消息人士称，SpaceX已选定美国银行、花旗集团、高盛集团、摩根大通和摩根士丹利担任此次IPO的主承销商，并已将更多银行纳入承销阵容。（快科技）

库克回顾：iPod 爆红需 3 个月产 1500 万台，倒逼苹果打造出世界级供应链

4 月 1 日，《华尔街日报》在苹果公司成立 50 周年之际，发布了最新专访视频。在视频中，**苹果首席执行官蒂姆 · 库克（Tim Cook）首次公开回顾了多款罕见的苹果原型机与内部绝密档案。**

库克指出在苹果产品发展历程中，iPod 绝对扮演浓墨重彩的角色。谈及 iPod 的量产，库克分享称早期供应链面临巨大挑战。他指出，苹果在此之前从未处理过如此庞大的生产网络。随着产品爆红，苹果必须在短短三个月内生产出 1400 万到 1500 万台设备。

面对激增的订单，苹果的供应链体系迎来了大考。库克强调，要完成这种级别的量产，整个生产线必须保持极高的精度与质量，容不得半点闪失。这也为苹果日后强大的供应链管理能力奠定了基础。（IT 之家）

绝版了！马斯克发文确认停产特斯拉 Model S/X

4月1日，马斯克通过社交平台宣布，特斯拉将正式停产Model S与Model X两款车型。目前官方已停止接受全新定制订单，消费者仅能选购渠道内的现有库存车辆。

作为特斯拉切入豪华纯电市场的代表，Model S与Model X分别于2012年和2015年上市。由于产品重心转移，两款车型的市场销量持续走低，2025年数据显示其合计销量占比已不足整体销量的4%，产能利用率与投入产出比较低。 依据规划，加州弗里蒙特工厂原有的Model S/X生产线将进行改造，转为Optimus人形机器人生产线，目标年产能可达100万台。

马斯克表示，第三代Optimus是特斯拉后续重点推进的量产设计项目。特斯拉方面称，此次停产属于产能结构调整，两款车型在完成阶段性使命后正式退出量产序列。（快科技）

树莓派因 LPDDR4 内存涨价七倍，上调多款产品价格并新增 Pi 4 3GB 版本
==========================================

4月1日，树莓派官方发布通知，因LPDDR4内存采购成本较一年前大幅攀升，涨幅约达七倍，公司决定对部分产品价格进行调整。此次调价范围为11.25美元至150美元不等，其中Raspberry Pi 500+键盘一体机版本的涨幅相对显著。

为帮助用户更精准匹配硬件配置与实际使用需求，避免为冗余内存支付额外费用，树莓派同步推出Raspberry Pi 4的3GB内存新版本，售价为83.75美元，约合人民币578.2元。至此，Raspberry Pi 4共提供五种内存规格：1GB、2GB、3GB、4GB及8GB，用户可根据具体应用场景和项目规模自主选择最适配的配置。（中关村在线）

ChatGPT 登陆苹果 CarPlay：无唤醒词，不能操控车辆功能

4 月 1 日消息，OpenAI 已更新 ChatGPT ，新增了对苹果 CarPlay 的支持。这意味着 CarPlay 用户如今可以直接在汽车中控屏上向 ChatGPT 提问并下达指令。

据了解，苹果在 iOS 26.4 系统中开放了第三方语音对话应用接入 CarPlay 的权限，但相关应用必须完成功能适配，并向苹果申请专属权限认证。对于 ChatGPT 及其他适配 CarPlay 的应用而言，语音必须作为核心交互方式。苹果规定，**聊天机器人应用在响应用户查询时，不得展示文字或图像内容。**

苹果提供了统一的语音控制模板，所有应用必须遵照使用。语音服务运行期间，应用界面需固定显示语音控制面板，且最多仅可设置四个功能操作按钮。若要在 CarPlay 中使用 ChatGPT，iPhone 必须升级至 iOS 26.4 及以上系统版本。

阿里巴巴图像生成与编辑统一模型 Wan2.7-Image 发布

智谱发布原生多模态 Coding 基座模型 GLM-5V-Turbo

智谱发布首个原生多模态Coding基座模型GLM-5V-Turbo。该模型最大突破在于深度融合视觉与编程能力，能够原生处理文本、图片、视频等多模态信息，同时擅长编程、长程规划、操作执行等复杂任务。GLM-5V-Turbo在多模态Coding、Agent等核心基准上取得领先表现，引入视觉能力的同时，纯文本编程与推理能力保持了同等水准。并且深度Claude Code与龙虾场景，让OpenClaw龙虾具备了真正的视觉能力，能看懂屏幕上的信息。目前该模型已通过智谱MaaS平台开放接入。（格隆汇）
