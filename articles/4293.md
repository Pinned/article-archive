# “你们都在偷软件！”50年前比尔·盖茨炮轰盗版软件的公开信曝光：硬件要钱，软件就该被“白嫖”？

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAujqPrtpN5QqqOuSG0AdUUx8wRg3vvr7aQJicdC0cHDBqZrJBZajQUXx97BwNaAE6L6x7OGlsTIpWjw/640?wx_fmt=gif)

50 年前的本周，一封名为《致电脑爱好者的一封公开信》在刚兴起的计算机爱好者圈子中流传开来。这封由 20 岁的比尔・盖茨撰写的信件，因非常直白地炮轰“软件盗版”问题，在当时掀起轩然大波，其彼时抱怨道：“你们这样做就是在偷窃。”

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjExLUjGUUU9AqvQmUEyFhXJWsWYIHaB83djVnlXfWNjOj96uGMIFCs7Df532HL8BX4lJaeTQctZZNkzWh5m7ia7Nasc30xPNqru4/640?wx_fmt=png&from=appmsg)

如今恰逢发布 50 周年，这封至今保存在《纽约时报》档案中的内容再次展现在大众视野上，随即也冲上了 HN 热榜，引发了新一轮讨论。

时代背景，Altair 8800 开启个人电脑革命

时间拉回 1975 年，搭载了 Intel 8080 CPU 和 S-100 总线卡的 Altair 8800 横空出世。

这款没有键盘、没有显示器，甚至默认仅为简陋套件的机器，却意外开启了桌面个人电脑革命，它催生了早期 PC 社区与生态的形成。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEyicqY4TV1IZtVB4Ae8HsCGI8XLoF2P9zUnmMLq6CuybM8TfekCLAeMT66lmJpwWUe7kXiaNHxDIibia6Biagp1h6QV0emc77XCSFicg/640?wx_fmt=png&from=appmsg)

正是这款机器，让 20 岁的比尔・盖茨与 22 岁的保罗・艾伦找到了合作契机。两人便主动联系 Altair 开发商 MITS，声称已经为这台微型计算机完成了一套 BASIC 解释器。

时任 MITS 公司负责人 Ed Roberts 很快同意了这次合作，邀请其二人为 Altair 编写软件。随后，艾伦前往 MITS 公司，全职参与与 MITS 的对接与支持；仍在哈佛大学就读的盖茨则以合同工身份为其远程开发代码。

值得注意的是，这套软件并非以个人名义交付，而是由两人新成立的公司对外授权。1975 年，他们为这家尚未真正运营的公司取名 Micro-Soft，意指“微型计算机的软件”。这家公司，后来便发展为微软。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

带有完整源代码的纸带被盗，还被免费分发了

随后，盖茨、艾伦与另一位程序员 Monte Davidoff 联手，为 Altair 8800 开发了定制版 BASIC 编程语言——Altair BASIC。

这款软件的开发耗时远超最初预期：初始编写仅用两个月，但后续一年里，三人将大部分精力投入到文档整理、功能迭代与版本扩展中，最终推出了 4K、8K、EXTENDED、ROM 和 DISK 五个版本。

据比尔·盖茨透露，「为此消耗的计算时间，价值已经超过 4 万美元」。

然而，这款投入巨大的软件，在正式发布前就遭遇了 “意外”。

外媒 TheNewStack 称，后来有一个名为 Steven Levy 作家记载，在一次帕洛阿尔托的演示活动后，存有 Altair BASIC 完整源代码的一卷纸带被人“顺手借走”。当时硅谷系统测试工程师 Dan Sokol 被人问及能否复制纸带，他给出了肯定答案，并在下一次 Homebrew 计算机俱乐部聚会上，带来了一整箱复制好的纸带——以“黑客文化里软件应有的价格”去免费分发了。

在当时的爱好者圈子里，“拿到即共享”是默认规则：大家普遍认为软件就该自由复制、互相传递，几乎没有“软件需要付费”的版权意识。

这导致当时 Altair BASIC 在爱好者群体中广泛流通，但真正购买这款软件的用户，却不足 Altair 使用者总数的 10%。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

盖茨怒发公开信引爆争议：“你们大多数人都在偷软件”

原本期待通过版税收回成本的盖茨，对此深感不满。

1976 年 2 月，他在 MITS 的《Altair 用户通讯》及 Homebrew 计算机俱乐部内部通讯中，发布了那封著名的《致电脑爱好者的一封公开信》，措辞激烈地控诉软件盗版行为。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/S1iaf4GgGjEz88M1zn4pGjHQborB7YbT2Fu1UnTtk1TGSNKEhcJicNOXSCrvsyBziaF9xCv2E5VrEPt2MHqHXla4J6E5xicGD3rPExn7cImuS7k/640?wx_fmt=png&from=appmsg)https://archive.nytimes.com/www.nytimes.com/library/cyber/surf/072397mind-letter.html

信件伊始，盖茨称：“当下爱好者市场中最关键的问题，是缺乏优质的软件课程、书籍，以及软件本身。如果没有好的软件，也没有一个懂编程的使用者，那么一台爱好者计算机几乎毫无意义。问题是：爱好者市场还能不能诞生高质量的软件？”

随后，他分享了其团队三人过去一年的工作：

将近一年前，保罗·艾伦和我预期爱好者市场会持续扩大，于是雇佣了 Monte Davidoff，共同开发了 Altair BASIC。

最初的开发只花了两个月时间，但在过去的一年里，我们三个人把大部分精力都投入到了 BASIC 的文档整理、功能改进和持续扩展中。

如今我们已经有了 4K、8K、EXTENDED、ROM 和 DISK 版本的 BASIC。我们为此消耗的时间，价值已经超过 4 万美元。

盖茨进一步列出数据，表达了自己的不满：“我们从数百名自称正在使用 BASIC 的人那里收到了正面反馈，但其中大多数从未购买过 BASIC；我们从爱好者销售中获得的版税，折算下来，让我们在 Altair BASIC 上投入的时间，时薪还不到 2 美元。”  

其也直接点出核心矛盾：“大多数爱好者应该都清楚：你们中的绝大多数人都在偷软件。硬件需要花钱购买，但软件却被认为是可以随意分享的。至于开发它的人能不能拿到报酬，又有谁在乎呢？”

盖茨怒斥这种行为是“盗窃”：

这样做公平吗？通过盗用软件，你并不能“报复” MITS——即便你曾在某些事情上对它不满。MITS 并不是靠卖软件赚钱的。付给我们的版税、手册、磁带以及各种运营成本，加在一起，也只是勉强打平。

你真正做到的，是阻止优质软件的诞生。谁能负担得起无偿做专业工作？哪个爱好者能投入三个人年的时间来写程序、修复所有漏洞、编写文档，然后免费分发？

事实是，除了我们之外，没有人真正为爱好者软件投入过大量资金。我们已经写完了 6800 BASIC，也正在开发 8080 APL 和 6800 APL，但几乎没有任何动力把这些软件提供给爱好者使用。说得再直接一点：你们的行为就是盗窃。

那那些转卖 Altair BASIC 的人呢？他们不是在靠爱好者软件赚钱吗？是的，但那些已经被我们掌握情况的人，最终可能会付出代价。他们给爱好者群体抹黑，也理应在任何俱乐部聚会上被请出去。

信件的末尾，盖茨表达了对行业的期待：“如果有一天我能雇上十名程序员，把大量优秀软件投向爱好者市场，那将是最让我感到高兴的事情。”

半个世纪后再引热议

这封信件在 1976 年就引发巨大争议。有人认为盖茨的指控“臭名昭著”，违背了崇尚共享的黑客文化，也有人开始反思软件价值与开发者回报的关系。

50 年后，这封信在 Hacker News 上再度登上热榜，讨论依旧没有降温。一些网友从结果论的角度回看这段历史，认为微软后来其实“想得很明白”。

有评论写道，微软很快学会了该把精力放在哪些战场上——Windows 的反盗版机制长期维持在“几乎谁都能绕过去”的水平，真正被重点打击的，往往只有那些大规模、问题明显的企业。更现实的一点是，微软也从发展中国家和青少年爱好者群体中广泛存在的盗版现象里获益匪浅，这些用户在无形中推动了 Windows 的全球普及。

![](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEzThUptOGuEaYOSBFJC82ZY8Cp2cSNpy3msyYQrdz19gLUPTb45iae1vzA69SBrQOlZTrm3lmGIEeLicNlP3cfzlOhA7Sic6y9RP0/640?wx_fmt=png&from=appmsg)

也有更尖锐的声音称：“说真的，这才是微软真正意义上的关键创新，让人们为软件付费，以及让人们不得不为软件付费。”

其实 50 年过去，计算机行业早已天翻地覆，软件从纸带走向云端，传播从物理复制变成一键分享，商业模式也从一次性买断演化为订阅、开源与赞助并存。但盖茨在那封公开信中抛出的核心问题依然悬而未决，即「如何让开发者获得合理回报，同时又不切断用户接触软件的途径？」
