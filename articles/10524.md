# 倒计时3天！2026 奇点智能技术大会高质量参会指南，请查收！

![](https://mmbiz.qpic.cn/mmbiz_gif/Pn4Sm0RsAugJkibeIkBwI7feN33tcTm1qPpuj2lvd72ibrAX8fmhZCzuPoUE3c3B4TPE2SZsSOicDkKSOSOQusEBQ/640?wx_fmt=gif)

从 Agent 工程化破局，到国产算力性能的极致压榨，再到 AI+行业落地的“深水区”复盘……经过数月的筹备与期待，由 CSDN 与奇点智能研究院联合举办的 「2026 奇点智能技术大会」终于要在本周五（4 月 17 日）正式拉开帷幕。

这不只是专家们聚集在一起讲讲 PPT，更像是一群在大模型一线的技术专家聚在一块聊聊，当 AI 彻底打破咱们过去那套轻车熟路的开发路径，如何搭建出一套能真正落地、经得起业务考验的实战新逻辑。在这里，你听到的将不再是理想化的指标，而是万亿级流量、复杂业务约束下的真实取舍。

核心议程：两日高密度干货全览

在大会首日的上午，奇点智能研究院院长，CSDN 高级副总裁**李建忠**、荣耀终端 AI 首席科学家和 CAIO **黄非**、蚂蚁集团副总裁，国家级创新领军人才**周俊**、以及上海人工智能实验室青年领军科学家，大模型中心负责人**陈恺**将相继登场，深度解析从 Agent 重构产业范式、AI 周期演进到百灵大模型 AGI 探索及书生万亿参数科学大模型的底层突破。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/iaM6L8b5B1Glv72eTib804WLnvMH7iapNpbRBsEOrGfhAW5MUkZjJzDYGPtt9D0eDAKr0LPZmkuNK116Loibck2abjQU9YBnAbcYleKkAgwAtEo/640?wx_fmt=jpeg)

在主会场最后的圆桌论坛环节，一场极具深度的圆桌对话将在李建忠、黄非、陈恺、沐曦高级副总裁**孙国梁之间展开，**围绕“推理拐点下的 Agent 产业新生态”进行巅峰对话，从学术、工程到算力全方位透视 AI 落地的新航向。

4 月 17 日下午，大会议程将正式进入高密度的“分会场深水区”。四大平行分会场——大语言模型技术演进、多模态与世界模型、AI 原生软件研发、智能体系统与工程将同步开启，从底层算子优化到上层 Agent 架构重构展开全方位探讨。

以下是大会首日的下午场日程：

![](https://mmbiz.qpic.cn/mmbiz_jpg/iaM6L8b5B1GnU8YmXWhMtgRKKibIiaIAbQ0L2oLRZ2Ps62RhjNGCm4ibv6NjdyDicFad2jiaGF0qtB5lgXHsic8lJcvHS4EA0fpCnjGbmiagZ9mYdy4/640?wx_fmt=jpeg)

4 月 18 日上午，大会的探讨将向“系统可靠性”与“软硬深度协同”进一步纵深。在大语言模型技术演进与行业落地专题中，我们将重新审视预训练阶段的数据工程逻辑，并探讨如何通过编译器优化压榨 vLLM 的推理极限，同时复盘 AI 搜索 Agent 对齐与工业场景下的 AI 工程化范式。

在备受期待的具身智能、AI Infra 与 OpenClaw 实践会场，技术焦点将全面转向“从思考到行动”的跨越。议程涵盖了空间理解、生物智能机器人以及无人装载机等复杂工业场景的算法闭环；同时，资深专家们将系统拆解 GPU 基础设施的规模化运营、软硬协同架构及以智能体为中心的 AI 软件栈优化。此外，针对开发者关注的 OpenClaw 生态，我们将集中展示可自我增强的 Agent 工程体系、全民 AI 工作台以及企业级数字员工的最新落地突破。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/iaM6L8b5B1GlHVTqWiasFIm86QuNlyjQR6y2CM41SSJSms3H3hkJnwDOzEiarsb6zNRFZn4DIEOh9VK1BY1kEu8icrBDqicI9fL7H91wMNm8gsgg/640?wx_fmt=jpeg&from=appmsg)

4 月 18 日下午，大会将进入收官阶段的最后一场“技术博弈”。AI+行业落地与应用创新专题将直击复杂业务的深水区，涵盖商业因果建模、重疾诊疗中的多 Agent 协同以及办公领域的多模态实践；同时，我们将全方位探讨下一代视频剪辑范式、AI PPT 的工程化路径，以及如何实现从“构建自治 AI”向“设计生产级 Agent 系统”的跨越。
