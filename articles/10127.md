# Claude年化收入首次反超OpenAI

##### 梦晨 发自 凹非寺  
量子位 | 公众号 QbitAI

Anthropic年化收入首超OpenAI！

最新披露的热乎数据，Claude背后这家公司年化营收已突破300亿美元。

作为对比的OpenAI最新数据，2月底披露年化收入为250亿美元。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGCtm014GHTeBmsjicnjGZZm2T8Xf3WEAPvYHzjbiaOicWk13OVYuZeKgaia8P1xIhOibbVJySGM1UBSEV3icCqUQs2n3fMXoQlvxakI/640?wx_fmt=png&from=appmsg)

Anthropic大部分收入来自API，其余来自Claude应用的订阅。

有计算称按照目前的增长速度，Anthropic预计将在2027年底实现盈利。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/A6fTew8FFGGCNicyn2u93XpVCKUZNmhhVHQQicerIU6WkedJ1A0oZTia9GJnuVXgFcScZh0UCpILodJV5VgQCibibTjrP3cTgxdGHjB3n2l3FPSA/640?wx_fmt=jpeg&from=appmsg)

同一时间，Anthropic还宣布了另一件大事，跟谷歌、博通签下了一份数GW级的TPU算力协议，2027年开始供货。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGF4yHEYLvxJzgPaiad4e94o99fbicpvLxcTM54f2TkP5ItWf59ntY3ia5XDm3MrdxxQGn4IoOKksh8rvKHFecBeoTbkQnvjstIZlE/640?wx_fmt=png&from=appmsg)

半年翻3倍，企业客户2个月翻倍
---------------

先看增长的时间线。

2025年底，Anthropic的年化收入是90亿美元。到了2026年2月G轮融资时，已经有超过500家企业客户年化消费超百万美元。

然后就是4月初，直接冲破了300亿美元。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGHkBheibao3Y8N4ib27n1t2w735ibExWw0nicA4cH0IKd3rvBlP2Zcy8xGUKxJZpAh9IUJI7sVORIp5zGvFOfkcHN33h4bH8cRFxEc/640?wx_fmt=png&from=appmsg)

而且增速还在加快。从2月底到4月初，短短6周，收入又涨了58%。最近这一个多月的增量，比之前好几个月加起来都多。

其中最突出的是企业客户的增长。年化消费超过100万美元的客户数量，2月份还刚宣布突破500家，到了4月已经超过了1000家。不到两个月，直接翻了一倍。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGFLbibBtgaEZqAQntIjXQeBwD0OxzLAFYZxiar8XBakuDmV78o5rdcQ5ZwpOhtSrpIGib5Ct6DxsPGIKZjfiaPQxwkDflmprF8xIPw/640?wx_fmt=png&from=appmsg)

Claude的API生意能爆发，还有一个关键前提：

它是唯一同时登陆AWS Bedrock、Google Cloud Vertex AI和Microsoft Azure Foundry三大云平台的前沿AI模型。至少目前，还没有其他前沿模型做到同样的全覆盖。

背后是三套算力平台并行：AWS Trainium、Google TPU、NVIDIA GPU。不同工作负载匹配不同的芯片，性能和弹性都拉满了。

但现在，TPU的分量正在其中加码。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGEnU4eqyC9wpJSic3Iox80muj7wqq6ia2Dyr0VMYh8xE8gmp3t5H071dwnQHuRPd2F1IicW7qric1CTA4prnW5AKSGqE9xzIajsksg/640?wx_fmt=png&from=appmsg)

数GW级算力大单，相当于中型城市供电
------------------

Anthropic宣布与Google、Broadcom签署了一份新一代TPU算力容量协议。规模是”数GW级”，预计从2027年开始上线。

1GW大致可以为几十万户家庭供电。数GW级算力已经不是”建一个数据中心”的级别了，而是相当于为好几座中型城市供电，全部用来跑AI模型。

  

谷歌持有Anthropic 17-20%的股权，并且上个月签署了为其数据中心提供融资的协议。

谷歌TPU路线已经发出相当明确的信号，将成为英伟达GPU的有力竞争对手。
