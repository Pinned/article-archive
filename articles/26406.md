# 刚刚，英伟达重新定义PC！史上最高效CPU来了

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

机器之心编辑部

  

「就像十年前我们重新定义了手机，从今天起，PC 的意义彻底变了，」黄仁勋说道。

  

刚刚，老黄掏出了我们期待已久的 CPU，百分百适配 Windows。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqG96JlNib7CftFMRVzYc0jAI6MjjPakb493LzBavHX0eiao6rSjw0NYGjxpZk1BLjhficocDbfvGq49zY556G2ldZ834QfnGGSXRw/640?wx_fmt=gif&from=appmsg)

  

上个周末，PC 市场被英伟达、Arm、Windows 等多方发布的预告消息搅动了！

  

传闻称，英伟达将与微软合作，推出首款采用 ARM 架构的英伟达芯片作为主处理器的 Windows PC，从而向传统的 x86 架构发起挑战。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqH7KeE4xEMJc5ZqZKVFibpe6dLB7ib9MuLqyK1WEfUEky001NvlqPiaccjB23cbdCJficrvP452qMRGSa178bUKd7iaL9GW18ev52GI/640?wx_fmt=png&from=appmsg)

  

今天，靴子终于落地。在上午 11 点开始的英伟达 GTC Taibei 2026 大会现场，黄仁勋拿出了英伟达与微软联手打造的 PC 产品。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqF5Gwh6bfsam1Okoz2dCnqFTiaArdOS9HC1c0kbXt9QYLl2iazBbBQ6uTc5D1QC4ChEZ05k9eRPP6u6meqLuhzxicia8iasN4xhloSU/640?wx_fmt=gif&from=appmsg)

  

不过，演讲开始之后，黄仁勋并没有直接进入正题，他回顾了此前三月份 GTC 大会上发布的 Vera Rubin 系列 GPU，目前该系列已经全面投入生产。此外还有被定位为「面向智能体时代」的 Vera CPU。

  

在细数了将近 1 个小时已有成果之后，黄仁勋终于开讲今天的重头戏：一款迄今为止全球性能最强、能效最高的轻薄型 Windows PC。

  

黄仁勋表示，英伟达和微软正在重新定义 PC 的硬件基础。未来面向本地 AI Agent 的新一代 PC，不再只是传统办公电脑，更是带有强 AI 推理能力的小型 AI 工作站 / Agent PC。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGCrXYAyHX7O1SDYP1CV9WfeUSaiaBVFakHohuBNYS4cz12Z77rr1GQJGRae4thAKp2Wb4s92o3jicquXfUk3r9ACwOwxzWlRtNQ/640?wx_fmt=png&from=appmsg)

  

支撑起全新 Windows PC 的是一款全新的超级芯片 NVIDIA RTX Spark，它带来一种全新的计算机形态：PC 不再只是工具，开始成为你的队友。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqEPSweoB8Xnm7hrtZgdLJHGzJd5VNTfMPpycWKicNRCurV2AUyDicPyQa6c2W3G4MWqSBgT1fuJxpquJcfqJhib8q5VunRXd0Xc5A/640?wx_fmt=png&from=appmsg)

  

从目前公布的数据可以看出，该芯片与英伟达去年发布的微型「个人 AI 超级计算机」DGX Spark 中使用的 GB10 芯片相同，基于 Arm 架构，CPU部分由联发科打造。其搭载了一颗 NVIDIA Blackwell RTX GPU，拥有 6144 个 CUDA 核心和支持 FP4 精度的第五代 Tensor Core，并通过 NVIDIA NVLink-C2C 芯片间互联连接到一颗高性能的 20 核 NVIDIA Grace CPU。

  

当然对于笔记本电脑来说，肯定会有价格更低、内存更小的配置。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqE3kLS6aibmskkULC78hIfgo2opwqIXHvtxxFammxmNAict4eMtb4bYfNJEQbUCROwhYwpex9oicq7YlvVic2cic4JCHVbRvtBltMNo/640?wx_fmt=png&from=appmsg)

  

RTX Spark 可以被用于轻薄的 Windows 笔记本，带来全天续航能力；也可以用于小型、高能效的台式 PC。

  

RTX Spark 笔记本最薄可做到 14 毫米，重量轻至 3 磅，提供 14 英寸至 16 英寸多种尺寸。机身采用精密加工的铝合金材质，保证耐用性的同时，呈现简洁、现代的设计语言。具备精准色彩表现的双层 OLED 显示屏，并支持 NVIDIA G-SYNC 技术，可为创意工作和沉浸式游戏带来出色视觉体验。

  

小巧、高能效的 RTX Spark 台式机则面向 Agent、创意工作负载、游戏和日常生产力场景打造。各大硬件厂商正在围绕 RTX Spark 加速布局，许多产品设计已经在开发之中。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqFrkicook4QN6QicDL8xZ9qlPb5ibucYAuOcxB08a9J63Pic1I2q812icdibf9aiaYxRuLZbxzrPsySNhyh0LRQMve8bryIx9HDjjSUgA/640?wx_fmt=png&from=appmsg)

  

黄仁勋讲到，「PC 正在被重新定义。过去 40 年里，你启动应用，点击，输入。而有了 RTX Spark 和 Windows，你只需要提出需求，PC 就会完成工作。RTX Spark 把 NVIDIA 已经打造的一切 ——CUDA、RTX、我们的 AI 平台 —— 整合进一颗超级芯片。本地 Agent、前沿模型、创意工作流、RTX 游戏，全部都能在一台笔记本上运行。这就是新一代 PC，个人 AI 计算机。」

  

另外，联发科作为 Arm 架构 SoC 设计领域的市场领导者，与英伟达共同参与了这一定制 CPU 的设计，为其带来同级领先的能效、性能和连接能力。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGiaXD2qTlOZHW1lrMFyv8H7ywHDCZ0HNAqwBrsicjlNfbNjb0KdsgKYFMYABAyQWnS3lWyV67IwnrbVPTSPtug2GA0CATp0RCMQ/640?wx_fmt=png&from=appmsg)

  

在博客中，英伟达详细介绍了其联合微软为端侧 Agent 打造强大、安全的 Windows 平台的初衷。

  

AI Agent 正处在一个关键拐点。OpenClaw、Hermes Agent 等开源项目，已经在 GitHub、OpenRouter 等开发者网络上取得了创纪录的增长。但 Agent 的大规模普及仍然受到限制，核心原因在于：用户很难在自己的主力 PC 上安全、私密地运行 Agent。

  

因此，双方合作的基础是包括新的 Windows 安全原语，以及 NVIDIA OpenShell 运行时，以确保 Agent 能够在安全环境下运行，并且始终处于用户的完全控制之下。

  

这一强大的安全和隐私层，正在被 Hermes Agent、OpenClaw 等领先 Agent 开发者用于他们的新 Windows 应用。这些新应用将让用户更轻松、更安全地使用强大的端侧 Agent：它们可以在 Windows 应用中执行任务，完成跨应用工作流推理，生成图像和视频，编写插件和应用，并对本地文件进行语义搜索。

  

要在本地设备上驱动 Agent，不仅需要强大的安全机制，也需要高性能硬件。RTX Spark 最高可提供 1 petaflop AI 算力和 128GB 统一内存，以满足端侧 Agent 的处理需求。

  

全新 Windows PC 能做什么？

  

具备这样的硬件配置，RTX Spark 要怎样颠覆 PC？英伟达声称，这构成了一种「AI 用户体验」，用户不再需要掌握复杂的应用程序用户界面，你只需要和电脑对话就能解决问题，也用不到鼠标和键盘。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqHw2LrV8IutlDgEsQ1g4xkm2Q6aOc3BhUrnfZoyuib66MTwticqXFk4EAzYMdnNaJnaoo3YX5Pql0msVtl6KILciaj4CHibPpZq1DM/640?wx_fmt=png&from=appmsg)

  

对未来的设想可能还有些远。不过对影视剪辑和 3D 建模人员而言，此类设备显然提升了移动办公环境下的处理能力。

  

系统最高支持 128GB 统一内存。在处理高分辨率视频合成或复杂 3D 几何图形时，GPU 可直接调用系统内存，缓解了传统独立显存容量受限的问题。

  

目前，包括 Adobe Premiere、DaVinci Resolve、Blender 和 Maxon Cinema4D 在内的多款专业创作软件已原生适配 Arm 架构。结合硬件加速技术，可提升 4K 视频导出与特效渲染的效率。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGZrIOyPGxVw9p61KiaQ3zH3rWNBdfyQM3Dx1JppzmiakseBXQwicXnfib5jaLVLNo1jPXUYX4xFLnaE6UhNJLOfOtKicEiaHuD0ict20/640?wx_fmt=png&from=appmsg)

  

依托于第五代 Tensor Core 的算力，Photoshop 中的「生成式填充」和 Premiere 中的「生成式扩展」等生成式 AI 工具获得了硬件级加速。数据显示，在实际创作流中，其 AI 处理、编辑与特效渲染速度最高可提升至 2 倍。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqEW0TJl7bl6Pm6EOzxUia2yU8BLhLzGHsBHAFhOicJyHib1xD3T2IBrDZ5iaibwrRKNMygeQHH8UChaq8ECZg68AQ7ia44RdrttfKLDc/640?wx_fmt=png&from=appmsg)

  

对开发者与专业用户，可实现实现本地化的 AI 模型与智能体运行。此类 PC 的主要特性之一是增强了对本地 AI 智能体的支持。

  

设备可提供最高 1 Petaflop 的 AI 算力，配置最多 6144 个 Blackwell RTX 核心，允许用户在离线状态下运行较大参数的 AI 模型，减少对云端计算的依赖及网络延迟。

  

借助 NVIDIA OpenShell 及 Windows 的安全隔离机制，开发者可在本地环境构建和测试代码分析、内容推理等 Agent 流程。本地化运算使得敏感数据和核心代码无需上传云端，满足了企业对数据隐私及合规性的要求。

  

对游戏玩家，提升轻薄型设备的图形游戏性能。虽然采用了 Arm 架构 CPU，但在微软优化的 Prism 模拟器与 Blackwell 显卡的配合下，该设备在图形游戏领域表现出了较强的兼容能力。

  

针对过往 Arm 架构运行游戏最大的痛点，微软与 Epic Easy Anti-Cheat、BattlEye 等反作弊系统厂商进行了底层深度合作，确保了主流多人在线游戏能够在系统内正常加载并运行。

  

该平台支持 DirectX 12 神经渲染与光线追踪技术。目前，包括《英雄联盟》、《无畏契约》（VALORANT）、《绝地求生》（PUBG）等电竞游戏，以及《心灵杀手 2》（Alan Wake 2）等高硬件要求的大型 3A 作品，均已确认兼容并在该架构下保持稳定的帧率与画面表现。
