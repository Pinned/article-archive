# 分手微软后，OpenAI 今天在 AWS 安家

昨天，[OpenAI 和微软，官宣分手](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247516360&idx=1&sn=a6e8fdc670757a6a734b899bb3567f88&scene=21#wechat_redirect)

今天，OpenAI 已在 AWS 把家安好

三件家具一起搬上：**模型、Codex、Managed Agents**

三件家具
----

### 第一件，OpenAI 模型上 Bedrock

包括 `GPT-5.5` 在内的模型，今天起可以在 Amazon Bedrock 直接调用

企业从已经买好的 AWS 环境就能拿到模型，省了再走 OpenAI API 或 Azure 一道

### 第二件，Codex 上 AWS

OpenAI 顺手交了一个数，Codex 现在每周 **4M** 活跃用户

Codex 在最常用的几个入口都能切到走 AWS：命令行、桌面 App、VS Code 扩展

### 第三件，Bedrock Managed Agents，由 OpenAI 提供能力

企业可以在 AWS 自己的环境里部署 Agent，让 Agent 维护上下文、执行多步流程、调用工具、跨业务系统办事

AWS 管基础设施、工具调用、编排、治理，开发者只管把 Agent 派出去

同一场大会，AWS 还摆了几件事
----------------

今天的发布在 SF 的「What's Next with AWS」大会上

AWS CEO Matt Garman 主持，除了 OpenAI 三件套，AWS 同场还出了

### Amazon Quick 大更新

AWS 自己的 AI 助手，加了桌面 App、自定义 dashboard，集成 Google Workspace、Microsoft 365、Zoom、Salesforce

业内对位的产品是 Anthropic 的 Claude Cowork

### Amazon Connect 改成 agentic 应用家族

Connect Decisions 做供应链预测和扰动管理，Connect Talent 做语音面试，自动给候选人打分
