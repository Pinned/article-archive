# 像Vibe Coding一样写论文！OpenAI发布免费科研写作平台

##### 克雷西 发自 凹非寺  
量子位 | 公众号 QbitAI

就在今天，OpenAI给肝论文的科研党送上了一份大礼——免费的科研写作平台Prism。

它把GPT-5.2模型深度集成到了在线LaTeX编辑器中，能够直接理解论文的完整结构、公式推导与参考文献。

现在，写论文都用上Vibe Coding了！

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZoktWfycxu5boadqgpiat2lH7yhZUWdiaP3qIcCj5zfkewVpzz3fa52Aew/640?wx_fmt=png&from=appmsg)

Prism里，找文献、看预览和写正文的流程都被整合在了一个网页里。

你随手拍下一张写满推导过程的白板照片，几秒钟后，它们就自动转化为文档里完美排版的LaTeX代码。

Hyperbolic Labs的联创兼CTO Yuchen Jin犀利评价，Prism已经杀死了overleaf（一种LaTeX编辑器），未来的每篇新论文都应该把ChatGPT列为共同作者。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZobVOcbZ55lx1Kic2mknJh3UViblsibK9Fjgia4ln0fM7tt2tE5tbMuHK3kQ/640?wx_fmt=png&from=appmsg)

要知道不久之前，奥特曼刚刚说OpenAI把写作搞砸了，后面的版本会重视写作能力。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZo5ibKevW2UhQrv3zzlecfBKSSDj5CUNGtdibibDOjQOSK0sF17gxUtL4Vw/640?wx_fmt=png&from=appmsg)

结果一个新的写作神器，就这么水灵灵地出现了。

用Vibe Coding模式写论文
-----------------

用Prism来写论文，有点像Vibe Coding。

它的主界面采用了直观的左右分屏布局，左侧是LaTeX代码编辑区，右侧是即时编译的PDF预览窗口。

代码编辑区的底部嵌入了一个AI对话框，在这里输入指令，系统会立即生成特定的公式代码、解释编译报错的具体原因，或者对选中的文本段落进行润色，另外这里还支持语音模式。

![](https://mmbiz.qpic.cn/mmbiz_png/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZoj4nupGmIiam71jbT62PKO6iaJP6TFDjOaxchPc5iaTiaCdAZUgkOLRdxLQ/640?wx_fmt=png&from=appmsg)

它还能连接外部数据库检索资料，一键生成BibTeX代码并插入正文，保证了检索与写作在视线上的连贯性。

![](https://mmbiz.qpic.cn/mmbiz_gif/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZoKLfn2y4niaxianR6sEZIlRbQhxtiaDPsQ6XyZlyia4FvHBwlaTkpWVnM3Q/640?wx_fmt=gif&from=appmsg)

针对复杂数学公式的录入，直接上传一张写满推导过程的白板照片或手写笔记，系统就能精准识别，并迅速将其转化为排版完美的LaTeX代码插入编辑器。

![](https://mmbiz.qpic.cn/mmbiz_gif/YicUhk5aAGtDSB8Ex7bxvSepA21zRsCZoe7RmvMfzcRxTeXzPYN5rqh64qZs0vFrD87QsQhWWSNRSFf4B5jCHnw/640?wx_fmt=gif&from=appmsg)

系统还具备数学推导能力，可用于核查公式的准确性。要求它检查某一特定公式是否导出了预期的物理对称性，后台会进行符号计算与推导，验证计算结果是否与文中结论一致。

最后，所有这些功能被封装在一个支持全实时协作的云端架构里，合作者可同时进入同一个文档，队友的光标位置和输入内容都会同步呈现在屏幕上。

而这次上线的Prism只是一个开端，背后暴露出的，还有OpenAI向AI for science（AI4S）领域进军的战略目标。

OpenAI的AI4S战略
-------------

以Prism为出发点，OpenAI正在将触角深入至科研的核心地带。

在接受MIT Technology Review的独家专访时，OpenAI副总裁、新成立的“OpenAI for Science”团队负责人Kevin Weil详细拆解了这一战略背后的逻辑。

他透露，公司于去年十月正式组建了这支专门团队，核心使命就是致力于“加速”科学发现的进程。

在Weil看来，现代科学研究的效率往往被大量非创造性的环节所拖累。

起草论文初稿、梳理晦涩的参考文献、清洗实验数据以及对初步假设进行验证性推导……这些工作占据了科学家大量的时间。

OpenAI的策略是利用AI接管这些高耗时、低认知的任务，让科学家能够从繁琐的事务中抽身，将精力集中在真正需要洞察力的核心问题上。

这一目标背后的技术支撑来自OpenAI最新迭代的GPT-5.2模型，它展现出了远超前代产品的推理能力。

然而，科学界对AI最大的顾虑始终在于幻觉问题。

针对这一点，OpenAI正在为模型引入一种被称为“认识论谦逊”（epistemological humility）的关键特性。

Weil解释说，科学研究需要的不是一个只会迎合用户的“马屁精”，而是一个敢于质疑和自我校对的合作者。

系统将建立一套自我评估机制，模型在生成答案后会进行内部的对抗性审查，甚至通过自我反驳来寻找逻辑漏洞。

基于技术成熟度与应用场景的匹配，Weil给出了一个极具分量的行业预判——

2026年将成为科学界的“AI变革年”。

他将这一即将到来的范式转移类比为2025年的软件工程领域——在短短一年内，AI辅助编程从少数极客的尝鲜迅速成为了开发者的行业标准。

在他的愿景中，未来一年内，熟练运用AI工具会迅速确立为科研工作的标准范式，不再只是早期采用者的专利。
