# SpaceX 600 亿美元收购 Cursor！马斯克不补课，直接买课代表

![](https://mmbiz.qpic.cn/mmbiz_png/lYDricib2fsm4m0DJy0Kl9icPRLeXPaslpovLic7D8diboVicDvBmMuic2BWCaLibxlkIpBPdl7XLicNBUXGW6RXS16FNqmhIkOqwlO5ZAkMewVuG3A8/640?wx_fmt=png&from=appmsg) 

作者｜冬梅

**SpaceX 斥资 600 亿美元拿下 Cursor**

马斯克在 AI 编程工具上的短板，最终没有靠慢慢补课解决，而是用了一种非常马斯克的方式：直接买下最关键的那家公司。

![](https://mmbiz.qpic.cn/mmbiz_png/lYDricib2fsm52QticP4yGp3eosFzy27q1tPIaOwEvMeAIOLqDiawpicegohAribpZ9gU2wTGJOwIuHYcCMj1j7HJ9hKeY35ibODypWpN0BjUFwhFI/640?wx_fmt=png&from=appmsg)

据彭博社等多家外媒消息，SpaceX 已正式同意收购 AI 编程初创公司 Cursor，交易估值达到 600 亿美元。这笔交易将以 SpaceX 股票完成，Cursor 投资者将按照 Cursor 隐含的 600 亿美元股权价值获得 SpaceX 股份。

若进展顺利，合并预计将在 2026 年第三季度完成。

Space X 刚刚完成 IPO 后，此次收购 Cursor 更像是马斯克对 AI 编程战场的一次“补位”。

过去一年，AI 编程工具几乎成了大模型商业化最先跑出来的场景。

Anthropic 的 Claude Code、OpenAI 的 Codex，以及 Cursor、Windsurf 等产品，都在争夺开发者的入口。相比聊天机器人，代码工具离企业付费更近，也更容易证明 ROI：它能直接进入工程师工作流，影响开发效率、代码审查、调试和软件交付周期。

马斯克在着急什么？

问题在于，xAI 在这个领域并不领先。

马斯克此前也承认，xAI 在编程工具方面落后于竞争对手。为了追赶 Anthropic 和 OpenAI，xAI 过去一直在从 Cursor 招募工程师。但挖人显然不够快，也不够彻底。

![](https://mmbiz.qpic.cn/mmbiz_png/lYDricib2fsm6C2ALGqsm89BvJykib3MKEuJtsnFXDVJSmtPKibJia0PexavjlwhBQ48nA46oAC6MCsw7SiapZ0s4BXyiaLF0NERKTRCWPzXfjEUpQ/640?wx_fmt=png&from=appmsg)

现在，马斯克干脆把 Cursor 整个买下来。

Cursor 的价值，首先在于开发者入口。它不是又一个单纯的大模型应用，而是嵌入程序员日常工作的 IDE 级工具。开发者每天在里面写代码、改代码、问模型、调项目，这些交互数据和真实工程场景，对训练下一代代码模型有极高价值。

对 Space X 旗下的 xAI 来说，Cursor 恰巧是那条进入企业软件开发流程的高速通道。

还有一个值得注意的点是，Cursor 的瓶颈也正好是马斯克体系能补的短板：算力。

随着用户增长，AI 编程工具对推理算力、模型迭代和企业级服务稳定性的要求越来越高。

Cursor 如果继续独立融资，需要不断为算力买单，但并入 SpaceX 与 xAI 体系后，它至少可以获得更强的基础设施支撑。这也是这笔交易看起来昂贵，但逻辑并不难理解的地方。

当然，代价同样巨大。

600 亿美元估值，已经远远超过大多数 AI 初创公司的融资规模。SpaceX 去年资本开支大幅增加，AI 已经成为其中重要支出项。马斯克正在把火箭、卫星、社交平台、AI 模型和开发者工具拧成一个更庞大的技术帝国，但这套体系也越来越烧钱。

这笔交易最有意思的地方在于，它再次体现了马斯克的一贯风格：不愿意在关键入口上长期落后。如果一个赛道足够重要，而自己短期内追不上，那就直接收购最靠前的玩家。

从特斯拉到 SpaceX，再到 xAI，马斯克一直擅长用极端资源配置换时间。Cursor 这一次也是如此。

AI 编程工具已经不只是程序员效率工具，而是大模型公司争夺企业入口、代码数据和软件生产方式控制权的核心战场。

马斯克现在给出的答案很直接：这张牌，不能只靠 Grok 慢慢追。

既然 Cursor 已经坐在牌桌中央，那就把牌桌买下来。
