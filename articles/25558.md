# 互联网变天：机器人流量首次反超人类，Agent 时代拉开大幕

![](https://mmbiz.qpic.cn/mmbiz_png/lYDricib2fsm4Oy0HHxhl0HRDFX6XprKtz71sicd8dW2Q87cEPa5DvUEyqdDQGE6mTaPjfwlDOicbuTian0jicQqjP6fMpsVcX0fa01RMy5rgwIlo/640?wx_fmt=png&from=appmsg)

作者 | 四月

过去半个多世纪，互联网的技术和商业体系，都是建立在“服务对象是人”这个前提之上的。

因为人会打开网页、点击链接，网站才需要精心的设计视觉与交互；因为人的注意力有限，搜索才需要排序、内容才需要推荐。我们习以为常的浏览、搜索和留痕，构成了整个互联网商业的基石。

但现在，这块前提正在松动。

根据 Cloudflare Radar 数据显示，在全球 HTML 网页请求里，Bot 机器人的流量占比已达 57.5%，人类流量只剩 42.5%。**互联网有史以来第一次，访问网页的请求里，机器比人更多。**

也就是说，**互联网的默认访客，正在从“人”变成“机器人”**。以前我们问“网站有多少人访问”，接下来要问的是：**有多少机器在替人访问？它们是谁派来的？它们想干嘛？**

![](https://mmbiz.qpic.cn/mmbiz_jpg/lYDricib2fsm5WJubQia5BwQhIcLWeB6TEGPMZicUo2tibicKjC3Iwxr800fpiaYdxNMT35f6ZCUsojiaQF03KEYJwDRXh1bNyzZnZHq0u11xkrJ2zQ/640?from=appmsg)

并且，机器人的访问分布是极不均匀的。全球均值是 57.5%，**但在美国，这个数字达到了 71.5%**。也就是说，在全球互联网最发达的市场里，**每十次网页访问请求，七次多是机器发出的**。

Cloudflare CEO Matthew Prince 在 x 上谈道：“Welp, that happened faster than I predicted.”

![](https://mmbiz.qpic.cn/mmbiz_png/lYDricib2fsm5G7ORXXyCzpejt6iaaVXaWibRD9VADTAY3G7q9YDYxSgz7W9YxFnczx3BJicL4gQqDc6sbJ262nxWqwINKjGhSfur7icBvjnRS4dc/640?from=appmsg)

三个月前，他在 SXSW 上预测，Bot 流量超过人类流量大概得等到 2027 年底。现在他不得不承认，这个速度比以往任何时刻都要更快。

值得注意的是，上述统计仅限 HTML 页面请求，不含图片、视频、API 和流媒体等。但即便加上限定，**这依然是个历史性的时刻**。因为 HTML 恰恰是传统网页世界最核心的入口。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/lYDricib2fsm5AwbLHCLKrFT6Yic9YTkhU6VbbOPW19aV2qtZmzONq7HpMlAb0D7K95Qjg8e3zqXrK90fZ3BwUvtwTrbwliaM70kHnvLu87W8iaU/640?from=appmsg)

过去我们说“上网”，本质就是人通过浏览器访问网页；而现在，**这个入口不再主要由人类打开**。越来越多网页，是被爬虫、监控脚本、AI 抓取器，以及正在兴起的 AI Agent 打开的。

Bot 超过人类，不是终点，而是开场

Bot 不是新玩意。搜索引擎爬虫、安全扫描脚本、价格监控，早就构成了互联网的暗流。今天真正的新变化，不是“有 Bot 了”，**而是 Bot 的角色变了**。

过去的 Bot，更多在为系统服务——抓取、索引、监控，任务清晰，边界稳定。但 Agent 出现后，**Bot 开始代表人的意图行动。**

一个购物 AI 不只是看价格，它会比较平台、筛除差评、核对优惠券，甚至结账；一个旅行 Agent 不仅搜航班，还会对比酒店、查库存、预订行程；研究 Agent 则能连开几十个网页，交叉验证后组织成报告。

**传统爬虫是在“读网页”，Agent 是在“使用网页”。**

![](https://mmbiz.qpic.cn/mmbiz_jpg/lYDricib2fsm6ej1VOicFMKJM6RV1oMQVAu0ne4sQG3KRvvqPrIutmK7ZGMsPKrr1N6FucIGH6dsofw82BM6IiaOvO8MqK4LKbBHGlmLibvNAMow/640?from=appmsg)

前者是读者，后者是代理。它不是替人看一眼，而是替人完成一连串动作。也正因如此，Bot 流量超过人类，并不是 Agent 时代的终点，而是它全面侵入互联网的前奏。

8000% 增长背后：从阅读走向行动

当前的 AI 流量里，训练爬虫仍是主力，但结构正在剧变。

HUMAN Security 的 2026 年 AI Traffic 报告显示：**训练爬虫占比正在下降，而 AI scrapers 增长了 597%，并且 AI agents 和 agentic browsers 的流量同比暴涨 7,851%！**

2025 年自动化流量同比增长 23.51%，是人类流量增速（3.10%）的近 8 倍。其中，AI 驱动流量年内增长 187%。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/lYDricib2fsm6WJYwuVI6wRPtpn99tG0szaPM0mKpcVkCo92WaFdhalRJ3gOWwkQcQicsBxqKOAUdK3Ntodj7MmVia1vlw81HVjKicpDcSZPzT2o/640?from=appmsg)

这说明，AI 访问互联网的动机变了。

早期 AI 访问网页，主要是为了训练，把内容抓进语料池；后来 AI 搜索兴起，需要实时抓取和总结；现在轮到 Agent，它不满足于把网页变成答案，而是要把网页变成任务的一环。**网页不再是信息源，而是操作界面。**

这也是为什么 AI 流量高度集中在三个行业：**零售电商、流媒体与媒体、旅游和酒店**。

它们的共同点是：信息高度结构化、变化频繁，且直接连接交易。商品价格、库存、房态、评价，都很适合机器读取、比较和行动。此时，网页不再是内容展示页，而变成了机器决策链条里的一个节点。

以前网站怕没人来，现在怕来的不是人

过去，网站最怕没流量，所有运营动作都指向“把人引过来”。但 Agent 时代，问题变了：**网站可能有大量访问，可来的不是人。**

机器读完网页，提取重点，把结果直接呈现在 ChatGPT、Perplexity 等入口里。用户得到了答案，却未必会点击原网页。**内容被消费了，流量却没有回来。**

搜索引擎抓网页，至少承诺一种交换：我抓你内容，给你带回点击。AI 入口更狠，它抓取、理解、总结，最后直接交出答案，原网站沦为模型背后的“隐形供应商”。

**这正是 Cloudflare 推出 Pay Per Crawl（按爬取付费）引发热议的原因**，它本质上是在给“机器访问内容”重新定价。过去互联网的账本按人记：UV、PV、DAU、转化率，全默认访问者是人。

一旦机器成为主要访客，这套账本就开始失灵——**机器能创造访问量，却不一定创造广告价值**；能消耗内容，却不一定带来品牌认知。流量，开始不再等于价值。

入口从搜索框，转向模型和 Agent

入口控制权也在转移。报告显示，AI 流量高度集中在少数平台：**OpenAI 相关 bots 占比约 69%，Meta 约 16%，Anthropic 约 11%。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/lYDricib2fsm70ic9ZjvT8T2Hkg8exgfqkyOYAkibO88iafib3eTdhNn8kk5IYWgUpTDqLZUkFABsh0acxKL4G7C0L8HLjbw90H1rEz3cPhml4gEE/640?from=appmsg)

PC 时代网站争夺 Google 入口，移动时代争夺 App 和信息流，Agent 时代的入口，正集中到少数模型公司手里。用户不再自己开十个网页比较，而是问模型一句话，模型替他筛选、压缩甚至执行。

网站面对的不再只是 SEO，而是新问题：**我的网页 Agent 读懂吗？信息模型能信任吗？服务机器能调用吗？内容被消费后能获回报吗？**

未来很多网站会演变成双层结构：**一层给人看，争夺注意力；一层给 Agent 读，争夺机器可用性**。给人看的需要故事、视觉、停留时间；给机器看的需要结构、权限、接口和责任边界。

流量主体转变的背后

我们看到，当互联网的流量主体从人变成 Agent，随即也带动了三个层面的深层变化：

网页变了，从供人阅读的信息源，变成供机器执行的操作界面；价值变了，从基于人类注意力的变现，转向基于机器调用的定价；入口变了，从搜索框的分发，转向大模型的一言而决。

57.5% 远不是终点，很有可能只是一个开始，是互联网从“人类的浏览场”走向“机器的执行场”的起跑线。人依然是意图的源头和价值的终点，但中间的触达、交互与交付，正不可逆地交由 Agent 完成。
