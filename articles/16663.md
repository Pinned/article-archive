# 港大×复旦×上交：视触觉融合+闭环纠错，让机器人双臂协作不再「盲操」

##### TAMEn团队 投稿   
量子位 | 公众号 QbitAI

在具身智能快速发展的今天，高质量数据已成为驱动能力提升的关键基础，然而一个核心问题也随之而来：

如何让机器人数据采集更快、更稳、更有效？

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGG1GGzKsbEWHbGXia0zZyLJpuvulCzBGqPVGGZpoQZ5XoSibYJdrlWXnkyUibXVUIRuLqpyTiaN0CbElZ7xAV6Q8SQMHu1UWyEmtvM/640?wx_fmt=png&from=appmsg)

港大联合复旦、上交大近期提出的**TAMEn**_（Tactile-Aware Manipulation Engine for Closed-Loop Data Collection in Contact-Rich Tasks）_给出的核心答案是：

在UMI框架基础上做了全方位升级——不是只给一个采集设备或训练代码，而是构建了**视触感知融合+精度与便携兼顾+数据可靠性增强+持续进化的数据闭环**。

从数据采集、策略训练，到部署纠错、再训练，整条链路被打通，让机器人学得更快、更准、更高效。

TAMEN视触觉闭环数据引擎总体流程
------------------

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGGPIHJ4hJuxJPZdjenOj44yP6XLKaUiah1LdITXPiaaG4MyjOytUAUevuf1VkbVmX4vglwMz9ymiczOLe5PZXuDick1dQd0VAkgvgg/640?wx_fmt=png&from=appmsg)

上图对应TAMEN的三层闭环：

**1\. Data Collection Hardware（数采硬件层）：**数采设备、传感器、灵活兼容的采集方式。

**2\. Data Recipe（人在环数据层）：**在线可行性检查，数据金字塔体系。

**3\. Data Flywheel（价值推理与训练层）：****AR**触觉反馈遥操作、介入纠错、闭环采集。

三者串联，形成了真实场景中的“数采-训练-部署-纠错-再训练”持续迭代闭环。

TAMEN的技术闭环
----------

可行性在线验证：

*   带拓扑约束的marker追踪，提升遮挡与复杂交互下的跟踪稳定性。
    
*   融合运动学与动态约束的在线可达性验证_（关节限制、速度约束等）_，确保数据可重播，告别数据后处理清洗。
    

![](https://mmbiz.qpic.cn/mmbiz_gif/A6fTew8FFGGQktM8sQXclriaib8Kj3mibnq0ojfrUToNUibbGzZ77PoZ9PyB7nicXUlteIamZdyvmtffz61C842gDgh5JXzWSWzaFPa1l4UpbBtw/640?wx_fmt=gif&from=appmsg)

双模式数据采集：

*   高精度动捕，实现亚毫米精度数采
    
*   便携VR，实现in-the-wild低成本采集
    
*   高精度与便携性间实现灵活切换
    

![](https://mmbiz.qpic.cn/mmbiz_gif/A6fTew8FFGGNr5j3mOMDHibpGxJ7bWZGd9BsoYF2KyicGCenXiaAp3MeGLpBOeBic0Q7C2GeEr6rvUeT9AVia6HWjbyNfOfFpfSsFCqE44baaeGA/640?wx_fmt=gif&from=appmsg)

持续进化闭环：

*   推理中AR介入，自研tAmeR APP，赋予遥操作者实时触觉反馈
    
*   记录recovery状态下的视觉、触觉、位姿信息，采集珍贵“纠错数据”
    

![](https://mmbiz.qpic.cn/mmbiz_gif/A6fTew8FFGF5zLhA2ybA0icUSiaq8WlYMKhRiaBZHVqcTKNAYbsQBjtJ10OYUMgMDVjKZGTo9jHgTteYa5FdMQHblptymVt0gQEldWk3DuIsL0/640?wx_fmt=gif&from=appmsg)

金字塔式数据体系：

*   从大规模单臂触觉预训练→双臂任务微调→失败恢复数据闭环优化
    

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHFWY4Ws0Z9pxibzpQwQCFzVibdOYqUaZictHZFIOfY76GxfarRTZiag9KYhUCSKLEU2vGel3757KfjsRktDqhy9OrXNhbZlk2ZHAc/640?wx_fmt=png&from=appmsg)

双臂协作任务验证
--------

在多项双臂协作任务中，TAMEn将平均成功率从34%提升至75%，这些任务并非简单堆叠，而是覆盖了接触丰富操作中的几类典型难点：从中草药倾倒中对可变形物体的协调控制，到线缆装配中的柔性物体的适应性抓握与插入判断，再到弹簧夹收纳中的多阶段协同时序配合，以及擦盘子中的持续接触。

![](https://mmbiz.qpic.cn/mmbiz_png/A6fTew8FFGHod1Cricb5Dku2t3icyoh5iclF76dyfzhlzSzVxIibkvJ0zTDA7qqP3PTVsIYqiaLhys2GxYZC3aueAE2bI6Y4KzcUHcjqmJBStMKE/640?wx_fmt=png&from=appmsg)

触觉输入、预训练和闭环纠错三者分别带来持续性能提升，且提高泛化性与鲁棒性。这标志着机器人数据采集正从“可用”迈向“可持续优化”的新阶段。
