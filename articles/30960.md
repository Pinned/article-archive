# Codex封号、Claude查证件，而它选择了免费且不限期

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

编辑｜Panda、冷猫

  

刚刚，Agnes 升级了！

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqEyQibHCd4PewgibP98zzNRahHCgoDbwg4JG5kLQRh1nU6BEkn41DXyzIPwhHmeUosdU4OfJNib60y9oYcVzHeHvEKYyiaUJnvd78o/640?wx_fmt=gif&from=appmsg)

  

不，不是这个 Agnes，而是 Agnes 大模型。刚刚，这个模型家族迎来了其最新成员：Agnes-2.5-Flash。这是 Agnes 全新一代高性能文本模型，围绕 Coding、Agent 和日常开发场景做了整体优化，目标是在性能、速度与成本三者之间找到更好的平衡点。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqGLEtJZXeLxw3OF8J4vjUxD3TpnkZoUCU7JlpOvgoB0wOHYJsw0GlJVYDj6RPPJcdTC6uWUAuynoGkctfF1b36FHHj7yKFLibIw/640?wx_fmt=png&from=appmsg)

  

相比上一代，Agnes-2.5-Flash 在代码理解、工程修复、多步骤任务执行以及复杂推理上都有明显提升，能够覆盖从代码生成到项目维护的大部分开发任务，定位是开发者日常使用的主力模型。值得一提的是，它延续了不限期免费开放的策略！

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqEG3ERBicnPfnDRRicW1Amvqic42H9RRibJ8ic14CtPKZiattW8mubiasxZBzlBUicslJMwtgU0LdUv0ibXlnMFGh90tFXBibSrZZ9Is8Cfo/640?wx_fmt=png&from=appmsg)

  

与此同时，Agnes 还首次预告了旗舰模型 Agnes-2.5-Pro。其专为复杂 Agent、专业开发和高难度工程任务打造，代表 Agnes 当前最高水平的文本能力：相比 Flash，它更强调复杂推理、长链路任务执行和大型项目协作，更擅长处理大型代码仓库、多文件修改、复杂架构理解以及系统级问题分析，能力上限更高。

  

使用 Agnes-2.5-Pro 生成的一个欢乐派对型 3D 障碍竞速游戏试玩视频

  

据了解，Agnes-2.5-Pro 将对标 Claude Opus-4.8、GLM-5.2，面向专业开发与复杂 Agent 场景，是一款付费的旗舰模型，预计将在近期正式发布。

  

不仅如此，Agnes 这一次还把工具与模型一起端了上来，同步推出了 AgnesCode 桌面端。该工具能「稳稳接住」模型能力，让其真正落到本地开发场景，让 AI 从一个聊天工具，升级成能够直接参与开发工作的 AI Coding Workspace。

  

具体来说，AgnesCode 把模型、Skill、应用连接、本地项目和生成物管理整合进了一个桌面 AI 工作台。用户不必理解底层工具怎么运作，只要把要做的事说清楚，Agnes 就会在项目里帮你完成。它的定位不是又一个聊天窗口，而是让你在桌面端发起任务、处理文件、调用工具、直接拿到结果。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqE2uLPzC1vSKJvxgYE3kfDxqfI0EkraHm1xkzmmyzTeibAQEeahg1mRS1uia3STFmPgIEfia9ncahS0KBsDEJyfHkMicRiabMOhEGEU/640?wx_fmt=png&from=appmsg)

  

它的设计亮点包括一个输入框就能完成任务、面向真实开发项目、不止写代码、生成结果可持续管理等。

  

不仅如此，AgnesCode 还默认搭载了最新的 Agnes-2.5-Flash 模型，并且不限期免费使用，同时与 Agnes 的 Web、App 共用账号、会员和积分体系。桌面端由此成为 Agnes 生态的一个重要入口。

  

实测之后，我们找到了新的工作搭子

  

那么，Agnes-2.5-Flash 与 AgnesCode 在真实任务上的实际表现如何呢？我们实测了一番。

  

首先，我们考验一下其代码能力，让其制作了贪吃蛇、太空侵略者等经典小游戏。整体来说，结果都相当可玩。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqHWaNPIwicAK8DYUkp5PN15parrTOiawPfu47TnyGpFPSXyib1icqTib3a4EfwLMdSdWkwnjrABBicAXNtkibkqwkIaia0QQogFoZDPPu0/640?wx_fmt=gif&from=appmsg)

  

胃口被养大之后，我们加了点难度：复现一个简装版的《超级马里奥》。这次它也相当出色地交了卷。

  

  

写游戏考的是「从零生成」，但 AgnesCode 主打的是 Coding Harness，因此真正的分水岭在于它能不能调用外部工具、在多步骤任务里把活干完。

  

正好，最近我们的官网正式上线了机器之心的数据服务，可通过 MCP/AI Skills 的方式访问和查询历史文章等数据。我们将其接入到了 AgnesCode 中，简单查询看看：机器之心最近三个月发了哪些关于 Loop Engineering 的文章？

  

![](https://mmbiz.qpic.cn/mmbiz_gif/5L8bhP5dIqFsKkz9XiaOVIje5jg2KV54ia35ok1NugCRRJtjQ9oeXS5MC1FGxxpGiaiaHWE7yicq1aOoP5vgY2vj2fPIUozho4JGxIAaCTAc4szA/640?wx_fmt=gif&from=appmsg)

  

AgnesCode 执行了 6 分多钟，中间完成了多次工具调用，最终给出了一个非常详实的报告。在这份报告中，Agnes-2.5-Flash 不仅列出了标题中带有「Loop Engineering」的两篇文章，还进一步检索扩展，找到了不少相关概念的文章。这表明 Agnes-2.5-Flash 做的不仅仅是关键词匹配，而是真正理解了我们真正想问的是什么。

  

另外，Agnes-2.5-Flash 在制作知识图谱方面也表现卓越，基于所提供的《明代文学家关系总谱》制作了一份相当精致的「明代文学人物关系图」：

  

  

AgnesCode 把自己定位成「桌面 AI 工作台」而非单纯的编程工具，那就得拿非代码任务验一验。接下来，我们就考验一下其在办公场景中的实力：做 PPT。

  

这不，前两天 ACL 公布了获奖论文——还真不少，有最佳论文、最佳主题论文、最佳资源论文、最佳社会影响论文……我们就直接丢给 AgnesCode 一个链接 https://2026.aclweb.org/program/best\_papers/ ，让其制作了一个绚丽多彩的 PPT 介绍三篇 Best Paper Awards 获奖论文，并且我们要求每篇论文至少有三页介绍。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqEGl9eLAnvWwyxFf8icnv9vib18kEesnCbLwnnRjjqImTSmfxiawcOUKhg6wBZRcAX4ZaxicyrMdxgcuepLdjeCzvPjCFptT2ibyiaro/640?wx_fmt=gif&from=appmsg)

  

整体来看，Agnes-2.5-Flash 与 AgnesCode 的成果还不完美，依然存在一些编码上的瑕疵，但整体来说，结构、内容乃至审美方面都非常出色，略作修改就可以直接使用。

  

从一个链接到一份成型的 PPT，中间的读取、筛选、提炼、排版，它自己走完了。

  

而这些，还只是免费的 Flash。Agnes-2.5-Pro 的表现又如何呢？由于该模型尚未上线，我们先来看几个官方 demo。

  

一句简单的「帮我做一个炫酷的粒子游戏」，Agnes-2.5-Pro 就交出了一个完整可玩的成品：包含可用的计分系统、正确粒子与错误粒子的判定机制，以及流畅的游戏画面。

  

  

2D 之外，官方也展示了它在空间和渲染上的表现：一个可以直接在网页中展示的 3D 小房间，包含完整的房间构造，房外几棵小树，整体走简单的卡通建模风格。

  

  

下面这个台风追踪器则更为炫酷，看起来颇有几分专业气象中心大屏的架势：地图、实时数据面板、风速仪表盘、图例、预警信息一应俱全，还配上了丰富的粒子特效，整体视觉精致流畅。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqEYDnKAcDGHvViclGHFeF3X5VEmqGq218OAFhw9JHuFdJbjYRSgNB64R5UdZaMFIX8leRu5c52XjgTsibJSkrDLPUT4WjwkCtwUY/640?wx_fmt=gif&from=appmsg)

  

这个展示漫威系列电影的网页也非常有用，包含清晰的时间线、角色介绍和人物关系，甚至还分阶段地列出了不同时期的电影，完全可以成为新影迷的漫威超级英雄电影之旅的「入坑指南」：

  

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/5L8bhP5dIqHjaC9GRheorVmkvqqM9Jibb19MQ23pKfdo9kwpVKNFtTp5UfDNJLCib4kIzl0xWGib63gBE4oAW76ZJYcdlgVR5Oib0j12jy8LBWc/640?wx_fmt=gif&from=appmsg)

  

整体来看，Agnes-2.5 系列模型与 AgnesCode 在产出成果的结构、内容乃至审美上都相当可用，可以说是已经真正从「玩具」进阶成了「生产力工具」。

  

上面这些 demo 横跨了代码、数据检索、信息梳理和办公文档等不同类型的任务，如果让人类来做，需要使用不同的工具和工作流，甚至可能还需要不同的专业人士，而在集成了 Agnes-2.5 系列模型的 AgnesCode 中，这些任务都只是同一个输入框里的不同句子，产出物也都规规矩矩落在各自的项目空间中。

  

所谓「工作搭子」，大概就是这个意思：你不需要为每件事切换一次身份，打开就能开始干活。

  

结语

  

把这次发布放回最近半年的行业背景里，脉络会更清楚。

  

今年 6 月初，OpenAI 迎来一波大面积（误）封号，多名用户反映账号在没有预警、没有解释的情况下被暂停，波及 ChatGPT、Codex、订阅乃至多年历史记录，社区统计估算受影响的订阅用户在 5% 到 10% 之间。如果你已经把 Codex 接进工作流，那可能就意味着某天早上你打开电脑，却发现自己的整条流水线一夜之间停摆了。

  

另一边，Anthropic 从 4 月起在 Claude 平台上线强制身份验证（KYC），需要证件加实时人脸自拍，而验证服务并不支持中国大陆签发的证件，官方支持中心甚至预置了「为什么我验证之后账号反而被封了」这样的条目。

  

对国内开发者来说，「使用顶级 AI 编程工具」这件事的门槛正越来越高。当越来越多的 Coding Agent 开始「下班」，人们越来越迫切地需要一个能稳定在国内用、不用担心封号、还持续免费的选项。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/5L8bhP5dIqE1JmZmEVvh6AwPgtzzy1oSJrqURBicicEr5Xs1q7iaKUnicdnfE3LcrVFWhmlI8mHgpnibJSmAJMicuf3Tic5iaYVjexOXI7fd0gs9KUg/640?wx_fmt=png&from=appmsg)

  

今天，Agnes 为我们提供了这样一套选项：Agnes-2.5-Flash 可让开发者以更低成本拿到更强的 Coding 能力；Agnes-2.5-Pro 预告在即，指向旗舰级的开发体验；AgnesCode 桌面端同步亮相，把模型、Harness Agent 与本地项目连成一条完整链路。

  

至于它们到底好不好用，上面的实测已经给出了一部分答案，剩下的，交给你自己上手去试。
