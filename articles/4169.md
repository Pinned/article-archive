# 小红书、AWS、商汤的一线实战：AI 应用如何从“能用”到“好用”｜奇点智能大会议题前瞻

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/S1iaf4GgGjEzNJeOR6Hj8ib43Nj6kXQdTZvp0LYTEkeWOvIIffiaQag5uG6Fmlz6ciaL3r3OMtWMLLKn9a4K7RWA02EU4CMlN6mQhLlqLRcjtlw/640?wx_fmt=jpeg&from=appmsg)

过去两年，我们见过了太多惊艳的 Demo 。

视频生成只需一句话，PPT 一秒成型，医疗问诊看起来比专家还快。但当你真正要把它们放进业务流里，问题才刚刚开始：生成的视频没法改细节，搜索成本烧得太快，PPT 生成后无法二次编辑，医疗 Agent 不敢真的交给医生用……

2026 年，AI 应用的下半场，不再是比谁模型更大，而是比谁的产品更“可用”。

4 月 17-18 日，CSDN 与奇点智能研究院举办的「2026 奇点智能技术大会」在上海·环球港凯悦酒店隆重举行。

在「AI 原生应用创新与开发实践」专题，我们邀请了小红书、商汤、平安科技、AWS、Dify 五位一线实战派，拆解如何把 AI 与业务结合，让产品真正落地。

下一代视频剪辑范式——OpenStoryline

🔹演讲嘉宾：许思杰｜小红书 SuperIntelligence AIGC 组算法工程师

🎯核心问题：大量剪辑工作其实是可结构化的重复劳动。如果这些步骤能被 AI 自动化，创作者能否只凭“说话”就完成一条视频？

💡为什么值得听：许思杰作为开源项目 OpenStoryline 的负责人，他致力于将复杂的视频制作流程转化为可交互的智能系统，让 AI 真正理解“剪辑思维”而非简单套用模板。现场，他将系统拆解 “从 Workflow 到 Agent ”的演进路径。

👉适合谁来听：正在从事多模态内容生成、AI 影视制作、数字创意工具开发的，苦于“生成很炫但不可控、不连贯、不可编”的 AIGC 应用开发者和技术人员。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

GenAI中搜索的极致性价比之路

🔹演讲嘉宾：杨扬｜AWS OpenSearch AI研发经理

🎯核心问题：当 RAG 成为标配，Token 消耗却越来越“任性”，推理成本水涨船高。如何在亿级并发下兼顾“理解力”与“性价比”？ 杨扬带来的正是OpenSearch 的最新探索。

💡为什么值得听：杨扬将在会场系统拆解 OpenSearch 高性价比 GenAI 落地的 “三驾马车”：

*   稀疏检索再进化：为何“稀疏搜索”正在回归？如何做到 Inference-Free 极致推理，用 SparseANN 算法平衡性价比与速度？
    
*   GraphRAG 工程突围：“启发探索”为何是当下最优解？OpenSearch Graph RAG 工程化路径首次公开。
    
*   用 GenAI 回馈GenAI ：如何用 Claude Code 反哺性能优化，让 AI 帮 AI 系统跑得更快、更稳、更省？
    

这是一份 “如何用最少的 Token 办最多的事”的实战手册。

👉适合谁来听：企业级搜索/问答应用的架构师与算法工程师；苦于 RAG 推理成本高、延迟难降的技术决策者；探索 “用 AI 造 AI ” 性能优化新范式的开发者。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

从生成到可编辑：AI PPT 在线编辑系统的工程化实践

🔹演讲嘉宾：马林｜商汤科技 技术总监

🎯核心问题：当行业共识“ AI 生成必须可编辑”已成定局，如何让大模型输出的非结构化内容，变成用户手中可拖拽、可修改、可导出的真实 PPT ？这不仅是算法问题，更是系统工程难题。

💡为什么值得听：马林将深入讲解 AI PPT 的 “双路径架构”的优劣势对比与场景适配策略，以及如何构建 “生成—解析—结构化—渲染—导出”的完整闭环，确保多轮编辑一致性。

👉适合谁来听：AI 办公/创意工具的产品经理与前端/全栈工程师；关注非结构化内容结构化转换、渲染引擎设计的技术架构师；探索视觉生成与结构化生成如何取舍的 AI 产品决策者。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q3SAo8dibFicWvibnB5u4gdHLgb2AbA6UJ8VImKnyW92hibqZefwpDbPYAQ/640?wx_fmt=png)

破局大模型落地“不可能三角”：平安科技 AI 在复杂医疗场景下的多 Agent 实战

🔹演讲嘉宾：董扬｜平安科技创新事业部 算法研究资深专家

🎯核心问题：当通用大模型在娱乐办公场景百花齐放，专业医疗成了检验能力的“试金石”。诊疗场景有三大铁律：准确性不可妥协、知识壁垒极高、路径极度碎片化。

传统 RAG 只能解决“知识检索”，却无法触及 “经验推理”的深水区。当医生需要的不是“查资料”，而是 “像专家一样思考”，AI 该如何破局？

💡为什么值得听：董扬是横跨搜索、金融、医疗三大领域的算法“老兵”。拥有超过 15 年一线架构经验。会场，董扬将揭秘平安重疾 MDT 诊疗产品的三大核心算法引擎：“专家临床经验”内化学习引擎、推理进化与幻觉控制引擎、多智能体协同引擎。

👉适合谁来听：负责金融、医疗、工业控制等高风险、高合规领域 AI 落地的算法专家与技术管理者。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0jqAkXB2hEohZkOqYzXVsmJtnNRQncYh54ZQOpcC5ZGSicYFgtBZP4w/640?wx_fmt=png)

从 Dify 1.14 RC 到企业 AI 自动化：工作流与沙盒 OS Agent 的落地实践  

🔹演讲嘉宾：陈叶枫｜Dify 产品工程师

🎯核心问题：企业 AI 落地，为何 Chat 模式撑不住了？企业不仅要“能用”，更要“可控”：权限怎么隔离？失败怎么回退？过程怎么审计？ 这些治理问题，成了横亘在 PoC 与生产之间的“天堑”。

💡为什么值得听：陈叶帆将在现场系统拆解 Dify 1.14 RC 企业落地路径：

*   三驾马车的能力边界：厘清 Workflow、Skill、沙盒 OS Agent 的分工，告别“盲目全 Agent 化”。
    
*   可执行工作流实战：以真实场景为例，将文件处理、命令执行、多步骤编排连接成业务闭环。
    
*   PoC 到生产的治理设计：揭秘权限隔离、失败回退、人工介入、操作审计，让AI 通过企业“安检”。
    

👉适合谁来听：负责企业内部数字化转型、AI 自动化流程设计的 IT 负责人与架构师；正在探索 RPA 升级、数字员工、企业级 AI 应用的产品经理与开发者。

这或许能帮你省下数月的试错时间，以及百万级的沉没成本。
