# 神秘模型「Pony Alpha」引外网热议，它会是国产大模型中的谁？

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg)

机器之心编辑部

  

这两天，外网都在好奇：

  

全球模型服务平台 OpenRouter 上这个搜索第一的神秘模型是哪家的？

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGkXasbnSk5PyPUib8yQyiaBgRlcyZHv0qHHEibfHedOsXr57uthGI9ex3hoiavG6icV9y6szchukK5harTJEicbjCIQ6ia6DtYTbYHY8/640?wx_fmt=png&from=appmsg)

  

这个匿名模型叫做「Pony Alpha」。根据 OpenRouter 官方的说法，它是新一代的通用大模型，在编程、逻辑推理和角色扮演方面表现突出，并针对 Agent 工作流进行了优化，具有极高的工具调用准确率。

  

目前，该模型可以免费使用。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqF5Ce0ickRmZss13VtiarIC7TbsHTva9ibZX7Sp4uSsc7iawB1icw1VP8DkmLdXF1jcl2GgRDRnKSibJb82oEG5OErL4BXPgvRl6eDRo/640?wx_fmt=png&from=appmsg)

  

每每出现这种能力强大的匿名模型，网友们刨根问底的热情就会被瞬间点燃。

  

有人翻模型参数、有人对比输出风格、有人逐字分析回复，试图从一丁点蛛丝马迹中还原它背后的训练数据。

  

知名 X 博主「karminski - 牙医」猜测 Pony Alpha 是国产大模型，要么是 DeepSeek-V4，要么是 GLM 新模型。
