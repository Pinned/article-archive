**Marvis，马维斯？**

牛马团队
----

马维斯Marvis出厂就自带了一个牛马六人组。

1.  PM（项目经理）：听你的指令，然后拆活、派活。你一开口，它琢磨这事儿该谁干；
    
2.  File Agent（文件管家）：专门搞定搜文件、改格式、去重、合并。你电脑里的几十个表格，它能一口气分析完出报表；
    
3.  Computer Agent（系统运维）：改默认浏览器、查硬件、设置开机关机时间，这种藏在设置里的琐碎事儿它管。 ；
    
4.  APP Agent（应用外卖员）：联动全网各类常用软件与平台，可以帮你查个机票、电商比个价啥的；
    
5.  Search Agent（搜索专家）：负责网络搜索与信息聚合能快速从公开资源中定位答案并给出关键引用；
    
6.  Browser Agent（网页交互专家）：接管网页，进行网页交互与数据抓取；
    

有这套阵容，你说要啥，PM就会带着剩下几位团队伙伴并行跑活。

能管你电脑掉电快的毛病
-----------

日常使用电脑总会遇上各类棘手小问题，马维斯也能接深入系统排查根源问题。

要是你问“电脑掉电快怎么办”，其它人可能会甩你一堆链接让你看，或者给你口头排查一下。

但马维斯会直接去检测你的电源运行、看哪个高耗电进程在偷电，顺便还能帮你调整下，从系统层面做到省电。

或者，你还可以直接让它帮你给电脑体检。

隐私、效率双模式
--------

当然了，AI深度进驻系统，大家最顾虑的就是个人数据安全和隐私。

对于这个问题，Marvis可以设置两种运行模式，交由用户自主按需选择：

*   隐私模式  
    全部对话内容与相关数据均在设备本地完成运算处理，全程不上传云端，实现数据本地闭环处理，敏感内容选用这个模式，能最大程度保障信息不外流。
    
*   效率模式  
    依托云端大模型完成语义理解、任务规划等思考环节，实际操作执行依旧在本地设备完成。相当于思维依托全网算力升级，实际操作仍把控在本机，使用速度更快，整体功能也更全面。
    

![](https://mmbiz.qpic.cn/sz_mmbiz_png/A6fTew8FFGGD5cuumVXOtrm5jXXNWtnRvUzqiarwia4njiaRVLapeeRf0f4rDrU5JbO5giahHv7B5MibHmTKiaCwX9J7ibbibQk4jicOjQnHdRWXJfrE/640?wx_fmt=png&from=appmsg)
