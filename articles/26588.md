Anthropic这几天的代码里，也显示出重大信息。

  

6月24日，据多位博主爆料，在最新鲜出炉的 **Claude Code v2.1.190** 版本更新中，新增了这样一行极其醒目的字符串。

  

**「You've used your included Fable 5 usage for this week」 （您已用完本周包含的 Fable 5 使用额度）**

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYUXyIBfdhsar9tWeAQJza6QZtHsMS0DpZ629aRiaGy0ibgxGR1XmWnLFYoXgicq0mad7dZnMa7kic0nlzqIPKeEjXPxzV09qsXUns8/640?from=appmsg)

  

更关键的是，旧版本中那句 「purchased separately from your plan」（需单独于您的套餐另行购买）， 被官方彻底移除了。

  

这释放了一个极其强烈的信号，很有可能， Fable 5将作为「每周配额」，被永久整合进现有的 Claude 订阅服务中！

  

这对于广大 Pro 用户来说，这简直是特大喜讯。

  

更有意思的是，这场回归的灰度测试充满各种玄学。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYXj5a2icSiaHIEElzmoTJyq0sZbHwicLm1R2U1Al3BSFiaEvUTichrnhm1EnfQtc4fF4iaibFwicVsFjzE4dJUhV3lLXGchldMEDwgKKbw/640?from=appmsg)

  

比如，只有部分对话可用。

另外，还发生了一个小插曲。

  

X网友Chetaslua发现，Fable 5在 AWS（亚马逊云）平台上重新上线，然而，并不对所有人开放。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYW0KQxmG3YPVVOR4Hib13pSECngoRwE309YRbRib9MREcDFGsJ0gvm1uG2KWGKVTzSox7IrT3lpyDaAM6zkHwZNcWicOmRnC8JkibI/640?from=appmsg)

  

第一关，你必须有美国身份证，提交验证。这一次使用，是绝对的实名制。

  

第二关，就是提交使用场景，等待人工审核。通过了身份验证，你还需要向Anthropic提交一份具体的应用场景说明，由官方人工审核。

  

审核通过，才能进入白名单，然后才能获得实际的API调用权限。

如果消息属实，AI圈又将被彻底改变。

  

如果你是 Claude 的老用户，你一定知道 Anthropic 的产品线矩阵：

  

**Haiku（俳句）：** 小巧轻快，主打性价比。

**Sonnet（十四行诗）：** 中坚力量，速度与智能的完美平衡，也是大部分日常用户的首选。

**Opus（宏篇巨著）：** 算力吞金兽，反应慢但极其深刻，代表着模型能力的最高天花板。

  

目前我们正在使用的是 3.5 时代的模型，而传闻中的 **Sonnet 5（Fennec）**，跨越了整整一个半的代差！

  

更可怕的就是那句「性能接近 Opus 4.8」。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYU0Uic6C3v7qHGQib32sczYLf2msGdDKBwALr6IIMl9kGicT0s3UTXbNbAoUyAQPXwSYXFtb89slupuG9AtdFHrVn79X6UF3Jaars/640?from=appmsg)

来源：https://www.cometapi.com/claude-sonnet-5-spotted/

  

要知道，当前最强的 Opus 模型在逻辑推演、长文本理解和复杂代码重构上，依然是难以逾越的高山。

  

如果一个中杯的Sonnet 模型，都能达到Opus 4.8 的水平，这意味着「算力平权」将真正到来。

  

很有可能，我们能在几秒钟内，获得以前需要等待几十秒、花费高昂 API 成本才能换来的深度思考结果。

  

而Anthropic 官方的一个微小动作，更是印证了新风暴的降临：现有的 Claude Sonnet 每周使用限制，刚刚被官方静悄悄地取消了！ 

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Rvq8Ow69CYW0e4YdoibpsVfMB6voicBzMKpNqKQxiaicTwjkG0c875PmXORiaMcM5R3OES0ib7e9D7WxBic7o7shibsTHUBNEYz2NwHEAwYmbbeSSt0/640?from=appmsg)

  

在 AI 算力比黄金还贵的今天，解除旧版 Sonnet 的限制，只可能有两个原因：

  

第一，是清理库存算力，为更大规模的新模型腾出服务器空间。

  

第二，就是对旧模型的稳定性已经拥有百分之百的信心，彻底放开手脚，准备迎接新王的加冕。

**风暴降临前的寂静，**

**24小时后的「生死大考」**

  

为什么 Fable 5 的各种代码泄露、iOS 端灰度测试以及 AWS 上的实名白名单审核，全都如此密集地集中在这几天爆发？

  

可以说，这是一场踩在红线上的极限试探。

  

因为，悬在 Anthropic 和整个美国 AI 产业头顶的达摩克利斯之剑，马上就要落下了——6月26日的大限将至！

  

为什么「6月26日」这个时间节点至关重要？

  

原因就是，来自美国国会山的极限施压。

  

就在上周，由美国重量级议员组成的一个跨党派小组，罕见地联名向美国商务部长Howard Lutnick发出了一封措辞极严厉的质询信。

  

![](https://mmbiz.qpic.cn/mmbiz_png/Rvq8Ow69CYXgs9Q0WSsK8d56Pa2luIX47VCWZVkDFo1zHtcFLC00wCqicIj5OibtNDTNFI9ZNwMXRmEp406afAx3CFdMkfwZUcn04LXaUlClk/640?from=appmsg)

  

在这份签名阵容堪称豪华的信件中，国会大佬们抛出连环大拷问。

  

1.商务部究竟依赖哪些具体的评估标准，来决定是否修改此前的出口管制禁令，并最终恢复公众对该大模型的访问权限？

2.做出这项生死决定的具体时间表，到底是什么？

  

信件的末尾，议员们给商务部下达了最后通牒：必须在 6月26日 之前，给出明确答复！

  

现在，6月26日已经到了。这个由国会设定的Deadline已经接近，让我们等来了Fable 5的疯狂试探。

  

24小时内，我们还会等来更重磅的大解禁吗？

  

让我们屏息等待。
