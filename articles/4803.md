# 旧地基，在松动

过去两天，有两件没有关联的事撞进了同一周

**第一件发生在中东：圣战打响了**

2月28日，美国和以色列联合对伊朗发动军事打击，目标包括导弹基地、海军设施和核相关地点。伊朗最高领袖哈梅内伊在行动中遇难。哈梅内伊执掌伊朗35年，是伊朗政治和宗教体系的最高权威，凌驾于总统之上

他死后不到24小时，伊朗宗教界最资深的学者之一、99岁的大阿亚图拉设拉子公开发出圣战令。大阿亚图拉是什叶派伊斯兰最高等级的宗教称号，全球只有几十人拥有。设拉子称，为最高领袖复仇是「全球所有穆斯林的宗教义务」。这个表述没有截止日期，没有地理边界，没有组织要求

**第二件发生在网上：OpenClaw 超过了 Linux**

GitHub 是全球最大的开源代码托管平台，开发者把代码放在上面，其他人可以给项目点一个「star」，相当于收藏加认可。star 数越高，说明这个项目在全球开发者中越受关注

Linux 操作系统1991年发布，是全世界绝大多数服务器、安卓手机和超级计算机运行的底层系统，三十多年积累了 GitHub 上最具代表性的 star 数字之一

这周，一个叫 OpenClaw 的项目超过了它。OpenClaw 是奥地利开发者 Peter Steinberger 在2025年11月发布的个人项目，让 AI 可以通过 WhatsApp、Telegram 这类日常消息软件来操控你的电脑。从发布到超过 Linux，不到四个月

  

旧系统以为自己还稳定
----------

伊朗谈判团队从2025年4月开始工作。在阿曼、罗马、马斯喀特，他们和美方代表一轮接一轮地谈，内容是核计划的限制方案

这和过去几十年里每一次核谈判的逻辑一样，多谈几轮，总有空间。这个判断在历史上每次都成立。直到2026年2月28日，它不再成立了

Steinberger 发布 OpenClaw 的时候，GitHub 上的排名格局已经稳定了很多年。Linux 在那里，React 在那里，各个项目按照各自积累的速度缓慢移动

两个系统在触发之前，都在正常运行

**稳定的前提，是从未遇到让它不稳定的那个条件**

没人知道触发条件在哪里
-----------

圣战令在历史上被发出过很多次

1914年，奥斯曼帝国向全球穆斯林发出圣战令，世界的走向没有因此改变。1998年，本·拉登发出圣战令，对象是美国人，措辞是「在任何能执行的地方执行」。三年后，发生了9·11

两次圣战令，形式上没有本质区别。一次世界照常运转，一次改变了世界。在1998年那一刻，没有任何人能从那份文件里读出它三年后会触发什么

**触发条件在哪里，没人知道。这本身就是它最危险的地方**

  

常识是有保质期的
--------

「大型开源项目需要多年集体积累」，这条规律从 Linux 1991年发布起成立，在 OpenClaw 出现之后开始松动

「打击一国最高领袖在政治上属于禁区」，在2月28日之后被重新定价。事后各方的反应证明，这件事发生了，世界还在运转，这本身就是新的参照点

这两条规律在失效之前，没有预警信号。某一天，它们不再成立了

**常识是有保质期的。保质期在哪里，没有标注**

有些门，推开就回不去了
-----------

这次圣战令的措辞是「直到这些罪犯的邪恶从世界上消除」。没有完成条件，没有撤回机制，没有过期日期

OpenClaw 在 Steinberger 加入 OpenAI 之后，被移交给独立基金会运营，代码已经被全球开发者 fork 了数万次。没有任何主体可以关掉它

哈梅内伊死后，伊朗进入权威真空期。谈判桌上某些在他在世时尚存的选项，从那一刻起就不在了

**有些门，推开就回不去了**

百年一遇的事，开始一件件发生
--------------

今年1月，世界经济论坛发布了2026年全球风险报告。68% 的受访专家预期未来十年世界进入「多极或碎片化秩序」，地缘经济对抗是年度头号风险。投资机构 Wellington Management 的判断是：地缘政治周期历史上持续80到100年，我们正在经历的结构性变化，每个世纪只出现一次

这份报告，在2月28日之前发布，它没有把这一周纳入计算

Linux 花三十年积累的数字，四个月被超越。美以联合打击一国最高领袖。圣战令向全球穆斯林发出

这些发生在同一周

**百年一遇的事，开始一件件发生**

我们判断「正常」和「异常」的基准线，建立在二战之后一段异常稳定的历史时期上。如果这个基准本身在失效，那我们对风险的所有直觉，都需要重新校准

* * *

战争还在继续，圣战令在传播  
OpenClaw 还在被各地开发者 fork

唯一确定的是：我们会遇到有更多的不确定
