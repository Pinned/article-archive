不同于以往单 Agent 的玩具演示，这次我们将向你展示：如何在一个终端型环境（OpenCode）中，通过职责分离（Separation of Concerns），让多个专用智能体分别负责游戏架构、视觉 UI 和交互逻辑的交付，实现真正意义上的“分层构建”。

你将获得以下核心技术能力的实操复刻：

*   Agentic 开发范式：学会用多个专职 Agent 替代单次长对话，构建高可靠的模块化代码交付链；
    
*   全栈本地链路：基于 Lemonade Server，依托 AMD Ryzen AI Max+ 395 (Strix Halo) 的端侧算力，完全脱离云端 API，实现代码生成与模型推理的 100% 本地闭环；
    
*   端侧落地范例：从视觉处理到交互逻辑，我们将现场演示如何利用端侧算力完成一款太空射击游戏的构建，所有推理调用均发生在你的本机。
