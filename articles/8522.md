# 万人AIFUT大会免费参会指南来了，我们，线下见。

先跟大家们汇报一下。

上周我预告了我们的[AIFUT大会](https://mp.weixin.qq.com/s?__biz=MzIyMzA5NjEyMA==&mid=2647680939&idx=1&sn=eb6d0cd640399b39908d4e89ea6cbe75&scene=21#wechat_redirect)，现在，还有最后一周时间就要开始了，我现在的心情其实慌的一笔。

上周一晚上门票开售，开售之后，虽然当时售票系统崩得很彻底（这个事情我已经在[致歉声明](https://mp.weixin.qq.com/s?__biz=MzIyMzA5NjEyMA==&mid=2647680941&idx=1&sn=3abf3c0b25f50655eb4917951eb92fea&scene=21#wechat_redirect)里跟大家详细说过了），但好在1天之内，我们将近1000张主论坛的双日票，还是全部都卖光了。

![](https://mmbiz.qpic.cn/mmbiz_jpg/2jjfQoZLoqUMzToYNZJQlS74xrGKzkv7rAKzlmac7gHHROc9TCwCJnaSYVkfYvmBHxZtg9XNPdQEeZVHTAUnPTOdye9WyLicbzVLRqb7lrZw/640?wx_fmt=jpeg)

现在还剩了100多张第2日的单日票，第二天其实偏影视和创作一点，你不仅可以看到赛博魅魔郭帆导演录制的VCR，还有《流浪地球》的主创沈今晶和演员王智老师来一起聊电影的创作，还有刚刚从国外回来就被我化缘喊来的郑林老师（就是神文《未来1500天，影视行业的钱会被这1%的人赚走？》的作者），还有我们的5v5 Prompt Battle AI视频极限电竞赛，还有等等等等，应该还是有不少干货可以学到一些东西的～

推荐大家还是可以来玩一玩的![](https://res.wx.qq.com/t/wx_fed/we-emoji/res/assets/newemoji/Yellowdog.png)

也谢谢大家的支持，真的。

然后有一个很重要的信息，可能还有很多朋友不知道。

就是我们的AIFUT大会，其实是主论坛+巨型户外展区空间组成的。

我们的展区空间，其实是完全免费对所有人开放的。

不要钱，不用报名，不用抢票，直接就可以来。

![](https://mmbiz.qpic.cn/mmbiz_png/2jjfQoZLoqV5FCs0wWicQUNln8Fs58uQrbxretgoJrTVqJcGXMEza7x35JiaJQ6Fgmdxv7rzqESwWKptnHoMZiaibGSQAykaTqIibI4qUnH5jic3c/640?wx_fmt=png&from=appmsg)

3500平方米，34个展位，从具身智能机器人到AI大模型应用到智能硬件，全都可以上手体验，还有游戏可以玩。

我坚持要搞整个线下展区呢，其实也有一个我自己很私人的原因。

就是我真的觉得AI时代有一个特别矛盾的地方，我们每天花大量的时间跟AI对话，跟屏幕对话，效率越来越高，信息越来越多，但是人跟人之间面对面的交流，反而越来越少了。

我自己就是这样，我每天跟Claude聊的时间，可能比我跟所有同事加起来聊的都多。

所以我在设计这个展区的时候，有一个很执念的想法，我想让它不只是一个你来看看展就走的地方，我想让它变成一个你愿意待着、愿意坐下来、愿意跟身边的陌生人聊两句的地方。

一个AI味很浓，但是人味更浓的地方。

所以我才做了整个的游戏互动和虚拟代币FUT币的机制（后面会详细说），还有很多的休息区，就是希望大家逛累了能坐下来，跟旁边的人聊聊天，聊聊你在做什么、你在用什么AI工具、你对这个行业怎么看。

我们还有免费的酒可以喝（此处感谢AGI Bar的赞助），在展区里逛着逛着来一杯，跟刚认识的朋友碰个杯，这个画面我想想就觉得还挺美好的。

然后别忘了，这是一个电竞场馆。

展区外面有一块曾经用来看电竞比赛的巨型大屏，两天活动期间，这块屏会全程直播主论坛的内容，你可以一群人一起坐在外面，看着大屏幕上的分享，讨论几句，再去逛逛展台，再回来继续看。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/2jjfQoZLoqWH9aqSMnicaVEjLmJzfPuVRafPcb1GZMHpPzGeicC9EsmUx7hFzDmGGtNf5zMtmQyoJjORSKMDiag7hI60LMqOLKicQ6HZCTPOicUc/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

这种体验，我自己还是很喜欢也很珍惜的。

整个展区，每天从早8点开到晚9点，整整13个小时，随来随走。

所以今天给大家写一篇稍微详细一点的我们的AIFUT游玩攻略，真的很希望你能带你的朋友，一起来玩！

  

**时间和地点**

4月8号和9号，周三和周四，两天。

地点在北京亦庄，**北京智慧电竞赛事中心**。

《英雄联盟》《王者荣耀》《无畏契约》等等的比赛，都在这里打过。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/2jjfQoZLoqVKxL2tPtLvz7WJEHEPB6cbicUr9mueSia8YsvicLeOx7OrxawuGyXyOK9IrfGSqRIJ7l8qAvAFktqd1AOCHy71QvxgbwcBXUZa4M/640?wx_fmt=jpeg&from=appmsg)

就是之前Faker和TheShy打表演赛的那个电竞场馆。

**主论坛**需要凭票入场，活动时间是每天9:00到18:00。

**展区**完全免费，活动时间是每天8:00到21:00，比主论坛多开4个小时，晚上也能来逛。

  

**交通指南**

4月8号和9号，周三和周四，两天。

### 地铁（最推荐）：

### 坐**地铁17号线**，到**北神树站**，从**C口**出站。

出站之后直线步行大概8分钟，700米左右，就到了。

![](https://mmbiz.qpic.cn/mmbiz_png/2jjfQoZLoqUOJqlPl9g4ZXhng02vNHM4du7fqib34ZsF2BOSKEsLiaxPEXBcvSTjoqaFjEgIGxicVqQ7baNfB3KAAibXZP2DkqfJyEQaUibugvAA/640?wx_fmt=png&from=appmsg)

17号线是一条南北走向的线，北边可以从望京东、太阳宫、东大桥这些站换乘，南边直接通到亦庄，如果你在朝阳、东城方向过来，17号线是最方便的选择。

### 自驾：

### 导航搜「北京智慧电竞赛事中心-西门」就行，场馆周边有停车位。

亦庄规划做的比较好，基本不咋堵车。不过如果你是从城里过来，路上还是要预留一些时间。

  

**签到流程**

到了现场之后，签到流程是这样的。

**主论坛用户**，需要带身份证原件（港澳台通行证或护照也可以），到签到处扫AIFUT小程序里的票券二维码，核验身份之后领取实体入场凭证，然后对号入座。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/2jjfQoZLoqWjyQdjJZpQNHY6p5OAdN8BTfBZ8XoUq2gXfibIeoNdy5ux2gz641m8yWpkqGM4wNHdLGG5hkFq94u0Xx48bahgoBjmepcI9qHY/640?wx_fmt=jpeg)

签到时间是每天早上8:00到9:00，建议早点来。

如果中途需要出去玩也没关系，凭小程序二维码可以再次入场，座位都会保留。

**展区用户**，不需要提前买票，可以直接来。到了现场直接进展区就行，在小程序上完成现场签到还能领FUT币。

  

**FUT币，展区的游戏币系统**

这个是我觉得我们展区最有意思的设计。

我们在展区里做了一套虚拟代币系统，叫FUT币，你可以把它理解成游乐场的游戏币，或者说游戏里的金币。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/2jjfQoZLoqXfGIQpnLS4iclqjohAz5zTAybLJyHU1r14ib2S2Aa1ziat4XDQHPtZTjPiaTM2ewgetvdrREicTLoL1qUnjMeuJnBc9BHRUFiahEfko/640?wx_fmt=jpeg)

拿着FUT币，你可以在各个展台体验独特的项目、兑换品牌方准备的礼品、参与一些特色互动。

还有我们自己准备的，很多实体礼品兑换。

甚至，还有清闲的人体工学椅、还有AI眼镜可以兑换。。。嗯。。。

我们真的大出血了。

那怎么获取FUT币呢，通过做一些小任务就行。

**现场签到**，每天在小程序完成展区签到，5个FUT币。

**集章打卡**，去指定的展位完成展位任务并交付，然后完成集章任务，每天20个FUT币，这个其实挺好玩的，相当于给你一个理由把每个展台都逛一遍来集章。

**AI问答挑战，在展区中心舞台参与AI知识问答，每天10个FUT币。**

**展台任务，每个展位都有自己的专属任务，完成了也能拿FUT币，具体数量以现场公示为准。**

**34个展位，基本上涵盖了所有的门类，有做AI音乐的、AI办公的、AI视频的、AI 3D的、Agent、大模型、具身智能、AI硬件，还有社区和媒体，比如观猹、差评X.PIN等等。**

**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/2jjfQoZLoqXC0n2Pqa6U2uXNeO0kO48ibEBhzMiaiaib0q7GRxYNegAyHQrd5Z4Nq48uyM0ozEsSTsJmtf7Ab6v0V3pjupLghanuKcYovu8AAyw/640?wx_fmt=jpeg&from=appmsg)**

**基本上覆盖了AI行业大多数的方向。**

**如果你是主论坛的付费用户，你的初始FUT币会比普通观众多很多，算是给买了票的朋友们的一个额外福利。**

FUT币不可充值、不可提现、不可转账、不可交易，没有任何现金价值。

就是单纯给你在线下展区玩的，别想多了哈哈哈。

4月9号晚上9点活动结束之后自动清零，所以来了就尽情花，别攒着。

挣币和花币的时候也很简单，在展位面前，向展位的工作人员出示你的AIFUT小程序里面的专属交易码，工作人员扫码就会进行增减。

  

**吃饭和住宿，亦庄给的福利**

这个要重点说一下，因为经开区的领导们真的给了我们很大的支持。

现场观众可以在活动指定点位领取一张**大会纪念票根**，凭这张票根，可以享受北京亦庄的专属礼遇。

![](https://mmbiz.qpic.cn/mmbiz_png/2jjfQoZLoqWm5X18mluUmGnWF14icBj5a84meCHsuPnBuH8ZGa1nExKv7uuibpmvqBMEr9EwZyWVExHkAz4gJO8167wkcqwEeuJBaLBFlJv3k/640?wx_fmt=png&from=appmsg)

**就餐礼遇，凭纪念票根可以乘坐现场的****摆渡巴士**，免费接送你到龙玥城、大族广场、华联亦庄购物中心、开饭群岛这几个地方的指定餐厅，享受优惠折扣或者免费领取特色餐品。具体合作商户可以到时候扫描票根上的二维码查询。

你没看错，有摆渡巴士，不用自己走过去，直接坐车给你送到饭店门口，而且餐饮吃饭全部打折。

**住宿礼遇，如果你是从外地来的，凭纪念票根还可以享受北京亦庄合作酒店的专属优惠。**

这波，我只能说，经开区的领导，真的太给力了。

哦对了，另外提一嘴，场馆里面禁止吃螺蛳粉、臭豆腐等这些味道比较浓烈的食物。

我知道写出来有点好笑，但真的是有必要的规定哈哈哈哈哈。

3500平方米的空间，来一碗螺蛳粉的话...那画面我不敢想。

  

**关于天气**

4月8号和9号我目前看天气预报，应该都是晴天，气温大概在10到20度之间，白天还行，早晚会有点凉。

如果你是来待一整天的话，建议带件外套，特别是晚上，可能会凉一些。
