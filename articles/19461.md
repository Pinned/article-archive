微信状态可显示访客记录？腾讯客服回应
==================

5月11日消息，今日，微信状态访客记录的话题冲上热搜，引发网友热议。有网友发帖询问：“微信状态为什么可以看访客了？”该网友称，这一功能目前应该处于内测阶段，浏览人数大概率显示的是“看了你状态的人”，而不是点击次数。点击后出现的头像，既包括浏览过你状态的人，也包括同样发布状态的人。此外，该网友认为，历史微信状态的浏览数据准确性有限，参考价值不高。

针对网友疑问，有记者咨询腾讯客服。客服回应称：在微信状态有效期内，用户可以在状态右下角查看浏览人数。如果当前浏览者也设置了状态，轻点头像即可查看对方当前的状态。不过，该功能目前仅支持在iOS客户端进行灰度测试。腾讯客服表示：“若您未看到相关功能，还请耐心等待后续版本更新。”（北京日报）

Linux 内核首个 AI 生成代码驱动诞生！OpenAI 帮 AMD 写了个温度驱动

5月11日消息，**Linux内核迎来首个明确标注 AI 生成代码的驱动程序，这款名为prom21-xhci的温度驱动已提交至内核邮件列表审核，部分核心代码由OpenAI的编码代理Codex GPT-5.5生成。**

**该驱动专门用于暴露AMD Promontory 21芯片组xHCI控制器上的温度传感器，可间接反映芯片组本身的工作温度。Promontory 21架构被AMD 600和800系列AM5芯片组采用，覆盖主流及高端X670E主板。**

驱动由开源开发者Jihong Min主导开发，补丁说明中明确标注了AI生成代码的部分。用户可通过内核Kconfig文件中的SENSORS\_PROM21\_XHCI选项启用该功能。驱动会将温度数据暴露给Linux硬件监控（HWMON）子系统，与所有现有的Linux用户空间硬件监控工具完全兼容，无需额外安装专用软件。（快科技）

===============================

**加密消息会显示小锁图标，用户可以直观判断通信是否安全，不过该功能仍依赖运营商支持，目前尚在逐步推广阶段。**在地图应用方面，苹果引入了“推荐地点”功能，系统会根据用户所在位置和近期搜索记录，智能推荐可能感兴趣的地点。此外，苹果还推出了全新的“Pride Luminance”动态壁纸，与Apple Watch表盘和表带形成呼应。****

**除了新功能之外，苹果还在iOS 26.5和iPadOS 26.5的更新中修复了50多个安全漏洞，**推荐所有能升级的用户尽快更新，其中存在多个图片漏洞、内核问题以及与快捷方式、聚焦和截图相关的漏洞。

资源配套方面，MiniMax 将为合作人员提供**完整的多模态模型能力、研发环境以及无限的 Token**。合作模式具备较高灵活性，开放全职入职、Fellowship 短期协作两种形式。激励机制上，该计划支持联合署名、学术成果共享，可推进机构联合研究；同时设置国际化竞争力薪酬与激励体系，参与 Fellowship 项目的人员亦可**获得股票激励**，共享企业长期发展收益。（IT 之家）

字节火山引擎 Agent Plan 发布：业界首个“Agent 套餐包”，每月 40 元起
=============================================

5 月 11 日消息，火山引擎正式发布 Agent Plan。相比 Coding Plan 提供主力编程模型的订阅服务，Agent Plan 在此基础上新增更多模态的模型和 Harness 工具，官方称其为“业界首个‘Agent 套餐包’”。

据官方介绍，Agent Plan 包含 Doubao-Seed、Doubao-Seedance、Doubao-Seedream 等字节跳动系列 SOTA 模型，GLM-5.1、Kimi-K2.6 等在内的主流三方模型，以及联网搜索等 Harness 工具，满足 Agent 场景更全面的需求。未来，Agent Plan 将持续更新超全模态模型、持续升级 Harness。  

相较于传统的模型订阅或算力套餐，**Agent Plan 首次将 Model 与 Harness 能力深度整合**，适配 Claude Code、OpenCode、TRAE、OpenClaw、Hermes Agent 等编程和 Agent 平台。完成 Agent Plan 的初始配置后，平台即可自动调用。当前，Agent Plan 包含四种阶梯订阅组合，分别为 **40 元 / 月、200 元 / 月、500 元 / 月、1000 元 / 月**。（IT 之家）
