微信 AI 搜索 Agent 实战

🔹 演讲嘉宾：王炳宁 ｜ 腾讯微信搜索 AI 算法研究方向负责人，专家研究员

🤔为什么值得听：中科院自动化所博士，主导并发布如 ReCO、ComQA、ChiQA、T2Ranking 等大规模中文问答数据，以及 Baichuan 系列预训练模型。主导发布模型在 GitHub 斩获万星，HuggingFace 下载超 2000 万次。

🎯 核心问题：实验室里跑得好好的 Agent，一到线上就频发幻觉、死循环，怎么破？

💡关键价值：王炳宁将深度拆解微信 AI 搜索 Agent 的架构设计，分享在超大规模并发下，如何保障 Agent 工作流的强韧性与可控性。

“在亿级流量面前，Agent 的可靠性比单纯的‘聪明’更重要。”

👉 适合谁来听：负责搜索/推荐/对话系统，正在探索 Agent 落地，且对系统稳定性有极高要求的技术人员。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06QDyP5HTUOXsPlJWd79yygiasaqXicpN7ibIfqiak5WFpaxE1mGxxpfmMjiaA/640?wx_fmt=png)

迈向数字生命：Computer Use Agent (EvoCUA) 的核心技术与自进化实践

### 🔹 演讲嘉宾：薛涛锋 ｜ 美团 Staff Researcher，EvoCUA 项目主导者

### 🤔为什么值得听：他主导的 EvoCUA 项目获得图灵奖得主、"AI教父" Yoshua Bengio 与 UC Berkeley 顶尖安全学者 Dawn Song 在论文中联合点赞：  

> 在针对全球 7 个前沿 CUA 的安全与鲁棒性评测里，EvoCUA-32B 以 35.0% 的最低受干扰率排名第一，展现出强大的抗扰动鲁棒性，正面击败了 Claude 4.5 Sonnet (41.0%) 和 OpenAI 的 Operator (48.7%)。

该项目更是以 56.7% 的成功率在 2026 年 1 月登顶 OSWorld 开源榜首，技术报告登顶 HuggingFace 榜首。

🎯 核心问题：怎么让 Agent 像人一样，自主完成跨应用、长链路的计算机操作（Computer Use），真正成为“数字员工”？

💡关键价值：薛涛锋将现场拆解 EvoCUA 的核心技术与自进化实践。从可验证的数据合成，到分钟级拉起十万级沙盒基建，再到让模型“从失败中进化”的经验学习机制。这不仅仅是一个前沿探索，更是如何攻克“长链路任务极易中断”的工程作业书。  

👉 适合谁来听： 正在探索“数字员工”、“RPA 升级”、“自动化办公”场景，急需突破复杂长链路任务执行瓶颈的 AI 产品/技术落地人员。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q0QtgZAx8xsoTReptvArfwbn9MvHGVfV98Qkl5PMRS6MCt2ljwLJwoQ/640?wx_fmt=png)

用强化学习让 Agent 越学越聪明

🔹马骁腾 ｜ Macaron AI 首席科学家，Mind Lab Director

🤔为什么值得听：清华大学自动化系博士后，在强化学习领域发表 30 余篇论文。他带领团队研发了面向万亿参数模型的 LoRA-RL 训练底座 MinT，把 GPU 成本打到了传统方式的 10%。

🎯 核心问题：每次都要靠人工编排冗长的 Prompt，Agent 无法从真实的交互中积累经验，怎么办？

💡关键价值：马骁腾将带来从 Context Engineering 到 Context Learning 的范式演进。他会结合 Macaron 模型的真实案例，教你如何用极低的成本，让 Agent 在动态环境（如复杂 UI 交互）中自我学习、沉淀经验。这套方案能大幅降低你后续的运维和调优成本。

👉 适合谁来听： 负责 Agent 后训练/持续优化，希望降低模型迭代成本、提升智能体自适应能力的算法与工程人员。

![](https://mmbiz.qpic.cn/mmbiz_png/Pn4Sm0RsAujX5kS5KQ6BaBUsy1RqR06Q3SAo8dibFicWvibnB5u4gdHLgb2AbA6UJ8VImKnyW92hibqZefwpDbPYAQ/640?wx_fmt=png)

从“作坊式”到“工业化”：建立可靠的 Agent 设计模式

🔹演讲嘉宾：黄佳 ｜ 新加坡科技研究局（A\*STAR）AI 研究员，《Agent 设计模式》作者

🤔为什么值得听：致力于将复杂的 AI 能力转化为可设计、可复用的系统架构，提出了覆盖六大认知维度的 21 种 Agent 设计模式。

🎯 核心问题：团队开发 Agent 犹如盲人摸象，充斥着“万能 Prompt”、“无记忆”、“无护栏”等反模式，导致项目极难维护和扩展。

💡关键价值：黄佳将基于 Claude Code 和 OpenClaw 的双案例拆解，为你提供一套经过验证的 Agent 架构设计框架。这套从感知、记忆到推理、行动的设计模式，能帮助你的团队建立标准化的开发流程，彻底告别“面条式”的作坊代码，提升交付效率与系统可靠性。

👉 适合谁来听： 负责 Agent 系统架构设计，希望建立标准化开发规范、提升团队工程交付质量的一线技术实践者与管理人员。

稳定性 × 可进化 × 可复用，这是 2026 智能体下半场的核心命题。

这场「Agent 系统与工程」专题的价值，不在于听几项炫酷的新技术，而在于带走一套经过头部大厂验证、可直接复用的决策框架与避坑清单。
