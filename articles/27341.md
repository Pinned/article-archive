现在能用
----

今天开始，Sonnet 5 今天起全平台能用，免费版和 Pro 版直接换成默认模型，Claude Code 和 API 里用 `claude-sonnet-5` 这个名字

上下文窗口上，还是 100 万，价格和上一代 Sonnet 保持不变，每百万输入 token 2 美元、输出 10 美元

  

听着是个促销价，但这里有个不太起眼的细节，Sonnet 5 换了新的分词器，同样一段内容切出来的 token 数比以前多，官方给的区间是 1 到 1.35 倍，看内容类型。促销价定在这个位置，官方自己的说法是，让这次切换「大致成本中性」（等于先承认了一句，**新模型嚼字更细，账不能只看单价**）

  

  

注意！这个模型其实不便宜
------------

这里得重点说一下  

*   Sonnet 5 换了新的分词器，同样一段内容切出来的 token 数比以前多，同样文本需要原本 1 到 1.35 倍的 token  
    
*   8 月 31 日之后，模型价格会涨价 50%，到 3 美元和 15 美元  
    
*   Artificial Analysis 做了一个测试，按满血 effort 跑，Sonnet 5 单个任务用的输出 token，比上一代多出四成左右
    

  

换句话说：**Sonnet 5 一次任务实际花的钱，可能比 Opus 4.8 还高**
