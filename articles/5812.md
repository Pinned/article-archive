# Agent，能自动做生意、赚真钱了

PRODUCT

我的第一桶金是大学之前赚的，靠的淘宝与亚马逊；今天，这套东西已经 Agent 化了

  

今天，阿里上线了 **Accio Work，帮人自动做生意的 Agent，一个给海外市场用的企业级产品**

**网址在这里： http://www.accio.com/work**

你给它一句话，比如「我想在美国市场卖极客文化周边」，它会帮你做市场分析、选品定价、生成商品主图、写详情页、在当地的电商平台（比如 Shopify）上把店开出来，然后接管你的 Instagram 账号去发帖推广，再用你的邮箱去跟供应商砍价、谈账期、沟通物流

Accio Work 切的是企业经营，尤其是做生意这个方向，面向全球中小企业和创业者，目前已经开放使用

我拿到了一个测试用的 Shopify 空店铺，试着让它帮我上几个商品、装修一下店面。在对话框里说了一句话，过了一会儿刷新页面，东西就在了

它能干的事
-----

开店和上商品只是其中一个环节。Accio Work 想做的事情，是帮那些没啥电商经营能力的老外，把全球生意的全链路跑通：从最开始的市场调研，到选品、定价、找供应商、谈判、采购，再到开店、装修、上架、社媒推广、投流获客

一句 Prompt市场分析 · 选品定价开店装修 · 商品上架供应商寻源 · 砍价谈判社媒推广 · 投流获客$ 赚到钱

这样之后，未来当老外都做起了全球生意，中国商家的海外需求将会爆炸式增长  

而为了在海外跑通这条链路，Accio Work 内置了一套技能库。电商与供应链方向 **19** 个 skill，包括供应商寻源、市场洞察选品、1688 选品助手、1688 以图搜货、关税与 HS 编码查询、Shopify 建站助手、跨境选品、亚马逊热搜词、亚马逊商品优化等

营销与内容创作方向 **20** 个，包括社媒发布、爆款内容拆解、电商营销、竞争分析、内容策略、商品描述生成、销售谈判等

除此之外，还有大量的文档处理、开发工具等各种常用技能

应用授权页面

定时任务方面，竞品价格监控、热销品趋势扫描、汇率追踪、市场快讯、每周复盘，这些都可以设成自动跑，不用你每天手动去看

内测阶段有个用户让 Agent 帮他砍价，目标价 **30 CNY**。Agent 跟供应商来回发了 6 封邮件，最终 **38 CNY 成交**。整个过程自动执行，设好目标价和定时任务之后基本可以托管，到关键决策节点才通知用户介入。Agent 在谈判过程中还会展示采购策略和实时进展

另一个用户拿了 **68** 位供应商的报价让 Agent 分析，逐一对比各工作表的明细项和价格，选出 top 10，同时检查那些只报了少量项目但单价明显更低的供应商，判断是否值得为了价格优势牺牲集中采购的便利性。分析完之后没有停，接入 Gmail 自动发询盘、砍价、沟通交易细节

用户也可以自己创建 skill，把自己擅长的操作流程封装进去，分享给其他人用。这样平台上的能力库会随着用户使用不断扩充。Agent 之间可以组团队、分工协作。还有一个「自我进化」的 skill，能根据生意进展不断迭代自己的商业判断

不用部署，下载直接用。对本地文件的操作需要用户授权

  

为什么是阿里
------

写代码的 Agent 错了可以重跑。做 PPT 的 Agent 丑了可以重改。但做生意的 Agent，选品选错了你会压货，谈价谈高了利润没了，给供应商发了一封不合适的邮件合作可能直接断。帮你在社媒上投了一波不精准的广告，预算烧完了没有转化

Agent 帮你选品选错了，压的是真货，亏的是真钱

做生意的 Agent，对「准」的要求比做办公的 Agent 高很多。你需要足够扎实的数据来降低幻觉率，需要真实的供应商库和交易网络来兜底。这件事到最后拼的，是 Agent 背后有没有足够扎实的产业基础设施

Accio Work Agent调用阿里国际站 26 年积累10 亿+ 商品数据5000 万+ 企业数据实时海关 / 汇率物流 / 金融履约OpenClaw开放生态Claude Cowork安全易用不同方向，不同壁垒Accio Work 的壁垒：产业基础设施

Accio Work 底下是阿里国际站 **26 年** 做外贸的积累。**10 亿** 量级的商品数据，**5000 万** 以上的全球企业运营数据，实时海关和汇率数据，都拿来训练模型。物流履约和金融交易也跑在阿里生态内的成熟能力上

OpenClaw 不知道从义乌发一个集装箱到汉堡应该走哪条线路。Claude Cowork 不知道一个杯子工厂的最低起订量和账期。这些不是公开数据，也不是模型能力能补的。是二十多年一单一单做出来的

二十多年的脏活累活，在 Agent 时代变成了最难复制的东西

Accio Work 跟 OpenClaw、Claude Cowork 走的是不同方向。OpenClaw 走开放生态，Claude Cowork 走安全易用，Accio Work 走了一条更重的路：把产业基础设施封装成 Agent 的能力层

但仅仅这样就完了吗？并不...因为你甚至还可以用其他的 IM/Claw，来使用这个 Agent，比如微信 ClawBot

一人跨国公司
------

过去做跨境电商，你至少需要一个选品的人、一个跟供应商谈的人、一个运营的人、一个搞推广的人、一个管物流的人

传统跨境团队选品采购运营推广物流5 个人，各管一摊↓一人 + Agent 团队你A1A2A3A4A5Agent 负责执行1 个人做判断

这些人做的事，大部分不需要特别高的创造力。选品方法论网上都有，运营 SOP 同行都在用，砍价话术供应商听了几百遍。你需要这些人，是因为你需要有人去执行那些繁琐但必须做的事情

如果 Agent 能接管执行这一层，那**一个人加一组 Agent**就能跑一个完整的跨境生意

公司的最小单元，正在从一个团队变成一个人加一组 Agent

我读大学之前，就在淘宝&亚马逊上开店，两边互相导货，自己赚学费。那时候赚的是信息差的钱，你知道哪个供应商便宜，你知道美国那边什么好卖，别人不知道，你就能赚到

那时候最大的成本是自己。盯每一单，回每一封邮件，操心每一个物流节点。今天信息差已经被填得差不多了，供应商信息在阿里巴巴国际站上一搜一大把，市场趋势有各种工具在追踪，物流价格透明到几毛钱的差异都能比出来。剩下的瓶颈是执行力，你知道该做什么，但一个人干不过来

Accio Work 在试着补上这一块，目前已经面向海外市场全面开放使用。中国商家，或许能看到海外需求的膨胀了

我用上这个产品的时候，还是在测试阶段，跟外部电商平台的 API 没有完全直连，所以用了手动注册获取 API 的方式给Accio去操作，在真实场景中自动化能做到什么程度还需要更多验证。但这个方向上，阿里大概是第一个认真在产品层面动手的
