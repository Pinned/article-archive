传字节跳动剪映产品负责人张琪智已离职创业

报道称，字节跳动旗下剪辑工具剪映产品负责人张琪智已于近期正式离职，开启个人创业生涯。张琪智已于 4 月中旬在深圳全资注册成立新公司，且已完成相关职务交接。截至发稿，字节跳动方面尚未对该事作出回应。据介绍，张琪智为原剪映职级 L7（对应旧职级 4-2）的负责人，此前她已卸任脸萌技术（深圳）有限公司法定代表人一职，该职位于 3 月 27 日由杜稳接任。（新浪财经）

腾讯客服回应微信同时打开两张图闪退：系内存压力导致，建议清理缓存

近日，有用户反馈同时点开两张图片时微信会闪退、强制退出。腾讯客服回应，该问题由同时加载大量图片导致内存压力引发，属适配层面逻辑问题，非设计缺陷。实测显示该漏洞几乎仅影响安卓设备，iOS 系统不受波及。业内人士分析，安卓端因未做好图片层级屏蔽判断，二次操作时触发程序崩溃。腾讯客服给出定期清理缓存、更新版本等四点解决方案，提醒用户避免过快双击图片，微信团队将持续优化系统适配性。（快科技）  

阿里云日均 Token 收入较 4 月初已增长五倍

截至 5 月 13 日，阿里云日均 Token 收入相比 4 月初已经增长超过五倍。月收入目前已经达到数亿元级别。阿里云今年 4 月初设定了一个短期 Token 收入增长目标。具体要求是，到 5 月 15 日前，日均 Token 收入较 4 月初要增长 5 倍。也就是说，阿里云要在两个月内完成目标。目前，这个目标已经达成。（财经）

**国际要闻**

Anthropic 高管：公司 90% 代码已由 AI 完成

Anthropic 首席财务官表示，AI 已编写公司 90% 的代码，使白领工作重心从执行转向监督。这些言论透露出这家硅谷顶级 AI 公司对未来白领工作的看法：AI 未必会直接消灭大量岗位，但会自动化掉员工日常工作中的大部分执行任务，使员工逐渐变成 AI 系统的“监督者”。（财联社）

英伟达 CEO 黄仁勋称五年前的 GPU 就像「老酒」：越陈越香，还会涨价

在 AI 需求进入爆发周期的背景下，GPU 成为最紧缺的算力资源之一，甚至五年前的旧芯片也开始持续涨价。据外媒 Wccftech 报道，英伟达首席执行官黄仁勋将这一现象称为“好酒效应（Fine Wine）”。该说法含义已从形容 GPU 通过驱动优化提升性能变为描述其价格随时间上涨。AI 需求推动 GPU 市场爆发式增长，全球运行 AI 业务的数据中心多依赖 GPU。持续增长的需求导致行业供需失衡、半导体厂商产能紧张，进一步推高包括 GPU 在内的硬件价格，旧一代 GPU 也持续升值，其涨价速度超过好酒陈化速度，市场需求强劲，整个产业链均面临产能紧张。（IT 之家）

郭明錤透露苹果与英特尔代工合作：iPhone 芯片为主力，台积电仍占九成供应

知名分析师郭明錤发文分析苹果与英特尔在芯片生产上的合作及对台积电的影响。苹果因台积电资源未来将向 AI 领域倾斜，早与英特尔洽谈合作，已在英特尔 18A-P 系列制程启动低端／旧款 iPhone、iPad 与 Mac 处理器项目，投片规划反映该系列技术生命周期。苹果还在评估英特尔其他先进制程技术，英特尔量产时间表与出货规模不明，对 2027 年生产良率有目标。英特尔迎来机遇与挑战，苹果是其唯一且完整的晶圆代工练兵机会，但执行难度大。未来数年台积电仍占 90% 以上供应比重，但领先地位成各方风险对冲焦点，先进制程关键玩家都在针对台积电进行风险对冲，台积电主要以卓越执行力应对。

微软寻求收购 AI 公司以摆脱对 OpenAI 依赖

知情人士称，微软正寻求收购人工智能初创公司，以摆脱对 OpenAI 的依赖，储备 AI 人才，推进明年前打造先进 AI 模型的目标。今年春季，微软曾考虑收购代码生成初创公司 Cursor，但因监管审查问题放弃。目前，微软正与 Inception 展开谈判，这是一家由斯坦福大学团队创立的小型初创公司，专注于采用一种名为“扩散”（diffusion）的技术开发大语言模型。微软旗下风险投资基金 M12 于 2025 年末参与了 Inception 5000 万美元的种子轮融资。微软在竞购中还面临来自其他科技巨头的强劲竞争，尤其是马斯克旗下的 SpaceX。知情人士表示，SpaceX 也曾接触 Inception。（格隆汇）  

Anthropic 企业采用率首超 OpenAI

Ramp AI Index 日前发布报告称，Anthropic 的企业采用率于 4 月上升 3.8%，达到 34.4%；OpenAI 采用率下降 2.9%，至 32.3%。过去一年中，Anthropic 的企业采用率增长 4 倍，而 OpenAI 的企业采用率仅增长了 0.3%。报告来自企业信用卡与财务自动化平台 Ramp，该平台追踪了超过 5 万家美国企业的支出模式。但该报告也指出，Anthropic 的优势可能并不像表面看起来那样稳固：不断上升的成本、算力限制，以及推动公司营收高速增长的“基于 Token 计费”模式本身，都可能对其构成威胁。

英伟达总市值达 5.5 万亿美元，超越德国 GDP

**程序员专区**

MiniMax Agent 正式更名 Mavis：推出多智能体团队协作功能

MiniMax 正式宣布将其 Agent 产品整体升级并更名为 Mavis（MiniMax as a Jarvis）。此次更新核心上线了 Agent Teams 功能，支持多个 Agent 并行协作处理复杂任务，并将 TokenPlan 与 Agent Plan 合并，实现 CLI、API 及全模型互通。

针对单 Agent 在长任务中易中断、上下文失焦及响应延迟等痛点，Mavis 推出了由 Leader、Worker 和 Verifier 构成的 Team Engine 架构。该架构通过 Leader 统筹拆解任务，Worker 专业化执行，Verifier 独立进行质量校验与对抗式迭代，将多 Agent 系统从简单的 Prompt 编排升级为具备状态管理与上下文隔离的基础设施。

Mavis 目前已落地于即时通讯、代码开发、行业调研及办公文档四大核心场景。通过将“即时响应”与“后台执行”分离，Mavis 有效解决了 IM 场景下的长任务等待难题；在 Coding 与文档生成中，则通过确定性的代码逻辑驱动与分阶段验证，大幅提升了复杂任务的交付质量与可靠性。（品玩）

OpenAI 宣布：ChatGPT 移动端支持 Codex 远程控制功能

OpenAI 宣布 ChatGPT 的 iOS 和 Android 应用现已集成 Codex 远程控制功能，用户可在手机上监控、管理 Codex 运行任务，实现移动端对云端 / 桌面端开发任务全流程互动。该更新将此前 Mac 独立应用中的 Codex 功能引入移动端，在应用内提供完整操作界面。此功能即日起面向 iPhone 和 Android 用户逐步推送，Windows 版本支持后续推出。Codex 最初以命令行界面发布，今年 2 月推出 Mac 版，上月新增不占用鼠标的应用控制能力，近期 OpenAI 还推出订阅方案，发布 GPT-5.5 模型提升 ChatGPT 和 Codex 能力。

阿里企业级 Agent 平台“悟空”开始逐步规模化放量

5 月 13 日，阿里巴巴集团发布 2026 财年 Q4 及全年财报。财报表示，阿里全栈 AI 技术投入已正式跨越初期培育阶段，进入正向的规模商业化回报周期。在财年第四季度，阿里 AI 在模型、云基础设施和应用各层实现加速突破。在 AI to B 方向，阿里旗下企业级Agent平台“悟空”已于近期逐步规模化放量。

Anthropic 宣布提升 Claude Code 额度：每周限额叠加增长 50%，持续至 7 月 13 日

Anthropic 近日宣布，旗下开发者工具 Claude Code 每周使用限额即日起临时提升 50%，持续至 2026 年 7 月 13 日，该增长可与上周生效的「加倍至 5 小时」限制累加，额度调整全自动生效。Claude Code 以深度理解代码库和高效执行编程任务著称，Anthropic 此举意在降低开发者试错成本，吸引高频用户。这反映了算力分配灵活性的重要性，也预示着 AI 编程辅助工具向「高强度生产力」阶段迈进，开发者对 Claude 系列模型的性能反馈或成衡量其市场号召力的重要参考。

马斯克旗下 xAI 推出首款编程智能体，对标 Anthropic

马斯克旗下 xAI 推出首款人工智能（AI）编程智能体，命名为 Grok Build，旨在简化软件开发流程。该人工智能模型目前处于早期测试阶段，仅供付费用户使用，是这家初创公司进军专业编程领域的首次尝试。根据 xAI 网站发布的消息，该智能体可以根据用户指令完成复杂的编程任务，目前仅面向付费订阅用户开放。马斯克和 xAI 正致力于在编程领域追赶其他 AI 公司，xAI 总裁 Michael Nicolls 将追平 Claude 的表现视为“短期目标”。

曝 GPT-5.6 在内测！OpenAI与Anthropic补贴大战同日开打，开发者坐收渔利

三周前GPT-5.5刚落地，GPT-5.6的消息已经提前炸出来了。知名爆料人Leo透露，GPT-5.6目前开发进度已全速推进，首批内部检查点在过去几天已启动测试，下个月很可能就会正式露面。更有意思的是，有人从OpenAI内部Codex日志里扒出了rollout mapping的痕迹——大部分调用还是指向GPT-5.5，但有一条记录赫然落在了GPT-5.6身上。换句话说，Codex环境里可能已经在偷偷用它跑测试了。内部代码名也被顺带曝光:ember-alpha 和 beacon-alpha。（AIbase基地）
