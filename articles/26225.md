# Claude Code史诗级更新，突破终端，把工作实时变成网页看板

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1#imgIndex=0)

机器之心编辑部

  

Claude Code 的输出终于不只是终端里无聊的文字了！

  

今天，Anthropic 宣布 Claude Code 的一个重磅更新，正式支持 Artifacts 功能。能够实现实时地把用户在终端里和 Claude Code 对话的过程，直接变成一个可交互、可分享的网页。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqHfelNIlnXB6ibS8behXJ5LeQpJibR2MBCjkfHJpEFreTD0nG0aWVibz67tTJibeyMgGcD5bznMpMxe5431oV9SAZWiaIMfLsHqHlnY/640?wx_fmt=png&from=appmsg)

  

从今天开始，Claude Code 可以将工作进展捕获为一个 Artifact，把 Claude Code 的工作转化为实时、可分享的可视化页面，这些页面会随着你的会话工作进展自动更新。

  

这个网页可以是：

  

*   一个 PR 代码走查，包含 diff、改动原因、测试情况
    
*   一个系统架构图，直接从代码的 import 关系中生成
    
*   一个事故调查时间线，串联日志、可疑 commit、错误曲线
    
*   一个发布清单，随着工作推进自动填充状态
    

  

  

Claude Code 生成 Artifact 时，使用的是你会话中的完整上下文，包括代码库、连接的外部工具、以及对话本身。不需要手动接数据源，也不需要搭建任何基础设施。

  

Artifacts 生成后会产出一个固定链接。当 Claude Code 更新 Artifact 内容时，打开链接的人会看到页面原地刷新，无需手动操作。

  

每次发布都是一个新版本，历史版本可随时回溯。此外还有一个画廊视图，方便浏览和管理你创建过的所有 Artifacts。

  

实时同步的效率场景

  

在 Claude 的内部测试中，最常见的使用场景之一是调试。

  

这类场景通常是这样的：一位工程师在会议前启动了一次事故调查。Claude Code 会分析日志，并发布一个 artifact：其中包含时间线、可疑提交以及错误率图表。

  

她可以直接从页面顶部把链接分享给团队。等到会议开始时，随着调查推进，Claude 已经重新发布了两次页面，并纳入了最新信息。

  

有了 artifacts，团队成员和相关方不再需要别人 “带大家走一遍智能体发现了什么”，因为所有人看到的都是同一个视图，以及同一份上下文。

  

Artifacts 的目标用户群远不止写代码的人。官方列出了一系列按角色分类的示例：

  

*   法务 / 开源合规：生成一份依赖许可证审计页面，自动标记 copyleft 协议。
    
*   隐私团队：追踪代码库中个人数据的收集、存储和日志路径，输出数据流图。
    
*   安全团队：生成安全审计结果页面，每个发现都直接链接到代码具体行号。
    
*   FinOps / 平台财务：从 Terraform 代码中映射云资源和成本驱动因素。
    
*   软件工程师：生成 PR 走查页面，包含 diff、设计决策和测试覆盖。
    
*   设计师和前端工程师：基于真实组件库生成多个 UX 方案变体，选中即可交付。
    
*   架构师：从真实代码的 import 关系生成服务拓扑图。
    
*   SRE 和值班工程师：事故页面随调查进展自动生长，最后直接变成复盘文档。
    
*   工程经理：从已合并的 PR 中自动生成团队周报页面，按项目分组。
    

  

Artifacts 目前以 Beta 形式面向 Claude Team 和 Enterprise 组织开放，支持 Claude Code CLI 和桌面客户端，生成的页面可在任何浏览器中查看。

  

在团队协作中，AI 工具带来的效率提升已经很明显，但围绕 AI 产出的沟通成本，始终居高不下。

  

如何将 AI 的工作过程和产出结果规范化、可视化，并实时同步给团队成员，打破信息差，对效率的提升极其重要。

  

这种探索的意义在于，它让 AI 不只是个人效率工具，而开始成为团队共同工作的基础。过程可见、结果可追踪、上下文可共享，团队能够突破 AI 单点提效的瓶颈，实现智能化的协作链条。

  

这种实时、共享、透明的新功能的探索，非常有意义。
