# 刚刚，微信掀桌！2步接入龙虾🦞，24小时聊天干活

刚刚，微信能用“龙虾”了！

智东西3月22日报道，今日，微信「ClawBot」插件正式发布，支持接入OpenClaw（昵称“龙虾”）。

连接后，用户就能在微信聊天中与OpenClaw收发消息，指挥“龙虾”干活，用于个人学习、工作问题、内容创作等场景。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcupTGia78lnicicMsMzD3HBgJHUZ5wPgAqm6ibRKzmO41iaWcolibibSmRqLSZBHeib0gRjzBh0Ff6FGWicibg0qCice83xnr5txUdicaFNcO0M/640?from=appmsg)

这就相当于在微信里接入了一个7x24小时随时可召唤的AI助手。  

腾讯公关总监张军发文说：“微信当年所努力的，就是成为一个连接人、连接设备、连接服务的工具。这个理念依然有效。这个连接，可以方便每个人更好地管理自己的龙虾。”

![](https://mmbiz.qpic.cn/mmbiz_jpg/8Eyv4o4Acuqz0Oic8icVvZvxdDCCTwibic2yZeTicHNRU0DibVz1JJ8ic1fsQ2KTrEdRfeDictVPPqbDsCibiaBQuYzHUVHIPaCn6AXJLEzho9xE4SwpQ/640?wx_fmt=jpeg&from=appmsg)

只需**2步**，即可在微信里调用OpenClaw，轻松对话。

**第一步：启用微信ClawBot插件。**

把微信更新到最新版本，在「我→设置→插件」页面就能看到ClawBot插件。根据页面提示，启用或授权该插件功能。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acuoia9pjibG95DXeaiceIMGIBMlp40DSeMPONcQ5pVKccNibEiahOaKIX3j8dOgTuMvQbMpNDsJSgfngTHKrgHTj83u6m0KaoIK2ELAo/640?from=appmsg)

**第二步，绑定OpenClaw账号。**

根据微信ClawBot插件卡片中的功能指引，在OpenClaw中添加插件。

终端安装指令如下：

    npx -y @tencent-weixin/openclaw-weixin-cli@latest install

扫描二维码，添加微信ClawBot。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurGH1KUuyq2VP9RkicEyvJOweCjaeusZeMDN0lcUlib14tpGCEFjdRgK7y7DTTo2NaJyFPCD9wy71U0ibia0wFVd4TB7icV441uZ9N8/640?from=appmsg)

添加成功后，你就能在微信聊天界面，直接跟“龙虾”对话啦。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acurb12gVzuRLl0dJibXHAiav0ZEPuLCsmY0gxfDCJDiba1EIXOtrEE4ORNIQzJmWPsYhFXGs5vEVFNA5TzjoGHP3M4nVmuX1KVHDAA/640?from=appmsg)

微信官方也给出了4项关键**提示**：

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuqvCq6UfTblGoMAWF4AfkLQUic8QTazbroSxHriaicoju0KksV5h1dMxRVhgI6icv9krrjcexmAv203qfQR2Yd4ibtBBFoyJzZnEB0o/640?from=appmsg)

最近，腾讯在“养龙虾”这件事上行动快准狠，火速组成一支“龙虾特工队”：

*   打通微信、企业微信、QQ、元宝派四大通道；
    
*   涵盖云端虾（腾讯轻量云）、自研虾（WorkBuddy）、本地虾（QClaw）、企业虾（腾讯云ADPxClaw）、云桌面虾（腾讯云桌面）、一站式龙虾安全工具箱（腾讯云AI Agent安全中心、腾讯iOA、腾讯电脑管家“龙虾管家”）；
    
*   拥有丰富的龙虾技能生态（龙虾技能社区Skill Hub和一众腾讯产品Skill）。

还有更多产品接入中。

微信推出官方ClawBot插件，是微信首次正式加入“龙虾大战”，也是腾讯在AI智能体领域的又一次战略性落地，有助于进一步强化其AI生态竞争力。

当前微信对OpenClaw操作微信的权限态度依然谨慎。微信ClawBot插件正在逐步放量中，仅限于个人单聊，不能加入微信群，也不会自动化操作用户微信，以降低安全风险。
