# 600亿美元，Cursor归马斯克了

![](https://mmbiz.qpic.cn/sz_mmbiz_png/KmXPKA19gW889cR13aBX42evqQIRibKlicoCrHPEpT0tQiceNphESCa2eJTqstP8G0yqMTkeMFrOGue6kOyCKdTkA/640?wx_fmt=png&from=appmsg&wxfrom=5&wx_lazy=1&tp=wxpic#imgIndex=0)

编辑 | 张倩、泽南

  

SpaceX 对 Cursor 的收购，终于尘埃落定了。

  

刚刚，美国证券交易委员会 SEC 公告显示，SpaceX 宣布将通过全股权交易收购 AI 编程工具 Cursor 的母公司 Anysphere。该交易使得 Cursor 的估值达到了 600 亿美元。

  

这一收购消息发布前仅数日，SpaceX 刚刚完成从火箭公司向 AI 企业的转型，并在纳斯达克轰动上市，首日市值即突破 2 万亿美元，使其一跃成为全球市值最高的企业之一。上市第 2 天，SpaceX 再涨近 20%，当前市值已达 2.5 万亿美元。此次收购旨在加速 SpaceX 在企业级 AI 市场的布局。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqFRd0oLMiaDpxnluib16qYBxt48kAcaqOOKSiarbCTh9Iq22Cdl0a2xnKSI73TXLGrX4VMbAJ3UfNxiczAbtb1xklSEfWBQnzT5ib9g/640?wx_fmt=png&from=appmsg)

  

此次交易后，Cursor 将成为 SpaceX 的全资子公司，Cursor 的股东将获得 SpaceX 的 A 类股份，交换比例将基于交易完成前 SpaceX 7 天平均股价。不过，该交易仍须经监管批准及符合惯常交易完成条件。

  

SpaceX 表示，预计该合并交易将于 2026 年第三季度完成。

  

其实，SpaceX 对 Cursor 的关注已持续数月。该公司曾在 4 月表示，已获得一项选择权：要么在今年晚些时候以 600 亿美元收购 Cursor，要么以 100 亿美元的价格达成新的合作伙伴关系，并提供算力支持。

  

这是一个较为罕见的交易结构设计。当时，Cursor 原本正处于一轮由 a16z、英伟达等领投的 20 亿美元融资的最后阶段，投后估值约 500 亿美元。SpaceX 强势介入并「截胡」，实现了算力与股权的深度绑定。

  

现在 SpaceX 在完成上市后，凭借公开市场充沛的资金与股票流动性，正式行使了这项 600 亿美元的看涨期权。

  

Cursor 与 OpenAI、Anthropic 等公司并列，是硅谷少数几家通过人工智能实现代码编写自动化、从而吸引大量开发者的初创企业。

  

根据 Cursor 本月早些时候向路透社分享的公司数据，自 2022 年成立以来，Cursor 业务迅速扩张，其年度 B2B 收入约为 26 亿美元，企业销售额增长尤为迅猛。

  

对于 SpaceX 旗下的 Grok 团队来说，收购 Cursor 有着重要意义。上个月马斯克官宣新模型时就提到，他们在新模型的后训练加入了大量来自 Cursor 的数据，并且这一数据源还将继续扩充。众所周知，Cursor 是开发者实际使用的 AI 代码编辑器，数据包含真实的工作流（写代码、调试、迭代、修复 bug 等），是「人类开发者如何思考」的映射，其价值非常独特。

  

![](https://mmbiz.qpic.cn/mmbiz_png/5L8bhP5dIqGN4sO1WjyiboCbTvuwG7Ok9U1BpFalGer9iaOcj3rO67eF4PYbNHMJDE32CucxJNpmKL07BKrj1vr31UYz0L9DQkKPHH18ACBCY/640?wx_fmt=png&from=appmsg)

  

因此，这笔交易有望为 Grok 团队在 AI 编程市场赢得更稳固的立足点，而该领域此前一直是他们相对落后的板块。

  

虽然 Cursor 是过去两年增长最快的 AI 产品之一，并在 2025 年引领了 Vibe Coding 风潮，但其命门在于底层模型高度依赖竞争对手。它的核心体验建立在调用 Anthropic 和 OpenAI 的模型之上，随着这些巨头推出自己的原生代码工具（Claude Code 和 Codex），Cursor 面临着被断供和利润率被极限挤压的双重危机。

  

并入 SpaceX 后，Cursor 可以直接使用马斯克旗下 xAI 位于孟菲斯的 Colossus 超算中心，有了近乎无限算力的托底，Cursor 可以从单纯的应用层开发，转向直接训练旗舰级代码大模型。

  

目前尚不清楚该交易是否会影响 SpaceX 对外出租数据中心的现有协议。最近数周，SpaceX 已与 Anthropic 及谷歌达成协议，合计以年度约 260 亿美元的规模租赁云计算能力。

  

这两项协议均包含 90 天终止条款，意味着 SpaceX 可在必要时迅速收回计算资源。
