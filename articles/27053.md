# 蚂蚁百灵大模型负责人领衔！AI智能体大会再曝重磅嘉宾，自进化智能体研讨会议程同步揭晓

![](https://mmbiz.qpic.cn/sz_mmbiz_png/VYy0wxV0rxcqt1Qib1ATDYQkJHfOQ0F8ib1OBFfoPPoCGfo11H37LluyvfiaiaMK0cohJicP1b615icN7UnnGldAX0rQzw6PdrDZYWBC5zibFWU8bQ/640?wx_fmt=png&from=appmsg)

Agentic AI浪潮奔涌而至，全新技术范式加速迭代，一场重塑产业生产力的变革已然到来。

在此时代风口下，2026中国AI智能体大会（AgenticAICon 2026）将于7月2-3日在杭州隆重启幕。

本次大会由智东西旗下智猩猩发起主办，为期两天，由开幕式+论坛+研讨会+交流晚宴+展览区组成，将以“范式跃迁 重塑世界”为主题，邀请70+位重量级嘉宾带来致辞、报告、演讲和对话，前瞻把脉Agentic AI的范式跃迁，深度解构AI智能体的生产力。

其中，开幕式将于首日上午拉开帷幕；企业级AI智能体专题论坛、AI智能体产品创新论坛则将在主会场进行。此外，大会还将举办Coding Agent、自进化智能体、深度研究智能体、Computer-Use Agent、多智能体协同、Agent Skills、Agent Harness七场技术研讨会，研讨会主要面向持有大会通票、VIP票以及贵宾票（SVIP）的观众开放。

此前，我们揭晓了大会的部分嘉宾（《[阿里技术大牛组团分享！AI智能体大会进展公布，Harness研讨会议程出炉](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652805008&idx=1&sn=9a5a7d2208538b994a2ffdc48d96a504&scene=21#wechat_redirect)》）。今天，将为大家公布大会的最新进展。

**自进化智能体技术研讨会议程出炉**

**拆解下一代 Agent 的六大进化能力**

  

过去两年，大模型已经从“会回答问题”的聊天系统，快速走向“能完成任务”的智能体系统。

  

但真正进入产业场景后，一个关键问题开始浮出水面。

智能体不能只靠人类不断写提示词、堆工作流、调上下文来变强。真实世界是动态的，任务会变化，工具会变化，环境也会变化。如果一个智能体每次遇到新问题，都只能等待人类重新设计流程、补充规则、修正错误，那么它依然只是一个高级自动化工具。

下一阶段真正值得关注的方向，是自进化智能体。因此，本次大会同期组织了一场自进化智能体技术研讨会。目前，该研讨会的议程也已出炉，由主题报告+圆桌Panel两个环节组成。

前腾讯大语言模型部Frontier团队专家研究员王琰，香港科技大学研究助理教授张永岗，香港大学在读博士、Evolvent AI创始人胡梦康，中国人民大学高瓴人工智能学院在读博士、ARPO一作董冠霆，厦门大学人工智能研究院在读博士、SkillClaw核心作者姬煜翔，上海交通大学计算机学院在读博士、Misevolution一作任启涵6位技术大咖与研究人员将作为报告嘉宾带来分享。其中，研讨会及圆桌Panel将由董冠霆博士主持。

此前，我们也公布了Agent Harness技术研讨会的议程（《[阿里技术大牛组团分享！AI智能体大会进展公布，Harness研讨会议程出炉](https://mp.weixin.qq.com/s?__biz=MzA4MTQ4NjQzMw==&mid=2652805008&idx=1&sn=9a5a7d2208538b994a2ffdc48d96a504&scene=21#wechat_redirect)》）。除此之外，大会同期举办的Coding Agent、深度研究智能体、Computer-Use Agent、多智能体协同、Agent Skills五场技术研讨会中更多确认参会嘉宾以及议程，后续也将陆续披露。
