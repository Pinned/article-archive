# 让你的 Claude 记忆，通过 Github 同步

上周开源了的 Agent 网盘，我又给加了个功能：【GitHub 同步】  
[开源个 Agent 用的网盘，让 Skill 跨设备同步](https://mp.weixin.qq.com/s?__biz=MzkzNDQxOTU2MQ==&mid=2247516741&idx=1&sn=c92a963dd105a5d6ad229f11128c8230&scene=142#wechat_redirect)  
  
这样，你的 skill 和记忆，就能同步到 GitHub 私有仓库了  
正因为是 Git 仓库，每次同步都有版本记录，可以回溯到任意时间点  
  
通过这个工具，你在 Claude、Codex、Cursor 这些工具里攒下来的 skill、记忆、文件，可以通过 neuDrive 一键备份、相互同步  
  
定位就是：Agent 的 “OneDrive”，你的东西跟着你，不被封  
  
开源地址：  
https://github.com/agi-bar/neuDrive  
  
部署好的版本，可以直接用：  
www.neudrive.ai
