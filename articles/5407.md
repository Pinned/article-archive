**国内要闻**

美团回应“APP 删除用户图片”：第三方 SDK 冲突导致，波及范围约数百人，承担全部费用和损失

近日，有部分安卓用户反映，在使用美团 APP 期间收到图片和视频被异常删除的提示。3 月 24 日，美团官方客服回应称：“该问题主要是安卓系统在极少数情况下，APP 自动缓存清理时遇到第三方 SDK 冲突导致异常。问题发生后至今，累计有 180 多位用户进线客服咨询，预计潜在波及用户范围约在数百人。我们在发现该问题后已第一时间进行紧急修复。”

美团客服表示，此次问题仅发生在用户本地手机系统层面，美团 App 不会读取、存储或转移任何用户个人数据，用户隐私及账户信息均未受到影响。美团在发现问题后已迅速成立专项技术支持小组，联合专业安全技术团队协助受影响用户找回数据。美团客服介绍，已向每一位受影响用户致歉，并将承担全部费用和赔偿相应损失。（澎湃新闻）

“Token”中文名定了：词元

在中国发展高层论坛 2026 年年会上，国家数据局局长刘烈宏表示，Token“词元”不仅是智能时代的价值锚点，更是连接技术供给与商业需求的“结算单位”，为商业模式的落地提供了可量化的可能。

这从官方角度给出了 Token 的中文翻译：“词元”。刘烈宏透露，截至 2026 年 3 月，我国日均词元调用量已攀升至 140 万亿次以上，这一数字较 2024 年初的 1000 亿次实现千倍跃升；即便与 2025 年底的 100 万亿次相比，短短三个月内增幅亦超过四成。（央视新闻）

卢伟冰谈小米手机为何还没涨价：公司产品很多元化，暂时可以自己去抗

小米集团发布 2025 年全年业绩并举行投资者电话会议，回顾公司去年业务表现，并展望未来发展计划。据微博博主 @机智的娜娜 发文，会议中有记者提问：“为什么 OPPO、vivo 手机都涨价了，小米还没涨价？”。

小米集团合伙人、手机部总裁卢伟冰对此回答道，首先小米的产品是很多元化的，不仅仅有手机、平板、PC，还有汽车、家电等等。所以内存涨价的压力是有的，但暂时还可以消化，还可以自己去扛。“但是我觉得我们扛不动了，我们肯定会去讲，我希望我们涨的时候，消费者、用户能够对我们有一些理解，我们只是比别人多扛了一会儿。”卢伟冰对此表示。同时他还认为：“我觉得（手机）涨价的这个趋势是不可避免的”。

猎豹移动董事长傅盛深夜发文向 360 周鸿祎致歉

猎豹移动董事长傅盛发布《关于近日言论的说明》，称前两天深夜在微信群发了情绪激动的言论，被媒体大量转载，后续发展出乎预料，让他反思自责。起因是当晚喝酒后想给周鸿祎打电话叙旧，周未接还删了他，他一时冲动发了那些话。他否认博流量，为冲动和失态向周鸿祎道歉，称跟周打仗的日子是宝贵记忆，周曾给他重要机会。他希望把精力放产品上，感谢刘亮调解。此前傅盛曾指责周鸿祎「欠钱不还、删好友」，360 回应将依法处理。傅盛 2005 年加入奇虎，后因理念分歧离开，双方曾有多年恩怨。

网易有道龙虾获 OpenClaw 创始人点赞，已全面接入主流即时通讯工具

全球爆火的 AI Agent 开源项目 OpenClaw 创始人 Peter Steinberger 点赞有道桌面级 Agent 产品 LobsterAI。网易有道宣布开源全场景个人助理 Agent——LobsterAI，春节期间火到海外。3 月，有道龙虾正式上线 Skill Store，覆盖基于 OpenClaw 的 5000 + 技能，支持 GitHub 全量 Skills 安装及实时更新等，已全面接入国内主流 IM 平台。「OpenClaw 生态流量榜-龙虾 Agent」榜单数据显示，有道龙虾上线首月访问量突破 27 万次，位列分类榜前五。

特斯拉副总裁陶琳：学历提升项目首批学员成功毕业

特斯拉公司副总裁陶琳宣布，特斯拉「我要上大学」学历提升项目迎来首批毕业生，62 名上海超级工厂员工通过与上海电力大学合作的新型学徒制项目完成学业，于 3 月 21 日举行仪式。该项目 2021 年发起，不设报名门槛，员工参加成人高考被上海电力大学录取后深造，采用「双元制」模式和「5+2」学习模式，毕业后获「双证书」。课程贴合行业和企业需求，公司提供支持和奖学金。此次毕业员工中，49 人获本科毕业证，13 人获专科毕业证，5 人获公司奖学金。

OpenAI 将停止 Sora AI 视频生成服务，精简产品线

OpenAI 在高调推出 Sora 独立应用程序六个月后，计划停止该视频生成服务，以简化其人工智能产品线。该公司于去年 9 月底推出 Sora 应用程序，承诺让用户在准社交网络中更便捷地生成并分享逼真的 AI 视频。这款免费应用程序一度迅速登顶苹果 App Store 排行榜，但此后排名已回落。除了关闭这项独立应用程序外，OpenAI 还将关闭开发者使用的 Sora 的应用程序接口（API）。

智能体大战升温：Claude 可代用户操作电脑 AI 代理走向实用化

美国人工智能企业 Anthropic 宣布旗下 Claude 模型可操纵用户电脑完成任务，如打开应用程序、浏览网页、填写电子表格等，该功能已在 macOS 版 Claude Cowork 和 Claude Code 中提供。其演示视频展示 Claude 成功完成用户指令。这一更新凸显 AI 公司加速推动「智能体」发展，此前 OpenClaw 发布后这类能力走红，英伟达推出支持 OpenClaw 的软件栈，OpenAI 聘请其创建者推动发展。不过，Anthropic 提醒「电脑操作」仍处早期阶段，已加入安全防护，Claude 访问新应用会请求用户许可。

曝 Epic 宣布裁员 1000 人，《堡垒之夜》玩家活跃度下滑成主因

Epic Games 首席执行官 Tim Sweeney 发内部邮件宣布裁员 1000 人，主因是《堡垒之夜》玩家活跃度下滑，公司支出高于收入，需降本增效。Epic 还削减了 5 亿美元成本。受影响员工将获至少 4 个月基本工资等补偿。Sweeney 称决定艰难，有行业增速放缓、玩家消费欲望减弱等行业原因，也有《堡垒之夜》未每个赛季都有「爆款体验」等自身原因，且本次裁员与 AI 无关。

OpenClaw 版本升级「翻车」

3 月 24 日，「龙虾」OpenClaw 迎来诞生以来最大规模更新，新版本定位为跨平台个人 AI 助手，重点调整底层架构，放弃 npm 转用 ClawHub。但此次以「安全与生态统一」为目标的激进重构演变成「升级事故」，导致大量用户反馈插件瘫痪、功能失效。OpenClaw 开发者 Peter Steinberger 回应称，因限流规则设置过严，大量用户涌入叠加可能的恶意流量攻击触发限流机制，加剧用户体验崩溃。

SpaceX 计划最早于本周提交 IPO 申请筹集超过 750 亿美元资金

3 月 25 日消息，据一名知情人士透露，SpaceX 计划于本周晚些时候或下周向监管机构提交首次公开发行招股说明书。此次秘密提交将正式敲定埃隆・马斯克旗下这家火箭与通信公司的 IPO 计划，目标是在 6 月挂牌上市。此次 IPO 将检验投资者对这桩有望成为美国史上规模最大 IPO 的认购热情，其体量将远超以往。该人士称，参与筹备工作的顾问预计，公司在 IPO 中或拟募资超 750 亿美元，高于此前报道的 500 亿美元预估。公司最新估值为 1.25 万亿美元，实际募资规模与估值将在 IPO 前数周最终确定。

亚马逊据悉开发新型 AI 智能体以自动执行部分部门职能

据报道，亚马逊旗下云计算部门 AWS 正在开发一种 AI 智能体，旨在自动执行销售、业务开发及其他部门的部分职能，而这些部门此前一直是该科技巨头的裁员重灾区。AWS 开发的这款智能体可以处理数千名技术专家在网络安全和服务器网络等领域的部分工作量。

苹果 MacBook Neo 同时打开 60 个应用仍能流畅播放视频

YouTube 频道 Hardware Canucks 在最新一期视频中，实测了苹果 MacBook Neo 的极限内存承受力，同时打开了 60 个应用程序，系统并未出现崩溃，且依然能流畅播放视频。苹果 Macbook Neo 推出后，最大的讨论焦点就是 8GB 内存，由于内存直接集成在芯片封装内部，因此用户无法通过常规方式升级。该频道为了测试 8GB 内存的极限，在 MacBook Neo 上同时打开了 60 个应用程序。结果显示，macOS 系统不仅没有崩溃，甚至在随后播放流媒体视频时依然保持流畅，鼠标指针也未出现任何卡顿。

Arm 发布首款自研 AI 芯片 AGI CPU

全球半导体 IP 巨头 Arm 在美国旧金山推出首款自主设计的数据中心 CPU——Arm AGI CPU，这是 Arm 成立 35 年来首次涉足自研芯片销售业务并切入数据中心核心算力市场。该芯片基于台积电 3nm 制程、采用双 Chiplet 架构，专为 AI 智能体基础设施设计，性能优越，单机架性能可达 x86 平台 2 倍以上，每 1GW AI 数据中心算力资本支出可节省高达 100 亿美元。它支持多种部署方案，满足不同规模数据中心需求。发布会上，英伟达 CEO 黄仁勋祝贺，Arm CEO 透露 Arm 架构芯片累计出货量已突破 3500 亿颗，远超全球人口总数，是非 Arm 架构 CPU 累计出货量的 7 倍。

OpenAI 推出 ChatGPT「Library」功能，实现文件云端持久化存储

OpenAI 官方向全球（除欧洲经济区、瑞士及英国外）的 Plus、Pro 及 Business 用户推出「Library」新功能，该功能可让用户在 OpenAI 云端存储个人文件与图像，提升文件管理便捷性与安全性。启用后侧边栏显示入口，系统默认将对话中上传或创建的文档等保存至专用安全位置供随时调用。ChatGPT 生成的 AI 图像在「Images」标签页，「Library」仅收录用户上传文件。文件永久保留，删除对话记录不清除库中存档，用户可手动删除，文件将在 30 天内从服务器彻底移除，确保资料持久可用，赋予用户数据控制权。

京东开源大模型 JoyAI-LLM Flash

京东宣布开源基础大模型JoyAI-LLM Flash的Instruct版本，该模型创新引入基于“纤维丛”理论的FiberPO强化学习方法，实现技术突破，在19个权威基准测试中表现优异。

微软发布 TypeScript 6.0，为 7.0 底层 Go 语言重写铺路

微软发布 TypeScript 6.0，这是基于 JavaScript 编译器代码库构建的最后一个版本，在未来 7.0 版本中，将迁移至 Go 语言。此次 TypeScript 6.0 最引人注目的标签在于，它是基于现有 JavaScript 编译器代码库运行的最后一个版本。

微软目前已经着手开发 TypeScript 7.0，并且决定使用 Go 语言全面重写其底层。微软强调，彻底更换编译器代码库预计能让 TypeScript 的编程与编译速度大幅提升 10 倍。作为承前启后的过渡版本，6.0 的核心任务是清理底层代码，并为开发者向 7.0 迁移做好准备。尽管如此，该版本依然带来了一些新特性，同时也包含了大量破坏性更改。例如，在默认设置方面，strict 现已默认开启（true），types 默认值变更为 \[\]，而 module 和 target 则分别默认指向 esnext 与 es2025。此外，官方还废弃支持 es5 等旧版目标环境。
