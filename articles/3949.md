# 刚刚，苹果祭出地表最强AI PC，顶配6万5，M5 Max芯核弹AI性能暴涨4倍

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqX3IPC7JGwRrqY7UMGFHtJgbRWRopWAO2CjoTyQicmH5x0DX3iaEKGg9hjMpeIJd8kahjtd28C2hEm6uv9AJQc1fh7RDRX6qpdI/640?from=appmsg)

**苹果一切向着AI，性能暴涨价格也暴涨。**

**作者 |**  云鹏  

**编辑 |**  李水青  

智东西3月4日消息，继3999元用上苹果A19芯片iPhone后，刚刚，苹果春季新品发布第二弹正式揭晓：

**两款Mac笔电芯片：M5 Pro和M5 Max；**

**两款新电脑：苹果MacBook Pro、MacBook Air；**

**两台新显示器：Studio Display、Studio Display XDR；**

没错，库克一口气端上来6款重磅新品，各个都看点十足。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuqYic2IjIGHEvicwb8fSP5FLsJ2W9E2qUf4G0teRctRkbSedc02cRS6vrOR5KQ3Yxlbib7J51zQ1kB8A7MZFVx92IK9OsnC6y4bzo/640?from=appmsg)

作为地表最强PC笔电芯片，M5 Pro和M5 Max的CPU、GPU都有大幅性能升级，基于新融合架构**“拼好芯”，苹果将两颗3纳米晶粒合二为一，**与其他模块共同构成SoC，兼顾高带宽和低延迟。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurexMPU6RqFUQtbTzZVxsKicAg23cusiccrKP0g1WdUvib9WbnuTeh9OEkDBiaYicZvtOicMXhAJRNyUHVzUReP4SMe3ksQ7F20KVfAM/640?from=appmsg)

两颗芯片的每个GPU内核都塞入了神经网络处理器，加上统一内存性能的提升，**M5 Max的AI峰值性能相比上代暴涨了4倍。**

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcurpDXuJZXPrw2fHaP2yzts1SJJJ0VAj3xy8y8gYvL1ydWkl8LyjQia3u5qFXab2x07oiauybbYXCD4xnh1N5hW74VAnbdhiaNictT4/640?from=appmsg)

比如在**AI图像生成**任务中，搭载M5 Max的MacBook Pro比上代快了近4倍，在**大语言模型提示词处理、AI视频处理**方面也有数倍提升。

AI，毫无疑问是两颗芯片升级到重点。苹果特别提到，他们**“从头到尾打造芯片都是为AI精心构建”。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcupTvMsyaVbX9OgSDPXXsFib7wnIvkGp4yX5ibl2FKlfgAjeTHVg6ZnWZ5k4efNNfyph6djAYdza18x7ObenPdymmDxjXffm0tQPc/640?from=appmsg)

基于这两颗芯片打造的新款MacBook Pro不仅有着Mac笔电**史上最长的24小时续航、**翻倍的固态硬盘速度，针对各类AI任务的处理速度都有大幅提升。

相比之下，搭载M5芯片的MacBook Air就没有太多升级点，轻薄、价格更低依旧是其核心优势。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acuoe6E2kU5cIrjiajCZniaaCWSMuJIMNlp34RbvAErM0iao92GKW3JfZr0icYHxFjYvbFqyciaDHvTLtRnSsQsL5dI8XVHxT8ETDMQJY/640?from=appmsg)

两款显示器的升级重点一个是画质，一个是接口扩展性，两者均为27英寸5K分辨率，**苹果称之为“视网膜级”显示，**画面非常细腻。XDR款背光技术为Mini LED，有2304个分区并支持120Hz高刷，标准版只有60Hz。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuoLxFAtbJgMsy8ENicsuwILxzBUFy7gvZbXrfvv72EbrPZA7EquyicmiafiaT7F8zb1b5sZOG6JSXiciagegzKhLMegx6qmufyxma4Ic/640?from=appmsg)

▲Studio Display XDR

此次XDR显示器的**HDR峰值亮度升级到了2000nit；**接口方面，两款显示器都支持**最多6台串联，显示6000万像素图像。**

价格方面，14英寸搭载新M5 Pro的MacBook Pro起售价为**17999元，**存储为24GB+1TB起步，如果是16英寸M5 Max版全部“拉满”，总价最高在**64719元，**当然，这包含了两款软件和Apple Care服务，去除后大约58124元。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcupWotEq8hEOnIBP4VV4l9d7YtCXusu6Yn6tywLdjia7JT5rTgPiczzAANBl7WtlMTtTpLicdAZHpDE4XBEe9f0LZushtBZloaYVrE/640?from=appmsg)

即使是17999元的起售价，相比上代同存储规格M4 Pro版MacBook Pro发售时的15699元，也**涨了2300元左右。**

MacBook Air是16GB+512GB存储规格起步，13英寸售价**8499元**起，上代同规格M4版MacBook Air售价9499元，**下降了1000元。**

但值得注意的是，新款MacBook Pro和MacBook Air的起步固态硬盘规格都涨了，**Pro从512GB涨到了1TB，Air从256GB涨到了512GB。**

哦对了，苹果生怕你不知道该不该升级、提升大不大，所以贴心的为你放上了“比一比”，但其中只有M2及以前的老机型，甚至还有英特尔处理器的机型对比。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcupSXyiczWVIVJqTuHcyf89MLW9tPheQ3r8Q6rfywstrSmicgZOGU3mkhYUqZELvIK4GM2HZT3Fiam8rzED1PBTkEAawOrdMhABvBk/640?from=appmsg)

**用户心理拿捏这块，库克是懂得。**

两款显示器，Studio Display起售价为11999元，Studio Display XDR起售价为24999元，Nano-texture纳米纹理显示屏玻璃版本售价27499元。好家伙，“一层膜”2500元，果然够苹果。

所有新品都是3月4日开始预购，3月11日正式发售。

  

**_01_****_._**

**芯片封装黑科技加持**

**单线程王者依旧，AI能力大提升**

作为M5系列的两款新品，M5 Pro和M5 Max的GPU同样加入了神经网络加速器，与M5相同，因此这两款芯片的AI性能也进行了重点升级。

苹果称这两款芯片使用了他们设计的新融合架构：将两颗晶粒结合为一个单片系统（SoC），也就是我们在Ultra上曾见到过的“拼好芯”，也是业内目前称之为“Chiplet”的热门芯片封装工艺。

这一工艺兼顾高带宽、低延迟，将两颗第三代3纳米晶粒合二为一。两颗晶粒集CPU、GPU、媒体处理引擎、统一内存控制器、神经网络引擎和雷雳5控制器于一体。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuoY90cicrPhhic2xNPibOczfvvC1YYcMvCTrkiaohPpMGgKTGj5gbg7zHVDK3ZdPqTiaicEyia98tZPLh6RclMmpxFoOGWkHog5DXNiacI/640?from=appmsg)

M5 Pro和M5 Max的CPU均为18核，包括6颗性能较高的“超级核心”和12颗针对高能效、多线程工作负载优化的性能核心。

因此也可以说是“全大核”架构。根据苹果官网信息，18核CPU专业工作复杂处理性能最高提升30%。

苹果特别提到，CPU的单线程性能首屈一指，这部分得益于更高的前端带宽、新缓存结构和强化的分支预测技术。

GPU方面，M5 Pro和M5 Max有较大差异。

M5 Max的GPU最多支持40颗核心，每个核心里都有神经网络加速器，GPU配备了更高的统一内存带宽，处理AI任务的峰值GPU计算性能相比前代机型提升了4倍。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acurr8xWQRjic7l6tZvCHgq2fUWicXKlHUqmNfQL354b44EeCvhwGJ6I0DzCRctDiaBicOm4rKOWyQtPQ7mpB1qCc1p8KuqLCIJsecdw/640?from=appmsg)

图形性能方面，对于使用光线追踪的App，GPU性能相较M4 Pro和M4 Max提升最高可达35%。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4Acuo1ruOjCXuhRKYsfd3nolmjkkkiaONayCov5noAv82ReIGqbXyFdOWzjM6DcYu8SRWGwf9qmEYLIUibNmzhzGqe3pIAicvdIHgom4/640?from=appmsg)

M5 Pro的GPU为20核，仅有Max版的一半，处理AI任务时的峰值GPU计算性能相比M4 Pro提升超过4倍。

其他方面，M5 Pro和M5 Max都集成了16核NPU，安全性方面，支持Memory Integrity Enforcement，苹果称这是一项行业首创的内存安全保护功能。

苹果硬件技术高级副总裁，我们的老熟人Johny Srouji称，M5 Pro和M5 Max代表了苹果芯片的里程碑式提升，MacBook Pro**性能、能效和设备端 AI 处理能力**的大幅提升都离不开两颗芯片的升级。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqzEia2nXVHiaacr3Qwmz5VxT6XdsMUDlBJMLDA8Kwmfl596iaB0ERx0fBRiaWyiciaTjO35CO9AcEIgs6ibQQcrStPXNxIGmqE3hwDGw/640?from=appmsg)

▲在Xcode等App中加速代码编译和设备端智能体编程

  

**_02_****_._**

**史上续航最长的Mac笔电**

**AI和专业生产力是重头戏**

新款MacBook Pro的芯片有M5、M5 Pro和M5 Max三个版本，AI能力有重点增强。苹果称这是史上续航最长的Mac笔电，最长可达24小时，并且快充只需要30分钟就能充50%电量。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcupAT8NTmsEfGfWUVmjiaoEJXjF4LJOjiba8t7on6e3HE9UQcWXzC3T4XDvReEUciat5ShPnZvpbScfcYW8JnEeiaUYarbzKsUvzKU8/640?from=appmsg)

14英寸机型可选配M5、M5 Pro或M5 Max芯片，16英寸机型可选配M5 Pro或M5 Max芯片。M5 Max芯片的GPU还分为32核和40核不同版本，库克这刀法，真的切麻了。

存储方面，MacBook Pro的固态硬盘速度最高提升了2倍。新款MacBook Pro的读写性能峰值速度为14.5GB/s，在处理4K与8K视频项目、大语言模型和复杂数据集等任务工作流时能提高效率。

此外，MacBook Pro搭载了N1芯片，支持Wi-Fi 7和蓝牙6。

苹果特别提到，M5系列芯片的GPU内都有神经网络加速器，对基于扩散模型生成图像、大语言模型提示词处理，设备端Transformer模型训练等各类AI任务的处理速度都有大幅提升。

在苹果看来，M5 Max是其为专业级Mac笔电打造的最强芯片，目标用户是3D特效艺术家、AI开发者和电影作曲家。

苹果强调称，苹果芯片以及驱动它的每个重要子系统，都是围绕AI所设计，“从头到尾打造芯片都是为AI精心构建”，其优势在于整合硬件、软件和生态系统。

在官网案例中，苹果提到了Msty Studio、LM Studio这些大语言模型App，App中的AI工具也可以在芯片加持下获得更好体验，比如Logic Pro中的大分轨拆分器AI功能。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acurh6B5eBQTbHJMkw8BzibnxYWbO0xVIbmDgyjJ7dK3PILia7RXwDehpNRXX0nHSHY7rrYWjWWuwsJ69mu8K5tf7HrV5jrSPdcb7w/640?from=appmsg)

▲AI图像、音频处理

提到AI处理能力提升，统一内存的升级也发挥了作用，M5 Pro支持最高307GB/s内存带宽，M5 Max则支持最高614GB/s内存带宽。

在很多高端PC笔记本电脑都升级了OLED屏幕的今天，苹果顶级MacBook Pro仍然采用了一块LCD屏幕，当然，这块屏幕的素质没得说，与上代参数基本看齐，不过“昏暗环境亮度可降到1尼特”算是苹果在护眼方面的一点升级。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcurOuVVthLpCwVaQ4LmvAOUSFW6bBicAAn5JokQwYuicib5ibGZhy0hVyibTUEicuqmbd2WHu1oiadIfrVrDFjTLiclQAY8Jiau24wm3qKmg/640?from=appmsg)

MacBook Pro的接口依然丰富，配备了三个雷雳4或雷雳5端口、一个HDMI端口、一个MagSafe 3端口、一个SDXC卡插槽和一个耳机插孔。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuoBQ2aaNweP7kxguLeiaHCmn80MDAH730UibxzibXaUHaYNicTPEPcKYI7eleB2dFaj93agRfotv6W1eLrdg1t9xiaMskpqt9ugH8sw/640?from=appmsg)

电脑手机互联方面，iPhone镜像、实时活动、通用剪贴板、电话App、接力都是我们很熟的功能了，不做赘述。

相比MacBook Pro，MacBook Air的配置相对简单，芯片只有M5一个版本，续航最长18小时。

在核心的轻薄方面，13英寸的MacBook Air重量为1.23千克，厚度不到1.2厘米，另外还有15英寸的版本。

![](https://mmbiz.qpic.cn/mmbiz_png/8Eyv4o4AcuqIdPyZpUm6CKOoYo0CamYcrq6czLnKCtJlbW0X5sWaZjyjsezuiceQTBddM8LynfhBqibRIxwOzcicA5icfAN3VRtZibyasqKO50Kw/640?from=appmsg)

在MacBook Air部分，苹果依然强调了AI能力的提升，当然，与Pro相比会少一些专业软件的介绍。

  

**_03_****_._**

**“视网膜”级显示器来了**

**连接扩展性大提升**

除了两款新MacBook，苹果这次上架的两款“专业”显示器同样可圈可点。

27英寸的Studio Display有着5K分辨率（5120x2880），XDR款HDR峰值亮度从此前苹果“祖传”的1600nit升级到2000nit，刷新率为120Hz。

今年新iPhone 18系列的屏幕峰值亮度或许也会同步提升至2000nit。

Studio Display XDR作为Mini LED显示器，调光分区增加到2304个。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4Acuqlzas3bLs8EHTI4rPvCwudj0MlKX5vQF24YntibFeFQd6HRdTibfVxI8VgItBsaOWLpDvLZy4ABicAAslxAJmSp1OTNwttoK622w/640?from=appmsg)

5K分辨率总共有1400多万个像素，在27寸的尺寸下，苹果称之为“视网膜显示屏”。

两台显示器都配备了1200万像素摄像头，支持人物居中跟踪。

接口方面，两款显示器都新增了雷雳5连接能力，总共提供2个端口，可以最多同时串联4台Studio Display显示器，实现接近6000万像素的画面显示。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/8Eyv4o4AcuoeOyCYysibopWngyoApMiaRqIUicsGbbgBTMNRfdQ7QwJ5DdwHGx4gIpibDYGwSyVSEImrPA5sWGQiciclEV6AjiazV9FULTN6FwUMG0/640?from=appmsg)

在专业性方面，Studio Display XDR在P3广色域之外新增了对Adobe RGB色域的支持，可以说是一款校准参考显示器，对印刷设计类专业人士是一大利好。

  

**_04_****_._**

**结语：软硬一体协同仍是苹果AI时代王牌**

此次苹果祭出的两款王炸芯片，以及基于芯片升级的新款Mac笔电，都花了不少篇幅提及AI方面的能力提升，以及在各类端侧AI应用方面的流畅表现，可以看到，苹果对AI是非常重视的，而芯片层的支持则成为苹果产品的独特优势——软硬协同一体。

面向未来的AI大战，苹果智能在国内的落地备受关注，3月4日苹果会在上海的活动中放出怎样的惊喜，我们拭目以待。
