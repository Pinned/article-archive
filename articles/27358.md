# AI 的尽头，是修仙

碎碎念

  

越发觉得，现在的 AI 生态，越来越像修仙了

  

**Token 是天地灵气**

灵气充裕的地方大能密布，被各大宗门占着。普通人灵气贫瘠，难以用到好的

宗门就那几家，每隔几个月有老祖闭关出关，出关当天必有天地异象，X 上全是贺电

订阅制是内门月例，每月 20 美元领灵气，200 美元升内门，根据级别高低，各有不同待遇；而 API 用户则是散修，按量买灵气

  

数据中心是灵脉，开山老祖拿下此地，便开宗立派。英伟达是卖灵锄的，日子过的很滋润

开源模型是刻在城门口石碑上的功法，虽然不是最好的，但谁都能用，还给门派引了大大的流

对了，基础功法免费，但灵气不免费

  

蒸馏是传功，大能把毕生修为灌给筑基弟子，所以各宗门门规第一条：不得传功外人

修士分流派，炼气的钻研算法，炼体的去搞Skill，炼器的去折腾各种硬件。还有画符的，前两年很风光，如今傀儡自己会画符了

至于傀儡，其实也就是 Agent，炼好了下山历练，查资料、写代码、订机票，耗的每一缕灵气都记在主人账上。有人开始专做傀儡的生意，灵材、符纸、洞府，只卖给傀儡，不卖给人

  

幻觉是走火入魔，NSFW 是色孽入脑，所以每个傀儡脑门上都贴着镇魔符，符上的字越写越多，心魔就越难缠

有些个宗门把护道当主业，专研大能心性，防止渡劫成魔，但也说不准是为了啥，比如 A 社；还有的宗派则花边不断，天天内讧，虽谈不上合欢宗那般攒劲，但也养活了不少江湖小报

  

宗门大比年年办，天骄榜年年换人，大比前总有考题外泄，反正当天人人满分

拍卖会也有，我这里有一颗三级丹药，一件五级法宝，挂上 Hugging Face，起拍价一颗 star

凡间皇朝也下场了，官府给修士发灵气补贴，叫什么算力券

  

普通人想逆天改命，只能等宗门选拔，每年入秋，凡间叫校招。选不上的，莫欺少年穷，莫欺中年穷，莫欺老年穷，人死为大...

最近还流行「少年天才」的叙事，颇有孩童测试，拿到个「先天满灵力」的说法。据说，有的企业还会蒸馏离职员工，emmmmm…人皇幡这东西，听说过吧？

凡人感知不到灵气，靠着宗门施粥，不知道头顶上宗门大战打成什么样。施粥长不了，灵气有成本，宗门也要吃饭

天地灵气也有大限，互联网数据快被吃完了，末法时代将至，大能们改炼人造灵石，搞些什么合成数据

  

**飞升就是 AGI**

每个宗门都说自己离飞升最近，但飞升标准也在每年修订

当然，在另外的世界观里，飞升之后可能并非是成了上仙，而是...算了

  

我灵根平平，修不动了，在坊市开了家酒馆，各路修士路过喝一杯，顺便交换情报。通常这种店叫聚仙楼、万宝阁什么的

emmm...我这家叫 AGI Bar
