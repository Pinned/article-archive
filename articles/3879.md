# 刚刚，阿里All in Token：成立全新事业群，吴泳铭直接负责

![](https://mmbiz.qpic.cn/mmbiz_png/z7ZD1WagSLgsRsrdDyQwMQFeic5uAcFg7nUPJuxZKtOOpVA2qEx77mXgLIZRcOIDgFRTT28ymhVVnUyawribP28g/640)![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/8Eyv4o4AcuphEEnD9nP8EykJT8LVSTJPc3RUP1gztYD4crRbRicq1vqH20JERKx4xQQG3Sic5DLDvWzz87o1rHibBT519EvpOGbKg2oAxHnPAg/640?from=appmsg)

**作者 |**  王涵

**刚刚，阿里新架构公布！**

智东西3月16日报道，今天，阿里巴巴正式成立**Alibaba Token Hub（ATH）事业群**，**由阿里巴巴CEO吴泳铭直接负责**。

据阿里官方透露，Alibaba Token Hub是一个以“创造Token、输送Token、应用Token”为核心目标的新组织。

ATH事业部旗下包括**通义实验室、MaaS业务线、千问事业部、悟空事业部及AI创新事业部等五大事业部**，覆盖从基础模型研发、模型服务平台，到个人与企业端AI应用的完整布局。

五大事业部的职责分别为：

**1、通义实验室：**创造领先的多模态模型，不断追求基础模型能力上限，为集团和业界提供最领先模型。

**2、MaaS业务线：**构建高效开放的模型服务平台和技术体系，支撑全行业AI生态。

**3、千问事业部：**打造最好的个人AI助手。

**4、悟空事业部：**打造B端AI原生工作平台，将模型能力深度融入企业工作流。

**5、AI创新事业部：**探索各类AI创新应用，快速验证新模式、新市场。

随着“龙虾”OpenClaw在今年的爆火出圈，Agent应用全面爆发。作为敏锐的第一批“吃上龙虾肉”的大厂，阿里在1月底便迅速反应，推出了一键部署OpenClaw的云服务，并构建了丰富的产品矩阵：

在**基础设施**层，1月28日，阿里云上线**OpenClaw全套云服务**，全面提供Agent所需的算力、模型和消息应用等，极大降低了OpenClaw的部署门槛。

在**个人应用**层，1月30日，阿里发布桌面**Agent应用QoderWork**，无需任何复杂部署工作，输入一句话，QoderWork就能按需调用授权的本地应用，完成文件整理、数据处理、文档生成等任务。

3月13日，阿里云推出手机**“一键养虾”产品JVS Claw**，消费者无需掌握任何代码知识，在手机上简单操作三步，就能拥有即开即用、智能进化、安全可控的AI智能体“龙虾”。

然而，随着Agent应用的指数级增长，一个严峻的问题浮出水面：**Token资源的供需矛盾**。高频的Agent调用和复杂的编程需求，使得**Token成为了AI时代的“水、电、煤”**。

今年1月，阿里云推出的**Coding Plan订阅服务**便是这一趋势的缩影。该服务整合了千问、GLM、Kimi等顶尖编程模型，因性价比极高，上线即引爆市场，迅速成为平台增长最快的产品之一。甚至因市场需求远超预期，阿里云在本月初不得不暂停了首购优惠活动。

当“Token焦虑”成为开发者和企业的普遍痛点，**如何高效地分配、调度和优化这一核心资源**，便成了阿里必须面对的考题。

而ATH事业群的本质，正是对分散在各部门的Token资源进行**“整合与再分配”**，以实现全局最优解。

“将资源合并再分配”这一操作似曾相识。2023年4月，谷歌将内部竞争多年的Google Brain和DeepMind两大AI实验室合并，统一为DeepMind。

合并后的谷歌 DeepMind在Demis Hassabis的领导下，仅用半年时间就推出了Gemini系列模型。2024年10月，谷歌将Gemini APP团队也并入DeepMind团队，实现**模型和应用层**的合并。

但阿里的ATH之路，与谷歌有着本质的不同。谷歌的合并侧重于技术与产品的纵向打通，旨在消除内部内耗，集中力量办大事。而阿里成立ATH，瞄准的是**底层资源“Token”的横向统筹**。

在阿里看来，AGI（通用人工智能）时代的竞争，不仅是模型能力的竞争，更是资源调度效率的竞争。通过将通义的模型能力、MaaS的分发能力、千问与悟空的应用场景全部纳入ATH架构，**阿里试图构建一个以Token为流通货币的内部大市场**。

**内部信原文：**

各位阿里人：

当下正处于AGI 爆发前夜。大量数字化工作将由数以百亿计的AI Agent来支撑，而这些AI Agent将由模型产生的Token支撑运行，成为人类与数字世界交互的主要载体。

面向这一历史性机遇，集团决定正式成立Alibaba Token Hub（ATH）事业群，建立以“创造 Token、输送 Token、应用Token”为核心目标的新组织。

Alibaba Token Hub事业群由我直接负责，强化各AI业务战略协同，以AI重塑工作方式，保持敏捷组织。具体业务包括：

通义实验室：创造领先的多模态模型，不断追求基础模型能力上限，为集团和业界提供最领先模型。

MaaS业务线：构建高效开放的模型服务平台和技术体系，支撑全行业AI生态。

悟空事业部：打造B端AI原生工作平台，将模型能力深度融入企业工作流。

AI创新事业部：探索各类AI创新应用，快速验证新模式、新市场。

以上调整即日生效。一起全力以赴！

吴泳铭

2026年3月16日

  

  

**结语：Tokens需求极速增长**

**阿里选择“模型-平台-应用”协同**

上一轮由谷歌推动的DeepMind与Brain合并，其后续效应表明：在AI领域，模型能力与应用生态的整合节奏，往往决定技术优势能否转化为市场优势。

如今，AI Agent正推动Tokens需求进入增长通道。阿里通过ATH事业群的整合，将原本分散的模型、平台及应用板块纳入统一架构，意在形成从技术到市场的闭环。

这种“模型-平台-应用”协同的模式，或是阿里应对AGI趋势所选择的核心路径。组织结构改革后，阿里会迸发出怎样的活力，我们拭目以待。
